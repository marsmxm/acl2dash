Modifying constructor for fty-type-option structures.

Syntax
(change-fty-type-option x 
                        [:some-type <some-type>])

This is an often useful alternative to make-fty-type-option.

We construct a new fty-type-option structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-fty-type-option

(defmacro
