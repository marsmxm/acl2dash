(vl-namefactory-plain-name name factory) returns (mv fresh-name factory-prime). When possible, fresh-name is just name. When this is not possible, a note is printed and fresh-name looks like name_n instead.

Signature
(vl-namefactory-plain-name name factory) 
  → 
(mv fresh-name new-factory)
Arguments
name — Guard (stringp name).
factory — Guard (vl-namefactory-p factory).
Returns
fresh-name — Type (stringp fresh-name).
new-factory — Type (vl-namefactory-p new-factory).
