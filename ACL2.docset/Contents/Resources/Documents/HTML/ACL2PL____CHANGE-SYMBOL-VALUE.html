Modifying constructor for symbol-value structures.

Syntax
(change-symbol-value x 
                     [:package <package>] 
                     [:name <name>])

This is an often useful alternative to make-symbol-value.

We construct a new symbol-value structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-symbol-value

(defmacro
     change-symbol-value (x &rest args)
     (std::change-aggregate 'symbol-value
                            x args
                            '((:package . symbol-value->package)
                              (:name . symbol-value->name))
                            'change-symbol-value
                            'nil))
