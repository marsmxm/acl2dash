Modifying constructor for scope-list-result-ok structures.

Syntax
(change-scope-list-result-ok x 
                             [:get <get>])

This is an often useful alternative to make-scope-list-result-ok.

We construct a new scope-list-result-ok structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
