Modifying constructor for integral-type-short structures.

Syntax
(change-integral-type-short x 
                            )

This is an often useful alternative to make-integral-type-short.

We construct a new integral-type-short structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
