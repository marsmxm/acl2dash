Check if a term is a (translated) call of and.

Signature
(check-and-call term) → (mv yes/no left right)
Arguments
term — Guard (pseudo-termp term).
Returns
yes/no — Type (booleanp yes/no).
left — Type (pseudo-termp left), given the guard.
right — Type (pseudo-termp right), given the guard.
If it is, return its conjuncts. If it is not, all results are nil.
