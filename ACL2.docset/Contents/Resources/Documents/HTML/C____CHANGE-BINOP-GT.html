Modifying constructor for binop-gt structures.

Syntax
(change-binop-gt x 
                 )

This is an often useful alternative to make-binop-gt.

We construct a new binop-gt structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
