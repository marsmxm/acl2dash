Modifying constructor for uchar-array structures.

Syntax
(change-uchar-array x 
                    [:elemtype <elemtype>] 
                    [:elements <elements>])

This is an often useful alternative to make-uchar-array.

We construct a new uchar-array structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-uchar-array

(defmacro
     change-uchar-array (x &rest args)
     (std::change-aggregate 'uchar-array
                            x args
                            '((:elemtype . uchar-array->elemtype)
                              (:elements . uchar-array->elements))
                            'change-uchar-array
                            'nil))
