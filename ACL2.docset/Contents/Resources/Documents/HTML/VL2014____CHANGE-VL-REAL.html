Modifying constructor for vl-real structures.

Syntax
(change-vl-real x 
                [:value <value>])

This is an often useful alternative to make-vl-real.

We construct a new vl-real structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-real

(defmacro change-vl-real (x &rest args)
          (std::change-aggregate 'vl-real
                                 x args '((:value . vl-real->value))
                                 'change-vl-real
                                 'remake-vl-real))
