Modifying constructor for tyspecseq-uint structures.

Syntax
(change-tyspecseq-uint x 
                       )

This is an often useful alternative to make-tyspecseq-uint.

We construct a new tyspecseq-uint structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
