Modifying constructor for jbinop-condor structures.

Syntax
(change-jbinop-condor x 
                      )

This is an often useful alternative to make-jbinop-condor.

We construct a new jbinop-condor structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
