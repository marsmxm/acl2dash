Modifying constructor for maybe-rlp-error-some structures.

Syntax
(change-maybe-rlp-error-some x 
                             [:val <val>])

This is an often useful alternative to make-maybe-rlp-error-some.

We construct a new maybe-rlp-error-some structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
