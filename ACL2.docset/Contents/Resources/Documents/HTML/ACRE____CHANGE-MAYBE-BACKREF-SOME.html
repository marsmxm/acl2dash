Modifying constructor for maybe-backref-some structures.

Syntax
(change-maybe-backref-some x 
                           [:val <val>])

This is an often useful alternative to make-maybe-backref-some.

We construct a new maybe-backref-some structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-maybe-backref-some

(defmacro
     change-maybe-backref-some (x &rest args)
     (std::change-aggregate 'maybe-backref-some
                            x
                            args '((:val . maybe-backref-some->val))
                            'change-maybe-backref-some
                            'nil))
