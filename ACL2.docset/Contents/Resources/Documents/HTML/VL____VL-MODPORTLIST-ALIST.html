Extend an alist by binding the names of vl-modports to their definitions.

Signature
(vl-modportlist-alist x acc)  alist
Arguments
x Guard (vl-modportlist-p x).
Returns
alist Type (equal (vl-modport-alist-p alist) (vl-modport-alist-p acc)).

This can be used as an alternative to vl-find-modport when you need to perform a lot of lookups.

Definitions and Theorems

Function: vl-modportlist-alist

(defun vl-modportlist-alist (x acc)
       (declare (xargs :guard (vl-modportlist-p x)))
       (let ((__function__ 'vl-modportlist-alist))
            (declare (ignorable __function__))
            (if (atom x)
                acc
                (cons (cons (vl-modport->name (car x))
                            (vl-modport-fix (car x)))
                      (vl-modportlist-alist (cdr x) acc)))))

     (syntaxp (not (equal acc ''nil)))

Theorem: lookup-in-vl-modportlist-alist-fast

(defthm
    lookup-in-vl-modportlist-alist-fast
    (implies
         (stringp name)
         (equal (hons-assoc-equal name (vl-modportlist-alist x nil))
                (let ((val (vl-find-modport name x)))
                     (and val (cons name val))))))
