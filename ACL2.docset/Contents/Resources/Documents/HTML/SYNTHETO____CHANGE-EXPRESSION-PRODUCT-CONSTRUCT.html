Modifying constructor for expression-product-construct structures.

Syntax
(change-expression-product-construct x 
                                     [:type <type>] 
                                     [:fields <fields>])

This is an often useful alternative to make-expression-product-construct.

We construct a new expression-product-construct structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-expression-product-construct

(defmacro change-expression-product-construct
          (x &rest args)
          (std::change-aggregate
               'expression-product-construct
               x args
               '((:type . expression-product-construct->type)
                 (:fields . expression-product-construct->fields))
               'change-expression-product-construct
               'nil))
