Retrieve the value of the :classicalp option of a function introduced via defun-sk.

Signature
(defun-sk-classicalp fn wrld)  strengthen
Arguments
fn Guard (defun-sk-namep fn wrld).
wrld Guard (plist-worldp wrld).
Returns
strengthen A booleanp.

This is only relevant for ACL2(r). This function returns t when not running ACL2(r).

Function: defun-sk-classicalp

(defun defun-sk-classicalp (fn wrld)
       (declare (xargs :guard (and (plist-worldp wrld)
                                   (defun-sk-namep fn wrld))))
       (let ((__function__ 'defun-sk-classicalp))
            (declare (ignorable __function__))
            (b* ((options (defun-sk-options fn wrld))
                 (pair (assoc-eq :classicalp options)))
                (if (null pair) t (cdr pair)))))
