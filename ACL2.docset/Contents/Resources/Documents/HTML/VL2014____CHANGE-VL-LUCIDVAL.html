Modifying constructor for vl-lucidval structures.

Syntax
(change-vl-lucidval x 
                    [:used <used>] 
                    [:set <set>] 
                    [:errors <errors>])

This is an often useful alternative to make-vl-lucidval.

We construct a new vl-lucidval structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-lucidval

(defmacro change-vl-lucidval (x &rest args)
          (std::change-aggregate 'vl-lucidval
