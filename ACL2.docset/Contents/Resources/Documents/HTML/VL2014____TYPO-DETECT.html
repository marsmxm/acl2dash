Build an alist associating any "bad" wires with any "good" wires that they may be typos for.

Signature
(typo-detect bad good)  alist
Arguments
bad Names of wires we think are somehow bad. It could be, e.g., all of the implicit wires in a module, or all of the implicit wires that are only partly used, etc.
Guard (string-listp bad).
good Names of all the wires we think are somehow good; e.g., declared, used, etc.
Guard (string-listp good).
Returns
alist Type (alistp alist).
