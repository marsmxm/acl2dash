Modifying constructor for vl-slicesize-type structures.

Syntax
(change-vl-slicesize-type x 
                          [:type <type>])

This is an often useful alternative to make-vl-slicesize-type.

We construct a new vl-slicesize-type structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-slicesize-type

(defmacro
     change-vl-slicesize-type (x &rest args)
     (std::change-aggregate 'vl-slicesize-type
                            x args
                            '((:type . vl-slicesize-type->type))
                            'change-vl-slicesize-type
