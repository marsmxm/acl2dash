(partsumlist-fix x) is a usual fty list fixing function.

Signature
(partsumlist-fix x) → fty::newx
Arguments
x — Guard (partsumlist-p x).
Returns
fty::newx — Type (partsumlist-p fty::newx).

In the logic, we apply partsum-elt-fix to each member of the x. In the execution, none of that is actually necessary and this is just an inlined identity function.
