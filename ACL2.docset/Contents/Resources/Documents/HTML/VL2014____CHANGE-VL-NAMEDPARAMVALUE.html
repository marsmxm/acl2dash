Modifying constructor for vl-namedparamvalue structures.

Syntax
(change-vl-namedparamvalue x 
                           [:name <name>] 
                           [:value <value>])

This is an often useful alternative to make-vl-namedparamvalue.

We construct a new vl-namedparamvalue structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-namedparamvalue

(defmacro
     change-vl-namedparamvalue (x &rest args)
     (std::change-aggregate 'vl-namedparamvalue
                            x args
                            '((:name . vl-namedparamvalue->name)
                              (:value . vl-namedparamvalue->value))
                            'change-vl-namedparamvalue
                            'remake-vl-namedparamvalue))
