All but the least significant bit of a number.

Signature
(logcdr i)  int
Arguments
i Guard (integerp i).
Returns
int Type (integerp int).

(logcdr i) is the cdr of an integer conceptualized as a bit-vector, where the least significant bit is at the head of the list.

In languages like C, this might be written as i >> 1.
