(object/data-list-fix x) is a usual ACL2::fty list fixing function.

Signature
(object/data-list-fix x) → fty::newx
Arguments
x — Guard (object/data-listp x).
Returns
fty::newx — Type (object/data-listp fty::newx).

In the logic, we apply object/data-fix to each member of the x. In the execution, none of that is actually necessary and this is just an inlined identity function.
