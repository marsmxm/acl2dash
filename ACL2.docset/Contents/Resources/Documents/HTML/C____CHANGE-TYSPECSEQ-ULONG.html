Modifying constructor for tyspecseq-ulong structures.

Syntax
(change-tyspecseq-ulong x 
                        [:int <int>])

This is an often useful alternative to make-tyspecseq-ulong.

We construct a new tyspecseq-ulong structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-tyspecseq-ulong

(defmacro
     change-tyspecseq-ulong (x &rest args)
     (std::change-aggregate 'tyspecseq-ulong

                            'nil))
