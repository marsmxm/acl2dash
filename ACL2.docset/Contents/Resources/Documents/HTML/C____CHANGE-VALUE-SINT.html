Modifying constructor for value-sint structures.

Syntax
(change-value-sint x 
                   [:get <get>])

This is an often useful alternative to make-value-sint.

We construct a new value-sint structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-value-sint

(defmacro change-value-sint (x &rest args)
          (std::change-aggregate 'value-sint
                                 x args '((:get . value-sint->get))
                                 'change-value-sint
                                 'nil))
