Modifying constructor for type-option-none structures.

Syntax
(change-type-option-none x 
                         )

This is an often useful alternative to make-type-option-none.

We construct a new type-option-none structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
