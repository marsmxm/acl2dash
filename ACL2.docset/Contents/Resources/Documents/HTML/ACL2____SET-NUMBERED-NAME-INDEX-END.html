Set the ending marker of the numeric index of numbered names.

This macro generates an event to override the default, or the previously set value.

Macro: set-numbered-name-index-end

(defmacro set-numbered-name-index-end (end)
          (cons 'table
                (cons 'numbered-name-index-end
                      (cons ''end (cons end 'nil)))))
