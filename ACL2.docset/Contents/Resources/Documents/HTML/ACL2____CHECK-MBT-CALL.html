Check if a term is a call of mbt.

Signature
(check-mbt-call term) → (mv yes/no arg)
Arguments
term — Guard (pseudo-termp term).
Returns
yes/no — Type (booleanp yes/no).
arg — Type (pseudo-termp arg), given the guard.

If it is, return its argument. If it is not, all results are nil.

In translated terms, (mbt x) is (return-last 'mbe1-raw 't x).

Definitions and Theorems

Function: check-mbt-call

(defun check-mbt-call (term)
       (declare (xargs :guard (pseudo-termp term)))
       (let ((__function__ 'check-mbt-call))
            (declare (ignorable __function__))
            (case-match term
                        (('return-last ''mbe1-raw ''t x)
                         (mv t x))
                        (& (mv nil nil)))))
