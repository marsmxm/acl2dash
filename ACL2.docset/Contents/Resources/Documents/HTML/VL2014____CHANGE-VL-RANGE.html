Modifying constructor for vl-range structures.

Syntax
(change-vl-range x 
                 [:msb <msb>] 
                 [:lsb <lsb>])

This is an often useful alternative to make-vl-range.

We construct a new vl-range structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-range

(defmacro change-vl-range (x &rest args)
          (std::change-aggregate 'vl-range
                                 x args
                                 '((:msb . vl-range->msb)
                                   (:lsb . vl-range->lsb))
                                 'change-vl-range
                                 'remake-vl-range))
