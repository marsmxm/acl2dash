Basic constructor macro for array-schar structures.

Syntax
(make-array-schar [:get <get>])

This is the usual way to construct array-schar structures. It simply conses together a structure with the specified fields.

This macro generates a new array-schar structure from scratch. See also change-array-schar, which can "change" an existing structure, instead.
