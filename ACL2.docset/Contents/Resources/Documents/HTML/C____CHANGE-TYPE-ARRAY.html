Modifying constructor for type-array structures.

Syntax
(change-type-array x 
                   [:of <of>] 
                   [:size <size>])

This is an often useful alternative to make-type-array.

We construct a new type-array structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-type-array

(defmacro change-type-array (x &rest args)
          (std::change-aggregate 'type-array
                                 x args
                                 '((:of . type-array->of)
                                   (:size . type-array->size))
                                 'change-type-array
                                 'nil))
