We formalize the Java boolean and integer operations [JLS:4.2.5] [JLS:4.2.2]. We also provide abstract notions of the Java floating-point operations, as a placeholder for a more precise formalization of them. Primitive conversions [JLS:5.1.2-4] are formalized here.
