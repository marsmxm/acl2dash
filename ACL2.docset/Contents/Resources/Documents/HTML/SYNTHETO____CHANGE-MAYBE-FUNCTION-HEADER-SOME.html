Modifying constructor for maybe-function-header-some structures.

Syntax
(change-maybe-function-header-some x 
                                   [:val <val>])

This is an often useful alternative to make-maybe-function-header-some.

We construct a new maybe-function-header-some structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
