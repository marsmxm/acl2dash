Modifying constructor for rw-pair structures.

Syntax
(change-rw-pair x 
                [:hyps <hyps>] 
                [:concl <concl>])

This is an often useful alternative to make-rw-pair.

We construct a new rw-pair structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-rw-pair

(defmacro change-rw-pair (x &rest args)
          (std::change-aggregate 'rw-pair
                                 x args
                                 '((:hyps . rw-pair->hyps)
                                   (:concl . rw-pair->concl))
                                 'change-rw-pair
                                 'nil))
