Modifying constructor for partsum-comp-idx structures.

Syntax
(change-partsum-comp-idx x 
                         [:val <val>])

This is an often useful alternative to make-partsum-comp-idx.

We construct a new partsum-comp-idx structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.
