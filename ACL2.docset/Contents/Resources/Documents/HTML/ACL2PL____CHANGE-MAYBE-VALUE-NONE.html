Modifying constructor for maybe-value-none structures.

Syntax
(change-maybe-value-none x 
                         )

This is an often useful alternative to make-maybe-value-none.

We construct a new maybe-value-none structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-maybe-value-none

(defmacro change-maybe-value-none (x &rest args)
          (std::change-aggregate 'maybe-value-none

                                 'nil))
