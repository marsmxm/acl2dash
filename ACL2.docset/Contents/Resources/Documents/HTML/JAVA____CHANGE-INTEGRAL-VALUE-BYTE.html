Modifying constructor for integral-value-byte structures.

Syntax
(change-integral-value-byte x 
                            [:get <get>])

This is an often useful alternative to make-integral-value-byte.

We construct a new integral-value-byte structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-integral-value-byte

(defmacro
     change-integral-value-byte
     (x &rest args)
     (std::change-aggregate 'integral-value-byte
                            x args
                            '((:get . integral-value-byte->get))
                            'change-integral-value-byte
                            'nil))
