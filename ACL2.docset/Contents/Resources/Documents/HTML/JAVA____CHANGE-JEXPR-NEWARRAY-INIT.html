Modifying constructor for jexpr-newarray-init structures.

Syntax
(change-jexpr-newarray-init x 
                            [:type <type>] 
                            [:init <init>])

This is an often useful alternative to make-jexpr-newarray-init.

We construct a new jexpr-newarray-init structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-jexpr-newarray-init

(defmacro
     change-jexpr-newarray-init
     (x &rest args)
     (std::change-aggregate 'jexpr-newarray-init
                            x args
                            '((:type . jexpr-newarray-init->type)
                              (:init . jexpr-newarray-init->init))
                            'change-jexpr-newarray-init
                            'nil))
