A rule is well-formed iff its name and definiens are well-formed.

Signature
(rule-wfp rule) → yes/no
Arguments
rule — Guard (rulep rule).
Returns
yes/no — Type (booleanp yes/no).

This condition is structural.

Function: rule-wfp

(defun rule-wfp (rule)
       (declare (xargs :guard (rulep rule)))
       (and (rulename-wfp (rule->name rule))
            (alternation-wfp (rule->definiens rule))))

(defthm booleanp-of-rule-wfp
        (b* ((yes/no (rule-wfp rule)))
