Modifying constructor for type-pointer structures.

Syntax
(change-type-pointer x 
                     [:referenced <referenced>])

This is an often useful alternative to make-type-pointer.

We construct a new type-pointer structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-type-pointer

(defmacro
  change-type-pointer (x &rest args)
  (std::change-aggregate 'type-pointer
                         x args
                         '((:referenced . type-pointer->referenced))
                         'change-type-pointer
                         'nil))
