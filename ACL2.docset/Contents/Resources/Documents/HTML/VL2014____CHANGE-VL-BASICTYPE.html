Modifying constructor for vl-basictype structures.

Syntax
(change-vl-basictype x 
                     [:kind <kind>])

This is an often useful alternative to make-vl-basictype.

We construct a new vl-basictype structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-basictype

(defmacro
     change-vl-basictype (x &rest args)
     (std::change-aggregate 'vl-basictype
                            x args '((:kind . vl-basictype->kind))
                            'change-vl-basictype
                            'nil))
