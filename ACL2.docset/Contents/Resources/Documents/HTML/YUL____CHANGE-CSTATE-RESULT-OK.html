Modifying constructor for cstate-result-ok structures.

Syntax
(change-cstate-result-ok x 
                         [:get <get>])

This is an often useful alternative to make-cstate-result-ok.

We construct a new cstate-result-ok structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
