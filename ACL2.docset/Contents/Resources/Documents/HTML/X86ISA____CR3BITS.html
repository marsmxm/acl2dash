An 64-bit unsigned bitstruct type.

This is a bitstruct type introduced by defbitstruct, represented as a unsigned 64-bit integer.

Fields
res1 — 3bits
pwt — bit
pcd — bit
res2 — 7bits
pdb — 40bits
res3 — 12bits

Source: Intel Manual, Feb-14, Vol. 3A, Section 2.5

 cr3bits$inline
 (res1 pwt pcd res2 pdb res3)

               :exec res3)))

           (logapp 1 pwt
