Modifying constructor for function-specifier-quantified structures.

Syntax
(change-function-specifier-quantified x 
                                      [:quantifier <quantifier>] 
                                      [:variables <variables>] 
                                      [:matrix <matrix>])

This is an often useful alternative to make-function-specifier-quantified.

We construct a new function-specifier-quantified structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-function-specifier-quantified

(defmacro
     change-function-specifier-quantified
     (x &rest args)
     (std::change-aggregate
          'function-specifier-quantified
          x args
          '((:quantifier .
                         function-specifier-quantified->quantifier)
            (:variables .
                        function-specifier-quantified->variables)
            (:matrix .
                     function-specifier-quantified->matrix))
          'change-function-specifier-quantified
          'nil))
