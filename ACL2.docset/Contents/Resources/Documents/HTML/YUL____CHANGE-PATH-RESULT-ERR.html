Modifying constructor for path-result-err structures.

Syntax
(change-path-result-err x 
                        [:get <get>])

This is an often useful alternative to make-path-result-err.

We construct a new path-result-err structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

(defmacro
