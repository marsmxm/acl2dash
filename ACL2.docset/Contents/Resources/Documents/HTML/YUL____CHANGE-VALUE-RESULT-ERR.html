Modifying constructor for value-result-err structures.

Syntax
(change-value-result-err x 
                         [:get <get>])

This is an often useful alternative to make-value-result-err.

We construct a new value-result-err structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-value-result-err

(defmacro
     change-value-result-err (x &rest args)
     (std::change-aggregate 'value-result-err

                            'change-value-result-err
                            'nil))
