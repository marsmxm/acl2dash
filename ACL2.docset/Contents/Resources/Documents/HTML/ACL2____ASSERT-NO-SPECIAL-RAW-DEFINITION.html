Assert that given symbols do not have a special raw function definition.

Example Form:
(assert-no-special-raw-definition my-fn your-fn)

General Form:
(assert-no-special-raw-definition fn1 fn2 ... fnk)

where each fni is a literal symbol. An error is raised if any fni is is flagged as having a special raw definition. See ensure-special-raw-definition-flag. This is a pseudo-event, meaning it can be used in an event context but does not (ever) change the world.
