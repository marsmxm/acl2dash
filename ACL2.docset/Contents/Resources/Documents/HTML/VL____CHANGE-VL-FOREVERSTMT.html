Modifying constructor for vl-foreverstmt structures.

Syntax
(change-vl-foreverstmt x 
                       [:body <body>] 
                       [:atts <atts>])

This is an often useful alternative to make-vl-foreverstmt.

We construct a new vl-foreverstmt structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-foreverstmt

(defmacro change-vl-foreverstmt (x &rest args)
          (std::change-aggregate 'vl-foreverstmt
                                 x args
                                 '((:body . vl-foreverstmt->body)
                                   (:atts . vl-foreverstmt->atts))
                                 'change-vl-foreverstmt
                                 'remake-vl-foreverstmt))
