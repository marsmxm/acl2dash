Annotate an ACL2 variable with a non-empty list of types.

Signature
(atj-type-annotate-var var types) → annotated-var
Arguments
var — Guard (symbolp var).
types — Guard (atj-type-listp types).
Returns
annotated-var — Type (symbolp annotated-var).

As mentioned in atj-pre-translation-type-annotation, we systematically add type information to each ACL2 variable. We do so by adding [types] before the variable name, where types identifies a list of ATJ types.

The result of this function is never the symbol nil, because the name of that symbol always starts with [.
