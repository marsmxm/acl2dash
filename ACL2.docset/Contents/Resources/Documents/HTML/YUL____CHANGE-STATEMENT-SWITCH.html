Modifying constructor for statement-switch structures.

Syntax
(change-statement-switch x 
                         [:target <target>] 
                         [:cases <cases>] 
                         [:default <default>])

This is an often useful alternative to make-statement-switch.

We construct a new statement-switch structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-statement-switch

(defmacro
    change-statement-switch (x &rest args)
    (std::change-aggregate 'statement-switch
                           x args
                           '((:target . statement-switch->target)
                             (:cases . statement-switch->cases)
                             (:default . statement-switch->default))
                           'change-statement-switch
                           'nil))
