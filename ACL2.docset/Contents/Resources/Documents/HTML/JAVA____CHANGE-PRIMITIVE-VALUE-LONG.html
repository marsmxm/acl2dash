Modifying constructor for primitive-value-long structures.

Syntax
(change-primitive-value-long x 
                             [:get <get>])

This is an often useful alternative to make-primitive-value-long.

We construct a new primitive-value-long structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-primitive-value-long

(defmacro
     change-primitive-value-long
     (x &rest args)
     (std::change-aggregate 'primitive-value-long
                            x args
                            '((:get . primitive-value-long->get))
                            'change-primitive-value-long
                            'nil))
