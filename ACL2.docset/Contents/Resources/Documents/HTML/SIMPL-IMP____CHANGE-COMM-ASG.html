Modifying constructor for comm-asg structures.

Syntax
(change-comm-asg x 
                 [:var <var>] 
                 [:exp <exp>])

This is an often useful alternative to make-comm-asg.

We construct a new comm-asg structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-comm-asg

(defmacro change-comm-asg (x &rest args)
          (std::change-aggregate 'comm-asg
                                 x args
                                 '((:var . comm-asg->var)
                                   (:exp . comm-asg->exp))
                                 'change-comm-asg
                                 'nil))
