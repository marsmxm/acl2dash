Basic constructor macro for ushort structures.

Syntax
(make-ushort [:get <get>])

This is the usual way to construct ushort structures. It simply conses together a structure with the specified fields.

This macro generates a new ushort structure from scratch. See also change-ushort, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by fty::defprod.

Macro: make-ushort

(defmacro make-ushort (&rest args)
          (std::make-aggregate 'ushort
                               args '((:get))
                               'make-ushort
                               nil))
