Modifying constructor for vl-dimension-queue structures.

Syntax
(change-vl-dimension-queue x 
                           [:maxsize <maxsize>])

This is an often useful alternative to make-vl-dimension-queue.

We construct a new vl-dimension-queue structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-dimension-queue

(defmacro
  change-vl-dimension-queue (x &rest args)
  (std::change-aggregate 'vl-dimension-queue
                         x args
                         '((:maxsize . vl-dimension-queue->maxsize))
                         'change-vl-dimension-queue
                         'nil))
