Modifying constructor for array-sint structures.

Syntax
(change-array-sint x 
                   [:get <get>])

This is an often useful alternative to make-array-sint.

We construct a new array-sint structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-array-sint

(defmacro change-array-sint (x &rest args)
          (std::change-aggregate 'array-sint
                                 x args '((:get . array-sint->get))
                                 'change-array-sint
                                 'nil))
