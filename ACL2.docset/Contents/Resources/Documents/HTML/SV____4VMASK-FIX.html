Fixing function for 4vmasks.

Signature
(4vmask-fix x) → x-fix
Arguments
x — Guard (4vmask-p x).
Returns
x-fix — Type (4vmask-p x-fix).

This is unlike ifix because we return -1 (``all bits are relevant'') in the default case.
