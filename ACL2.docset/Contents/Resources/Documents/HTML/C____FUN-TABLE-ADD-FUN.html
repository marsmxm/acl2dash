Add a function with a function type to a function table.

Signature
(fun-table-add-fun fun type funtab) → new-funtab
Arguments
fun — Guard (identp fun).
type — Guard (fun-typep type).
funtab — Guard (fun-tablep funtab).
Returns
new-funtab — Type (fun-table-resultp new-funtab).

If the table already has a function with that name, it is an error. Otherwise, we add the function and return the function table.

Definitions and Theorems

Function: fun-table-add-fun

(defun fun-table-add-fun (fun type funtab)
       (declare (xargs :guard (and (identp fun)
                                   (fun-typep type)
                                   (fun-tablep funtab))))
       (let ((__function__ 'fun-table-add-fun))
            (declare (ignorable __function__))
            (b* ((fun (ident-fix fun))
                 (type (fun-type-fix type))
                 (funtab (fun-table-fix funtab))
                 ((when (in fun funtab))
                  (error (list :duplicate-fun fun))))
                (omap::update fun type funtab))))
