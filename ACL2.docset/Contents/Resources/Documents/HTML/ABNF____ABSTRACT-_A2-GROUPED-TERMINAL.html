A list of zero or more ( term1 / ... / termN ) parse trees, where term1, ..., termN are terminal numeric or character value notations that all denote single natural numbers, are abstracted to the list of its natural numbers.

Signature
(abstract-*-grouped-terminal trees) → nats
Arguments
trees — Guard (tree-listp trees).
Returns
nats — Type (nat-listp nats).

Each parse tree must have a root labeled by no rule name, with a single leaf subtree labeled by a list of one natural number. That natural number is returned in correspondence to the parse tree.
