Extend an alist by binding the names of vl-genelements to their definitions.

Signature
(vl-genelementlist-alist x acc) → alist
Arguments
x — Guard (vl-genelementlist-p x).
Returns
alist — Type (equal (vl-genelement-alist-p alist) (vl-genelement-alist-p acc)).

This can be used as an alternative to vl-find-genelement when you need to perform a lot of lookups.

Definitions and Theorems

Function: vl-genelementlist-alist

(defun
     vl-genelementlist-alist (x acc)
     (declare (xargs :guard (vl-genelementlist-p x)))
     (let ((__function__ 'vl-genelementlist-alist))
          (declare (ignorable __function__))
          (b* (((when (atom x)) acc)
               (name (hons-copy (vl-genelement->blockname (car x))))
               ((when name)
                (cons (cons name (vl-genelement-fix (car x)))
                      (vl-genelementlist-alist (cdr x) acc))))
              (vl-genelementlist-alist (cdr x) acc))))

Theorem: lookup-in-vl-genelementlist-alist-fast

      (equal (hons-assoc-equal name (vl-genelementlist-alist x nil))
