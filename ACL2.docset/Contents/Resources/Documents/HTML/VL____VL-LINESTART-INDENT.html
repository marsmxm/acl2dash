Compatible with seq. If we are at the first token on a line, return its column number. Otherwise return NIL.

Signature
(vl-linestart-indent &key (tokstream 'tokstream)) 
  → 
(mv errmsg startcol new-tokstream)
Returns
errmsg — Type (not errmsg).
startcol — Type (maybe-natp startcol).
new-tokstream — Type (equal new-tokstream tokstream).

Function: vl-linestart-indent-fn

(defun
   vl-linestart-indent-fn (tokstream)
   (declare (xargs :stobjs (tokstream)))
   (declare (xargs :guard t))
   (let ((__function__ 'vl-linestart-indent))
        (declare (ignorable __function__))
        (b* ((tokens (vl-tokstream->tokens))
             ((unless (and (consp tokens)
                           (vl-token->breakp (car tokens))))
              (mv nil nil tokstream))
             (echar1 (car (vl-token->etext (car tokens))))
             (col1 (vl-echarpack->col (vl-echar-raw->pack echar1))))
            (mv nil col1 tokstream))))
