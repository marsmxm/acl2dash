(bitscan-fwd src) returns the bit position of the least significant bit in src that is set, or 0 when src is zero (and hence has no such bit).

Signature
(bitscan-fwd src) → position
Arguments
src — Guard (natp src).
Returns
position — Type (natp position).

Examples:

(bitscan-fwd #b1001) = 0
(bitscan-fwd #b1010) = 1
(bitscan-fwd #b1100) = 2
(bitscan-fwd #b1000) = 3
