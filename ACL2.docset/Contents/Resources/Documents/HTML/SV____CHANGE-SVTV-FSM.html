Modifying constructor for svtv-fsm structures.

Syntax
(change-svtv-fsm x 
                 [:base-fsm <base-fsm>] 
                 [:namemap <namemap>])

This is an often useful alternative to make-svtv-fsm.

We construct a new svtv-fsm structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-svtv-fsm

(defmacro change-svtv-fsm (x &rest args)
          (std::change-aggregate 'svtv-fsm
                                 x args
                                 '((:base-fsm . svtv-fsm->base-fsm)
                                   (:namemap . svtv-fsm->namemap))
                                 'change-svtv-fsm
                                 'nil))
