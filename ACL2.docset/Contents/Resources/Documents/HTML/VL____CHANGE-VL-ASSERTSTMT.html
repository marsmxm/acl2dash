Modifying constructor for vl-assertstmt structures.

Syntax
(change-vl-assertstmt x 
                      [:assertion <assertion>] 
                      [:atts <atts>])

This is an often useful alternative to make-vl-assertstmt.

We construct a new vl-assertstmt structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-assertstmt

(defmacro
    change-vl-assertstmt (x &rest args)
    (std::change-aggregate 'vl-assertstmt
                           x args
                           '((:assertion . vl-assertstmt->assertion)
                             (:atts . vl-assertstmt->atts))
                           'change-vl-assertstmt
                           'remake-vl-assertstmt))
