Modifying constructor for regex-start structures.

Syntax
(change-regex-start x 
                    )

This is an often useful alternative to make-regex-start.

We construct a new regex-start structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-regex-start

(defmacro change-regex-start (x &rest args)
          (std::change-aggregate 'regex-start
                                 x args 'nil
                                 'change-regex-start
                                 'nil))
