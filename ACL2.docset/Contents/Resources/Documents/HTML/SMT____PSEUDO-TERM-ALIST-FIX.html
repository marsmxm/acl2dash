(pseudo-term-alist-fix x) is an ACL2::fty alist fixing function that follows the fix-keys strategy.

Signature
(pseudo-term-alist-fix x) → fty::newx
Arguments
x — Guard (pseudo-term-alistp x).
Returns
fty::newx — Type (pseudo-term-alistp fty::newx).

Note that in the execution this is just an inline identity function.
