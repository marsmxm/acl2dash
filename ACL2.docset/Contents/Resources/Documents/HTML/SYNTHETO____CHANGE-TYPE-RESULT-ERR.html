Modifying constructor for type-result-err structures.

Syntax
(change-type-result-err x 
                        [:info <info>])

This is an often useful alternative to make-type-result-err.

We construct a new type-result-err structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-type-result-err

(defmacro
     change-type-result-err (x &rest args)
     (std::change-aggregate 'type-result-err
                            x
                            args '((:info . type-result-err->info))
                            'change-type-result-err
                            'nil))
