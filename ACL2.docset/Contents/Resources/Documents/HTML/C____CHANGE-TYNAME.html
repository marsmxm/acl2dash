Modifying constructor for tyname structures.

Syntax
(change-tyname x 
               [:tyspec <tyspec>] 
               [:declor <declor>])

This is an often useful alternative to make-tyname.

We construct a new tyname structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-tyname

(defmacro change-tyname (x &rest args)
          (std::change-aggregate 'tyname
                                 x args
                                 '((:tyspec . tyname->tyspec)
                                   (:declor . tyname->declor))
                                 'change-tyname
                                 'nil))
