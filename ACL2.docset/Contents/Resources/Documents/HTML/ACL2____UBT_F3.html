Undo commands, with queries as appropriate

See undo for an introduction to undoing commands.

Example:
:ubt? :x-4

The keyword command :ubt? is the same as :ubt, but with appropriate queries made. :Oops will undo the last :ubt?. See ubt!, ubt, ubu!, ubu, ubu?, and u.
