Modifying constructor for svexllist structures.

Syntax
(change-svexllist x 
                  [:top-nodelist <top-nodelist>] 
                  [:node-array <node-array>])

This is an often useful alternative to make-svexllist.

We construct a new svexllist structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-svexllist

(defmacro
  change-svexllist (x &rest args)
  (std::change-aggregate 'svexllist
                         x args
                         '((:top-nodelist . svexllist->top-nodelist)
                           (:node-array . svexllist->node-array))
                         'change-svexllist
                         'nil))
