Implementation of strnat<.

(strnat<-aux x y xn yn xl yl) is basically the adaptation of charlistnat< for strings. Here, X and Y are the strings being compared, and XL and YL are their pre-computed lengths. XN and YN are the indices into the two strings that are our current positions.

BOZO why do we have XN and YN separately? It seems like we should only need one index.

Function: strnat<-aux

(defun
 strnat<-aux (x y xn yn xl yl)
 (declare (type string x)
          (type string y)
          (type integer xn)
          (type integer yn)
          (type integer xl)
          (type integer yl)
          (xargs :guard (and (stringp x)
                             (stringp y)
                             (natp xn)
                             (natp yn)
                             (equal xl (length x))
                             (equal yl (length y))
                             (<= xn xl)
                             (<= yn yl)))
          (ignorable xl yl))
 (mbe
  :logic (let* ((x (if (stringp x) x ""))
                (y (if (stringp y) y ""))
                (xn (nfix xn))
                (yn (nfix yn))
                (xl (length x))
                (yl (length y)))
               (cond ((zp (- yl yn)) nil)
                     ((zp (- xl xn)) t)
                     ((and (dec-digit-char-p (char x xn))
                           (dec-digit-char-p (char y yn)))
                      (b* (((mv v1 l1)
                            (parse-nat-from-string x 0 0 xn xl))
                           ((mv v2 l2)
                            (parse-nat-from-string y 0 0 yn yl)))
                          (cond ((or (< v1 v2)
                                     (and (int= v1 v2) (< l1 l2)))
                                 t)
                                ((or (< v2 v1)
                                     (and (int= v1 v2) (< l2 l1)))
                                 nil)
                                (t (strnat<-aux x y (+ xn l1)
                                                (+ yn l2)
                                                xl yl)))))
                     ((char< (char x xn) (char y yn)) t)
                     ((char< (char y yn) (char x xn)) nil)
                     (t (strnat<-aux x y (+ 1 xn)
                                     (+ 1 yn)
                                     xl yl))))
  :exec
  (cond
   ((int= yn yl) nil)

Theorem: strnat<-aux-correct

(defthm strnat<-aux-correct
        (implies (and (stringp x)
                      (stringp y)
                      (natp xn)
                      (natp yn)
                      (equal xl (length x))
                      (equal yl (length y))
                      (<= xn xl)
                      (<= yn yl))
                 (equal (strnat<-aux x y xn yn xl yl)
                        (charlistnat< (nthcdr xn (explode x))
                                      (nthcdr yn (explode y))))))
