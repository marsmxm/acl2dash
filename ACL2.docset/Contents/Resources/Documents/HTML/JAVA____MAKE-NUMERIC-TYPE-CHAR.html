Basic constructor macro for numeric-type-char structures.

This is the usual way to construct numeric-type-char structures. It simply conses together a structure with the specified fields.

This macro generates a new numeric-type-char structure from scratch. See also change-numeric-type-char, which can "change" an existing structure, instead.
