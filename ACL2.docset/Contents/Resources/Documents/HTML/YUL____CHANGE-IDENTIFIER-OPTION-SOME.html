Modifying constructor for identifier-option-some structures.

Syntax
(change-identifier-option-some x 
                               [:val <val>])

This is an often useful alternative to make-identifier-option-some.

We construct a new identifier-option-some structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-identifier-option-some

(defmacro
     change-identifier-option-some
     (x &rest args)
     (std::change-aggregate 'identifier-option-some

                            'nil))
