Modifying constructor for const-enum structures.

Syntax
(change-const-enum x 
                   )

This is an often useful alternative to make-const-enum.

We construct a new const-enum structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-const-enum

(defmacro change-const-enum (x &rest args)
          (std::change-aggregate 'const-enum
                                 x args 'nil
                                 'change-const-enum
                                 'nil))
