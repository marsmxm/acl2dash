Basic constructor macro for pointer structures.

Syntax
(make-pointer [:address? <address?>] 
              [:reftype <reftype>])

This is the usual way to construct pointer structures. It simply conses together a structure with the specified fields.

This macro generates a new pointer structure from scratch. See also change-pointer, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by fty::defprod.

Macro: make-pointer

(defmacro make-pointer (&rest args)
          (std::make-aggregate 'pointer
                               args '((:address?) (:reftype))
                               'make-pointer
                               nil))

(defun pointer (address? reftype)
       (declare (xargs :guard (and (address-optionp address?)
                                   (typep reftype))))
       (declare (xargs :guard t))
       (let ((__function__ 'pointer))
            (declare (ignorable __function__))
            (b* ((address? (mbe :logic (address-option-fix address?)
                                :exec address?))
                 (reftype (mbe :logic (type-fix reftype)
                               :exec reftype)))
                (cons :pointer (list (cons 'address? address?)
                                     (cons 'reftype reftype))))))
