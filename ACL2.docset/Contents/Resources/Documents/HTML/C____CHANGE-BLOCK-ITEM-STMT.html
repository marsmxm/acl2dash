Modifying constructor for block-item-stmt structures.

Syntax
(change-block-item-stmt x 
                        [:get <get>])

This is an often useful alternative to make-block-item-stmt.

We construct a new block-item-stmt structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-block-item-stmt

(defmacro
     change-block-item-stmt (x &rest args)
     (std::change-aggregate 'block-item-stmt
                            x args '((:get . block-item-stmt->get))
                            'change-block-item-stmt
                            'nil))
