Modifying constructor for literal-boolean structures.

Syntax
(change-literal-boolean x 
                        [:value <value>])

This is an often useful alternative to make-literal-boolean.

We construct a new literal-boolean structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-literal-boolean

(defmacro
     change-literal-boolean (x &rest args)
     (std::change-aggregate 'literal-boolean
                            x args
                            '((:value . literal-boolean->value))
                            'change-literal-boolean
                            'nil))
