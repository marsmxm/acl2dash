Modifying constructor for tmp-occ-assign structures.

Syntax
(change-tmp-occ-assign x 
                       [:inputs <inputs>] 
                       [:delayed-inputs <delayed-inputs>] 
                       [:outputs <outputs>] 
                       [:svex <svex>])

This is an often useful alternative to make-tmp-occ-assign.

We construct a new tmp-occ-assign structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-tmp-occ-assign

(defmacro change-tmp-occ-assign (x &rest args)
          (std::change-aggregate

                 (:svex . tmp-occ-assign->svex))

               'nil))
