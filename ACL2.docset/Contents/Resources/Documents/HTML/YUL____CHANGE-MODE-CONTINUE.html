Modifying constructor for mode-continue structures.

Syntax
(change-mode-continue x 
                      )

This is an often useful alternative to make-mode-continue.

We construct a new mode-continue structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-mode-continue

(defmacro change-mode-continue (x &rest args)
          (std::change-aggregate 'mode-continue
                                 x args 'nil

                                 'nil))
