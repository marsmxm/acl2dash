Modifying constructor for binop-lt structures.

Syntax
(change-binop-lt x 
                 )

This is an often useful alternative to make-binop-lt.

We construct a new binop-lt structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-binop-lt

(defmacro change-binop-lt (x &rest args)
          (std::change-aggregate 'binop-lt
