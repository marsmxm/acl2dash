Modifying constructor for vl-propunary structures.

Syntax
(change-vl-propunary x 
                     [:op <op>] 
                     [:arg <arg>])

This is an often useful alternative to make-vl-propunary.

We construct a new vl-propunary structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-propunary

(defmacro change-vl-propunary (x &rest args)
          (std::change-aggregate 'vl-propunary
                                 x args
                                 '((:op . vl-propunary->op)
                                   (:arg . vl-propunary->arg))
                                 'change-vl-propunary
                                 'nil))
