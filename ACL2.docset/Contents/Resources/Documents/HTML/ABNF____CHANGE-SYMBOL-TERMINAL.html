Modifying constructor for symbol-terminal structures.

Syntax
(change-symbol-terminal x 
                        [:get <get>])

This is an often useful alternative to make-symbol-terminal.

We construct a new symbol-terminal structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-symbol-terminal

(defmacro
     change-symbol-terminal (x &rest args)
     (std::change-aggregate 'symbol-terminal
                            x args '((:get . symbol-terminal->get))
                            'change-symbol-terminal
                            'nil))
