Theorems about integers-from-to as a finite set.

The function integers-from-to is defined, and verified, to return a list of integers in increasing order. Because of this order, this list is also a set represented according to the Std/osets library.

Thus, here we introduce some theorems about integers-from-to viewed as a set:

  • The result of integers-from-to is (also) a set.
  • Membership in integers-from-to is equivalent to being an integer in the range.
  • Membership of all the elements in an integers-from-to list in an integers-from-to set is equivalent to the first range being contained in the second one.

Definitions and Theorems

Theorem: setp-of-integers-from-to

(defthm setp-of-integers-from-to
        (set::setp (integers-from-to min max)))

Theorem: in-of-integers-from-to

(defthm in-of-integers-from-to
        (iff (set::in x (integers-from-to min max))
             (and (integerp x)
                  (<= (ifix min) x)
                  (<= x (ifix max)))))

Theorem: integers-from-to-list-in-integers-from-to

(defthm integers-from-to-list-in-integers-from-to
        (implies (and (integerp min1)
                      (integerp max1)
                      (integerp min2)
                      (integerp max2)
                      (<= min1 max1))
                 (equal (set::list-in (integers-from-to min1 max1)
                                      (integers-from-to min2 max2))
                        (and (<= min2 min1) (<= max1 max2)))))
