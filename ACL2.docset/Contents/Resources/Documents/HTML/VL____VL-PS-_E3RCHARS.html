(vl-ps->rchars) --> rchars

This is misnamed for historic reasons. It used to return a character list in reverse order. It now returns a vl-printedlist-p, also in reverse order.

(defmacro vl-ps->rchars
          nil '(vl-ps->rchars-fn ps))
