Match as many characters of some type as possible.

Signature
(vl-read-while-ctype echars)  (mv prefix remainder)
Arguments
echars Guard (vl-echarlist-p echars).
Returns
prefix All of the leading characters from echars that satisfy vl-ctype-p.
remainder The remaining characters afterwards.

This is a generic function. See defchar for a convenient way to instantiate this function, and its corresponding theory, for your own character recognizers.

Function: vl-read-while-ctype

(defun
 vl-read-while-ctype (echars)
 (declare (xargs :guard (vl-echarlist-p echars)))
 (let
  ((__function__ 'vl-read-while-ctype))
  (declare (ignorable __function__))
  (mbe
    :logic (b* (((when (atom echars)) (mv nil echars))
                ((unless (vl-ctype-p (vl-echar->char (car echars))))
                 (mv nil echars))
                ((mv prefix remainder)
                 (vl-read-while-ctype (cdr echars))))
               (mv (cons (car echars) prefix)
                   remainder))
    :exec (b* (((mv prefix-rev remainder)
                (vl-read-while-ctype-impl echars nil)))
              (mv (reverse prefix-rev) remainder)))))
