Basic constructor macro for rlp-tree-branch structures.

Syntax
(make-rlp-tree-branch [:subtrees <subtrees>])

This is the usual way to construct rlp-tree-branch structures. It simply conses together a structure with the specified fields.

This macro generates a new rlp-tree-branch structure from scratch. See also change-rlp-tree-branch, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by fty::defprod.

(defun rlp-tree-branch (subtrees)

       (declare (xargs :guard t))
       (let ((__function__ 'rlp-tree-branch))
