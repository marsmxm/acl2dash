Modifying constructor for expr-grade-primary structures.

Syntax
(change-expr-grade-primary x 
                           )

This is an often useful alternative to make-expr-grade-primary.

We construct a new expr-grade-primary structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
