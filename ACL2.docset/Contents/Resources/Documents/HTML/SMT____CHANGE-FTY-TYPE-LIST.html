Modifying constructor for fty-type-list structures.

Syntax
(change-fty-type-list x 
                      [:elt-type <elt-type>])

This is an often useful alternative to make-fty-type-list.

We construct a new fty-type-list structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-fty-type-list

(defmacro
     change-fty-type-list (x &rest args)
     (std::change-aggregate 'fty-type-list
                            x args
                            '((:elt-type . fty-type-list->elt-type))
                            'change-fty-type-list
                            'nil))
