This is our preferred way to construct glcp-bit-ctrex-p structures. It simply conses together a structure with the specified fields.

This macro generates a new glcp-bit-ctrex-p structure from scratch. See also change-glcp-bit-ctrex, which can "change" an existing structure, instead.

The glcp-bit-ctrex-p structures we create here are just constructed with ordinary cons. If you want to create honsed structures, see make-honsed-glcp-bit-ctrex instead.
