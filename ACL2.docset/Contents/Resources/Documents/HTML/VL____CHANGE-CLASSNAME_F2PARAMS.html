Modifying constructor for classname/params structures.

Syntax
(change-classname/params x 
                         [:name <name>] 
                         [:params <params>])

This is an often useful alternative to make-classname/params.

We construct a new classname/params structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-classname/params

(defmacro
     change-classname/params (x &rest args)
     (std::change-aggregate 'classname/params
                            x args
                            '((:name . classname/params->name)
                              (:params . classname/params->params))
                            'change-classname/params
                            'nil))
