Generate code to parse an instance of an ABNF concatenation.

Signature
(def-parse-gen-code-for-concatenation 
     conc fn-name 
     group-fns option-fns repetition-fns) 
 
  → 
code
Arguments
conc — Guard (concatenationp conc).
fn-name — Guard (common-lisp::symbolp fn-name).
group-fns — Guard (alternation-symbol-alistp group-fns).
option-fns — Guard (alternation-symbol-alistp option-fns).
repetition-fns — Guard (repetition-symbol-alistp repetition-fns).
Returns
code — Type (true-listp code).

A concatenation is parsed by parsing each repetition in order. We also generate a final b* binder that puts all the lists of trees from the repetiitons into a list of lists of trees for the concatenation.

          (mv (append code1 code2)
              (cons var vars)))))

Function: def-parse-gen-code-for-concatenation

(defun
 def-parse-gen-code-for-concatenation
 (conc fn-name
       group-fns option-fns repetition-fns)
 (declare
     (xargs :guard (and (concatenationp conc)
                        (common-lisp::symbolp fn-name)
                        (alternation-symbol-alistp group-fns)
                        (alternation-symbol-alistp option-fns)
                        (repetition-symbol-alistp repetition-fns))))
 (let ((__function__ 'def-parse-gen-code-for-concatenation))
      (declare (ignorable __function__))
      (b* (((when (endp conc))
            (raise "Empty concatenation."))
           ((mv code vars)
            (def-parse-gen-code-for-concatenation-aux
                 conc 1 fn-name
                 group-fns option-fns repetition-fns)))
          (append code
                  (cons (cons 'treess
                              (cons (cons 'list vars) 'nil))
                        'nil)))))
