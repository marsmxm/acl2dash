Modifying constructor for vl-define-formal structures.

Syntax
(change-vl-define-formal x 
                         [:name <name>] 
                         [:default <default>])

This is an often useful alternative to make-vl-define-formal.

We construct a new vl-define-formal structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-define-formal

(defmacro
    change-vl-define-formal (x &rest args)
    (std::change-aggregate 'vl-define-formal

                             (:default . vl-define-formal->default))
                           'change-vl-define-formal
                           'nil))
