Modifying constructor for tag-info-option-some structures.

Syntax
(change-tag-info-option-some x 
                             [:val <val>])

This is an often useful alternative to make-tag-info-option-some.

We construct a new tag-info-option-some structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
