Modifying constructor for maybe-jexpr-none structures.

Syntax
(change-maybe-jexpr-none x 
                         )

This is an often useful alternative to make-maybe-jexpr-none.

We construct a new maybe-jexpr-none structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

This is an ordinary change- macro introduced by fty::defprod.
