Modifying constructor for constraint-instance structures.

Syntax
(change-constraint-instance x 
                            [:thmname <thmname>] 
                            [:subst <subst>])

This is an often useful alternative to make-constraint-instance.

We construct a new constraint-instance structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-constraint-instance

(defmacro
  change-constraint-instance
  (x &rest args)
  (std::change-aggregate 'constraint-instance
                         x args
                         '((:thmname . constraint-instance->thmname)
                           (:subst . constraint-instance->subst))
                         'change-constraint-instance
                         'remake-constraint-instance))
