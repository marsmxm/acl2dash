Fixtype of optional Syntheto type products.

This is an option type introduced by fty::defoption. Note that defoption is just a wrapper for fty::defflexsum, so there are :none and :some member types, a case macro, and so forth.

Member Types
:none → maybe-type-product-none
Represents that no maybe-type-product is available, i.e., Nothing or None.
:some → maybe-type-product-some
An available maybe-type-product, i.e., Just val or Some val.
