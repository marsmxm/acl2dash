Wrapper to indicate a C local variable assignment in an mv-let for 4 bound variables.

See atc-let-designations.

Macro: assign4

(defmacro
 assign4 (x)
 (cons
  'mv-let
  (cons
     (cons '*1
           (cons '*2 (cons '*3 (cons '*4 'nil))))
     (cons x
           (cons (cons 'mv
                       (cons (cons 'assign (cons '*1 'nil))
                             (cons '*2 (cons '*3 (cons '*4 'nil)))))
                 'nil)))))
