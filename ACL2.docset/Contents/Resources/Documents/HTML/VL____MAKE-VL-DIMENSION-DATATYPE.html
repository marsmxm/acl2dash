Basic constructor macro for vl-dimension-datatype structures.

Syntax
(make-vl-dimension-datatype [:type <type>])

This is the usual way to construct vl-dimension-datatype structures. It simply conses together a structure with the specified fields.

This macro generates a new vl-dimension-datatype structure from scratch. See also change-vl-dimension-datatype, which can "change" an existing structure, instead.
