Modifying constructor for numeric-value-byte structures.

Syntax
(change-numeric-value-byte x 
                           [:get <get>])

This is an often useful alternative to make-numeric-value-byte.

We construct a new numeric-value-byte structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-numeric-value-byte

(defmacro
     change-numeric-value-byte (x &rest args)
     (std::change-aggregate 'numeric-value-byte
                            x
                            args '((:get . numeric-value-byte->get))
                            'change-numeric-value-byte
                            'nil))
