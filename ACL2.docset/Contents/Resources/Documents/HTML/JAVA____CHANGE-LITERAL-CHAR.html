Modifying constructor for literal-char structures.

Syntax
(change-literal-char x 
                     [:get <get>])

This is an often useful alternative to make-literal-char.

We construct a new literal-char structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-literal-char

(defmacro
     change-literal-char (x &rest args)
     (std::change-aggregate 'literal-char
                            x args '((:get . literal-char->get))
                            'change-literal-char
                            'nil))
