Gather all of the operators used throughout an expression.

Signature
(vl-expr-ops x) → ops
Arguments
x — Guard (vl-expr-p x).
Returns
ops — Type (vl-oplist-p ops).

We simply gather all of the unary and binary operators used in the expression, with repetition.

        (b* ((?ops (vl-expr-ops x)))
