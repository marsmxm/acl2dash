Modifying constructor for vl-propcase structures.

Syntax
(change-vl-propcase x 
                    [:condition <condition>] 
                    [:cases <cases>])

This is an often useful alternative to make-vl-propcase.

We construct a new vl-propcase structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-propcase

(defmacro
     change-vl-propcase (x &rest args)
     (std::change-aggregate 'vl-propcase
                            x args
                            '((:condition . vl-propcase->condition)
                              (:cases . vl-propcase->cases))
                            'change-vl-propcase
                            'nil))
