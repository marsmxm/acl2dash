Modifying constructor for vl-clocking-direction-head structures.

Syntax
(change-vl-clocking-direction-head x 
                                   [:inp <inp>] 
                                   [:outp <outp>] 
                                   [:iskew <iskew>] 
                                   [:oskew <oskew>])

This is an often useful alternative to make-vl-clocking-direction-head.

We construct a new vl-clocking-direction-head structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.
