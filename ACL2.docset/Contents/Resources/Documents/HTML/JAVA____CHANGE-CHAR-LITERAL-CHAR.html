Modifying constructor for char-literal-char structures.

Syntax
(change-char-literal-char x 
                          [:get <get>])

This is an often useful alternative to make-char-literal-char.

We construct a new char-literal-char structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-char-literal-char

(defmacro
     change-char-literal-char (x &rest args)
     (std::change-aggregate 'char-literal-char
                            x
                            args '((:get . char-literal-char->get))
                            'change-char-literal-char
                            'nil))
