Basic constructor macro for value-schar structures.

Syntax
(make-value-schar [:get <get>])

This is the usual way to construct value-schar structures. It simply conses together a structure with the specified fields.

This macro generates a new value-schar structure from scratch. See also change-value-schar, which can "change" an existing structure, instead.
