Modifying constructor for string-result-err structures.

Syntax
(change-string-result-err x 
                          [:get <get>])

This is an often useful alternative to make-string-result-err.

We construct a new string-result-err structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-string-result-err

(defmacro
     change-string-result-err (x &rest args)
     (std::change-aggregate 'string-result-err
                            x
                            args '((:get . string-result-err->get))
                            'change-string-result-err
                            'nil))
