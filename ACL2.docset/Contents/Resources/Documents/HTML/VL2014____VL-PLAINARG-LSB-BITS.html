Build the list of vl-emodwire-ps for a vl-plainarg-p, in LSB-first order.

Signature
(vl-plainarg-lsb-bits x walist warnings) 
  → 
(mv successp warnings lsb-bits)
Arguments
x — Guard (vl-plainarg-p x).
walist — Guard (vl-wirealist-p walist).
warnings — Guard (vl-warninglist-p warnings).
Returns
warnings — Type (vl-warninglist-p warnings).
lsb-bits — Type (vl-emodwirelist-p lsb-bits).

See vl-msb-expr-bitlist. This function just makes sure a vl-plainarg-p isn't blank and then calls vl-msb-expr-bitlist to do the work. We return the bits in LSB-first order to match the convention throughout E.

Function: vl-plainarg-lsb-bits

(defun
 vl-plainarg-lsb-bits (x walist warnings)
 (declare (xargs :guard (and (vl-plainarg-p x)
                             (vl-wirealist-p walist)
                             (vl-warninglist-p warnings))))
 (let
  ((__function__ 'vl-plainarg-lsb-bits))
  (declare (ignorable __function__))
  (b*
   ((warnings (vl-warninglist-fix warnings))
    (expr (vl-plainarg->expr x))
    ((unless expr)
     (mv
      nil
      (fatal
           :type :vl-unsupported
           :msg "In vl-plainarg-lsb-bits, expected no blank ports.")
      nil))
    ((mv successp warnings bits)
     (vl-msb-expr-bitlist expr walist warnings)))
   (mv successp warnings (reverse bits)))))
