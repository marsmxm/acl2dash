Update the dimensions of any datatype, no matter its kind.

Signature
(vl-datatype-update-dims pdims udims x) → newx
Arguments
pdims — New packed dimensions to install.
    Guard (vl-dimensionlist-p pdims).
udims — New unpacked dimensions to install.
    Guard (vl-dimensionlist-p udims).
x — Datatype to update.
    Guard (vl-datatype-p x).
Returns
newx — Updated version of x with new dimensions installed.
    Type (and (vl-datatype-p newx) (eq (vl-datatype-kind newx) (vl-datatype-kind x))) .

Definitions and Theorems

Function: vl-datatype-update-dims

(defun vl-datatype-update-dims (pdims udims x)
       (declare (xargs :guard (and (vl-dimensionlist-p pdims)
                                   (vl-dimensionlist-p udims)
                                   (vl-datatype-p x))))
       (let ((__function__ 'vl-datatype-update-dims))
            (declare (ignorable __function__))
            (vl-datatype-case
                 x
                 :vl-coretype (change-vl-coretype x
                                                  :pdims pdims
                                                  :udims udims)
                 :vl-struct (change-vl-struct x
                                              :pdims pdims
                                              :udims udims)
                 :vl-union (change-vl-union x
                                            :pdims pdims
                                            :udims udims)
                 :vl-enum (change-vl-enum x
                                          :pdims pdims
                                          :udims udims)
                 :vl-usertype (change-vl-usertype x
                                                  :pdims pdims
                                                  :udims udims))))

(defthm return-type-of-vl-datatype-update-dims
