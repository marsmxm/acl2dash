A tagged union expression, e.g., tagged Some (12+34) or tagged None.

This is a product type, introduced by deftagsum in support of vl-expr.

Fields
tag — stringp
The tag name, e.g., Some or None above.
expr — vl-maybe-expr
The expression being tagged.
atts — vl-atts
Any (* foo = bar, baz *) style attributes.
