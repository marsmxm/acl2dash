Modifying constructor for scope-list-result-err structures.

Syntax
(change-scope-list-result-err x 
                              [:get <get>])

This is an often useful alternative to make-scope-list-result-err.

We construct a new scope-list-result-err structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
