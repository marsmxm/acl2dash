(bfr-varnamelist-fix x) is a usual ACL2::fty list fixing function.

Signature
(bfr-varnamelist-fix x) → fty::newx
Arguments
x — Guard (bfr-varnamelist-p x).
Returns
fty::newx — Type (bfr-varnamelist-p fty::newx).

In the logic, we apply bfr-varname-fix to each member of the x. In the execution, none of that is actually necessary and this is just an inlined identity function.
