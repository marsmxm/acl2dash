(final-cdr x) returns the atom in the "cdr-most branch" of x.

For example, (final-cdr '(a b c . d)) is d.

This function is related to list-fix. It is occasionally useful for strengthening rewrite rules by removing hypotheses.

(defun final-cdr (x)
       (declare (xargs :guard t))
       (if (atom x) x (final-cdr (cdr x))))
