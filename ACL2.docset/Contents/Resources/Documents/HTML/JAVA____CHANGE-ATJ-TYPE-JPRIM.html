Modifying constructor for atj-type-jprim structures.

Syntax
(change-atj-type-jprim x 
                       [:get <get>])

This is an often useful alternative to make-atj-type-jprim.

We construct a new atj-type-jprim structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
