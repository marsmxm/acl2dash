An executable lexer of Yul.

This is a simple lexer for the Yul lexical grammar. The grammar is defined in ABNF. See grammar-new.

The primary API for lexing Yul is lexemeize-yul and lexemeize-yul-bytes.

The lexer is defined in three sections:

  1. Named pieces of ABNF abstract syntax that occur internally to Yul grammar rules.
  2. Parser generator control constants (see parser-generators) that are used to help generate the lexing functions:
    • a constant containing the ABNF abstract syntax form of the lexical grammar
    • a constant containing the prefix that is attached to an ABNF rule name to generate its lexing function name
    • three constants (for groups, options, and repetitions) that map the abstract syntax structures in section 1 to names of the lexing functions
  3. Definitions of lexing functions, most of which are just calls to macros that generate the functions, but some of which are hand-written due to limitations in the current parser generators.
