Get the size for a data type, including unpacked dimensions.

Signature
(vl-datatype-size x) → (mv warning size)
Arguments
x — Guard (vl-datatype-p x).
Returns
warning — Type (iff (vl-warning-p warning) warning).
size — Type (implies (not warning) (posp size)).

The type should be fully resolved (i.e. no usertypes) or we'll fail.
