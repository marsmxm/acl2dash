Modifying constructor for fgl-rule-meta structures.

Syntax
(change-fgl-rule-meta x 
                      [:name <name>])

This is an often useful alternative to make-fgl-rule-meta.

We construct a new fgl-rule-meta structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-fgl-rule-meta

(defmacro
     change-fgl-rule-meta (x &rest args)
     (std::change-aggregate 'fgl-rule-meta
                            x args '((:name . fgl-rule-meta->name))
                            'change-fgl-rule-meta
                            'nil))
