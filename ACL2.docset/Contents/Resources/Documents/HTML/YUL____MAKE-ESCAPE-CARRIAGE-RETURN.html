Basic constructor macro for escape-carriage-return structures.

Syntax
(make-escape-carriage-return )

This is the usual way to construct escape-carriage-return structures. It simply conses together a structure with the specified fields.

This macro generates a new escape-carriage-return structure from scratch. See also change-escape-carriage-return, which can "change" an existing structure, instead.
