Turn an ACL2 boolean into an int value 0 or 1.

Signature
(sint-from-boolean b) → x
Arguments
b — Guard (booleanp b).
Returns
x — Type (sintp x).

This is essentially (but not exactly) the inverse of boolean-from-sint. Together with boolean-from-sint and other boolean-from-... operations, it can be used to represent in ACL2 shallowly embedded C logical conjunctions and disjunctions, which must be integers in C, but must be booleans in ACL2 to represent their non-strictness.

Function: sint-from-boolean

(defun sint-from-boolean (b)
       (declare (xargs :guard (booleanp b)))
       (if b (sint 1) (sint 0)))
