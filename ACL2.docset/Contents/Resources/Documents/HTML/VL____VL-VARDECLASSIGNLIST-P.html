(vl-vardeclassignlist-p x) recognizes lists where every element satisfies vl-vardeclassign-p.

Signature
(vl-vardeclassignlist-p x)  std::bool

This is an ordinary deflist. It is "loose" in that it does not care whether x is nil-terminated.
