Modifying constructor for vttree-warnings structures.

Syntax
(change-vttree-warnings x 
                        [:warnings <warnings>])

This is an often useful alternative to make-vttree-warnings.

We construct a new vttree-warnings structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vttree-warnings

(defmacro
   change-vttree-warnings (x &rest args)
   (std::change-aggregate 'vttree-warnings
                          x args
                          '((:warnings . vttree-warnings->warnings))
                          'change-vttree-warnings
                          'nil))
