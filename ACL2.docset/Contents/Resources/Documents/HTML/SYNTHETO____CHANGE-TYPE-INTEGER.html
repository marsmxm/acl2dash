Modifying constructor for type-integer structures.

Syntax
(change-type-integer x 
                     )

This is an often useful alternative to make-type-integer.

We construct a new type-integer structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-type-integer

(defmacro change-type-integer (x &rest args)
          (std::change-aggregate 'type-integer
                                 x args 'nil
                                 'change-type-integer
                                 'nil))
