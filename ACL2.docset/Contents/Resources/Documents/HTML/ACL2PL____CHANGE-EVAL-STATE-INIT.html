Modifying constructor for eval-state-init structures.

Syntax
(change-eval-state-init x 
                        [:function <function>] 
                        [:arguments <arguments>])

This is an often useful alternative to make-eval-state-init.

We construct a new eval-state-init structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-eval-state-init

(defmacro
 change-eval-state-init (x &rest args)
 (std::change-aggregate 'eval-state-init
                        x args
                        '((:function . eval-state-init->function)
                          (:arguments . eval-state-init->arguments))
                        'change-eval-state-init
                        'nil))
