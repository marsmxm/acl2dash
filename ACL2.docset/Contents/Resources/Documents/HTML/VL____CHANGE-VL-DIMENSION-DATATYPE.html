Modifying constructor for vl-dimension-datatype structures.

Syntax
(change-vl-dimension-datatype x 
                              [:type <type>])

This is an often useful alternative to make-vl-dimension-datatype.

We construct a new vl-dimension-datatype structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-dimension-datatype

(defmacro
