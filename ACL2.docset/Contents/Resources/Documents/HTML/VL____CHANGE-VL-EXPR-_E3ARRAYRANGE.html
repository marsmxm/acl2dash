Modifying constructor for vl-arrayrange-index structures.

Syntax
(change-vl-expr->arrayrange x 
                            [:expr <expr>])

This is an often useful alternative to make-vl-expr->arrayrange.

We construct a new vl-arrayrange-index structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-expr->arrayrange

(defmacro
     change-vl-expr->arrayrange
     (x &rest args)
     (std::change-aggregate 'vl-expr->arrayrange
                            x args
                            '((:expr . vl-expr->arrayrange->expr))
                            'change-vl-expr->arrayrange
                            'nil))
