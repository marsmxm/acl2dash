Basic constructor macro for jtype-class structures.

Syntax
(make-jtype-class [:name <name>])

This is the usual way to construct jtype-class structures. It simply conses together a structure with the specified fields.

This macro generates a new jtype-class structure from scratch. See also change-jtype-class, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by fty::defprod.

Macro: make-jtype-class

(defmacro make-jtype-class (&rest args)
