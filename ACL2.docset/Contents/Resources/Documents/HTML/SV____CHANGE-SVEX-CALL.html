Modifying constructor for svex-call structures.

Syntax
(change-svex-call x 
                  [:fn <fn>] 
                  [:args <args>])

This is an often useful alternative to make-svex-call.

We construct a new svex-call structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-svex-call

(defmacro change-svex-call (x &rest args)
          (std::change-aggregate 'svex-call
                                 x args
                                 '((:fn . svex-call->fn)
                                   (:args . svex-call->args))
                                 'change-svex-call
                                 'nil))
