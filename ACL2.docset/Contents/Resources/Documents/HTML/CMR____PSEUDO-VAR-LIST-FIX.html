(pseudo-var-list-fix x) is a usual ACL2::fty list fixing function.

Signature
(pseudo-var-list-fix x)  fty::newx
Arguments
x Guard (pseudo-var-list-p x).
Returns
fty::newx Type (pseudo-var-list-p fty::newx).

In the logic, we apply pseudo-var-fix to each member of the x. In the execution, none of that is actually necessary and this is just an inlined identity function.
