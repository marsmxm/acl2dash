Modifying constructor for function-definer-regular structures.

Syntax
(change-function-definer-regular x 
                                 [:body <body>] 
                                 [:measure <measure>])

This is an often useful alternative to make-function-definer-regular.

We construct a new function-definer-regular structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-function-definer-regular

(defmacro change-function-definer-regular
          (x &rest args)
          (std::change-aggregate
               'function-definer-regular
               x args
               '((:body . function-definer-regular->body)
                 (:measure . function-definer-regular->measure))
               'change-function-definer-regular
               'nil))
