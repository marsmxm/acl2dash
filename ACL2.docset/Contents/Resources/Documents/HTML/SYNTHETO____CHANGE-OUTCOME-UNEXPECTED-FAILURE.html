Modifying constructor for outcome-unexpected-failure structures.

Syntax
(change-outcome-unexpected-failure x 
                                   [:message <message>])

This is an often useful alternative to make-outcome-unexpected-failure.

We construct a new outcome-unexpected-failure structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-outcome-unexpected-failure

(defmacro change-outcome-unexpected-failure
          (x &rest args)
          (std::change-aggregate
               'outcome-unexpected-failure
               x args
               '((:message . outcome-unexpected-failure->message))
               'change-outcome-unexpected-failure
               'nil))
