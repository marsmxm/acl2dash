Execute a function call or assignment expression.

Signature
(exec-expr-call-or-asg e compst fenv limit)  new-compst
Arguments
e Guard (exprp e).
compst Guard (compustatep compst).
fenv Guard (fun-envp fenv).
limit Guard (natp limit).
Returns
new-compst Type (compustate-resultp new-compst).

This is used for expressions used as expression statements. Thus, in the case of a function call, we discard the returned value, if any.
