Modifying constructor for primitive-value-short structures.

Syntax
(change-primitive-value-short x 
                              [:get <get>])

This is an often useful alternative to make-primitive-value-short.

We construct a new primitive-value-short structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-primitive-value-short

(defmacro
     change-primitive-value-short
     (x &rest args)
     (std::change-aggregate 'primitive-value-short
                            x args
                            '((:get . primitive-value-short->get))
                            'change-primitive-value-short
                            'nil))
