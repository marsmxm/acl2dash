Look up a function in a function table.

Signature
(fun-table-lookup fun funtab)  fun-type
Arguments
fun Guard (identp fun).
funtab Guard (fun-tablep funtab).
Returns
fun-type Type (fun-type-optionp fun-type).

We return the type of the function, if the function is present. Otherwise, we return nil.

(defun
     fun-table-lookup (fun funtab)
     (declare (xargs :guard (and (identp fun) (fun-tablep funtab))))
     (let ((__function__ 'fun-table-lookup))

          (cdr (omap::in (ident-fix fun)
