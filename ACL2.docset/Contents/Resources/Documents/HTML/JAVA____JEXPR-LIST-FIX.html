(jexpr-list-fix x) is a usual ACL2::fty list fixing function.

Signature
(jexpr-list-fix x) → fty::newx
Arguments
x — Guard (jexpr-listp x).
Returns
fty::newx — Type (jexpr-listp fty::newx).

In the logic, we apply jexpr-fix to each member of the x. In the execution, none of that is actually necessary and this is just an inlined identity function.
