Modifying constructor for tag-env-result-err structures.

Syntax
(change-tag-env-result-err x 
                           [:get <get>])

This is an often useful alternative to make-tag-env-result-err.

We construct a new tag-env-result-err structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
