Constructor macro for vl-realtoken-p structures.

Syntax:

(make-vl-realtoken [:etext <etext>] 
                   [:breakp <breakp>])

This is our preferred way to construct vl-realtoken-p structures. It simply conses together a structure with the specified fields.

This macro generates a new vl-realtoken-p structure from scratch. See also change-vl-realtoken, which can "change" an existing structure, instead.

The vl-realtoken-p structures we create here are just constructed with ordinary cons. If you want to create honsed structures, see make-honsed-vl-realtoken instead.

Definition

This is an ordinary make- macro introduced by defaggregate.

Macro: make-vl-realtoken

(defmacro make-vl-realtoken (&rest args)
          (std::make-aggregate 'vl-realtoken
                               args '((:etext) (:breakp))
                               'make-vl-realtoken
                               nil))
