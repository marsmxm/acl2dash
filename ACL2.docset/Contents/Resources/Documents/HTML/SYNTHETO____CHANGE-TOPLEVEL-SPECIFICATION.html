Modifying constructor for toplevel-specification structures.

Syntax
(change-toplevel-specification x 
                               [:get <get>])

This is an often useful alternative to make-toplevel-specification.

We construct a new toplevel-specification structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-toplevel-specification

(defmacro
     change-toplevel-specification
     (x &rest args)
     (std::change-aggregate 'toplevel-specification
                            x args
                            '((:get . toplevel-specification->get))
                            'change-toplevel-specification
                            'nil))
