Execute a list of pure expression.

Signature
(exec-expr-pure-list es compst) → result
Arguments
es — Guard (expr-listp es).
compst — Guard (compustatep compst).
Returns
result — Type (value-list-resultp result).

This is used, in particular, for the argument expressions a function call.

Given that the expression have no side effects (if there is no error), the order of evaluation does not matter. Thus, we proceed left to right.

Definitions and Theorems

Function: exec-expr-pure-list

(defun exec-expr-pure-list (es compst)
       (declare (xargs :guard (and (expr-listp es)
                                   (compustatep compst))))
       (let ((__function__ 'exec-expr-pure-list))
            (declare (ignorable __function__))
            (b* (((when (endp es)) nil)
                 (val (exec-expr-pure (car es) compst))
                 ((when (errorp val)) val)
                 (vals (exec-expr-pure-list (cdr es) compst))
                 ((when (errorp vals)) vals))
                (cons val vals))))
