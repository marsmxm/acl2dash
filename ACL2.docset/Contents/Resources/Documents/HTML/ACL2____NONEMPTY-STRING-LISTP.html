Recognize true lists of nonempty strings.

Signature
(nonempty-string-listp x)  std::bool

This is an ordinary std::deflist. It is "strict" in that it requires x to be a "properly" nil-terminated list.

Definitions and Theorems

Function: nonempty-string-listp

(defun nonempty-string-listp (x)
       (declare (xargs :guard t))
       (let ((__function__ 'nonempty-string-listp))
            (declare (ignorable __function__))
            (if (consp x)
                (and (nonempty-stringp (car x))
                     (nonempty-string-listp (cdr x)))
                (null x))))
