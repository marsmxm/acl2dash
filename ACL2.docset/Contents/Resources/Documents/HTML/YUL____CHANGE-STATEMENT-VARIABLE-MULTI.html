Modifying constructor for statement-variable-multi structures.

Syntax
(change-statement-variable-multi x 
                                 [:names <names>] 
                                 [:init <init>])

This is an often useful alternative to make-statement-variable-multi.

We construct a new statement-variable-multi structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-statement-variable-multi

(defmacro
