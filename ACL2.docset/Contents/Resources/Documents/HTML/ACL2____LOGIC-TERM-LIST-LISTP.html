Recognizer for lists of lists of terms that call only :logic-mode function symbols

This predicate strengthens (term-list-listp x wrld), as it also requires (logic-fns-list-listp x wrld), i.e., that every function symbol called in x is in :logic mode in the world, wrld.

Function: logic-term-list-listp

(defun logic-term-list-listp (x w)
       (declare (xargs :guard (plist-worldp-with-formals w)))
       (and (term-list-listp x w)
            (logic-fns-list-listp x w)))
