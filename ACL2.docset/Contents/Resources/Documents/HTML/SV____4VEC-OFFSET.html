Negation, except Z bits become 0.

Signature
(4vec-offset x) → res
Arguments
x — Guard (4vec-p x).
Returns
res — Type (4vec-p res).

The closest monotonic approximation of the offset of each of the bits of the input. It differs from the actual offset in that the 4vec-offset of an X bit is X.

Definitions and Theorems

Function: 4vec-offset

(defun 4vec-offset (x)
       (declare (xargs :guard (4vec-p x)))
       (let ((__function__ '4vec-offset))
            (declare (ignorable __function__))
            (b* (((4vec x)))
                (4vec (lognot x.lower)
                      (lognot (logior x.upper x.lower))))))

Theorem: 4vec-offset-bits

(defthm 4vec-offset-bits
        (equal (4vec-idx->4v n (4vec-offset x))
               (let ((in (4vec-idx->4v n x)))
