Proof of grammar-dec-digitp from dec-digitp.

This is proved using dec-digit-tree as witness for the existential quantification: if x is a digit then we can use its tree as witness, since its leaves are the digit x as well.

Definitions and Theorems

Theorem: grammar-dec-digitp-when-dec-digitp

(defthm grammar-dec-digitp-when-dec-digitp
        (implies (dec-digitp x)
                 (grammar-dec-digitp (list x))))
