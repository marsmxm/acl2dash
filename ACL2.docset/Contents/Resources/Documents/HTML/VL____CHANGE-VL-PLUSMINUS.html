Modifying constructor for vl-plusminus structures.

Syntax
(change-vl-plusminus x 
                     [:base <base>] 
                     [:width <width>] 
                     [:minusp <minusp>])

This is an often useful alternative to make-vl-plusminus.

We construct a new vl-plusminus structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-plusminus

(defmacro change-vl-plusminus (x &rest args)
          (std::change-aggregate 'vl-plusminus
                                 x args
                                 '((:base . vl-plusminus->base)
                                   (:width . vl-plusminus->width)
                                   (:minusp . vl-plusminus->minusp))
                                 'change-vl-plusminus
                                 'remake-vl-plusminus))
