Basic constructor macro for bip32-ext-priv-key structures.

Syntax
(make-bip32-ext-priv-key [:key <key>] 
                         [:chain-code <chain-code>])

This is the usual way to construct bip32-ext-priv-key structures. It simply conses together a structure with the specified fields.

This macro generates a new bip32-ext-priv-key structure from scratch. See also change-bip32-ext-priv-key, which can "change" an existing structure, instead.
