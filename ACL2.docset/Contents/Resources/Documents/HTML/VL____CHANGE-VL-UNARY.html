Modifying constructor for vl-unary structures.

Syntax
(change-vl-unary x 
                 [:op <op>] 
                 [:arg <arg>] 
                 [:atts <atts>])

This is an often useful alternative to make-vl-unary.

We construct a new vl-unary structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-unary

(defmacro change-vl-unary (x &rest args)
          (std::change-aggregate 'vl-unary
                                 x args
                                 '((:op . vl-unary->op)
                                   (:arg . vl-unary->arg)
                                   (:atts . vl-unary->atts))
                                 'change-vl-unary
                                 'remake-vl-unary))
