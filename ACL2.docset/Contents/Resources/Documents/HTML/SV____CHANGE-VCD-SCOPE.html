Modifying constructor for vcd-scope structures.

Syntax
(change-vcd-scope x 
                  [:name <name>] 
                  [:wires <wires>] 
                  [:subscopes <subscopes>])

This is an often useful alternative to make-vcd-scope.

We construct a new vcd-scope structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vcd-scope

(defmacro
     change-vcd-scope (x &rest args)
     (std::change-aggregate 'vcd-scope
                            x args
                            '((:name . vcd-scope->name)
                              (:wires . vcd-scope->wires)
                              (:subscopes . vcd-scope->subscopes))
                            'change-vcd-scope
                            'nil))
