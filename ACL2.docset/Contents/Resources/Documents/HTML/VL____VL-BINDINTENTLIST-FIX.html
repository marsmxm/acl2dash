(vl-bindintentlist-fix x) is a usual fty list fixing function.

Signature
(vl-bindintentlist-fix x) → fty::newx
Arguments
x — Guard (vl-bindintentlist-p x).
Returns
fty::newx — Type (vl-bindintentlist-p fty::newx).

In the logic, we apply vl-bindintent-fix to each member of the x. In the execution, none of that is actually necessary and this is just an inlined identity function.
