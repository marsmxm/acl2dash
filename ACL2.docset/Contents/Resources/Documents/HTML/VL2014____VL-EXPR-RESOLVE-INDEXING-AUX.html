Core routine for introducing :vl-array-index and :vl-bitselect operators throughout a vl-expr-p.

Signature
(vl-expr-resolve-indexing-aux x ss warnings) 
  → 
(mv new-warnings changedp new-x)
Arguments
x — Guard (vl-expr-p x).
ss — Guard (vl-scopestack-p ss).
warnings — Guard (vl-warninglist-p warnings).
Returns
new-warnings — Type (vl-warninglist-p new-warnings).
changedp — Optimization, don't re-cons index-free expresions.
    Type (booleanp changedp).
new-x — Type (vl-expr-p new-x).
