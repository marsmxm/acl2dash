Basic constructor macro for type-pointer structures.

Syntax
(make-type-pointer [:referenced <referenced>])

This is the usual way to construct type-pointer structures. It simply conses together a structure with the specified fields.

This macro generates a new type-pointer structure from scratch. See also change-type-pointer, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by fty::defprod.

Macro: make-type-pointer

(defmacro make-type-pointer (&rest args)
          (std::make-aggregate 'type-pointer
                               args '((:referenced))
                               'make-type-pointer
                               nil))

(defun type-pointer (referenced)
       (declare (xargs :guard (typep referenced)))
       (declare (xargs :guard t))
       (let ((__function__ 'type-pointer))
            (declare (ignorable __function__))
            (b* ((referenced (mbe :logic (type-fix referenced)
                                  :exec referenced)))
                (cons :pointer (list referenced)))))
