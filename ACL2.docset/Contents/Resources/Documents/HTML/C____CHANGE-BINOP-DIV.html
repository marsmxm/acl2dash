Modifying constructor for binop-div structures.

Syntax
(change-binop-div x 
                  )

This is an often useful alternative to make-binop-div.

We construct a new binop-div structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
