Transform the recursive calls in the body of old.

Signature
(parteval-transform-rec-calls-in-term 
     body-term old$ new-name$ y1...ym wrld) 
 
  → 
new-body-term
Arguments
body-term — Guard (pseudo-termp body-term).
old$ — Guard (symbolp old$).
new-name$ — Guard (symbolp new-name$).
y1...ym — Guard (symbol-listp y1...ym).
wrld — Guard (plist-worldp wrld).

This applies to case 2 in the reference documentation. Each call of old is replaced with a call of new with the static arguments removed. This code recursively processes the terms in the body of old.
