Modifying constructor for numeric-type-int structures.

Syntax
(change-numeric-type-int x 
                         )

This is an often useful alternative to make-numeric-type-int.

We construct a new numeric-type-int structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
