Modifying constructor for atj-type-jprimarr structures.

Syntax

                          [:comp <comp>])

This is an often useful alternative to make-atj-type-jprimarr.

We construct a new atj-type-jprimarr structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
