Basic constructor macro for vl-paramvalue-expr structures.

Syntax
(make-vl-paramvalue-expr [:expr <expr>])

This is the usual way to construct vl-paramvalue-expr structures. It simply conses together a structure with the specified fields.

This macro generates a new vl-paramvalue-expr structure from scratch. See also change-vl-paramvalue-expr, which can "change" an existing structure, instead.
