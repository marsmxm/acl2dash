(svex-contextlist-fix x) is a usual fty list fixing function.

Signature
(svex-contextlist-fix x) → fty::newx
Arguments
x — Guard (svex-contextlist-p x).
Returns
fty::newx — Type (svex-contextlist-p fty::newx).

In the logic, we apply svex-context-fix to each member of the x. In the execution, none of that is actually necessary and this is just an inlined identity function.
