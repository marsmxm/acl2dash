Modifying constructor for outcome-function-success structures.

Syntax
(change-outcome-function-success x 
                                 [:message <message>])

This is an often useful alternative to make-outcome-function-success.

We construct a new outcome-function-success structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-outcome-function-success

(defmacro change-outcome-function-success
          (x &rest args)
          (std::change-aggregate
               'outcome-function-success
               x args
               '((:message . outcome-function-success->message))
               'change-outcome-function-success
               'nil))
