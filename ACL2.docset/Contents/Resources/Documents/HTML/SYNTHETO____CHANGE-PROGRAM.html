Modifying constructor for program structures.

Syntax
(change-program x 
                [:tops <tops>])

This is an often useful alternative to make-program.

We construct a new program structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-program

(defmacro change-program (x &rest args)
          (std::change-aggregate 'program
                                 x args '((:tops . program->tops))
                                 'change-program
                                 'nil))
