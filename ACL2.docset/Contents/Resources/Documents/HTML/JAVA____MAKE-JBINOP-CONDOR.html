Basic constructor macro for jbinop-condor structures.

Syntax
(make-jbinop-condor )

This is the usual way to construct jbinop-condor structures. It simply conses together a structure with the specified fields.

This macro generates a new jbinop-condor structure from scratch. See also change-jbinop-condor, which can "change" an existing structure, instead.
