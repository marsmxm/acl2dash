Modifying constructor for jtype-class structures.

Syntax
(change-jtype-class x 
                    [:name <name>])

This is an often useful alternative to make-jtype-class.

We construct a new jtype-class structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-jtype-class

(defmacro
     change-jtype-class (x &rest args)
     (std::change-aggregate 'jtype-class
                            x args '((:name . jtype-class->name))
                            'change-jtype-class
                            'nil))
