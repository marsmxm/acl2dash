Modifying constructor for command-error-malformed-mnemonic structures.

Syntax
(change-command-error-malformed-mnemonic x 
                                         [:mnemonic <mnemonic>])

This is an often useful alternative to make-command-error-malformed-mnemonic.

We construct a new command-error-malformed-mnemonic structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
