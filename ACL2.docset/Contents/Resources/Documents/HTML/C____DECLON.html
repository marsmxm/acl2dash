Declarations are perhaps the most complex part of the C syntax. For now we capture only a very limited form of declarations, namely:

  • The ones consisting of a type specifier sequence from tyspecseq, no storage class specifiers, no type qualifiers, no function specifiers, no alignment specifiers, and a single declarator (see declor) with an initializer expression. These declare variables.
  • The ones consisting of a single structure type specifier consisting of the tag identifier and a list of structure declarations from struct-declon. These declare structure types.

We will support richer forms of declarations when needed.

We do not support static assertions for now.
