Modifying constructor for jexpr-postinc structures.

Syntax
(change-jexpr-postinc x 
                      [:arg <arg>])

This is an often useful alternative to make-jexpr-postinc.

We construct a new jexpr-postinc structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-jexpr-postinc

(defmacro
     change-jexpr-postinc (x &rest args)
     (std::change-aggregate 'jexpr-postinc
                            x args '((:arg . jexpr-postinc->arg))
                            'change-jexpr-postinc
                            'nil))
