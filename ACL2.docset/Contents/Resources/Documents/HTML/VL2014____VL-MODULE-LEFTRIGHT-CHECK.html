(vl-module-leftright-check x) carries our our leftright-check on all the expressions in a module, and adds any resulting warnings to the module.

Arguments
x — Guard (vl-module-p x).
Returns
new-x — Type (vl-module-p new-x), given the guard.

Definitions and Theorems

Function: vl-module-leftright-check

(defun
 vl-module-leftright-check (x)
 (declare (xargs :guard (vl-module-p x)))
 (let
  ((__function__ 'vl-module-leftright-check))
  (declare (ignorable __function__))
  (b*
    ((ctxexprs (vl-module-ctxexprs x))
     (new-warnings (vl-exprctxalist-leftright-check ctxexprs)))
    (change-vl-module x
                      :warnings (append new-warnings
                                        (vl-module->warnings x))))))
