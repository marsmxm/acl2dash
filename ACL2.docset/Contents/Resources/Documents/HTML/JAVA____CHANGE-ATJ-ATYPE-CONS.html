Modifying constructor for atj-atype-cons structures.

Syntax
(change-atj-atype-cons x 
                       )

This is an often useful alternative to make-atj-atype-cons.

We construct a new atj-atype-cons structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
