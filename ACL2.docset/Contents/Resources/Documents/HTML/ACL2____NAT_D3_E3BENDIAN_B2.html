Convert a natural number to its non-empty minimum-length big-endian list of digits.

Signature
(nat=>bendian+ base nat) → digits
Arguments
base — Guard (dab-basep base).
nat — Guard (natp nat).
Returns
digits — Type (dab-digit-listp base digits).

The resulting list is never empty; it is (0) if nat is 0. The + in the name of this function can be read as `one or more' (as in typical regular expression notation).

See also nat=>bendian* and nat=>bendian.

Function: nat=>bendian+

(defun nat=>bendian+ (base nat)
       (declare (xargs :guard (and (dab-basep base) (natp nat))))
       (let ((__function__ 'nat=>bendian+))
            (declare (ignorable __function__))
            (b* ((digits (nat=>bendian* base nat)))
                (or digits (list 0)))))
