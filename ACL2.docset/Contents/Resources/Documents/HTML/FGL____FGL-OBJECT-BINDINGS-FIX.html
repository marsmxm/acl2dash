(fgl-object-bindings-fix x) is an ACL2::fty alist fixing function that follows the drop-keys strategy.

Signature
(fgl-object-bindings-fix x)  fty::newx
Arguments
x Guard (fgl-object-bindings-p x).
Returns
fty::newx Type (fgl-object-bindings-p fty::newx).

Note that in the execution this is just an inline identity function.
