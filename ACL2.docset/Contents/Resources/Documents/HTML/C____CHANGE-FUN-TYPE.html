Modifying constructor for fun-type structures.

Syntax
(change-fun-type x 
                 [:inputs <inputs>] 
                 [:output <output>])

This is an often useful alternative to make-fun-type.

We construct a new fun-type structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-fun-type

(defmacro change-fun-type (x &rest args)
          (std::change-aggregate 'fun-type
                                 x args
                                 '((:inputs . fun-type->inputs)
                                   (:output . fun-type->output))
                                 'change-fun-type
                                 'nil))
