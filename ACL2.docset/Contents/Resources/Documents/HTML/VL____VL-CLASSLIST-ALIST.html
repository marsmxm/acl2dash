Extend an alist by binding the names of vl-classs to their definitions.

Signature
(vl-classlist-alist x acc) → alist
Arguments
x — Guard (vl-classlist-p x).
Returns
alist — Type (equal (vl-class-alist-p alist) (vl-class-alist-p acc)).

This can be used as an alternative to vl-find-class when you need to perform a lot of lookups.

Definitions and Theorems

Function: vl-classlist-alist

(defun vl-classlist-alist (x acc)
       (declare (xargs :guard (vl-classlist-p x)))
       (let ((__function__ 'vl-classlist-alist))
            (declare (ignorable __function__))
            (if (atom x)
                acc
                (cons (cons (vl-class->name (car x))
                            (vl-class-fix (car x)))
                      (vl-classlist-alist (cdr x) acc)))))

Theorem: return-type-of-vl-classlist-alist

(defthm return-type-of-vl-classlist-alist
        (b* ((alist (vl-classlist-alist x acc)))
            (equal (vl-class-alist-p alist)
                   (vl-class-alist-p acc)))

Theorem: lookup-in-vl-classlist-alist-acc-elim

(defthm
  lookup-in-vl-classlist-alist-acc-elim
  (implies
       (syntaxp (not (equal acc ''nil)))
       (equal (hons-assoc-equal name (vl-classlist-alist x acc))
              (or (hons-assoc-equal name (vl-classlist-alist x nil))
                  (hons-assoc-equal name acc)))))

Theorem: lookup-in-vl-classlist-alist-fast

(defthm
  lookup-in-vl-classlist-alist-fast
  (implies (stringp name)
           (equal (hons-assoc-equal name (vl-classlist-alist x nil))
                  (let ((val (vl-find-class name x)))
                       (and val (cons name val))))))
