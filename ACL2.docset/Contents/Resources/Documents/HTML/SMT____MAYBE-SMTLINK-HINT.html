Option type; smtlink-hint or nil.

This is an option type introduced by defoption. Note that defoption is just a wrapper for fty::defflexsum, so there are :none and :some member types, a case macro, and so forth.

Member Types
:none → maybe-smtlink-hint-none
Represents that no maybe-smtlink-hint is available, i.e., Nothing or None.
:some → maybe-smtlink-hint-some
An available maybe-smtlink-hint, i.e., Just val or Some val.
