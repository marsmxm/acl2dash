Basic constructor macro for theorem structures.

Syntax
(make-theorem [:name <name>] 
              [:variables <variables>] 
              [:formula <formula>])

This is the usual way to construct theorem structures. It simply conses together a structure with the specified fields.

This macro generates a new theorem structure from scratch. See also change-theorem, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by fty::defprod.

Macro: make-theorem

(defmacro
     make-theorem (&rest args)
     (std::make-aggregate 'theorem
                          args '((:name) (:variables) (:formula))
                          'make-theorem
                          nil))

(defun
 theorem (name variables formula)
 (declare (xargs :guard (and (identifierp name)
                             (typed-variable-listp variables)
                             (expressionp formula))))
 (declare (xargs :guard t))
 (let
     ((__function__ 'theorem))
     (declare (ignorable __function__))
     (b* ((name (mbe :logic (identifier-fix name)
                     :exec name))
          (variables (mbe :logic (typed-variable-list-fix variables)
                          :exec variables))
          (formula (mbe :logic (expression-fix formula)
                        :exec formula)))
         (list (cons 'name name)
               (cons 'variables variables)
               (cons 'formula formula)))))
