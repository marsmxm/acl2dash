Modifying constructor for rlp-error-branch-tree structures.

Syntax
(change-rlp-error-branch-tree x 
                              [:fragment <fragment>])

This is an often useful alternative to make-rlp-error-branch-tree.

We construct a new rlp-error-branch-tree structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-rlp-error-branch-tree

(defmacro change-rlp-error-branch-tree
          (x &rest args)
          (std::change-aggregate
               'rlp-error-branch-tree
               x args
               '((:fragment . rlp-error-branch-tree->fragment))
               'change-rlp-error-branch-tree
               'nil))
