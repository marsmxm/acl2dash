Modifying constructor for 4vec-driver structures.

Syntax
(change-4vec-driver x 
                    [:value <value>] 
                    [:strength <strength>])

This is an often useful alternative to make-4vec-driver.

We construct a new 4vec-driver structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-4vec-driver

(defmacro
     change-4vec-driver (x &rest args)
     (std::change-aggregate '4vec-driver
                            x args
                            '((:value . 4vec-driver->value)
                              (:strength . 4vec-driver->strength))
                            'change-4vec-driver
                            'remake-4vec-driver))
