Check a list of pure expressions.

Signature
(check-expr-pure-list es vartab tagenv) → types
Arguments

vartab — Guard (var-tablep vartab).
tagenv — Guard (tag-envp tagenv).

This is used for arguments of function calls. The expression types returned by the expressions are subjected to lvalue conversion and array-to-pointer conversion.

Definitions and Theorems

Function: check-expr-pure-list

(defun check-expr-pure-list (es vartab tagenv)
       (declare (xargs :guard (and (expr-listp es)
                                   (var-tablep vartab)
                                   (tag-envp tagenv))))
       (let ((__function__ 'check-expr-pure-list))
            (declare (ignorable __function__))
            (b* (((when (endp es)) nil)
                 (etype (check-expr-pure (car es)
                                         vartab tagenv))
                 ((when (errorp etype)) etype)
                 (type (expr-type->type etype))
                 (type (apconvert-type type))
                 (types (check-expr-pure-list (cdr es)
                                              vartab tagenv))
                 ((when (errorp types)) types))
                (cons type types))))
