Modifying constructor for svex-context-top structures.

Syntax
(change-svex-context-top x 
                         )

This is an often useful alternative to make-svex-context-top.

We construct a new svex-context-top structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
