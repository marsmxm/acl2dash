Modifying constructor for iconst-base-oct structures.

Syntax
(change-iconst-base-oct x 
                        )

This is an often useful alternative to make-iconst-base-oct.

We construct a new iconst-base-oct structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

This is an ordinary change- macro introduced by fty::defprod.
