Modifying constructor for const-float structures.

Syntax
(change-const-float x 
                    )

This is an often useful alternative to make-const-float.

We construct a new const-float structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-const-float

(defmacro change-const-float (x &rest args)
          (std::change-aggregate 'const-float
                                 x args 'nil
                                 'change-const-float
                                 'nil))
