Fixtype of optional addresses.

This is an option type introduced by fty::defoption. Note that defoption is just a wrapper for fty::defflexsum, so there are :none and :some member types, a case macro, and so forth.

Member Types
:none → address-option-none
Represents that no address-option is available, i.e., Nothing or None.
:some → address-option-some
An available address-option, i.e., Just val or Some val.
