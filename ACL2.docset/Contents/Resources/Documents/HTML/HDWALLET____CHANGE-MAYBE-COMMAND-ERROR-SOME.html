Modifying constructor for maybe-command-error-some structures.

Syntax
(change-maybe-command-error-some x 
                                 [:val <val>])

This is an often useful alternative to make-maybe-command-error-some.

We construct a new maybe-command-error-some structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
