Modifying constructor for vl-warningtree-pair structures.

Syntax
(change-vl-warningtree-pair x 
                            [:left <left>] 
                            [:right <right>])

This is an often useful alternative to make-vl-warningtree-pair.

We construct a new vl-warningtree-pair structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

(defmacro
