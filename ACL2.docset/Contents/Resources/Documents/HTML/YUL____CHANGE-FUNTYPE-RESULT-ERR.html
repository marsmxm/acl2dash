Modifying constructor for funtype-result-err structures.

Syntax
(change-funtype-result-err x 
                           [:get <get>])

This is an often useful alternative to make-funtype-result-err.

We construct a new funtype-result-err structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
