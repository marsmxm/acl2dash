Modifying constructor for label-cas structures.

Syntax
(change-label-cas x 
                  [:get <get>])

This is an often useful alternative to make-label-cas.

We construct a new label-cas structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-label-cas

(defmacro change-label-cas (x &rest args)
          (std::change-aggregate 'label-cas
                                 x args '((:get . label-cas->get))
                                 'change-label-cas
                                 'nil))
