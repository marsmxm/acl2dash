Modifying constructor for atj-atype-string structures.

Syntax
(change-atj-atype-string x 
                         )

This is an often useful alternative to make-atj-atype-string.

We construct a new atj-atype-string structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
