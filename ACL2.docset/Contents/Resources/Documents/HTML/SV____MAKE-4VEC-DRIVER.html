Basic constructor macro for 4vec-driver structures.

Syntax
(make-4vec-driver [:value <value>] 
                  [:strength <strength>])

This is the usual way to construct 4vec-driver structures. It simply conses together a structure with the specified fields.

This macro generates a new 4vec-driver structure from scratch. See also change-4vec-driver, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by defprod.

Macro: make-4vec-driver

(defmacro make-4vec-driver (&rest args)
          (std::make-aggregate '4vec-driver
                               args '((:value) (:strength . 6))
                               'make-4vec-driver
                               nil))

(defun 4vec-driver (value strength)
       (declare (xargs :guard (and (4vec-p value) (natp strength))))
       (declare (xargs :guard t))
       (let ((__function__ '4vec-driver))
            (declare (ignorable __function__))
            (b* ((value (mbe :logic (4vec-fix value)
                             :exec value))
                 (strength (mbe :logic (nfix strength)
                                :exec strength)))
                (std::prod-cons value strength))))
