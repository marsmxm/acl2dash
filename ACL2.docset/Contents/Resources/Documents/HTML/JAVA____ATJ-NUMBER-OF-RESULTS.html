Number of results returned by a function.

Signature
(atj-number-of-results fn wrld)  numres
Arguments
fn Guard (symbolp fn).
wrld Guard (plist-worldp wrld).
Returns
numres Type (posp numres).

This is similar to number-of-results+, but that function has a guard disallowing the function symbol to be a member of the built-in constant *stobjs-out-invalid*, i.e. to be if or return-last. For ATJ's purpose, we totalize number-of-results+ by having it return 1 on these two functions. The actual result is irrelevant, because return-last is removed by one of ATJ's pre-translation steps, and if is treated specially by ATJ.
