Gathers the names of all design elements that particular descriptions directly depend on.

Signature
(vl-necessary-elements-direct superiors design) → subs
Arguments
superiors — Names of the superior design elements we're interested in.
    Guard (string-listp superiors).
design — The design.
    Guard (vl-design-p design).
Returns
subs — Names of the sub design elements that are directly needed by these superior elements.
    Type (and (string-listp subs) (setp subs)).

Function: vl-necessary-elements-direct

(defun vl-necessary-elements-direct
       (superiors design)
       (declare (xargs :guard (and (string-listp superiors)
                                   (vl-design-p design))))
       (let ((__function__ 'vl-necessary-elements-direct))
            (declare (ignorable __function__))
            (vl-collect-dependencies (string-list-fix superiors)
                                     (vl-design-downgraph design))))
