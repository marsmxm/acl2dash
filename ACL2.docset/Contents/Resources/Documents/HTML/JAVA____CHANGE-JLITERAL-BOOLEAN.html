Modifying constructor for jliteral-boolean structures.

Syntax
(change-jliteral-boolean x 
                         [:value <value>])

This is an often useful alternative to make-jliteral-boolean.

We construct a new jliteral-boolean structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-jliteral-boolean

(defmacro
     change-jliteral-boolean (x &rest args)
     (std::change-aggregate 'jliteral-boolean
                            x args
                            '((:value . jliteral-boolean->value))
                            'change-jliteral-boolean
                            'nil))
