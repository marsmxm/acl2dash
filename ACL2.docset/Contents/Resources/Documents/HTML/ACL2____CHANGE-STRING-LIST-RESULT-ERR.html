Modifying constructor for string-list-result-err structures.

Syntax
(change-string-list-result-err x 
                               [:get <get>])

This is an often useful alternative to make-string-list-result-err.

We construct a new string-list-result-err structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
