Modifying constructor for atj-atype-value structures.

Syntax
(change-atj-atype-value x 
                        )

This is an often useful alternative to make-atj-atype-value.

We construct a new atj-atype-value structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

This is an ordinary change- macro introduced by fty::defprod.
