Modifying constructor for vl-propthen structures.

Syntax
(change-vl-propthen x 
                    [:delay <delay>] 
                    [:left <left>] 
                    [:right <right>])

This is an often useful alternative to make-vl-propthen.

We construct a new vl-propthen structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-propthen

(defmacro change-vl-propthen (x &rest args)
          (std::change-aggregate 'vl-propthen
                                 x args
                                 '((:delay . vl-propthen->delay)
                                   (:left . vl-propthen->left)
                                   (:right . vl-propthen->right))
                                 'change-vl-propthen
                                 'nil))
