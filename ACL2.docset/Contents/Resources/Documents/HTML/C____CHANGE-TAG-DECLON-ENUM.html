Modifying constructor for tag-declon-enum structures.

Syntax
(change-tag-declon-enum x 
                        [:tag <tag>] 
                        [:enumerators <enumerators>])

This is an often useful alternative to make-tag-declon-enum.

We construct a new tag-declon-enum structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-tag-declon-enum

(defmacro change-tag-declon-enum (x &rest args)
          (std::change-aggregate
               'tag-declon-enum
               x args
               '((:tag . tag-declon-enum->tag)
                 (:enumerators . tag-declon-enum->enumerators))
               'change-tag-declon-enum
               'nil))
