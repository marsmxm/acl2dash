One-way unification for svexes.

Signature
(svex-unify pat x al) → (mv successp al1)
Arguments
pat — Pattern to match.
    Guard (svex-p pat).
x — Target expression to match against.
    Guard (svex-p x).
al — Accumulator for bindings so far. Slow alist.
    Guard (svex-alist-p al).
Returns
successp — Type (booleanp successp).
al1 — Updated alist.
    Type (svex-alist-p al1).

We expect to use this function when applying rewrite rules, which typically have only a few variables, so we don't use a fast alist for the accumulator.
