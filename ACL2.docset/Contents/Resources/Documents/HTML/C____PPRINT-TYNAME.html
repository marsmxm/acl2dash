Pretty-print a type name.

Signature
(pprint-tyname tn) → part
Arguments
tn — Guard (tynamep tn).
Returns
part — Type (msgp part).

If there is a declarator, we add a space beween it and the type specifier sequence. Otherwise, we just print the type specifier sequence.

Definitions and Theorems

Function: pprint-tyname

(defun pprint-tyname (tn)
       (declare (xargs :guard (tynamep tn)))
       (let ((__function__ 'pprint-tyname))
            (declare (ignorable __function__))
            (b* (((tyname tn) tn))
                (if (obj-adeclor-case tn.declor :none)
                    (pprint-tyspecseq tn.tyspec)
                    (msg "~@0 ~@1" (pprint-tyspecseq tn.tyspec)
                         (pprint-obj-adeclor tn.declor))))))

(defthm msgp-of-pprint-tyname
        (b* ((part (pprint-tyname tn)))
