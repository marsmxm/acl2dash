Prove that an SVTV is monotonic in all variables except override tests.

Usage:

(def-svtv-partial-monotonic <svtv-name> <export-name>)

Prerequisite: The SVTV must be defined with defsvtv$ or defsvtv$-phasewise (or otherwise result from populating a svtv-data stobj), and the contents of the stobj thus populated must be exported into a regular object <export-name> using def-svtv-data-export. Additionally, the setting for the monotonify argument of the defsvtv$ form must have been t (the default).

This proves two theorems about the SVTV:

(svex-alist-partial-monotonic (<svtv>-override-test-vars)
                              (svtv->outexprs (<svtv>)))

and a direct consequence:

(implies (and (svex-envs-agree (<svtv>-override-test-vars) env1 env2)
              (svex-env-<<= env1 env2))
         (svex-env-<<= (svtv-run (<svtv>) env1)
                       (svtv-run (<svtv>) env2)))

This is useful for overrides; see def-svtv-overrides-correct.
