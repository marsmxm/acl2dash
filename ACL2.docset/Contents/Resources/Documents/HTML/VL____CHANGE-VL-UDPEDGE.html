Modifying constructor for vl-udpedge structures.

Syntax
(change-vl-udpedge x 
                   [:prev <prev>] 
                   [:next <next>])

This is an often useful alternative to make-vl-udpedge.

We construct a new vl-udpedge structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-udpedge

(defmacro change-vl-udpedge (x &rest args)
          (std::change-aggregate 'vl-udpedge
                                 x args
                                 '((:prev . vl-udpedge->prev)
                                   (:next . vl-udpedge->next))
                                 'change-vl-udpedge
                                 'remake-vl-udpedge))
