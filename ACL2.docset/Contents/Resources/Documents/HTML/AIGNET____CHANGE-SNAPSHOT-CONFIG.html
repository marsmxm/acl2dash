Modifying constructor for snapshot-config structures.

Syntax
(change-snapshot-config x 
                        [:filename <filename>])

This is an often useful alternative to make-snapshot-config.

We construct a new snapshot-config structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-snapshot-config

(defmacro
   change-snapshot-config (x &rest args)
   (std::change-aggregate 'snapshot-config
                          x args
                          '((:filename . snapshot-config->filename))
                          'change-snapshot-config
                          'remake-snapshot-config))
