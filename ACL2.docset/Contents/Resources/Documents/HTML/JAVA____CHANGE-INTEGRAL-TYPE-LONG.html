Modifying constructor for integral-type-long structures.

Syntax
(change-integral-type-long x 
                           )

This is an often useful alternative to make-integral-type-long.

We construct a new integral-type-long structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
