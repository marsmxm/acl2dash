Modifying constructor for unary-op-minus structures.

Syntax
(change-unary-op-minus x 
                       )

This is an often useful alternative to make-unary-op-minus.

We construct a new unary-op-minus structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

This is an ordinary change- macro introduced by fty::defprod.
