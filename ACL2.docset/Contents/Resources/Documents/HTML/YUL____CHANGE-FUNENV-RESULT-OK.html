Modifying constructor for funenv-result-ok structures.

Syntax
(change-funenv-result-ok x 
                         [:get <get>])

This is an often useful alternative to make-funenv-result-ok.

We construct a new funenv-result-ok structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
