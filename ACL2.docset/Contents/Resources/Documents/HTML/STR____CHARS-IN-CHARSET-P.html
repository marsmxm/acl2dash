(chars-in-charset-p x set) recognizes lists of characters x where every character is a member of the charset-p set.

Signature
(chars-in-charset-p x set)  std::bool

This is an ordinary std::deflist. It is "loose" in that it does not care whether x is nil-terminated.

(defun chars-in-charset-p (x set)
