Modifying constructor for command-error-malformed-nonce structures.

Syntax
(change-command-error-malformed-nonce x 
                                      [:nonce <nonce>])

This is an often useful alternative to make-command-error-malformed-nonce.

We construct a new command-error-malformed-nonce structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
