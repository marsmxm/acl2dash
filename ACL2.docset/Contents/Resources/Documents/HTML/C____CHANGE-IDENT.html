Modifying constructor for ident structures.

Syntax
(change-ident x 
              [:name <name>])

This is an often useful alternative to make-ident.

We construct a new ident structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-ident

(defmacro change-ident (x &rest args)
          (std::change-aggregate 'ident
                                 x args '((:name . ident->name))
                                 'change-ident
                                 'nil))
