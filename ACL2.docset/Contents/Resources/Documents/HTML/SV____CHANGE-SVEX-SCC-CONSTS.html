Modifying constructor for svex-scc-consts structures.

Syntax
(change-svex-scc-consts x 
                        [:final-masks <final-masks>] 
                        [:loop-vars <loop-vars>] 
                        [:updates <updates>])

This is an often useful alternative to make-svex-scc-consts.

We construct a new svex-scc-consts structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-svex-scc-consts

(defmacro change-svex-scc-consts (x &rest args)
