Modifying constructor for value-ushort structures.

Syntax
(change-value-ushort x 
                     [:get <get>])

This is an often useful alternative to make-value-ushort.

We construct a new value-ushort structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-value-ushort

(defmacro
     change-value-ushort (x &rest args)
     (std::change-aggregate 'value-ushort
                            x args '((:get . value-ushort->get))
                            'change-value-ushort
                            'nil))
