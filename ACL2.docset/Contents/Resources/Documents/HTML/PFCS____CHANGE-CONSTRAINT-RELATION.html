Modifying constructor for constraint-relation structures.

Syntax
(change-constraint-relation x 
                            [:name <name>] 
                            [:args <args>])

This is an often useful alternative to make-constraint-relation.

We construct a new constraint-relation structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-constraint-relation

(defmacro
     change-constraint-relation
     (x &rest args)
     (std::change-aggregate 'constraint-relation
                            x args
                            '((:name . constraint-relation->name)
                              (:args . constraint-relation->args))
                            'change-constraint-relation
                            'nil))
