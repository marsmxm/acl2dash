A ( term1 / ... / termN ) parse tree, where term1, ..., termN are terminal numeric and character notations that all denote single natural numbers, is sometimes abstracted to its only natural number.

Signature
(abstract-grouped-terminal tree) → nat
Arguments
tree — Guard (treep tree).
Returns
nat — Type (natp nat).

The parse tree must have a root labeled by no rule name, with a single leaf subtree labeled by a list of one natural number. That natural number is returned.

Function: abstract-grouped-terminal

(defun abstract-grouped-terminal (tree)
       (declare (xargs :guard (treep tree)))
       (b* ((nats (abstract-grouped-terminals tree)))
           (if (consp nats)
               (car nats)
               (prog2$ (abstract-fail) 0))))
