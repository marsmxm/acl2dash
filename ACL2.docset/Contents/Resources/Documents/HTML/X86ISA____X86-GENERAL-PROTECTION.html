Semantic function corresponding to a general protection fault

(x86-general-protection message start-rip temp-rip x86) → x86
Returns
x86 — Type (x86p x86), given (x86p x86).

Note that the #GP (general protection) exception should be thrown here, which is why the fault field is populated with message.
