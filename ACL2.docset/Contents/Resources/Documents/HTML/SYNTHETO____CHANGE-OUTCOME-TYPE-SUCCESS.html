Modifying constructor for outcome-type-success structures.

Syntax
(change-outcome-type-success x 
                             [:message <message>])

This is an often useful alternative to make-outcome-type-success.

We construct a new outcome-type-success structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-outcome-type-success

(defmacro change-outcome-type-success
          (x &rest args)
          (std::change-aggregate
               'outcome-type-success
               x args
               '((:message . outcome-type-success->message))
               'change-outcome-type-success
               'nil))
