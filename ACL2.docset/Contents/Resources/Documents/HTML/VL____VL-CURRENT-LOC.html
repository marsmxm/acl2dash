Compatible with seq. Get the current location.

Signature
(vl-current-loc &key (tokstream 'tokstream)) 
  → 
(mv errmsg loc new-tokstream)
Returns
errmsg — Type (not errmsg).
loc — Type (vl-location-p loc).
new-tokstream — Type (equal new-tokstream tokstream).

We just return the location of the first token, if there is one. Or, if the token stream is empty, we just return *vl-fakeloc*

Definitions and Theorems

Function: vl-current-loc-fn

(defun vl-current-loc-fn (tokstream)
       (declare (xargs :stobjs (tokstream)))
       (declare (xargs :guard t))
       (let ((__function__ 'vl-current-loc))
            (declare (ignorable __function__))
            (b* ((tokens (vl-tokstream->tokens)))
                (mv nil
                    (if (consp tokens)
                        (vl-token->loc (car tokens))
                        *vl-fakeloc*)
                    tokstream))))
