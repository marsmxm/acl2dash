Modifying constructor for vl-maybe-define-some structures.

Syntax
(change-vl-maybe-define-some x 
                             [:val <val>])

This is an often useful alternative to make-vl-maybe-define-some.

We construct a new vl-maybe-define-some structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.
