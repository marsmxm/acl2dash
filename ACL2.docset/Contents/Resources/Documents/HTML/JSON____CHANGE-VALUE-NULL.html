Modifying constructor for value-null structures.

Syntax
(change-value-null x 
                   )

This is an often useful alternative to make-value-null.

We construct a new value-null structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-value-null

(defmacro change-value-null (x &rest args)
          (std::change-aggregate 'value-null
                                 x args 'nil
                                 'change-value-null
                                 'nil))
