Modifying constructor for ext-declon-tag-declon structures.

Syntax
(change-ext-declon-tag-declon x 
                              [:get <get>])

This is an often useful alternative to make-ext-declon-tag-declon.

We construct a new ext-declon-tag-declon structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
