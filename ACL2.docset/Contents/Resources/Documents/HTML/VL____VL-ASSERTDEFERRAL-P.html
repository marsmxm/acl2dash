Indicates whether this assertion is deferred.

  • nil — this is not a deferred assertion.
  • :vl-defer-0 — this is a #0 deferred assertion.
  • :vl-defer-final — this is a final deferred assertion.

This is an ordinary defenum.

Function: vl-assertdeferral-p

(defun vl-assertdeferral-p (x)
       (declare (xargs :guard t))
       (or (eq x 'nil)
           (eq x ':vl-defer-0)
           (eq x ':vl-defer-final)))
