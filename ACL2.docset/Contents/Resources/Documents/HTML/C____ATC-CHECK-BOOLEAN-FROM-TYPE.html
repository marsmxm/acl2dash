Check if a term may represent a conversion from a C integer value to an ACL2 boolean.

Signature
(atc-check-boolean-from-type term) → (mv yes/no arg in-type)
Arguments
term — Guard (pseudo-termp term).
Returns
yes/no — Type (booleanp yes/no).
arg — Type (pseudo-termp arg).
in-type — Type (typep in-type).

We also return the input C type of the conversion. The output type is known (boolean), and it is in fact an ACL2 type.

Function: atc-check-boolean-from-type

(defun atc-check-boolean-from-type (term)
       (declare (xargs :guard (pseudo-termp term)))
       (let ((__function__ 'atc-check-boolean-from-type))
            (declare (ignorable __function__))
            (b* (((acl2::fun (no))
                  (mv nil nil (irr-type)))
                 ((mv okp fn args)
                  (fty-check-fn-call term))
                 ((unless okp) (no))
                 ((mv okp boolean from type)
                  (atc-check-symbol-3part fn))
                 ((unless (and okp (eq boolean 'boolean)
                               (eq from 'from)))
                  (no))
                 (in-type (atc-integer-fixtype-to-type type))
                 ((when (not in-type)) (no))
                 ((unless (list-lenp 1 args)) (no)))
                (mv t (first args) in-type))))
