Modifying constructor for vl-elabtask structures.

Syntax
(change-vl-elabtask x 
                    [:stmt <stmt>])

This is an often useful alternative to make-vl-elabtask.

We construct a new vl-elabtask structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-elabtask

(defmacro
     change-vl-elabtask (x &rest args)
     (std::change-aggregate 'vl-elabtask
                            x args '((:stmt . vl-elabtask->stmt))
                            'change-vl-elabtask
                            'remake-vl-elabtask))
