Modifying constructor for vl-parsed-interface-head structures.

Syntax
(change-vl-parsed-interface-head x 
                                 [:ifname <ifname>] 
                                 [:modport <modport>])

This is an often useful alternative to make-vl-parsed-interface-head.

We construct a new vl-parsed-interface-head structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-parsed-interface-head

(defmacro change-vl-parsed-interface-head
          (x &rest args)
          (std::change-aggregate
               'vl-parsed-interface-head
               x args
               '((:ifname . vl-parsed-interface-head->ifname)
                 (:modport . vl-parsed-interface-head->modport))
               'change-vl-parsed-interface-head
               'remake-vl-parsed-interface-head))
