Modifying constructor for objdesign-address structures.

Syntax
(change-objdesign-address x 
                          [:get <get>])

This is an often useful alternative to make-objdesign-address.

We construct a new objdesign-address structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-objdesign-address

(defmacro
     change-objdesign-address (x &rest args)
     (std::change-aggregate 'objdesign-address
                            x
                            args '((:get . objdesign-address->get))
                            'change-objdesign-address
                            'nil))
