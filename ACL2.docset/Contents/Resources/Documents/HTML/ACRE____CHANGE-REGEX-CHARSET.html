Modifying constructor for regex-charset structures.

Syntax
(change-regex-charset x 
                      [:chars <chars>] 
                      [:negp <negp>])

This is an often useful alternative to make-regex-charset.

We construct a new regex-charset structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-regex-charset

(defmacro change-regex-charset (x &rest args)
          (std::change-aggregate 'regex-charset
                                 x args
                                 '((:chars . regex-charset->chars)
                                   (:negp . regex-charset->negp))
                                 'change-regex-charset
                                 'nil))
