(svar-boolmasks-fix x) is an fty alist fixing function that follows the drop-keys strategy.

Signature
(svar-boolmasks-fix x) → fty::newx
Arguments
x — Guard (svar-boolmasks-p x).
Returns
fty::newx — Type (svar-boolmasks-p fty::newx).

Note that in the execution this is just an inline identity function.
