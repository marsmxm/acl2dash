Modifying constructor for vl-scopecontext-package structures.

Syntax
(change-vl-scopecontext-package x 
                                [:pkg <pkg>])

This is an often useful alternative to make-vl-scopecontext-package.

We construct a new vl-scopecontext-package structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.
