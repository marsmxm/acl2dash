Basic constructor macro for tag-declon-struct structures.

Syntax
(make-tag-declon-struct [:tag <tag>] 
                        [:members <members>])

This is the usual way to construct tag-declon-struct structures. It simply conses together a structure with the specified fields.

This macro generates a new tag-declon-struct structure from scratch. See also change-tag-declon-struct, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by fty::defprod.
