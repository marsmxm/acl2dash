Modifying constructor for floating-point-type-float structures.

Syntax
(change-floating-point-type-float x 
                                  )

This is an often useful alternative to make-floating-point-type-float.

We construct a new floating-point-type-float structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
