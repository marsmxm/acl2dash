Basic constructor macro for fty-type-list structures.

Syntax
(make-fty-type-list [:elt-type <elt-type>])

This is the usual way to construct fty-type-list structures. It simply conses together a structure with the specified fields.

This macro generates a new fty-type-list structure from scratch. See also change-fty-type-list, which can "change" an existing structure, instead.

This is an ordinary make- macro introduced by defprod.
