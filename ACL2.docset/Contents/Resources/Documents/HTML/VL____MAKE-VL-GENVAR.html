Basic constructor macro for vl-genvar structures.

Syntax
(make-vl-genvar [:name <name>] 
                [:atts <atts>] 
                [:loc <loc>])

This is the usual way to construct vl-genvar structures. It simply conses together a structure with the specified fields.

This macro generates a new vl-genvar structure from scratch. See also change-vl-genvar, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by defprod.

Macro: make-vl-genvar

(defmacro make-vl-genvar (&rest args)
          (std::make-aggregate 'vl-genvar
                               args '((:name) (:atts) (:loc))
                               'make-vl-genvar
                               nil))

Function: vl-genvar

(defun vl-genvar (name atts loc)
       (declare (xargs :guard (and (stringp name)
                                   (vl-atts-p atts)
                                   (vl-location-p loc))))
       (declare (xargs :guard t))
       (let ((__function__ 'vl-genvar))
            (declare (ignorable __function__))
            (b* ((name (mbe :logic (str-fix name) :exec name))
                 (atts (mbe :logic (vl-atts-fix atts)
                            :exec atts))
                 (loc (mbe :logic (vl-location-fix loc)
                           :exec loc)))
                (cons :vl-genvar (list (cons 'name name)
                                       (cons 'atts atts)
                                       (cons 'loc loc))))))
