Basic constructor macro for svtv*-phase structures.

Syntax
(make-svtv*-phase [:delay <delay>] 
                  [:label <label>] 
                  [:inputs <inputs>] 
                  [:overrides <overrides>] 
                  [:outputs <outputs>])

This is the usual way to construct svtv*-phase structures. It simply conses together a structure with the specified fields.

This macro generates a new svtv*-phase structure from scratch. See also change-svtv*-phase, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by defprod.

Macro: make-svtv*-phase

(defmacro make-svtv*-phase (&rest args)
          (std::make-aggregate 'svtv*-phase
                               args
                               '((:delay . 1)
                                 (:label quote acl2::?)
                                 (:inputs)
                                 (:overrides)
                                 (:outputs))
                               'make-svtv*-phase
                               nil))

(defun
  svtv*-phase
  (delay label inputs overrides outputs)
  (declare (xargs :guard (and (posp delay)
                              (symbolp label)
                              (svtv*-input-alist-p inputs)
                              (svtv*-input-alist-p overrides)
                              (svtv*-output-alist-p outputs))))
  (declare (xargs :guard t))
  (let ((__function__ 'svtv*-phase))
       (declare (ignorable __function__))
       (b* ((delay (mbe :logic (pos-fix delay)
                        :exec delay))
            (label (mbe :logic (acl2::symbol-fix label)
                        :exec label))
            (inputs (mbe :logic (svtv*-input-alist-fix inputs)
                         :exec inputs))
            (overrides (mbe :logic (svtv*-input-alist-fix overrides)
                            :exec overrides))
            (outputs (mbe :logic (svtv*-output-alist-fix outputs)
                          :exec outputs)))
           (list delay label inputs overrides outputs))))
