Modifying constructor for funcall structures.

Syntax
(change-funcall x 
                [:name <name>] 
                [:args <args>])

This is an often useful alternative to make-funcall.

We construct a new funcall structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-funcall

(defmacro change-funcall (x &rest args)
          (std::change-aggregate 'funcall
                                 x args
                                 '((:name . funcall->name)
                                   (:args . funcall->args))
                                 'change-funcall
                                 'nil))
