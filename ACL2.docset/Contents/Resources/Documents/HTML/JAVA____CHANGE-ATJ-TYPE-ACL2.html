Modifying constructor for atj-type-ACL2 structures.

Syntax
(change-atj-type-ACL2 x 
                      [:get <get>])

This is an often useful alternative to make-atj-type-ACL2.

We construct a new atj-type-ACL2 structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
