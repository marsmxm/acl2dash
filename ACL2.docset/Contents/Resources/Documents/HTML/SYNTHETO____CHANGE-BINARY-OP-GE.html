Modifying constructor for binary-op-ge structures.

Syntax
(change-binary-op-ge x 
                     )

This is an often useful alternative to make-binary-op-ge.

We construct a new binary-op-ge structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
