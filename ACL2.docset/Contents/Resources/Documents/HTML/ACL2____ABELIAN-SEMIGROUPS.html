Axiomatization of an associative and commutative binary operation, developed by John Cowles.

Theory of Abelian SemiGroups

ACL2-ASG::op is an associative and commutative binary operation on the set (of equivalence classes formed by the equivalence relation, ACL2-ASG::equiv, on the set)

{ x | (ACL2-ASG::pred x) != nil }

Exclusive-or on the set of Boolean values with the equivalence relation, EQUAL, is an example.

Note, it is recommended that a second version of commutativity, called Commutativity-2, be added as a :rewrite rule for any operation which has both Associative and Commutative :rewrite rules. The macro add-commutativity-2 may be used to add such a rule.

Axioms of Abelian Semigroups

Using encapsulate, we introduce constrained functions:

  • (equiv x y)
  • (pred x)
  • (op x y)

with the following, constraining axioms:

Theorem: equiv-is-an-equivalence

(defthm equiv-is-an-equivalence
        (and (booleanp (equiv x y))
             (equiv x x)
             (implies (equiv x y) (equiv y x))
             (implies (and (equiv x y) (equiv y z))
                      (equiv x z)))
        :rule-classes (:equivalence))

Theorem: equiv-implies-iff-pred-1

(defthm equiv-implies-iff-pred-1
        (implies (equiv x x-equiv)
                 (iff (pred x) (pred x-equiv)))
        :rule-classes (:congruence))

Theorem: equiv-implies-equiv-op-1

(defthm equiv-implies-equiv-op-1
        (implies (equiv x x-equiv)
                 (equiv (op x y) (op x-equiv y)))
        :rule-classes (:congruence))

Theorem: equiv-implies-equiv-op-2

(defthm equiv-implies-equiv-op-2
        (implies (equiv y y-equiv)
                 (equiv (op x y) (op x y-equiv)))
        :rule-classes (:congruence))

Theorem: closure-of-op-for-pred

(defthm closure-of-op-for-pred
        (implies (and (pred x) (pred y))
                 (pred (op x y))))

Theorem: associativity-of-op

(defthm associativity-of-op
        (implies (and (pred x) (pred y) (pred z))
                 (equiv (op (op x y) z)
                        (op x (op y z)))))

Theorem: commutativity-of-op

(defthm commutativity-of-op
        (implies (and (pred x) (pred y))
                 (equiv (op x y) (op y x))))

Theorem of the theory of Abelian Groups

Given the above constraints, we prove the following generic theorems.

Theorem: commutativity-2-of-op

(defthm commutativity-2-of-op
        (implies (and (pred x) (pred y) (pred z))
                 (equiv (op x (op y z))
                        (op y (op x z)))))
