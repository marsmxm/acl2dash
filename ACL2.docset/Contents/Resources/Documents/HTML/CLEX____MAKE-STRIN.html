Constructor macro for strin-p structures.

Syntax:

(make-strin [:chars <chars>] 
            [:line <line>] 
            [:col <col>] 
            [:file <file>])

This is our preferred way to construct strin-p structures. It simply conses together a structure with the specified fields.

This macro generates a new strin-p structure from scratch. See also change-strin, which can "change" an existing structure, instead.

The strin-p structures we create here are just constructed with ordinary cons. If you want to create honsed structures, see make-honsed-strin instead.

Definition

This is an ordinary make- macro introduced by defaggregate.

Macro: make-strin

(defmacro
     make-strin (&rest args)
     (std::make-aggregate 'strin
                          args '((:chars) (:line) (:col) (:file))
                          'make-strin
                          nil))
