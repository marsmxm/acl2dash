(tyspecseq-list-fix x) is a usual ACL2::fty list fixing function.

Signature
(tyspecseq-list-fix x) → fty::newx
Arguments
x — Guard (tyspecseq-listp x).
Returns
fty::newx — Type (tyspecseq-listp fty::newx).

In the logic, we apply tyspecseq-fix to each member of the x. In the execution, none of that is actually necessary and this is just an inlined identity function.
