Approximate event name matches

General Form:

(near-misses name)

where name is a symbol. Community books "xdoc/spellcheck" and "system/event-names" are used automatically (by temporarily including them) to compute a list of names of events that approximate the given name. Example:

ACL2 !>(near-misses nt)
(NOT NTH)
ACL2 !>
