Modifying constructor for scratchobj-bfrlist structures.

Syntax
(change-scratchobj-bfrlist x 
                           [:val <val>])

This is an often useful alternative to make-scratchobj-bfrlist.

We construct a new scratchobj-bfrlist structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-scratchobj-bfrlist

(defmacro
     change-scratchobj-bfrlist (x &rest args)
     (std::change-aggregate 'scratchobj-bfrlist
                            x

                            'change-scratchobj-bfrlist
