Basic constructor macro for vl-namedarg structures.

Syntax
(make-vl-namedarg [:name <name>] 
                  [:expr <expr>] 
                  [:nameonly-p <nameonly-p>] 
                  [:atts <atts>])

This is the usual way to construct vl-namedarg structures. It simply conses together a structure with the specified fields.

This macro generates a new vl-namedarg structure from scratch. See also change-vl-namedarg, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by defprod.

Macro: make-vl-namedarg

(defmacro
     make-vl-namedarg (&rest args)
     (std::make-aggregate 'vl-namedarg
                          args
                          '((:name) (:expr) (:nameonly-p) (:atts))
                          'make-vl-namedarg
                          nil))

Function: vl-namedarg

(defun
 vl-namedarg (name expr nameonly-p atts)
 (declare (xargs :guard (and (stringp name)
                             (vl-maybe-expr-p expr)
                             (booleanp nameonly-p)
                             (vl-atts-p atts))))
 (declare (xargs :guard t))
 (let
     ((__function__ 'vl-namedarg))
     (declare (ignorable __function__))
     (b* ((name (mbe :logic (str-fix name) :exec name))
          (expr (mbe :logic (vl-maybe-expr-fix expr)
                     :exec expr))
          (nameonly-p (mbe :logic (acl2::bool-fix nameonly-p)
                           :exec nameonly-p))
          (atts (mbe :logic (vl-atts-fix atts)
                     :exec atts)))
         (cons :vl-namedarg
               (std::prod-cons (std::prod-cons name expr)
                               (std::prod-cons nameonly-p atts))))))
