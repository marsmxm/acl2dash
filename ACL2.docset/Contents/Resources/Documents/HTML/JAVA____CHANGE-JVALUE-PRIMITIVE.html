Modifying constructor for jvalue-primitive structures.

Syntax
(change-jvalue-primitive x 
                         [:get <get>])

This is an often useful alternative to make-jvalue-primitive.

We construct a new jvalue-primitive structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-jvalue-primitive

(defmacro
     change-jvalue-primitive (x &rest args)
     (std::change-aggregate 'jvalue-primitive
                            x args '((:get . jvalue-primitive->get))
                            'change-jvalue-primitive
                            'nil))
