Recognizer for pairs of segment register indices and values for the hidden portions containing the base addresses of the registers

Signature
(seg-hidden-basei-alistp alst)  *

Note that the register values are required to be n64p

Definitions and Theorems

Function: seg-hidden-basei-alistp

(defun seg-hidden-basei-alistp (alst)
       (declare (xargs :guard t))
       (let ((__function__ 'seg-hidden-basei-alistp))
            (declare (ignorable __function__))
            (if (atom alst)
                (equal alst nil)
                (if (atom (car alst))
                    nil
                    (let ((index (caar alst))
                          (value (cdar alst))
                          (rest (cdr alst)))
                         (and (natp index)
                              (< index 6)
                              (unsigned-byte-p 64 value)
                              (seg-hidden-basei-alistp rest)))))))
