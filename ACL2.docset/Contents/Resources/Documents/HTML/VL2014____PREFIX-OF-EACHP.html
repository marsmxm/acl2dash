(prefix-of-eachp p x) returns true exactly when p is a prefix of every member of x.

This is an ordinary deflist. It is "loose" in that it does not care whether x is nil-terminated.

Function: prefix-of-eachp

(defun prefix-of-eachp (p x)
       (declare (xargs :guard t))
       (let ((__function__ 'prefix-of-eachp))
            (declare (ignorable __function__))
            (if (consp x)
                (and (prefixp p (car x))
                     (prefix-of-eachp p (cdr x)))
                t)))
