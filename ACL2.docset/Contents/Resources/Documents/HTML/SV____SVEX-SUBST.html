Basic substitution operation for svexes. Not memoized.

Signature
(svex-subst pat al) → x
Arguments
pat — Pattern to substitute into.
    Guard (svex-p pat).
al — Substitution: binds variables to replacement svexes. Need not be a fast alist.
    Guard (svex-alist-p al).
Returns
x — Rewritten pattern with variables replaced by their bindings.
    Type (svex-p x).

Any variables that aren't bound get replaced by all Xes.

We expect to use this function when applying rewrite rules, which typically have only a few variables, so we don't use a fast alist here.

See also svex-subst-memo for a memoized version.
