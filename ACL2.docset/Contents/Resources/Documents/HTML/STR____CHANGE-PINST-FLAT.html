Modifying constructor for pinst-flat structures.

Syntax
(change-pinst-flat x 
                   [:guts <guts>])

This is an often useful alternative to make-pinst-flat.

We construct a new pinst-flat structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-pinst-flat

(defmacro
     change-pinst-flat (x &rest args)
     (std::change-aggregate 'pinst-flat
                            x args '((:guts . pinst-flat->guts))
                            'change-pinst-flat
                            'remake-pinst-flat))
