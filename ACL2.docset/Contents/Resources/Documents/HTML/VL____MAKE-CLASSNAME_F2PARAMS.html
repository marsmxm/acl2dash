Basic constructor macro for classname/params structures.

Syntax
(make-classname/params [:name <name>] 
                       [:params <params>])

This is the usual way to construct classname/params structures. It simply conses together a structure with the specified fields.

This macro generates a new classname/params structure from scratch. See also change-classname/params, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by defprod.
