Modifying constructor for atj-maybe-type-some structures.

Syntax

                            [:val <val>])

This is an often useful alternative to make-atj-maybe-type-some.

We construct a new atj-maybe-type-some structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
