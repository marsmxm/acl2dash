Modifying constructor for identifier-set-result-err structures.

Syntax
(change-identifier-set-result-err x 
                                  [:get <get>])

This is an often useful alternative to make-identifier-set-result-err.

We construct a new identifier-set-result-err structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
