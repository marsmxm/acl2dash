Modifying constructor for jbinop-div structures.

Syntax
(change-jbinop-div x 
                   )

This is an often useful alternative to make-jbinop-div.

We construct a new jbinop-div structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
