Modifying constructor for statement-assign-multi structures.

Syntax
(change-statement-assign-multi x 
                               [:targets <targets>] 
                               [:value <value>])

This is an often useful alternative to make-statement-assign-multi.

We construct a new statement-assign-multi structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-statement-assign-multi

(defmacro change-statement-assign-multi
          (x &rest args)
          (std::change-aggregate
               'statement-assign-multi
               x args
               '((:targets . statement-assign-multi->targets)
                 (:value . statement-assign-multi->value))
               'change-statement-assign-multi
               'nil))
