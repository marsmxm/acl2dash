Basic constructor macro for maybe-function-some structures.

Syntax
(make-maybe-function-some [:val <val>])

This is the usual way to construct maybe-function-some structures. It simply conses together a structure with the specified fields.

This macro generates a new maybe-function-some structure from scratch. See also change-maybe-function-some, which can "change" an existing structure, instead.
