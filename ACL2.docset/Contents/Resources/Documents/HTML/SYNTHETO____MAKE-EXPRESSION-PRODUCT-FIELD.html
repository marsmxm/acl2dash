Basic constructor macro for expression-product-field structures.

Syntax
(make-expression-product-field [:type <type>] 
                               [:target <target>] 
                               [:field <field>])

This is the usual way to construct expression-product-field structures. It simply conses together a structure with the specified fields.

This macro generates a new expression-product-field structure from scratch. See also change-expression-product-field, which can "change" an existing structure, instead.
