Modifying constructor for type-ullong structures.

Syntax
(change-type-ullong x 
                    )

This is an often useful alternative to make-type-ullong.

We construct a new type-ullong structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-type-ullong

(defmacro change-type-ullong (x &rest args)
          (std::change-aggregate 'type-ullong
