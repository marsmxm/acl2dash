(vl-fwdtypedeflist-fix x) is a usual fty list fixing function.

Signature
(vl-fwdtypedeflist-fix x)  fty::newx
Arguments
x Guard (vl-fwdtypedeflist-p x).
Returns
fty::newx Type (vl-fwdtypedeflist-p fty::newx).

In the logic, we apply vl-fwdtypedef-fix to each member of the x. In the execution, none of that is actually necessary and this is just an inlined identity function.
