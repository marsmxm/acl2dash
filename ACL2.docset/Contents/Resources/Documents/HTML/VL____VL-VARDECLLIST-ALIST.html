Extend an alist by binding the names of vl-vardecls to their definitions.

Signature
(vl-vardecllist-alist x acc)  alist
Arguments
x Guard (vl-vardecllist-p x).
Returns
alist Type (equal (vl-vardecl-alist-p alist) (vl-vardecl-alist-p acc)).

This can be used as an alternative to vl-find-vardecl when you need to perform a lot of lookups.

Definitions and Theorems

Function: vl-vardecllist-alist

(defun vl-vardecllist-alist (x acc)
       (declare (xargs :guard (vl-vardecllist-p x)))
       (let ((__function__ 'vl-vardecllist-alist))
            (declare (ignorable __function__))
            (if (atom x)
                acc
                (cons (cons (vl-vardecl->name (car x))
                            (vl-vardecl-fix (car x)))
                      (vl-vardecllist-alist (cdr x) acc)))))

Theorem: lookup-in-vl-vardecllist-alist-fast

(defthm
    lookup-in-vl-vardecllist-alist-fast
    (implies
         (stringp name)
         (equal (hons-assoc-equal name (vl-vardecllist-alist x nil))
                (let ((val (vl-find-vardecl name x)))
                     (and val (cons name val))))))
