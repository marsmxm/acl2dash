Definitions and Theorems

Function: letter-char-p

(defun letter-char-p (x)
       (declare (xargs :guard t))
       (and (characterp x)
            (let ((code (char-code x)))
                 (or (and (<= (char-code #\a) code)
                          (<= code (char-code #\z)))
                     (and (<= (char-code #\A) code)
                          (<= code (char-code #\Z)))))))
