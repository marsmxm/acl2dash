Abstract representation of a scope that just has vl-blockitems in it, such as a function, task, or block statement.

This is a product type introduced by defprod.

Fields
imports vl-importlist
Package imports in this scope.
paramdecls vl-paramdecllist
Parameter declarations in this scope.
vardecls vl-vardecllist
Variable declarations in this scope.
name maybe-string
Just a debugging aide. This lets us see the name of this scope when we want to print scopes, etc.
