(alternative-list-fix x) is a usual ACL2::fty list fixing function.

Signature
(alternative-list-fix x) → fty::newx
Arguments
x — Guard (alternative-listp x).
Returns
fty::newx — Type (alternative-listp fty::newx).

In the logic, we apply alternative-fix to each member of the x. In the execution, none of that is actually necessary and this is just an inlined identity function.
