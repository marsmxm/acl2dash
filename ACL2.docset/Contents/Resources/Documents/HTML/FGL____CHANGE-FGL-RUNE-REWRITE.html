Modifying constructor for fgl-rune-rewrite structures.

Syntax
(change-fgl-rune-rewrite x 
                         [:name <name>])

This is an often useful alternative to make-fgl-rune-rewrite.

We construct a new fgl-rune-rewrite structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-fgl-rune-rewrite

(defmacro
