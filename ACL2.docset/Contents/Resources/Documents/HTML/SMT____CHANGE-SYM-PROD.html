Modifying constructor for sym-prod structures.

Syntax
(change-sym-prod x 
                 [:sym <sym>])

This is an often useful alternative to make-sym-prod.

We construct a new sym-prod structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-sym-prod

(defmacro change-sym-prod (x &rest args)
          (std::change-aggregate 'sym-prod
                                 x args '((:sym . sym-prod->sym))
                                 'change-sym-prod
                                 'nil))
