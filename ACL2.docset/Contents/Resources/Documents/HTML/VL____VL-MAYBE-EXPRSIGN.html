Option type; vl-exprsign-p or nil.

This is an option type introduced by defoption. Note that defoption is just a wrapper for defflexsum, so there are :none and :some member types, a case macro, and so forth.

Member Types
:none vl-maybe-exprsign-none
Represents that no vl-maybe-exprsign is available, i.e., Nothing or None.
:some vl-maybe-exprsign-some
An available vl-maybe-exprsign, i.e., Just val or Some val.
