Modifying constructor for maybe-svex-some structures.

Syntax
(change-maybe-svex-some x 
                        [:val <val>])

This is an often useful alternative to make-maybe-svex-some.

We construct a new maybe-svex-some structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-maybe-svex-some

(defmacro
     change-maybe-svex-some (x &rest args)
     (std::change-aggregate 'maybe-svex-some
                            x args '((:val . maybe-svex-some->val))
                            'change-maybe-svex-some
                            'nil))
