Pretty-print an integer constant.

Signature
(pprint-iconst c) → part
Arguments
c — Guard (iconstp c).
Returns
part — Type (msgp part).

If the constant is unsigned, we print the uppercase U, not the lowercase u.

Definitions and Theorems

Function: pprint-iconst

(defun
    pprint-iconst (c)
    (declare (xargs :guard (iconstp c)))
    (let ((__function__ 'pprint-iconst))
         (declare (ignorable __function__))
         (b* (((iconst c) c))
             (msg "~@0~@1~@2"
                  (iconst-base-case c.base
                                    :dec (pprint-dec-const c.value)
                                    :oct (pprint-oct-const c.value)
                                    :hex (pprint-hex-const c.value))
                  (if c.unsignedp "U" "")
                  (pprint-iconst-tysuffix c.type)))))

Theorem: msgp-of-pprint-iconst

(defthm msgp-of-pprint-iconst
        (b* ((part (pprint-iconst c)))
            (msgp part))
        :rule-classes :rewrite)

(defthm pprint-iconst-of-iconst-fix-c
