Basic constructor macro for array-uchar structures.

Syntax
(make-array-uchar [:get <get>])

This is the usual way to construct array-uchar structures. It simply conses together a structure with the specified fields.

This macro generates a new array-uchar structure from scratch. See also change-array-uchar, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by fty::defprod.
