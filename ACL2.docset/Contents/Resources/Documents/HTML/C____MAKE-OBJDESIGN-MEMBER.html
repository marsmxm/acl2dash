Basic constructor macro for objdesign-member structures.

Syntax
(make-objdesign-member [:super <super>] 
                       [:name <name>])

This is the usual way to construct objdesign-member structures. It simply conses together a structure with the specified fields.

This macro generates a new objdesign-member structure from scratch. See also change-objdesign-member, which can "change" an existing structure, instead.
