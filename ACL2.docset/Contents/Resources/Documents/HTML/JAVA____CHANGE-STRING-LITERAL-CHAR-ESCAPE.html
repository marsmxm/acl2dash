Modifying constructor for string-literal-char-escape structures.

Syntax
(change-string-literal-char-escape x 
                                   [:get <get>])

This is an often useful alternative to make-string-literal-char-escape.

We construct a new string-literal-char-escape structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
