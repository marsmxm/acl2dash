Modifying constructor for class-type-nested structures.

Syntax
(change-class-type-nested x 
                          [:enclosing <enclosing>] 
                          [:name <name>] 
                          [:arguments <arguments>])

This is an often useful alternative to make-class-type-nested.

We construct a new class-type-nested structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
