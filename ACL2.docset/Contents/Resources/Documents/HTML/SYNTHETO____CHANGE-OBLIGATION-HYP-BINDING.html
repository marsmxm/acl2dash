Modifying constructor for obligation-hyp-binding structures.

Syntax
(change-obligation-hyp-binding x 
                               [:get <get>])

This is an often useful alternative to make-obligation-hyp-binding.

We construct a new obligation-hyp-binding structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-obligation-hyp-binding

(defmacro
     change-obligation-hyp-binding
     (x &rest args)
     (std::change-aggregate 'obligation-hyp-binding
                            x args

                            'change-obligation-hyp-binding
                            'nil))
