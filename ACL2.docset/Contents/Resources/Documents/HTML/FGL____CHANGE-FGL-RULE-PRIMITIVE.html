Modifying constructor for fgl-rule-primitive structures.

Syntax
(change-fgl-rule-primitive x 
                           [:name <name>])

This is an often useful alternative to make-fgl-rule-primitive.

We construct a new fgl-rule-primitive structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

(defmacro
