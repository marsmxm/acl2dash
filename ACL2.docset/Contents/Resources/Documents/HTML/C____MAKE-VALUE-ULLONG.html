Basic constructor macro for value-ullong structures.

Syntax
(make-value-ullong [:get <get>])

This is the usual way to construct value-ullong structures. It simply conses together a structure with the specified fields.

This macro generates a new value-ullong structure from scratch. See also change-value-ullong, which can "change" an existing structure, instead.
