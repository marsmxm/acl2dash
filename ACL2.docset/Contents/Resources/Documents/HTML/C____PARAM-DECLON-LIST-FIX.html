(param-declon-list-fix x) is a usual ACL2::fty list fixing function.

Signature
(param-declon-list-fix x) → fty::newx
Arguments
x — Guard (param-declon-listp x).
Returns
fty::newx — Type (param-declon-listp fty::newx).

In the logic, we apply param-declon-fix to each member of the x. In the execution, none of that is actually necessary and this is just an inlined identity function.
