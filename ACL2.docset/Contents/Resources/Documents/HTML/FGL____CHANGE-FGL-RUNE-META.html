Modifying constructor for fgl-rune-meta structures.

Syntax
(change-fgl-rune-meta x 
                      [:name <name>])

This is an often useful alternative to make-fgl-rune-meta.

We construct a new fgl-rune-meta structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-fgl-rune-meta

(defmacro
     change-fgl-rune-meta (x &rest args)
     (std::change-aggregate 'fgl-rune-meta
                            x args '((:name . fgl-rune-meta->name))
                            'change-fgl-rune-meta
                            'nil))
