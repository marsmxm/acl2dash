test?/prover timeout (in seconds)

Maximum allowed time (in seconds) to be spent in the ACL2 prover on behalf of Cgen/test? macro. This value is used as the first argument of the with-timeout macro around the call to prove/cgen. The default timeout limit is set to 60 sec. Guard : Timeout should be a non-negative rational. 
Usage: 
(acl2s-defaults :set cgen-timeout 60) 
(acl2s-defaults :get cgen-timeout)
