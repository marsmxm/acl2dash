Modifying constructor for array-slong structures.

Syntax
(change-array-slong x 
                    [:get <get>])

This is an often useful alternative to make-array-slong.

We construct a new array-slong structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-array-slong

(defmacro change-array-slong (x &rest args)
          (std::change-aggregate 'array-slong
                                 x args '((:get . array-slong->get))
                                 'change-array-slong
                                 'nil))
