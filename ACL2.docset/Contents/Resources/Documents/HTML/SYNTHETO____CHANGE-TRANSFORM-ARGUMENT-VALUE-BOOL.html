Modifying constructor for transform-argument-value-bool structures.

Syntax
(change-transform-argument-value-bool x 
                                      [:val <val>])

This is an often useful alternative to make-transform-argument-value-bool.

We construct a new transform-argument-value-bool structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

This is an ordinary change- macro introduced by fty::defprod.
