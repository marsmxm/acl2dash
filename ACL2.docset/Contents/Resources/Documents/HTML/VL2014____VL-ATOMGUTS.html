The main contents of a vl-atom-p.

This is a ``transparent'' sum type introduced using deftranssum. It is simply any one of the following types:

  • vl-constint
  • vl-weirdint
  • vl-extint
  • vl-string
  • vl-real
  • vl-id
  • vl-hidpiece
  • vl-funname
  • vl-sysfunname
  • vl-typename
  • vl-keyguts
  • vl-time
  • vl-basictype
  • vl-tagname

The guts of an atom are its main contents. See vl-expr-p for a discussion of the valid types.
