Option type; vl-paramargs or nil.

This is an option type introduced by defoption. Note that defoption is just a wrapper for defflexsum, so there are :none and :some member types, a case macro, and so forth.

Member Types
:none → vl-maybe-paramargs-none
Represents that no vl-maybe-paramargs is available, i.e., Nothing or None.
:some → vl-maybe-paramargs-some
An available vl-maybe-paramargs, i.e., Just val or Some val.
