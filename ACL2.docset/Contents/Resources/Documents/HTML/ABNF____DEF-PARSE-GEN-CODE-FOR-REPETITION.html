Generate code to parse an instance of an ABNF repetition.

Signature

fn-name — Guard (common-lisp::symbolp fn-name).

option-fns — Guard (alternation-symbol-alistp option-fns).

code — Type (true-listp code).
var — Type (common-lisp::symbolp var).

A repetition is part of a concatenation, and a concatenation (see def-parse-gen-code-for-concatenation) is parsed by parsing each of its repetitions; the results of parsing the repetitions must be put together to yield the result of parsing the concatenation. The result of parsing each repetition is bound to a different variable, called trees<index> where <index> is a numeric index that starts with 1 and is incremented as we go through the concatenation. The index is passed to this code generation function, which generates b* binders and also returns the variable name.

If the repetition is in the alist, its parsing function is retrieved from there and we generate code to bind its result to trees<index>. We also propagate any error.

If the repetition is not in the alist, it must be a singleton repetition, in which case we generate code to parse the element, and we put the resulting tree into a singleton list if successful. Note that we propagate the error when parsing the element, i.e. we pass t as the check-error-p flag.

Function: def-parse-gen-code-for-repetition

(defun
 def-parse-gen-code-for-repetition
 (rep index fn-name
      group-fns option-fns repetition-fns)
 (declare
     (xargs :guard (and (repetitionp rep)
                        (natp index)
                        (common-lisp::symbolp fn-name)
                        (alternation-symbol-alistp group-fns)
                        (alternation-symbol-alistp option-fns)
                        (repetition-symbol-alistp repetition-fns))))
 (let
  ((__function__ 'def-parse-gen-code-for-repetition))
  (declare (ignorable __function__))
  (b*
   ((trees-index (add-suffix 'trees
                             (str::nat-to-dec-string index)))
    (parse-repetition-fn? (cdr (assoc-equal rep repetition-fns)))
    ((when parse-repetition-fn?)
     (mv
      (cons
       (cons (cons 'mv (cons trees-index '(input)))
             (cons (cons parse-repetition-fn? '(input))
                   'nil))
       (cons
        (cons
          (cons 'when
                (cons (cons 'resulterrp
                            (cons trees-index 'nil))
                      'nil))
          (cons (cons 'mv
                      (cons (cons 'err-push (cons trees-index 'nil))
                            '(input)))
                'nil))
        'nil))
      trees-index))
    ((repetition rep) rep)
    ((when (equal rep.range
                  (make-repeat-range :min 1
                                     :max (nati-finite 1))))
     (mv (append (def-parse-gen-code-for-element
                      rep.element
                      t fn-name group-fns option-fns)
                 (cons (cons trees-index '((list tree)))
                       'nil))
         trees-index)))
   (prog2$ (raise "Repetition ~x0 not supported yet." rep)
           (mv nil nil)))))
