Modifying constructor for vl-subexpr-type-error structures.

Syntax
(change-vl-subexpr-type-error x 
                              [:expr <expr>] 
                              [:type <type>] 
                              [:type-err <type-err>])

This is an often useful alternative to make-vl-subexpr-type-error.

We construct a new vl-subexpr-type-error structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.
