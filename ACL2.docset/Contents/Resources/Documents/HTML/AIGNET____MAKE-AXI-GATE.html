Basic constructor macro for axi-gate structures.

Syntax
(make-axi-gate [:op <op>] 
               [:left <left>] 
               [:right <right>])

This is the usual way to construct axi-gate structures. It simply conses together a structure with the specified fields.

This macro generates a new axi-gate structure from scratch. See also change-axi-gate, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by fty::defprod.

(defmacro make-axi-gate (&rest args)

Function: axi-gate

(defun axi-gate (op left right)
       (declare (xargs :guard (and (axi-op-p op)
                                   (axi-lit-p left)
                                   (axi-lit-p right))))
       (declare (xargs :guard t))
       (let ((__function__ 'axi-gate))
            (declare (ignorable __function__))
            (b* ((op (mbe :logic (axi-op-fix op) :exec op))
                 (left (mbe :logic (axi-lit-fix left)
                            :exec left))
                 (right (mbe :logic (axi-lit-fix right)
                             :exec right)))
                (list op left right))))
