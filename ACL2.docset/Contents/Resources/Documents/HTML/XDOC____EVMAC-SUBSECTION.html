Construct a subsection of the user documentation of an event macro.

This provides more abstraction than using a specific heading.

Macro: evmac-subsection

(defmacro xdoc::evmac-subsection
          (title &rest content)
          (cons 'xdoc::&&
                (cons (cons 'xdoc::h4 (cons title 'nil))
                      content)))
