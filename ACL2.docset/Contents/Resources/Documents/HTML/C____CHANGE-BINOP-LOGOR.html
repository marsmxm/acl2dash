Modifying constructor for binop-logor structures.

Syntax
(change-binop-logor x 
                    )

This is an often useful alternative to make-binop-logor.

We construct a new binop-logor structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-binop-logor

(defmacro change-binop-logor (x &rest args)
          (std::change-aggregate 'binop-logor

                                 'nil))
