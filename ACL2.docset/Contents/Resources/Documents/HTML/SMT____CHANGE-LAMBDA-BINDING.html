Modifying constructor for lambda-binding structures.

Syntax
(change-lambda-binding x 
                       [:formals <formals>] 
                       [:actuals <actuals>])

This is an often useful alternative to make-lambda-binding.

We construct a new lambda-binding structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-lambda-binding

(defmacro
     change-lambda-binding (x &rest args)
     (std::change-aggregate 'lambda-binding
                            x args
                            '((:formals . lambda-binding->formals)
                              (:actuals . lambda-binding->actuals))
                            'change-lambda-binding
                            'nil))
