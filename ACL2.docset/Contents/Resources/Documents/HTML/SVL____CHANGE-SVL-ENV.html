Modifying constructor for svl-env structures.

Syntax
(change-svl-env x 
                [:wires <wires>] 
                [:modules <modules>])

This is an often useful alternative to make-svl-env.

We construct a new svl-env structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-svl-env

(defmacro change-svl-env (x &rest args)
          (std::change-aggregate 'svl-env
                                 x args
                                 '((:wires . svl-env->wires)
                                   (:modules . svl-env->modules))
                                 'change-svl-env
                                 'nil))
