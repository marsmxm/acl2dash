Efficiently revappend the characters from base64-decode onto an accumulator.

Signature
(base64-revappend-decode x acc) → (mv okp acc)
Arguments
x — String to be decoded.
    Guard (stringp x).
acc — Accumulator for decoded characters (in reverse order).
Returns
