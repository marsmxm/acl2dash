Check if a type definer is statically well-formed.

Signature
(check-type-definer definer ctxt) → (mv err? obligs)
Arguments
definer — Guard (type-definerp definer).
ctxt — Guard (contextp ctxt).
Returns
obligs — Type (proof-obligation-listp obligs).

The guard assumptions on the context are motivated as in check-type-product.

Definitions and Theorems

Function: check-type-definer

(defun
 check-type-definer (definer ctxt)
 (declare (xargs :guard (and (type-definerp definer)
                             (contextp ctxt))))
 (declare
      (xargs :guard (and (null (context->functions ctxt))
                         (omap::empty (context->variables ctxt))
                         (null (context->obligation-vars ctxt))
                         (null (context->obligation-hyps ctxt)))))
 (let
  ((__function__ 'check-type-definer))
  (declare (ignorable __function__))
  (type-definer-case definer
                     :product (check-type-product definer.get ctxt)
                     :sum (check-type-sum definer.get ctxt)
                     :subset (check-type-subset definer.get ctxt))))
