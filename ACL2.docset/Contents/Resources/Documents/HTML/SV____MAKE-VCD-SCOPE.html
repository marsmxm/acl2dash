Basic constructor macro for vcd-scope structures.

Syntax
(make-vcd-scope [:name <name>] 
                [:wires <wires>] 
                [:subscopes <subscopes>])

This is the usual way to construct vcd-scope structures. It simply conses together a structure with the specified fields.

This macro generates a new vcd-scope structure from scratch. See also change-vcd-scope, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by defprod.

Macro: make-vcd-scope

(defmacro make-vcd-scope (&rest args)
          (std::make-aggregate 'vcd-scope
                               args '((:name) (:wires) (:subscopes))
                               'make-vcd-scope
                               nil))

Function: vcd-scope

(defun
     vcd-scope (name wires subscopes)
     (declare (xargs :guard (and (stringp name)
                                 (vcd-wirelist-p wires)
                                 (vcd-scopelist-p subscopes))))
     (declare (xargs :guard t))
     (let ((__function__ 'vcd-scope))
          (declare (ignorable __function__))
          (b* ((name (mbe :logic (str-fix name) :exec name))
               (wires (mbe :logic (vcd-wirelist-fix wires)
                           :exec wires))
               (subscopes (mbe :logic (vcd-scopelist-fix subscopes)
                               :exec subscopes)))
              (list (cons 'name name)
                    (cons 'wires wires)
                    (cons 'subscopes subscopes)))))
