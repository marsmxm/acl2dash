(svexlist-fix x) is a usual fty list fixing function.

Signature
(svexlist-fix x) → fty::newx
Arguments
x — Guard (svexlist-p x).
Returns
fty::newx — Type (svexlist-p fty::newx).

In the logic, we apply svex-fix to each member of the x. In the execution, none of that is actually necessary and this is just an inlined identity function.
