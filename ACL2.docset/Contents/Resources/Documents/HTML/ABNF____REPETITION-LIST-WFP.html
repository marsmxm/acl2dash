Check if all the repetitions in a list of repetitions are well-formed.

Signature
(repetition-list-wfp repetitions) → yes/no
Arguments
repetitions — Guard (concatenationp repetitions).
Returns
yes/no — Type (booleanp yes/no).

Function: repetition-list-wfp

(defun repetition-list-wfp (repetitions)
       (declare (xargs :guard (concatenationp repetitions)))
       (or (endp repetitions)
           (and (repetition-wfp (car repetitions))
                (repetition-list-wfp (cdr repetitions)))))
