(vl-defines-fix x) is an fty alist fixing function that follows the drop-keys strategy.

Signature
(vl-defines-fix x) → fty::newx
Arguments
x — Guard (vl-defines-p x).
Returns
fty::newx — Type (vl-defines-p fty::newx).

Note that in the execution this is just an inline identity function.

Function: vl-defines-fix$inline

(defun
 vl-defines-fix$inline (x)
 (declare (xargs :guard (vl-defines-p x)))
 (let
  ((__function__ 'vl-defines-fix))
  (declare (ignorable __function__))
  (mbe
   :logic
   (if
     (atom x)
     nil
     (let ((rest (vl-defines-fix (cdr x))))
          (if (and (consp (car x)) (stringp (caar x)))
              (let ((fty::first-key (caar x))
                    (fty::first-val (vl-maybe-define-fix (cdar x))))
                   (cons (cons fty::first-key fty::first-val)
                         rest))
              rest)))
   :exec x)))
