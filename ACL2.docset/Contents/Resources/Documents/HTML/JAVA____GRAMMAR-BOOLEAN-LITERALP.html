Definition of boolean literals based on the grammar.

This defines a boolean literal as a string at the leaves of a terminated tree rooted at the boolean-literal nonterminal.

Definitions and Theorems

Theorem: grammar-boolean-literalp-suff

(defthm grammar-boolean-literalp-suff
        (implies (and (abnf-tree-with-root-p tree "boolean-literal")
                      (equal (abnf::tree->string tree) x))
                 (grammar-boolean-literalp x)))

Theorem: booleanp-of-grammar-boolean-literalp

(defthm booleanp-of-grammar-boolean-literalp
