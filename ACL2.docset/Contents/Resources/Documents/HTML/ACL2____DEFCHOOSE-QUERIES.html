Utilities to query defchoose functions.

Since defchoose is a primitive event, functions introduced via defchoose can always be recognized precisely. Besides the non-defchoose-specific constituents (e.g. formal arguments), which can be queried with more general utilities, these functions have defchoose-specific constituent, which can be queried with these defchoose query utilities.
