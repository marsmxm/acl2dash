(pseudo-term-subst-fix x) is an ACL2::fty alist fixing function that follows the drop-keys strategy.

Signature
(pseudo-term-subst-fix x) → fty::newx
Arguments
x — Guard (pseudo-term-subst-p x).
Returns
fty::newx — Type (pseudo-term-subst-p fty::newx).

Note that in the execution this is just an inline identity function.

 (let
  ((__function__ 'pseudo-term-subst-fix))
  (declare (ignorable __function__))
  (mbe
     :logic
     (if (atom x)
         nil
         (let ((rest (pseudo-term-subst-fix (cdr x))))
              (if (and (consp (car x))

                       (cons (cons fty::first-key fty::first-val)
                             rest))
                  rest)))
     :exec x)))
