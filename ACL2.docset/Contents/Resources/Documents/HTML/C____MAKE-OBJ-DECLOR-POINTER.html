Basic constructor macro for obj-declor-pointer structures.

Syntax
(make-obj-declor-pointer [:decl <decl>])

This is the usual way to construct obj-declor-pointer structures. It simply conses together a structure with the specified fields.

This macro generates a new obj-declor-pointer structure from scratch. See also change-obj-declor-pointer, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by fty::defprod.
