Modifying constructor for repetition structures.

Syntax
(change-repetition x 
                   [:range <range>] 
                   [:element <element>])

This is an often useful alternative to make-repetition.

We construct a new repetition structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-repetition

(defmacro change-repetition (x &rest args)
          (std::change-aggregate 'repetition
                                 x args
                                 '((:range . repetition->range)
                                   (:element . repetition->element))
                                 'change-repetition
                                 'nil))
