Modifying constructor for literal-option-some structures.

Syntax
(change-literal-option-some x 
                            [:val <val>])

This is an often useful alternative to make-literal-option-some.

We construct a new literal-option-some structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-literal-option-some

(defmacro
     change-literal-option-some
     (x &rest args)
     (std::change-aggregate 'literal-option-some
                            x args
                            '((:val . literal-option-some->val))
                            'change-literal-option-some
                            'nil))
