Cause an error if a term is not a call of if.

Signature
(ensure-term-if-call term description error-erp error-val ctx state) 
  → 
(mv erp val state)
Arguments
term — Term to check.
    Guard (pseudo-termp term).
description — Guard (msgp description).
error-erp — Flag to return in case of error.
error-val — Value to return in case of error.
ctx — Context for errors.
Returns
erp — Type (implies erp (equal erp error-erp)).
val — Type (and (implies erp (equal val error-val)) (implies (and (not erp) error-erp (pseudo-termp term)) (and (pseudo-term-listp val) (equal (len val) 3)))) .

If the term is a call to if, return its test, `then' branch, and `else' branch.

Definitions and Theorems

Function: ensure-term-if-call

(defun ensure-term-if-call
       (term description
             error-erp error-val ctx state)
       (declare (xargs :stobjs (state)))
       (declare (xargs :guard (and (pseudo-termp term)
                                   (msgp description))))
       (b* (((unless (and (nvariablep term)
                          (not (fquotep term))
                          (eq (ffn-symb term) 'if)))
             (er-soft+ ctx error-erp
                       error-val "~@0 must be a call of IF."
                       description)))
           (value (list (fargn term 1)
                        (fargn term 2)
                        (fargn term 3)))))
