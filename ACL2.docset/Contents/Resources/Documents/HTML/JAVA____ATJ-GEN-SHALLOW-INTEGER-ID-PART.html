Turn an ACL2 integer into a Java identifier part.

Signature
(atj-gen-shallow-integer-id-part integer) → core
Arguments
integer — Guard (integerp integer).
Returns
core — Type (stringp core).

This is part of the names of the static final fields for quoted numbers.

We turn the integer into its (base 10) digits, prepended by minus if negative.

Definitions and Theorems

Function: atj-gen-shallow-integer-id-part

(defun atj-gen-shallow-integer-id-part
       (integer)
       (declare (xargs :guard (integerp integer)))
       (let ((__function__ 'atj-gen-shallow-integer-id-part))
            (declare (ignorable __function__))
            (if (>= integer 0)
                (nat-to-dec-string integer)
                (str::cat "minus"
                          (nat-to-dec-string (- integer))))))
