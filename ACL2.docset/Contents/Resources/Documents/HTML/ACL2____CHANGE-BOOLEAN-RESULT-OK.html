Modifying constructor for boolean-result-ok structures.

Syntax
(change-boolean-result-ok x 
                          [:get <get>])

This is an often useful alternative to make-boolean-result-ok.

We construct a new boolean-result-ok structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-boolean-result-ok

(defmacro
     change-boolean-result-ok (x &rest args)
     (std::change-aggregate 'boolean-result-ok
                            x
                            args '((:get . boolean-result-ok->get))
                            'change-boolean-result-ok
                            'nil))
