Like logcdr for signed bit vectors.

Signature
(scdr v) → cdr
Arguments
v — Guard (true-listp v).
Returns
cdr — Type (true-listp cdr).

For a signed bit vector, the final bit is the sign bit, which we must implicitly extend out to infinity.
