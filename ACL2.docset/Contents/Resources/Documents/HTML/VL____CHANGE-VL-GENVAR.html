Modifying constructor for vl-genvar structures.

Syntax
(change-vl-genvar x 
                  [:name <name>] 
                  [:atts <atts>] 
                  [:loc <loc>])

This is an often useful alternative to make-vl-genvar.

We construct a new vl-genvar structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-genvar

(defmacro change-vl-genvar (x &rest args)
          (std::change-aggregate 'vl-genvar
                                 x args
                                 '((:name . vl-genvar->name)
                                   (:atts . vl-genvar->atts)
                                   (:loc . vl-genvar->loc))
                                 'change-vl-genvar
                                 'nil))
