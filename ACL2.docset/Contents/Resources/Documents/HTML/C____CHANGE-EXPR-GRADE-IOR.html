Modifying constructor for expr-grade-ior structures.

Syntax
(change-expr-grade-ior x 
                       )

This is an often useful alternative to make-expr-grade-ior.

We construct a new expr-grade-ior structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
