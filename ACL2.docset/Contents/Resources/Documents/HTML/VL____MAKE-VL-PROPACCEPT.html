Basic constructor macro for vl-propaccept structures.

Syntax
(make-vl-propaccept [:op <op>] 
                    [:condition <condition>] 
                    [:prop <prop>])

This is the usual way to construct vl-propaccept structures. It simply conses together a structure with the specified fields.

This macro generates a new vl-propaccept structure from scratch. See also change-vl-propaccept, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by defprod.

(defun vl-propaccept (op condition prop)
       (declare (xargs :guard (and (vl-property-acceptop-p op)
                                   (vl-exprdist-p condition)
                                   (vl-propexpr-p prop))))
       (declare (xargs :guard t))
       (let ((__function__ 'vl-propaccept))
            (declare (ignorable __function__))
            (b* ((op (mbe :logic (vl-property-acceptop-fix op)
                          :exec op))
                 (condition (mbe :logic (vl-exprdist-fix condition)
                                 :exec condition))
                 (prop (mbe :logic (vl-propexpr-fix prop)
                            :exec prop)))
                (cons :vl-propaccept (list op condition prop)))))
