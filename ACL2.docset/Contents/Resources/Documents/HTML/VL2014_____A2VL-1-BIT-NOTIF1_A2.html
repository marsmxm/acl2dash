Primitive conditional driver.

The Verilog meaning of this module is:

module VL_1_BIT_NOTIF1 (out, data, ctrl);
   output out;
   input data;
   input ctrl;
   assign out = ctrl ? ~data : 1'bz;
endmodule

VL takes this as a primitive. The gate-elim transform converts certain notif1 gates into instances of this module.

The corresponding esim primitive is ACL2::*esim-notif1*.
