Temporary structure for parsing UDPs.

(vl-udp-body-p x) is a defaggregate of the following fields.

  • init — Initial value for the sequential UDP register, if applicable.
        Invariant (vl-maybe-expr-p init).
  • table — The parsed state table.
        Invariant (vl-udptable-p table).
