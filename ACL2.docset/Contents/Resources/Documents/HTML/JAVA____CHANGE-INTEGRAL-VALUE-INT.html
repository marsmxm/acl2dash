Modifying constructor for integral-value-int structures.

Syntax
(change-integral-value-int x 
                           [:get <get>])

This is an often useful alternative to make-integral-value-int.

We construct a new integral-value-int structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-integral-value-int

(defmacro
     change-integral-value-int (x &rest args)
     (std::change-aggregate 'integral-value-int
                            x
                            args '((:get . integral-value-int->get))
                            'change-integral-value-int
                            'nil))
