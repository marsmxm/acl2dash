Modifying constructor for transform-argument-value-term structures.

Syntax
(change-transform-argument-value-term x 
                                      [:get <get>])

This is an often useful alternative to make-transform-argument-value-term.

We construct a new transform-argument-value-term structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-transform-argument-value-term

(defmacro change-transform-argument-value-term
          (x &rest args)

               'transform-argument-value-term
               x args
               '((:get . transform-argument-value-term->get))
               'change-transform-argument-value-term
               'nil))
