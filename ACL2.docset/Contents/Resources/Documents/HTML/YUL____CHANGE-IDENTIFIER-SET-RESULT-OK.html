Modifying constructor for identifier-set-result-ok structures.

Syntax
(change-identifier-set-result-ok x 
                                 [:get <get>])

This is an often useful alternative to make-identifier-set-result-ok.

We construct a new identifier-set-result-ok structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
