Option type; vl-paramvalue or nil.

This is an option type introduced by defoption. Note that defoption is just a wrapper for defflexsum, so there are :none and :some member types, a case macro, and so forth.

Member Types
:none → vl-maybe-paramvalue-none
Represents that no vl-maybe-paramvalue is available, i.e., Nothing or None.
:some → vl-maybe-paramvalue-some
An available vl-maybe-paramvalue, i.e., Just val or Some val.
