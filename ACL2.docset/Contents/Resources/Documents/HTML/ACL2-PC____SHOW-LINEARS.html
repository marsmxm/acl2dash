(macro) display the applicable linear rules

Example:
show-linears

General Form:
(show-linears &optional rule-id enabled-only-flg)

This command displays linear rules with a trigger term that matches the current subterm, and shows how they can be applied. This command is analogous to the show-rewrites proof-builder command; see ACL2-pc::show-rewrites. Also see ACL2-pc::apply-linear for how to apply linear rules.
