Modifying constructor for bexp-less structures.

Syntax
(change-bexp-less x 
                  [:left <left>] 
                  [:right <right>])

This is an often useful alternative to make-bexp-less.

We construct a new bexp-less structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-bexp-less

(defmacro change-bexp-less (x &rest args)
          (std::change-aggregate 'bexp-less
                                 x args
                                 '((:left . bexp-less->left)
                                   (:right . bexp-less->right))
                                 'change-bexp-less
                                 'nil))
