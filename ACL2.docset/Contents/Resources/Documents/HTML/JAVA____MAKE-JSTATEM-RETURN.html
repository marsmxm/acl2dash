Basic constructor macro for jstatem-return structures.

Syntax
(make-jstatem-return [:expr? <expr?>])

This is the usual way to construct jstatem-return structures. It simply conses together a structure with the specified fields.

This macro generates a new jstatem-return structure from scratch. See also change-jstatem-return, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by fty::defprod.
