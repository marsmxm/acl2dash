Modifying constructor for expr-option-none structures.

Syntax
(change-expr-option-none x 
                         )

This is an often useful alternative to make-expr-option-none.

We construct a new expr-option-none structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
