Basic constructor macro for vl-enum structures.

Syntax
(make-vl-enum [:basetype <basetype>] 
              [:items <items>] 
              [:pdims <pdims>] 
              [:udims <udims>])

This is the usual way to construct vl-enum structures. It simply conses together a structure with the specified fields.

This macro generates a new vl-enum structure from scratch. See also change-vl-enum, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by defprod.

Macro: make-vl-enum

(defmacro

Function: vl-enum

(defun
 vl-enum (basetype items pdims udims)
 (declare (xargs :guard (and (vl-enumbasetype-p basetype)
                             (vl-enumitemlist-p items)
                             (vl-packeddimensionlist-p pdims)
                             (vl-packeddimensionlist-p udims))))
 (declare (xargs :guard t))
 (let
  ((__function__ 'vl-enum))
  (declare (ignorable __function__))
  (b*
    ((basetype (mbe :logic (vl-enumbasetype-fix basetype)
                    :exec basetype))
     (items (mbe :logic (vl-enumitemlist-fix items)
                 :exec items))
     (pdims (mbe :logic (vl-packeddimensionlist-fix pdims)
                 :exec pdims))
     (udims (mbe :logic (vl-packeddimensionlist-fix udims)
                 :exec udims)))
    (cons :vl-enum (std::prod-cons (std::prod-cons basetype items)
                                   (std::prod-cons pdims udims))))))
