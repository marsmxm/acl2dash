Modifying constructor for regex-zerolength structures.

Syntax
(change-regex-zerolength x 
                         [:pat <pat>] 
                         [:lookback <lookback>] 
                         [:negp <negp>])

This is an often useful alternative to make-regex-zerolength.

We construct a new regex-zerolength structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-regex-zerolength

(defmacro
   change-regex-zerolength (x &rest args)
   (std::change-aggregate 'regex-zerolength
                          x args
                          '((:pat . regex-zerolength->pat)
                            (:lookback . regex-zerolength->lookback)
                            (:negp . regex-zerolength->negp))
                          'change-regex-zerolength
                          'nil))
