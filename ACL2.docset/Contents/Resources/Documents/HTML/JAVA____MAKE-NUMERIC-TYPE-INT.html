Basic constructor macro for numeric-type-int structures.

Syntax
(make-numeric-type-int )

This is the usual way to construct numeric-type-int structures. It simply conses together a structure with the specified fields.

This macro generates a new numeric-type-int structure from scratch. See also change-numeric-type-int, which can "change" an existing structure, instead.
