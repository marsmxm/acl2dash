Remove the underscores from a list of hexadecimal digits and underscores.

Signature
(hexdig/uscores-to-digits dus) → ds
Arguments
dus — Guard (hexdig/uscore-listp dus).
Returns
ds — Type (hex-digit-listp ds).

Any underscores in a hex-numeral in the grammar are just used for separation. This function returns the underlying sequence of hexadecimal digits, exluding also the prefix of the hex-numeral.
