Option type; ACL2::int or nil.

This is an option type introduced by defoption. Note that defoption is just a wrapper for fty::defflexsum, so there are :none and :some member types, a case macro, and so forth.

Member Types
:none → maybe-integer-none
Represents that no maybe-integer is available, i.e., Nothing or None.
:some → maybe-integer-some
An available maybe-integer, i.e., Just val or Some val.
