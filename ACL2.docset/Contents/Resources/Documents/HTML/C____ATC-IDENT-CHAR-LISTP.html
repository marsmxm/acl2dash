Check if a list of ACL2 characters is not empty, consists only of ASCII letters, digits, and underscores, and does not start with a digit.

Signature
(atc-ident-char-listp chs) → yes/no
Arguments
chs — Guard (character-listp chs).
Returns
yes/no — Type (booleanp yes/no).

Sequences of characters satisfying these conditions may be portable ASCII C identifiers, provided they are distinct from C keywords.

(defun atc-ident-char-listp (chs)
       (declare (xargs :guard (character-listp chs)))
       (let ((__function__ 'atc-ident-char-listp))
            (declare (ignorable __function__))
            (and (consp chs)
                 (atc-letter/digit/uscore-char-listp chs)
                 (not (digit-char-p (car chs))))))
