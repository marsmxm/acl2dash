Modifying constructor for regex-end structures.

Syntax
(change-regex-end x 
                  )

This is an often useful alternative to make-regex-end.

We construct a new regex-end structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-regex-end

(defmacro change-regex-end (x &rest args)
          (std::change-aggregate 'regex-end
                                 x args 'nil
                                 'change-regex-end
                                 'nil))
