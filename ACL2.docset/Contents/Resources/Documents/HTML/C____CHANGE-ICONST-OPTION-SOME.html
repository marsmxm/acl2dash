Modifying constructor for iconst-option-some structures.

Syntax
(change-iconst-option-some x 
                           [:val <val>])

This is an often useful alternative to make-iconst-option-some.

We construct a new iconst-option-some structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-iconst-option-some

(defmacro
