Basic constructor macro for array-ullong structures.

Syntax
(make-array-ullong [:get <get>])

This is the usual way to construct array-ullong structures. It simply conses together a structure with the specified fields.

This macro generates a new array-ullong structure from scratch. See also change-array-ullong, which can "change" an existing structure, instead.
