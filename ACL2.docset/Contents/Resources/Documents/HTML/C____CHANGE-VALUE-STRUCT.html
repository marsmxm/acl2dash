Modifying constructor for value-struct structures.

Syntax
(change-value-struct x 
                     [:tag <tag>] 
                     [:members <members>])

This is an often useful alternative to make-value-struct.

We construct a new value-struct structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-value-struct

(defmacro
     change-value-struct (x &rest args)
     (std::change-aggregate 'value-struct
                            x args
                            '((:tag . value-struct->tag)
                              (:members . value-struct->members))
                            'change-value-struct
                            'nil))
