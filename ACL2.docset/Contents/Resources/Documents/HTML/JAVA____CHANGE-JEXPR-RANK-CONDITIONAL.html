Modifying constructor for jexpr-rank-conditional structures.

Syntax
(change-jexpr-rank-conditional x 
                               )

This is an often useful alternative to make-jexpr-rank-conditional.

We construct a new jexpr-rank-conditional structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
