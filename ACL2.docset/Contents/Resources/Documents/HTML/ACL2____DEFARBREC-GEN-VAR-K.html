Generate the variable k that counts the iterated updates of the arguments.

Signature
(defarbrec-gen-var-k fn$ x1...xn$) → k
Arguments
fn$ — Guard (symbolp fn$).
x1...xn$ — Guard (symbol-listp x1...xn$).
Returns
k — A symbolp.

This is used as a formal parameter of the iterated argument update functions, the bound variable in the termination testing predicate, and a formal parameter of the measure function. This variable must be distinct from the formal parameters of fn.
