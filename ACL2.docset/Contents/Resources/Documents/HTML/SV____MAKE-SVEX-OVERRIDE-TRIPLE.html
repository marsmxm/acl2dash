Basic constructor macro for svex-override-triple structures.

Syntax
(make-svex-override-triple [:testvar <testvar>] 
                           [:valvar <valvar>] 
                           [:valexpr <valexpr>])

This is the usual way to construct svex-override-triple structures. It simply conses together a structure with the specified fields.

This macro generates a new svex-override-triple structure from scratch. See also change-svex-override-triple, which can "change" an existing structure, instead.

This is an ordinary make- macro introduced by defprod.
