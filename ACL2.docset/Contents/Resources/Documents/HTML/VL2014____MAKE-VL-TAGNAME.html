Basic constructor macro for vl-tagname structures.

Syntax
(make-vl-tagname [:name <name>])

This is the usual way to construct vl-tagname structures. It simply conses together a structure with the specified fields.

This macro generates a new vl-tagname structure from scratch. See also change-vl-tagname, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by defprod.

Macro: make-vl-tagname

(defmacro make-vl-tagname (&rest args)

(defun vl-tagname (name)
       (declare (xargs :guard (stringp name)))
       (declare (xargs :guard t))
       (let ((__function__ 'vl-tagname))
            (declare (ignorable __function__))
            (b* ((name (mbe :logic (str-fix name) :exec name)))
