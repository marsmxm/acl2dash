Raw constructor for honsed vl-extinttoken-p structures.

Syntax:

(honsed-vl-extinttoken etext value)

This is identical to vl-extinttoken, except that we hons the structure we are creating.

Definition

This is an ordinary honsing constructor introduced by defaggregate.

Function: honsed-vl-extinttoken

(defun honsed-vl-extinttoken (etext value)
       (declare (xargs :guard (and (and (vl-echarlist-p etext)
                                        (consp etext)
                                        (true-listp etext))
                                   (vl-bit-p value))))
       (mbe :logic (vl-extinttoken etext value)
            :exec (hons :vl-extinttoken (hons etext value))))
