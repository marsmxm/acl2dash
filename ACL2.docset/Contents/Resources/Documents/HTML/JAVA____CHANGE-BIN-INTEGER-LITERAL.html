Modifying constructor for bin-integer-literal structures.

Syntax
(change-bin-integer-literal x 
                            [:digits/uscores <digits/uscores>] 
                            [:prefix-upcase-p <prefix-upcase-p>] 
                            [:suffix? <suffix?>])

This is an often useful alternative to make-bin-integer-literal.

We construct a new bin-integer-literal structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-bin-integer-literal

(defmacro
    change-bin-integer-literal
    (x &rest args)
    (std::change-aggregate
         'bin-integer-literal
         x args
         '((:digits/uscores . bin-integer-literal->digits/uscores)
           (:prefix-upcase-p . bin-integer-literal->prefix-upcase-p)
           (:suffix? . bin-integer-literal->suffix?))
         'change-bin-integer-literal
         'nil))
