Modifying constructor for vl-scopecontext-root structures.

Syntax
(change-vl-scopecontext-root x 
                             )

This is an often useful alternative to make-vl-scopecontext-root.

We construct a new vl-scopecontext-root structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

(defmacro change-vl-scopecontext-root
