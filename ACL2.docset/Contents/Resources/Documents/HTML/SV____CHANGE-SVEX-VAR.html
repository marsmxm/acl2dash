Modifying constructor for svex-var structures.

Syntax
(change-svex-var x 
                 [:name <name>])

This is an often useful alternative to make-svex-var.

We construct a new svex-var structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-svex-var

(defmacro change-svex-var (x &rest args)
          (std::change-aggregate 'svex-var
                                 x args '((:name . svex-var->name))
                                 'change-svex-var
                                 'nil))
