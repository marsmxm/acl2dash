A copying macro that lets you create new wcp-example-app-p structures, based on existing structures.

Syntax:

(change-wcp-example-app x 
                        [:instrule <instrule>] 
                        [:bindings <bindings>])

This is a sometimes useful alternative to make-wcp-example-app. It constructs a new wcp-example-app-p structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by std::defaggregate.

Macro: change-wcp-example-app

(defmacro
   change-wcp-example-app (x &rest args)
   (std::change-aggregate 'wcp-example-app
                          x args
                          '((:instrule . wcp-example-app->instrule)
                            (:bindings . wcp-example-app->bindings))
                          'change-wcp-example-app
                          'nil))
