Basic constructor macro for integer-literal-hex structures.

Syntax
(make-integer-literal-hex [:get <get>])

This is the usual way to construct integer-literal-hex structures. It simply conses together a structure with the specified fields.

This macro generates a new integer-literal-hex structure from scratch. See also change-integer-literal-hex, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by fty::defprod.

(defun integer-literal-hex (get)
