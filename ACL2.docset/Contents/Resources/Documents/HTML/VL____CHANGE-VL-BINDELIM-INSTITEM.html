Modifying constructor for vl-bindelim-institem structures.

Syntax
(change-vl-bindelim-institem x 
                             [:inst <inst>] 
                             [:genp <genp>] 
                             [:bindp <bindp>] 
                             [:ctx <ctx>])

This is an often useful alternative to make-vl-bindelim-institem.

We construct a new vl-bindelim-institem structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

This is an ordinary change- macro introduced by defprod.
