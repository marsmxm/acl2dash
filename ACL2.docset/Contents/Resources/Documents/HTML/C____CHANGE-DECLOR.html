Modifying constructor for declor structures.

Syntax
(change-declor x 
               [:pointerp <pointerp>] 
               [:ident <ident>])

This is an often useful alternative to make-declor.

We construct a new declor structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-declor

(defmacro change-declor (x &rest args)
          (std::change-aggregate 'declor
                                 x args
                                 '((:pointerp . declor->pointerp)
                                   (:ident . declor->ident))
                                 'change-declor
                                 'nil))
