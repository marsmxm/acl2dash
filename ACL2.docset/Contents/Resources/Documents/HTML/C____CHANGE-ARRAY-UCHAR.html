Modifying constructor for array-uchar structures.

Syntax
(change-array-uchar x 
                    [:get <get>])

This is an often useful alternative to make-array-uchar.

We construct a new array-uchar structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-array-uchar

(defmacro change-array-uchar (x &rest args)
          (std::change-aggregate 'array-uchar
                                 x args '((:get . array-uchar->get))
                                 'change-array-uchar
                                 'nil))
