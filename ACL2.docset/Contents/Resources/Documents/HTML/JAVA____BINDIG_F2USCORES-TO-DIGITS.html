Remove the underscores from a list of binary digits and underscores.

Signature
(bindig/uscores-to-digits dus)  ds
Arguments
dus Guard (bindig/uscore-listp dus).
Returns
ds Type (bin-digit-listp ds).

Any underscores in a binary-numeral in the grammar are just used for separation. This function returns the underlying sequence of binary digits, exluding also the prefix of the binary-numeral.
