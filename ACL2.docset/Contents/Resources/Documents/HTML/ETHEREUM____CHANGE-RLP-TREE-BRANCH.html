Modifying constructor for rlp-tree-branch structures.

Syntax
(change-rlp-tree-branch x 
                        [:subtrees <subtrees>])

This is an often useful alternative to make-rlp-tree-branch.

We construct a new rlp-tree-branch structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-rlp-tree-branch

(defmacro
   change-rlp-tree-branch (x &rest args)
   (std::change-aggregate 'rlp-tree-branch
                          x args
                          '((:subtrees . rlp-tree-branch->subtrees))
                          'change-rlp-tree-branch
                          'nil))
