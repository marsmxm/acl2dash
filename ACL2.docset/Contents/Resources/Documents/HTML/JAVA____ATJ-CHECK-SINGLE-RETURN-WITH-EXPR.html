Check if a Java block consists of a single return with an expression.

Signature
(atj-check-single-return-with-expr block) → (mv yes/no expr)
Arguments
block — Guard (jblockp block).
Returns
yes/no — Type (booleanp yes/no).
expr — Type (jexprp expr).

If the check is successful, we return the expression of the return. If the check is not successful, i.e. the first result is nil, the second result is irrelevant.

       (block)

       (let ((__function__ 'atj-check-single-return-with-expr))
            (declare (ignorable __function__))
            (b* (((fun (fail)) (mv nil (jexpr-name "")))
