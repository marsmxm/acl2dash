Raw constructor for vl-plaintoken-p structures.

Syntax:

(vl-plaintoken type etext breakp)

This is the lowest-level constructor for vl-plaintoken-p structures. It simply conses together a structure with the specified fields.

Note: It's generally better to use macros like make-vl-plaintoken or change-vl-plaintoken instead. These macros lead to more readable and robust code, because you don't have to remember the order of the fields.

The vl-plaintoken-p structures we create here are just constructed with ordinary cons. If you want to create honsed structures, see honsed-vl-plaintoken instead.

Definition

This is an ordinary constructor function introduced by defaggregate.

Function: vl-plaintoken

(defun vl-plaintoken (type etext breakp)
       (declare (xargs :guard (and (vl-plaintokentype-p type)
                                   (and (vl-echarlist-p etext)
                                        (consp etext))
                                   (booleanp breakp))))
       (cons type (cons etext breakp)))
