Add a variable to a variable table.

Signature
(add-var var vartab) → vartab?
Arguments
var — Guard (identifierp var).
vartab — Guard (vartablep vartab).
Returns
vartab? — Type (vartable-resultp vartab?).

If a variable with the same name is already in the table, an error is returned.

Definitions and Theorems

Function: add-var

(defun add-var (var vartab)
       (declare (xargs :guard (and (identifierp var)
                                   (vartablep vartab))))
       (let ((__function__ 'add-var))
            (declare (ignorable __function__))
            (b* ((var (identifier-fix var))
                 (vartab (vartable-fix vartab)))
                (if (in var vartab)
                    (err (list :duplicate-variable var))
                    (insert var vartab)))))
