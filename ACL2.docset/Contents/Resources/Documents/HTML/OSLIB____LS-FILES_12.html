Get a listing of only the regular files within a directory, or cause a hard error.

Signature
(ls-files! path &key (state 'state)) → (mv val state)
Arguments
path — Guard (stringp path).
Returns
val — Type (string-listp val).
state — Type (state-p1 state), given (force (state-p1 state)).

This is just a wrapper for ls-files that causes a hard error if there are any problems.

Definitions and Theorems

Function: ls-files!-fn

(defun ls-files!-fn (path state)
       (declare (xargs :stobjs (state)))
       (declare (xargs :guard (stringp path)))
       (let ((__function__ 'ls-files!))
            (declare (ignorable __function__))
            (b* (((mv err val state) (ls-files path))
                 ((when err)
                  (er hard? 'ls-files! "~@0" err)
                  (mv nil state)))
                (mv val state))))
