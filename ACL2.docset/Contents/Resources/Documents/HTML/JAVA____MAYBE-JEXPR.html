Java expressions or nil.

This is an option type introduced by fty::defoption. Note that defoption is just a wrapper for fty::defflexsum, so there are :none and :some member types, a case macro, and so forth.

Member Types
:none → maybe-jexpr-none
Represents that no maybe-jexpr is available, i.e., Nothing or None.
:some → maybe-jexpr-some
An available maybe-jexpr, i.e., Just val or Some val.
