Check if a decimal numeral is well-formed.

Signature
(decdig/uscore-list-wfp dus) → yes/no
Arguments
dus — Guard (decdig/uscore-listp dus).
Returns
yes/no — Type (booleanp yes/no).

A decimal-numeral in the grammar is a list of decimal digits and underscores subject to the following constraints, derived from the grammar:

  • It must start with a digit, not an underscore.
  • It must end with a digit, not an underscore. (Note that first and last digit may be the same, if the decimal numeral is just 0.)
  • If the starting digit is 0, then it must be the only digit (which implies that there cannot be any underscores.
