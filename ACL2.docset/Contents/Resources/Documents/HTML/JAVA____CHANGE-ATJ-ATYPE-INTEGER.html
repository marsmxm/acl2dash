Modifying constructor for atj-atype-integer structures.

Syntax
(change-atj-atype-integer x 
                          )

This is an often useful alternative to make-atj-atype-integer.

We construct a new atj-atype-integer structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
