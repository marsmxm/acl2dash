Modifying constructor for svtv-assigns-override-config-omit structures.

Syntax
(change-svtv-assigns-override-config-omit x 
                                          [:vars <vars>])

This is an often useful alternative to make-svtv-assigns-override-config-omit.

We construct a new svtv-assigns-override-config-omit structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
