(make-vl-paramvalue-type [:type <type>])

This is the usual way to construct vl-paramvalue-type structures. It simply conses together a structure with the specified fields.

This macro generates a new vl-paramvalue-type structure from scratch. See also change-vl-paramvalue-type, which can "change" an existing structure, instead.
