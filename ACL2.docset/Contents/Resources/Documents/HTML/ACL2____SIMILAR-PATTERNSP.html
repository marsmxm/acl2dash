(similar-patternsp pat1 pat2) determines whether pat1 and pat2 are compatible with the same data.

Definitions and Theorems

Function: similar-patternsp

(defun similar-patternsp (pat1 pat2)
       (declare (xargs :guard t))
       (if pat1
           (and pat2
                (if (atom pat1)
                    (atom pat2)
                    (and (consp pat2)
                         (similar-patternsp (car pat1)
                                            (car pat2))
                         (similar-patternsp (cdr pat1)
                                            (cdr pat2)))))
           (not pat2)))

(defthm similar-patternsp-commutes

Theorem: similar-patternsp-is-an-equivalence

(defthm similar-patternsp-is-an-equivalence
        (and (booleanp (similar-patternsp x y))
             (similar-patternsp x x)
             (implies (similar-patternsp x y)
                      (similar-patternsp y x))
             (implies (and (similar-patternsp x y)
                           (similar-patternsp y z))
                      (similar-patternsp x z)))
        :rule-classes (:equivalence))
