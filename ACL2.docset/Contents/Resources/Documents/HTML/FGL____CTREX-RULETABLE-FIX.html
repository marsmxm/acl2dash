(ctrex-ruletable-fix x) is an ACL2::fty alist fixing function that follows the drop-keys strategy.

Signature
(ctrex-ruletable-fix x) → fty::newx
Arguments
x — Guard (ctrex-ruletable-p x).
Returns
fty::newx — Type (ctrex-ruletable-p fty::newx).

Note that in the execution this is just an inline identity function.

Definitions and Theorems

Function: ctrex-ruletable-fix$inline

(defun
 ctrex-ruletable-fix$inline (x)
 (declare (xargs :guard (ctrex-ruletable-p x)))
 (let
  ((__function__ 'ctrex-ruletable-fix))
  (declare (ignorable __function__))
  (mbe
   :logic
   (if
      (atom x)
      nil
      (let ((rest (ctrex-ruletable-fix (cdr x))))
           (if (and (consp (car x))
                    (pseudo-fnsym-p (caar x)))
               (let ((fty::first-key (caar x))
                     (fty::first-val (ctrex-rulelist-fix (cdar x))))
                    (cons (cons fty::first-key fty::first-val)
                          rest))
               rest)))
   :exec x)))
