Check if a case is safe.

Signature
(check-safe-swcase case vartab funtab) → modes
Arguments
case — Guard (swcasep case).
vartab — Guard (vartablep vartab).
funtab — Guard (funtablep funtab).
Returns
modes — Type (mode-set-resultp modes).

We check its literal and its block. We return the termination modes of the block.
