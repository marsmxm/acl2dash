Check if a multi-valued expression is statically well-formed.

Signature
(check-multi-expression args args-result) → result
Arguments
args — Guard (expression-listp args).
args-result — Guard (type-resultp args-result).
Returns
result — Type (type-resultp result).

There must be two or more expressions. We return the same type result, unchanged.

Definitions and Theorems

Function: check-multi-expression

(defun
 check-multi-expression
 (args args-result)
 (declare (xargs :guard (and (expression-listp args)
                             (type-resultp args-result))))
 (declare
      (xargs :guard (type-result-case args-result
                                      :err t
                                      :ok (= (len args-result.types)
                                             (len args)))))
 (let
  ((__function__ 'check-multi-expression))
  (declare (ignorable __function__))
  (type-result-case
   args-result
   :err (type-result-err args-result.info)
   :ok
   (if
    (< (len args-result.types) 2)
    (type-result-err
      (list :multi-expression-non-multi (expression-list-fix args)))
    (type-result-fix args-result)))))
