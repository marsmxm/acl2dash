(bfr-listp$ x bfrstate) recognizes lists where every element satisfies bfr-p.

This is an ordinary deflist. It is "loose" in that it does not care whether x is nil-terminated.

Function: bfr-listp$

(defun bfr-listp$ (x bfrstate)
       (declare (xargs :guard (bfrstate-p bfrstate)))
       (let ((__function__ 'bfr-listp$))
            (declare (ignorable __function__))
            (if (consp x)
                (and (bfr-p (car x))
                     (bfr-listp$ (cdr x) bfrstate))
                t)))
