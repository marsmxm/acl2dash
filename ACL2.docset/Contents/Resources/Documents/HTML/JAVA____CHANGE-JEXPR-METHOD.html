Modifying constructor for jexpr-method structures.

Syntax
(change-jexpr-method x 
                     [:name <name>] 
                     [:args <args>])

This is an often useful alternative to make-jexpr-method.

We construct a new jexpr-method structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-jexpr-method

(defmacro change-jexpr-method (x &rest args)
          (std::change-aggregate 'jexpr-method
                                 x args
                                 '((:name . jexpr-method->name)
                                   (:args . jexpr-method->args))
                                 'change-jexpr-method
                                 'nil))
