Modifying constructor for boolean-result-err structures.

Syntax
(change-boolean-result-err x 
                           [:get <get>])

This is an often useful alternative to make-boolean-result-err.

We construct a new boolean-result-err structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-boolean-result-err

(defmacro
     change-boolean-result-err (x &rest args)
     (std::change-aggregate 'boolean-result-err
                            x
                            args '((:get . boolean-result-err->get))
                            'change-boolean-result-err
                            'nil))
