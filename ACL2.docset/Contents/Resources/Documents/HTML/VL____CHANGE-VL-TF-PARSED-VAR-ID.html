A copying macro that lets you create new vl-tf-parsed-var-id-p structures, based on existing structures.

Syntax:

(change-vl-tf-parsed-var-id x 
                            [:name <name>] 
                            [:udims <udims>] 
                            [:default <default>])

This is a sometimes useful alternative to make-vl-tf-parsed-var-id. It constructs a new vl-tf-parsed-var-id-p structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defaggregate.

Macro: change-vl-tf-parsed-var-id

(defmacro
 change-vl-tf-parsed-var-id
 (x &rest args)
 (std::change-aggregate 'vl-tf-parsed-var-id
                        x args
                        '((:name . vl-tf-parsed-var-id->name)
                          (:udims . vl-tf-parsed-var-id->udims)
                          (:default . vl-tf-parsed-var-id->default))
                        'change-vl-tf-parsed-var-id
                        'nil))
