Basic constructor macro for vl-typename structures.

Syntax
(make-vl-typename [:name <name>])

This is the usual way to construct vl-typename structures. It simply conses together a structure with the specified fields.

This macro generates a new vl-typename structure from scratch. See also change-vl-typename, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by defprod.

Macro: make-vl-typename

(defmacro make-vl-typename (&rest args)
          (std::make-aggregate 'vl-typename
                               args '((:name))
                               'make-vl-typename
                               nil))

(defun vl-typename (name)
       (declare (xargs :guard (stringp name)))
       (declare (xargs :guard t))
       (let ((__function__ 'vl-typename))
            (declare (ignorable __function__))
            (b* ((name (mbe :logic (str-fix name) :exec name)))
