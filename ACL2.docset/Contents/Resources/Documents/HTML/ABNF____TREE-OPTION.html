Union of trees and nil.

This is an option type introduced by fty::defoption. Note that defoption is just a wrapper for fty::defflexsum, so there are :none and :some member types, a case macro, and so forth.

Member Types
:none → tree-option-none
Represents that no tree-option is available, i.e., Nothing or None.
:some → tree-option-some
An available tree-option, i.e., Just val or Some val.
