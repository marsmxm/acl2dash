Modifying constructor for vl-ifdef-context structures.

Syntax
(change-vl-ifdef-context x 
                         [:loc <loc>])

This is an often useful alternative to make-vl-ifdef-context.

We construct a new vl-ifdef-context structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-ifdef-context

(defmacro
     change-vl-ifdef-context (x &rest args)
     (std::change-aggregate 'vl-ifdef-context
