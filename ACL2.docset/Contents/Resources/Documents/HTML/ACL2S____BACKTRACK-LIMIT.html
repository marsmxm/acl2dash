Maximum number of backtracks allowed (per variable)

Maximum number of backtracks allowed by a variable. The default backtrack limit is set to 3. Setting this parameter to 0 disables the backtracking. 
Usage: 
(acl2s-defaults :set backtrack-limit 3) 
(acl2s-defaults :get backtrack-limit)
