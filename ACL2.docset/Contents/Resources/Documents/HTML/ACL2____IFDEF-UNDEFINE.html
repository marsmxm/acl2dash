Undefine an environment variable for use with ifdef and ifndef.

Analogous to ifdef-define, but sets the environment variable to the empty string instead of to "1", so that it will be unset for the purposes of ifdef and ifndef.

Similarly to ifdef-define, the effects of ifdef-undefine are book-local. See ifdef-undefine! for a version that persists across include-books.
