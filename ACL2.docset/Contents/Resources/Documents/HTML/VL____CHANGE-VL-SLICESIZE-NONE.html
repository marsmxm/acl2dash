Modifying constructor for vl-slicesize-none structures.

Syntax
(change-vl-slicesize-none x 
                          )

This is an often useful alternative to make-vl-slicesize-none.

We construct a new vl-slicesize-none structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
