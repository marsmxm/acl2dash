Specify sampling method to be used to instantiate variables

Specify which of the following methods to use for instantiating free variables of the conjecture under test: :be or :random or :uniform-random or :mixed By default this parameter is set to the symbol :random 
Usage: 
(acl2s-defaults :set sampling-method :random) 
(acl2s-defaults :get sampling-method)
