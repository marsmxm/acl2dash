Modifying constructor for vl-assignpat-positional structures.

Syntax
(change-vl-assignpat-positional x 
                                [:vals <vals>])

This is an often useful alternative to make-vl-assignpat-positional.

We construct a new vl-assignpat-positional structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-assignpat-positional

(defmacro
   change-vl-assignpat-positional
   (x &rest args)
   (std::change-aggregate 'vl-assignpat-positional
