Modifying constructor for axi-const structures.

Syntax
(change-axi-const x 
                  )

This is an often useful alternative to make-axi-const.

We construct a new axi-const structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-axi-const

(defmacro change-axi-const (x &rest args)
          (std::change-aggregate 'axi-const
                                 x args 'nil
                                 'change-axi-const
                                 'nil))
