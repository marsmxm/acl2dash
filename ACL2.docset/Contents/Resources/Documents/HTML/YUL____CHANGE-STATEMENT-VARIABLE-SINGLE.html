Modifying constructor for statement-variable-single structures.

Syntax
(change-statement-variable-single x 
                                  [:name <name>] 
                                  [:init <init>])

This is an often useful alternative to make-statement-variable-single.

We construct a new statement-variable-single structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

(defmacro
