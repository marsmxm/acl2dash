Modifying constructor for numeric-type-byte structures.

Syntax
(change-numeric-type-byte x 
                          )

This is an often useful alternative to make-numeric-type-byte.

We construct a new numeric-type-byte structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
