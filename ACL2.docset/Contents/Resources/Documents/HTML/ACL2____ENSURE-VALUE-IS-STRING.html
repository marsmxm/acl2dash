Cause an error if a value is not a string.

Signature
(ensure-value-is-string x description error-erp error-val ctx state) 
  → 
(mv erp val state)
Arguments
x — Value to check.
description — Guard (msgp description).
error-erp — Flag to return in case of error.
error-val — Value to return in case of error.
ctx — Context for errors.
Returns
erp — Type (implies erp (equal erp error-erp)).
val — Type (and (implies erp (equal val error-val)) (implies (and (not erp) error-erp) (not val))) .

Function: ensure-value-is-string

(defun
     ensure-value-is-string
     (x description
        error-erp error-val ctx state)
     (declare (xargs :stobjs (state)))
     (declare (xargs :guard (msgp description)))
     (b* (((unless (stringp x))
           (er-soft+ ctx error-erp error-val
                     "~@0 must be a string, but it is ~x1 instead."
                     description x)))
         (value nil)))
