Modifying constructor for address structures.

Syntax
(change-address x 
                [:path <path>] 
                [:index <index>] 
                [:scope <scope>])

This is an often useful alternative to make-address.

We construct a new address structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-address

(defmacro change-address (x &rest args)
          (std::change-aggregate 'address
                                 x args
                                 '((:path . address->path)
                                   (:index . address->index)
                                   (:scope . address->scope))
                                 'change-address
                                 'nil))
