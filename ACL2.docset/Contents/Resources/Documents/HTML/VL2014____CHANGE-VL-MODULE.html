Syntax
(change-vl-module x 
                  [:name <name>] 
                  [:params <params>] 
                  [:imports <imports>] 
                  [:ports <ports>] 
                  [:portdecls <portdecls>]

                  [:loaditems <loaditems>] 
                  [:esim <esim>])

This is an often useful alternative to make-vl-module.

We construct a new vl-module structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-module
