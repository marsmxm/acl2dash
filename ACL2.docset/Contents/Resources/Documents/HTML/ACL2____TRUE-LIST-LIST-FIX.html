(true-list-list-fix x) is a usual fty list fixing function.

Signature
(true-list-list-fix x) → fty::newx
Arguments
x — Guard (true-list-listp x).
Returns
fty::newx — Type (true-list-listp fty::newx).

In the logic, we apply list-fix to each member of the x. In the execution, none of that is actually necessary and this is just an inlined identity function.
