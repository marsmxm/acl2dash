Modifying constructor for maybe-symbol-value-some structures.

Syntax
(change-maybe-symbol-value-some x 
                                [:val <val>])

This is an often useful alternative to make-maybe-symbol-value-some.

We construct a new maybe-symbol-value-some structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
