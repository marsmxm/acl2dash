Modifying constructor for element-rulename structures.

Syntax
(change-element-rulename x 
                         [:get <get>])

This is an often useful alternative to make-element-rulename.

We construct a new element-rulename structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-element-rulename

(defmacro
     change-element-rulename (x &rest args)
     (std::change-aggregate 'element-rulename
                            x args '((:get . element-rulename->get))
                            'change-element-rulename
                            'nil))
