Modifying constructor for vl-type-error-incompat structures.

Syntax
(change-vl-type-error-incompat x 
                               [:actual-type <actual-type>] 
                               [:detail <detail>])

This is an often useful alternative to make-vl-type-error-incompat.

We construct a new vl-type-error-incompat structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
