Modifying constructor for vl-nonansi-ports structures.

Syntax
(change-vl-nonansi-ports x 
                         [:ports <ports>])

This is an often useful alternative to make-vl-nonansi-ports.

We construct a new vl-nonansi-ports structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-nonansi-ports

(defmacro
     change-vl-nonansi-ports (x &rest args)
     (std::change-aggregate 'vl-nonansi-ports
                            x args
                            '((:ports . vl-nonansi-ports->ports))
                            'change-vl-nonansi-ports
                            'nil))
