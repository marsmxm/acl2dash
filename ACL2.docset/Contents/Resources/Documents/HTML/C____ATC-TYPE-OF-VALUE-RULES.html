Rules about type-of-value.

These rules rewrite type-of-value to specific types under hypotheses on different types of values that occur during symbolic execution.

Definitions and Theorems

Theorem: type-of-value-when-uchar-arrayp

(defthm type-of-value-when-uchar-arrayp
        (implies (uchar-arrayp x)
                 (equal (type-of-value x)
                        (type-array (type-uchar)
                                    (value-array->length x)))))
