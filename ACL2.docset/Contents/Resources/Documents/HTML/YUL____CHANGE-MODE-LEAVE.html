Modifying constructor for mode-leave structures.

Syntax
(change-mode-leave x 
                   )

This is an often useful alternative to make-mode-leave.

We construct a new mode-leave structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-mode-leave

(defmacro change-mode-leave (x &rest args)
          (std::change-aggregate 'mode-leave
                                 x args 'nil

                                 'nil))
