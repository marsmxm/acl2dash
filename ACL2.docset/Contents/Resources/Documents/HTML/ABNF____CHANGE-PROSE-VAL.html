Modifying constructor for prose-val structures.

Syntax
(change-prose-val x 
                  [:get <get>])

This is an often useful alternative to make-prose-val.

We construct a new prose-val structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-prose-val

(defmacro change-prose-val (x &rest args)
          (std::change-aggregate 'prose-val
                                 x args '((:get . prose-val->get))
                                 'change-prose-val
                                 'nil))
