Modifying constructor for vl-scopestack-global structures.

Syntax
(change-vl-scopestack-global x 
                             [:design <design>])

This is an often useful alternative to make-vl-scopestack-global.

We construct a new vl-scopestack-global structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-scopestack-global

(defmacro
