Modifying constructor for literal-hex-number structures.

Syntax
(change-literal-hex-number x 
                           [:get <get>])

This is an often useful alternative to make-literal-hex-number.

We construct a new literal-hex-number structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-literal-hex-number

(defmacro
     change-literal-hex-number (x &rest args)
     (std::change-aggregate 'literal-hex-number
                            x
                            args '((:get . literal-hex-number->get))
                            'change-literal-hex-number
                            'nil))
