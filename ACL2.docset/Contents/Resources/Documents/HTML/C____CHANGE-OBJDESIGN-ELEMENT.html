Modifying constructor for objdesign-element structures.

Syntax
(change-objdesign-element x 
                          [:super <super>] 
                          [:index <index>])

This is an often useful alternative to make-objdesign-element.

We construct a new objdesign-element structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-objdesign-element

(defmacro
     change-objdesign-element (x &rest args)
