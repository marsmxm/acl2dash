Modifying constructor for maybe-function-none structures.

Syntax
(change-maybe-function-none x 
                            )

This is an often useful alternative to make-maybe-function-none.

We construct a new maybe-function-none structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-maybe-function-none

(defmacro change-maybe-function-none
          (x &rest args)
          (std::change-aggregate 'maybe-function-none
                                 x args 'nil
                                 'change-maybe-function-none
                                 'nil))
