Modifying constructor for vl-elabinstruction-root structures.

Syntax
(change-vl-elabinstruction-root x

This is an often useful alternative to make-vl-elabinstruction-root.

We construct a new vl-elabinstruction-root structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
