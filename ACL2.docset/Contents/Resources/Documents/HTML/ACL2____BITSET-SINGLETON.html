(bitset-singleton a) constructs the singleton set {a}.

Signature
(bitset-singleton a) → bitset
Arguments
a — Guard (natp a).
Returns
bitset — Type (natp bitset).

This is perhaps slightly more efficient than the equivalent, (bitset-insert A 0).
