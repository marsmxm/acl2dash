Check if a binary numeral (without prefix) is well-formed.

Signature
(bindig/uscore-list-wfp dus) → yes/no
Arguments
dus — Guard (bindig/uscore-listp dus).
Returns
yes/no — Type (booleanp yes/no).

A binary-numeral in the grammar, without the prefix, is a list of binary digits and underscores subject to the following constraints, derived from the grammar:

  • It must start with a digit, not an underscore.
  • It must end with a digit, not an underscore. (Note that first and last digit may be the same, if the binary numeral is just 0b0 or 0B0.)
