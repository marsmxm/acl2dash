(svtv-lines-fix x) is a usual fty list fixing function.

Signature
(svtv-lines-fix x) → fty::newx
Arguments
x — Guard (svtv-lines-p x).
Returns
fty::newx — Type (svtv-lines-p fty::newx).

In the logic, we apply svtv-line-fix to each member of the x. In the execution, none of that is actually necessary and this is just an inlined identity function.
