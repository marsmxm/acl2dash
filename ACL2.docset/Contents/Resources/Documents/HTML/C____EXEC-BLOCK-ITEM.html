Execute a block item.

Signature
(exec-block-item item compst fenv limit) 
  → 
(mv result new-compst)
Arguments
item — Guard (block-itemp item).
compst — Guard (compustatep compst).
fenv — Guard (fun-envp fenv).
limit — Guard (natp limit).
Returns
result — Type (value-option-resultp result).
new-compst — Type (compustatep new-compst).

Besides an optional value result, we also return a possibly updated computation state.

If the block item is a declaration, we ensure it is a variable (not a structure type) declaration, then we execute the expression, then we add the variable to the top scope of the top frame. The initializer value must have the same type as the variable, which automatically excludes the case of the variable being void, since type-of-value never returns void (under the guard).

If the block item is a statement, we execute it like any other statement.
