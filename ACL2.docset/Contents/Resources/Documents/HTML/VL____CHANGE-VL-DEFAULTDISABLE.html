Modifying constructor for vl-defaultdisable structures.

Syntax
(change-vl-defaultdisable x 
                          [:exprdist <exprdist>] 
                          [:loc <loc>] 
                          [:atts <atts>])

This is an often useful alternative to make-vl-defaultdisable.

We construct a new vl-defaultdisable structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-defaultdisable

(defmacro
  change-vl-defaultdisable (x &rest args)
  (std::change-aggregate 'vl-defaultdisable
                         x args
                         '((:exprdist . vl-defaultdisable->exprdist)
                           (:loc . vl-defaultdisable->loc)
                           (:atts . vl-defaultdisable->atts))
                         'change-vl-defaultdisable
                         'remake-vl-defaultdisable))
