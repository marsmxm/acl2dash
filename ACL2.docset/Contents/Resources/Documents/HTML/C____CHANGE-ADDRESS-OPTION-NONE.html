Modifying constructor for address-option-none structures.

Syntax
(change-address-option-none x 
                            )

This is an often useful alternative to make-address-option-none.

We construct a new address-option-none structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
