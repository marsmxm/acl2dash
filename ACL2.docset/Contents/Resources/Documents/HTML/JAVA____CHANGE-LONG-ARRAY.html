Modifying constructor for long-array structures.

Syntax
(change-long-array x 
                   [:components <components>])

This is an often useful alternative to make-long-array.

We construct a new long-array structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-long-array

(defmacro
    change-long-array (x &rest args)
    (std::change-aggregate 'long-array
                           x args
                           '((:components . long-array->components))
                           'change-long-array
                           'nil))
