Coerces a dec-digit-char-p character into a number.

Signature
(dec-digit-char-value x) → val
Arguments
x — Guard (dec-digit-char-p x).
Returns
val — Type (natp val).

For instance, (dec-digit-char-value #\3) is 3. For any non-dec-digit-char-p, 0 is returned.
