Modifying constructor for jliteral-integer structures.

Syntax
(change-jliteral-integer x 
                         [:get <get>])

This is an often useful alternative to make-jliteral-integer.

We construct a new jliteral-integer structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-jliteral-integer

(defmacro
     change-jliteral-integer (x &rest args)
     (std::change-aggregate 'jliteral-integer
                            x args '((:get . jliteral-integer->get))
                            'change-jliteral-integer
                            'nil))
