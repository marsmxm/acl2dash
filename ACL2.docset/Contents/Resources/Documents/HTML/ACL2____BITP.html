A recognizer for the set of bits, {0,1}

Bitp returns t if and only its argument is 0 or 1, and nil otherwise.

Note that this is a predicate form of the type-spec declaration (TYPE BIT b).

Function: bitp

(defun bitp (x)
       (declare (xargs :guard t))
       (or (eql x 0) (eql x 1)))
