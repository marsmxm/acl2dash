(prof-entrylist-fix x) is a usual ACL2::fty list fixing function.

Signature
(prof-entrylist-fix x)  fty::newx
Arguments
x Guard (prof-entrylist-p x).
Returns
fty::newx Type (prof-entrylist-p fty::newx).

In the logic, we apply prof-entry-fix to each member of the x. In the execution, none of that is actually necessary and this is just an inlined identity function.
