Modifying constructor for ulong-array structures.

Syntax
(change-ulong-array x 
                    [:elements <elements>])

This is an often useful alternative to make-ulong-array.

We construct a new ulong-array structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-ulong-array

(defmacro
     change-ulong-array (x &rest args)
     (std::change-aggregate 'ulong-array
                            x args
                            '((:elements . ulong-array->elements))
                            'change-ulong-array
                            'nil))
