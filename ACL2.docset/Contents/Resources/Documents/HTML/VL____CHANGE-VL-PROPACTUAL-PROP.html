Modifying constructor for vl-propactual-prop structures.

Syntax
(change-vl-propactual-prop x 
                           [:name <name>] 
                           [:prop <prop>])

This is an often useful alternative to make-vl-propactual-prop.

We construct a new vl-propactual-prop structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.
