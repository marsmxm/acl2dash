Identity function for 2vecs, or all Xes when there are any X or Z bits.

Signature
(4vec-xdet x) → res
Arguments
x — Guard (4vec-p x).
Returns
res — Type (4vec-p res).

This (arguably) matches the Verilog specification for unary +.

BOZO this is identical to 2vecx-fix.
