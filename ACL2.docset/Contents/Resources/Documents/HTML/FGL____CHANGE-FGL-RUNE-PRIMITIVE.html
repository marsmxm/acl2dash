Modifying constructor for fgl-rune-primitive structures.

Syntax
(change-fgl-rune-primitive x 
                           [:name <name>])

This is an often useful alternative to make-fgl-rune-primitive.

We construct a new fgl-rune-primitive structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-fgl-rune-primitive

(defmacro
     change-fgl-rune-primitive (x &rest args)
     (std::change-aggregate 'fgl-rune-primitive
                            x args
                            '((:name . fgl-rune-primitive->name))
                            'change-fgl-rune-primitive
                            'nil))
