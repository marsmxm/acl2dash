Modifying constructor for vl-timeprecisiondecl structures.

Syntax
(change-vl-timeprecisiondecl x 
                             [:precision <precision>] 
                             [:loc <loc>])

This is an often useful alternative to make-vl-timeprecisiondecl.

We construct a new vl-timeprecisiondecl structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-timeprecisiondecl

(defmacro change-vl-timeprecisiondecl
          (x &rest args)
