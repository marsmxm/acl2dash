Raw constructor for vl-extinttoken-p structures.

Syntax:

(vl-extinttoken etext value)

This is the lowest-level constructor for vl-extinttoken-p structures. It simply conses together a structure with the specified fields.

Note: It's generally better to use macros like make-vl-extinttoken or change-vl-extinttoken instead. These macros lead to more readable and robust code, because you don't have to remember the order of the fields.

The vl-extinttoken-p structures we create here are just constructed with ordinary cons. If you want to create honsed structures, see honsed-vl-extinttoken instead.

Definition

This is an ordinary constructor function introduced by defaggregate.

Function: vl-extinttoken

(defun vl-extinttoken (etext value)
       (declare (xargs :guard (and (and (vl-echarlist-p etext)
                                        (consp etext)
                                        (true-listp etext))
                                   (vl-bit-p value))))
       (cons :vl-extinttoken (cons etext value)))
