Modifying constructor for fun-table-result-err structures.

Syntax
(change-fun-table-result-err x 
                             [:get <get>])

This is an often useful alternative to make-fun-table-result-err.

We construct a new fun-table-result-err structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
