For now we only model the void type, the plain char type, and the standard signed and unsigned integer types (except _Bool, as well as pointer types. The referenced type of a pointer type may be any type (that we model), including a pointer type. The recursion bottoms out at the integer types.

This semantic model is more general than its syntactic counterpart tyname: the latter only allows one level of pointers currently. In any case, initially we make a limited use of pointer types.
