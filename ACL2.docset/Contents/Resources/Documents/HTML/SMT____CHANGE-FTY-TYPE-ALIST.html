Modifying constructor for fty-type-alist structures.

Syntax
(change-fty-type-alist x 
                       [:key-type <key-type>] 
                       [:val-type <val-type>])

This is an often useful alternative to make-fty-type-alist.

We construct a new fty-type-alist structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-fty-type-alist

(defmacro
    change-fty-type-alist (x &rest args)
    (std::change-aggregate 'fty-type-alist
                           x args
                           '((:key-type . fty-type-alist->key-type)
                             (:val-type . fty-type-alist->val-type))
                           'change-fty-type-alist
                           'nil))
