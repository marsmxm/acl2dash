Modifying constructor for montgomery-curve structures.

Syntax
(change-montgomery-curve x 
                         [:p <p>] 
                         [:a <a>] 
                         [:b <b>])

This is an often useful alternative to make-montgomery-curve.

We construct a new montgomery-curve structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-montgomery-curve

(defmacro change-montgomery-curve (x &rest args)
          (std::change-aggregate 'montgomery-curve
                                 x args
                                 '((:p . montgomery-curve->p)
                                   (:a . montgomery-curve->a)
                                   (:b . montgomery-curve->b))
                                 'change-montgomery-curve
                                 'nil))
