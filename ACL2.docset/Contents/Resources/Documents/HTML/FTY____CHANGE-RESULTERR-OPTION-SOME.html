Modifying constructor for resulterr-option-some structures.

Syntax
(change-resulterr-option-some x 
                              [:val <val>])

This is an often useful alternative to make-resulterr-option-some.

We construct a new resulterr-option-some structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

This is an ordinary change- macro introduced by defprod.
