Basic constructor macro for stmt-type structures.

Syntax
(make-stmt-type [:return-types <return-types>] 
                [:variables <variables>])

This is the usual way to construct stmt-type structures. It simply conses together a structure with the specified fields.

This macro generates a new stmt-type structure from scratch. See also change-stmt-type, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by fty::defprod.

Macro: make-stmt-type

(defmacro make-stmt-type (&rest args)
          (std::make-aggregate 'stmt-type
                               args '((:return-types) (:variables))
                               'make-stmt-type
                               nil))

 stmt-type (return-types variables)
 (declare (xargs :guard (and (type-setp return-types)
                             (var-tablep variables))))
 (declare (xargs :guard (not (empty return-types))))
 (let
    ((__function__ 'stmt-type))
