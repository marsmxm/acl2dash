Recognize the Java null literal.

Signature
(null-literalp x) → yes/no
Returns
yes/no — Type (booleanp yes/no).

The Java null literal is a list of Java Unicode characters that consists of the (ASCII) codes of *null-literal*.

Definitions and Theorems

Function: null-literalp

(defun null-literalp (x)
       (declare (xargs :guard t))
       (let ((__function__ 'null-literalp))
            (declare (ignorable __function__))
            (and (ascii-listp x)
                 (equal (ascii=>string x)
                        *null-literal*))))

Theorem: booleanp-of-null-literalp

(defthm booleanp-of-null-literalp
        (b* ((yes/no (null-literalp x)))
            (booleanp yes/no))
        :rule-classes :rewrite)
