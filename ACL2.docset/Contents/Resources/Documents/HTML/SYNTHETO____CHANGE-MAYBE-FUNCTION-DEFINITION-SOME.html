Modifying constructor for maybe-function-definition-some structures.

Syntax
(change-maybe-function-definition-some x 
                                       [:val <val>])

This is an often useful alternative to make-maybe-function-definition-some.

We construct a new maybe-function-definition-some structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-maybe-function-definition-some

(defmacro change-maybe-function-definition-some
          (x &rest args)

               'maybe-function-definition-some
               x args
               '((:val . maybe-function-definition-some->val))
               'change-maybe-function-definition-some
               'nil))
