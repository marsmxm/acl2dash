Modifying constructor for definition-option-none structures.

Syntax
(change-definition-option-none x 
                               )

This is an often useful alternative to make-definition-option-none.

We construct a new definition-option-none structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
