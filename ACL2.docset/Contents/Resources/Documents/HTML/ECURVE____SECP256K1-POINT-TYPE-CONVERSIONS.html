Conversions between secp256k1-point and pointp.

The type secp256k1-point represents elliptic curve points as FTY products, with the point at infinity as (0, 0). On the other hand, the predicate pointp represents points as cons pairs, except for the point at infinity :infinity. Here we define functions to convert between these two representations.
