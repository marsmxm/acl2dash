(axi-termlist-fix x) is a usual ACL2::fty list fixing function.

Signature
(axi-termlist-fix x) → fty::newx
Arguments
x — Guard (axi-termlist-p x).
Returns
fty::newx — Type (axi-termlist-p fty::newx).

In the logic, we apply axi-term-fix to each member of the x. In the execution, none of that is actually necessary and this is just an inlined identity function.
