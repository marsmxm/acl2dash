(svarlist-list-fix x) is a usual fty list fixing function.

Signature
(svarlist-list-fix x) → fty::newx
Arguments
x — Guard (svarlist-list-p x).
Returns
fty::newx — Type (svarlist-list-p fty::newx).

In the logic, we apply svarlist-fix to each member of the x. In the execution, none of that is actually necessary and this is just an inlined identity function.
