Build a description of the :new-name input for the user documentation of an APT transformation.

Macro: desc-apt-input-new-name

(defmacro
 xdoc::desc-apt-input-new-name
 (&rest additional)
 (cons
  'xdoc::desc
  (cons
   '"@(':new-name') &mdash; default @(':auto')"
   (cons
    '(xdoc::p "Determines the name of the generated new function.")
    (cons
     '(xdoc::p "It must be one of the following:")
     (cons
      '(xdoc::ul
        (xdoc::li
         "@(':auto'), to generate the name automatically
       as described in @(see function-name-generation).")
        (xdoc::li
           "Any other symbol, to use as the name of the function."))
      (cons
       '(xdoc::p
         "In the rest of this documentation page,
      let @('new') be this function.")
       additional)))))))
