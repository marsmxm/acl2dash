Modifying constructor for package-name structures.

Syntax
(change-package-name x 
                     [:identifiers <identifiers>])

This is an often useful alternative to make-package-name.

We construct a new package-name structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-package-name

(defmacro change-package-name (x &rest args)
          (std::change-aggregate
               'package-name
               x args
               '((:identifiers . package-name->identifiers))
               'change-package-name
               'nil))
