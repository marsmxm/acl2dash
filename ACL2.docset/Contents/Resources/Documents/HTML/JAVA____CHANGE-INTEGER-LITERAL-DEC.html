Modifying constructor for integer-literal-dec structures.

Syntax
(change-integer-literal-dec x 
                            [:get <get>])

This is an often useful alternative to make-integer-literal-dec.

We construct a new integer-literal-dec structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-integer-literal-dec

(defmacro
     change-integer-literal-dec
     (x &rest args)
     (std::change-aggregate 'integer-literal-dec
                            x args
                            '((:get . integer-literal-dec->get))
                            'change-integer-literal-dec
                            'nil))
