Modifying constructor for jlocvar structures.

Syntax
(change-jlocvar x 
                [:final? <final?>] 
                [:type <type>] 
                [:name <name>] 
                [:init? <init?>])

This is an often useful alternative to make-jlocvar.

We construct a new jlocvar structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-jlocvar

(defmacro change-jlocvar (x &rest args)
          (std::change-aggregate 'jlocvar
                                 x args
                                 '((:final? . jlocvar->final?)
                                   (:type . jlocvar->type)
                                   (:name . jlocvar->name)
                                   (:init? . jlocvar->init?))
                                 'change-jlocvar
                                 'nil))
