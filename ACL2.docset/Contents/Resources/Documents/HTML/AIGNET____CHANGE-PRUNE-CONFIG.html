Modifying constructor for prune-config structures.

Syntax
(change-prune-config x 
                     [:gatesimp <gatesimp>])

This is an often useful alternative to make-prune-config.

We construct a new prune-config structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-prune-config

(defmacro
     change-prune-config (x &rest args)
     (std::change-aggregate 'prune-config
                            x args
                            '((:gatesimp . prune-config->gatesimp))
                            'change-prune-config
                            'remake-prune-config))
