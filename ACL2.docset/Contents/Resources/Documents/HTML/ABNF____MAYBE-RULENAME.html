Fixtype of rule names and nil.

This is an option type introduced by fty::defoption. Note that defoption is just a wrapper for fty::defflexsum, so there are :none and :some member types, a case macro, and so forth.

Member Types
:none → maybe-rulename-none
Represents that no maybe-rulename is available, i.e., Nothing or None.
:some → maybe-rulename-some
An available maybe-rulename, i.e., Just val or Some val.
