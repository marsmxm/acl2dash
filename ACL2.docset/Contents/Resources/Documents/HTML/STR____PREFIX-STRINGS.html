Concatenates a prefix onto every string in a list of strings.

Signature
(prefix-strings prefix x) → *
Arguments
prefix — Guard (stringp prefix).
x — Guard (string-listp x).

(prefix-strings prefix x) produces a new string list by concatenating prefix onto every member of x.

Definitions and Theorems

Function: prefix-strings

(defun prefix-strings (prefix x)
       (declare (xargs :guard (and (stringp prefix)
                                   (string-listp x))))
       (let ((acl2::__function__ 'prefix-strings))
            (declare (ignorable acl2::__function__))
            (if (atom x)
                nil
                (cons (cat prefix (car x))
                      (prefix-strings prefix (cdr x))))))

Theorem: prefix-strings-when-atom

(defthm prefix-strings-when-atom
        (implies (atom x)
                 (equal (prefix-strings prefix x) nil)))

Theorem: prefix-strings-of-cons

(defthm prefix-strings-of-cons
        (equal (prefix-strings prefix (cons a x))
               (cons (cat prefix a)
                     (prefix-strings prefix x))))

(defthm len-of-prefix-strings
