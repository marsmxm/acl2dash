Modifying constructor for tree-leafrule structures.

Syntax
(change-tree-leafrule x 
                      [:get <get>])

This is an often useful alternative to make-tree-leafrule.

We construct a new tree-leafrule structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-tree-leafrule

(defmacro
     change-tree-leafrule (x &rest args)
     (std::change-aggregate 'tree-leafrule
                            x args '((:get . tree-leafrule->get))
                            'change-tree-leafrule
                            'nil))
