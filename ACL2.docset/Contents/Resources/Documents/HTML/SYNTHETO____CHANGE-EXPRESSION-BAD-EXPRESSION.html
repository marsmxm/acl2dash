Modifying constructor for expression-bad-expression structures.

Syntax
(change-expression-bad-expression x 
                                  [:info <info>])

This is an often useful alternative to make-expression-bad-expression.

We construct a new expression-bad-expression structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-expression-bad-expression

(defmacro
