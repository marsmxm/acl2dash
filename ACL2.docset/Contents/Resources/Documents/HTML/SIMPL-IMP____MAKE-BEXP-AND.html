Basic constructor macro for bexp-and structures.

Syntax
(make-bexp-and [:left <left>] 
               [:right <right>])

This is the usual way to construct bexp-and structures. It simply conses together a structure with the specified fields.

This macro generates a new bexp-and structure from scratch. See also change-bexp-and, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by fty::defprod.

(defun bexp-and (left right)
       (declare (xargs :guard (and (bexpp left) (bexpp right))))
       (declare (xargs :guard t))
       (let ((__function__ 'bexp-and))
            (declare (ignorable __function__))
            (b* ((left (mbe :logic (bexp-fix left) :exec left))
                 (right (mbe :logic (bexp-fix right)
                             :exec right)))
                (cons :and (list left right)))))
