Modifying constructor for vl-lucidctx structures.

Syntax
(change-vl-lucidctx x 
                    [:modname <modname>] 
                    [:elem <elem>] 
                    [:ss <ss>] 
                    [:portname <portname>])

This is an often useful alternative to make-vl-lucidctx.

We construct a new vl-lucidctx structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-lucidctx

(defmacro
