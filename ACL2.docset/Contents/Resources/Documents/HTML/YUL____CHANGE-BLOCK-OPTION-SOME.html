Modifying constructor for block-option-some structures.

Syntax
(change-block-option-some x 
                          [:val <val>])

This is an often useful alternative to make-block-option-some.

We construct a new block-option-some structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-block-option-some

(defmacro
     change-block-option-some (x &rest args)
     (std::change-aggregate 'block-option-some
                            x
                            args '((:val . block-option-some->val))
                            'change-block-option-some
                            'nil))
