Modifying constructor for jaccess-public structures.

Syntax
(change-jaccess-public x 
                       )

This is an often useful alternative to make-jaccess-public.

We construct a new jaccess-public structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-jaccess-public

(defmacro change-jaccess-public (x &rest args)
          (std::change-aggregate 'jaccess-public
                                 x args 'nil
                                 'change-jaccess-public
                                 'nil))
