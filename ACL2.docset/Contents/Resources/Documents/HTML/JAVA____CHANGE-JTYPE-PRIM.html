Modifying constructor for jtype-prim structures.

Syntax
(change-jtype-prim x 
                   [:type <type>])

This is an often useful alternative to make-jtype-prim.

We construct a new jtype-prim structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-jtype-prim

(defmacro
     change-jtype-prim (x &rest args)
     (std::change-aggregate 'jtype-prim
                            x args '((:type . jtype-prim->type))
                            'change-jtype-prim
                            'nil))
