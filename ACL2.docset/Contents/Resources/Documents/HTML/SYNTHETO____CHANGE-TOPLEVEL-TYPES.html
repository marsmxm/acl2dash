Modifying constructor for toplevel-types structures.

Syntax
(change-toplevel-types x 
                       [:get <get>])

This is an often useful alternative to make-toplevel-types.

We construct a new toplevel-types structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-toplevel-types

(defmacro
     change-toplevel-types (x &rest args)
     (std::change-aggregate 'toplevel-types
                            x args '((:get . toplevel-types->get))
                            'change-toplevel-types
                            'nil))
