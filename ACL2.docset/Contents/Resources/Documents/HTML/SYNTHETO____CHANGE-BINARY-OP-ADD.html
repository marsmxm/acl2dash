Modifying constructor for binary-op-add structures.

Syntax
(change-binary-op-add x 
                      )

This is an often useful alternative to make-binary-op-add.

We construct a new binary-op-add structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
