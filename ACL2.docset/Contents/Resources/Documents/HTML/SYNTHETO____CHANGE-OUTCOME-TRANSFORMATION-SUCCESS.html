Modifying constructor for outcome-transformation-success structures.

Syntax
(change-outcome-transformation-success x 
                                       [:message <message>] 
                                       [:toplevels <toplevels>])

This is an often useful alternative to make-outcome-transformation-success.

We construct a new outcome-transformation-success structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
