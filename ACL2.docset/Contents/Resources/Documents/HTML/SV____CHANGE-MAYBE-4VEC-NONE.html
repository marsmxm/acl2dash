Modifying constructor for maybe-4vec-none structures.

Syntax
(change-maybe-4vec-none x 
                        )

This is an often useful alternative to make-maybe-4vec-none.

We construct a new maybe-4vec-none structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.
