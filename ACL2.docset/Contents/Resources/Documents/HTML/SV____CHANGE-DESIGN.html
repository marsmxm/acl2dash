Modifying constructor for design structures.

Syntax
(change-design x 
               [:modalist <modalist>] 
               [:top <top>])

This is an often useful alternative to make-design.

We construct a new design structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-design

(defmacro change-design (x &rest args)
          (std::change-aggregate 'design
                                 x args
                                 '((:modalist . design->modalist)
                                   (:top . design->top))
                                 'change-design
                                 'nil))
