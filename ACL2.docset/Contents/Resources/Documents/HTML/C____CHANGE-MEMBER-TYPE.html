Modifying constructor for member-type structures.

Syntax
(change-member-type x 
                    [:name <name>] 
                    [:type <type>])

This is an often useful alternative to make-member-type.

We construct a new member-type structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-member-type

(defmacro change-member-type (x &rest args)
          (std::change-aggregate 'member-type
                                 x args
                                 '((:name . member-type->name)
                                   (:type . member-type->type))
                                 'change-member-type
                                 'nil))
