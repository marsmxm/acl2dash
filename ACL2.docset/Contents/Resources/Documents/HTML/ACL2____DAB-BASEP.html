Recognize valid bases for representing natural numbers as digits.

Signature
(dab-basep x) → yes/no
Returns
yes/no — Type (booleanp yes/no).

The fixing function for this predicate is dab-base-fix and the fixtype for this predicate is dab-base.

Any integer above 1 raised to a positive power is a valid base, e.g. binary, octal, and hexadecimal bases.

(defun dab-basep (x)
       (declare (xargs :guard t))
       (let ((__function__ 'dab-basep))
            (declare (ignorable __function__))
