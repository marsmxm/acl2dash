Modifying constructor for integer-literal-bin structures.

Syntax
(change-integer-literal-bin x 
                            [:get <get>])

This is an often useful alternative to make-integer-literal-bin.

We construct a new integer-literal-bin structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-integer-literal-bin

(defmacro
     change-integer-literal-bin
     (x &rest args)
     (std::change-aggregate 'integer-literal-bin
                            x args
                            '((:get . integer-literal-bin->get))
                            'change-integer-literal-bin
                            'nil))
