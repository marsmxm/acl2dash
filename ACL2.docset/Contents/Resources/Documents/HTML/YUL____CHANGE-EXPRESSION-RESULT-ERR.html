Modifying constructor for expression-result-err structures.

Syntax
(change-expression-result-err x 
                              [:get <get>])

This is an often useful alternative to make-expression-result-err.

We construct a new expression-result-err structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-expression-result-err

(defmacro
     change-expression-result-err
     (x &rest args)
     (std::change-aggregate 'expression-result-err
                            x args
                            '((:get . expression-result-err->get))
                            'change-expression-result-err
                            'nil))
