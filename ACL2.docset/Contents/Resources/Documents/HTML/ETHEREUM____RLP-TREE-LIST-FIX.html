(rlp-tree-list-fix x) is a usual ACL2::fty list fixing function.

Signature
(rlp-tree-list-fix x) → fty::newx
Arguments
x — Guard (rlp-tree-listp x).
Returns
fty::newx — Type (rlp-tree-listp fty::newx).

In the logic, we apply rlp-tree-fix to each member of the x. In the execution, none of that is actually necessary and this is just an inlined identity function.
