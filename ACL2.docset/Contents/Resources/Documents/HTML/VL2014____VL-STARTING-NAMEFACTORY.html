(vl-starting-namefactory mod) creates a name factory for a module.

Signature
(vl-starting-namefactory mod) → nf
Arguments
mod — Guard (vl-module-p mod).
Returns
nf — Type (vl-namefactory-p nf).

This function is very cheap to call because the real work of initializing the name factory is deferred to its first use. See vl-namefactory-p for all name factory documentation.

Definitions and Theorems

Function: vl-starting-namefactory

(defun vl-starting-namefactory (mod)
       (declare (xargs :guard (vl-module-p mod)))
       (let ((__function__ 'vl-starting-namefactory))
            (declare (ignorable __function__))
            (make-vl-namefactory :mod (vl-module-fix mod)
                                 :namedb (vl-empty-namedb))))
