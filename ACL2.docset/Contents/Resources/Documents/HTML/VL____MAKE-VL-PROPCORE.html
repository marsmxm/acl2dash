Basic constructor macro for vl-propcore structures.

Syntax
(make-vl-propcore [:guts <guts>])

This is the usual way to construct vl-propcore structures. It simply conses together a structure with the specified fields.

This macro generates a new vl-propcore structure from scratch. See also change-vl-propcore, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by defprod.

Macro: make-vl-propcore

(defmacro make-vl-propcore (&rest args)
