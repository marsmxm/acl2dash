Modifying constructor for oct-integer-literal structures.

Syntax
(change-oct-integer-literal x 
                            [:digits/uscores <digits/uscores>] 
                            [:prefix-upcase-p <prefix-upcase-p>] 
                            [:suffix? <suffix?>])

This is an often useful alternative to make-oct-integer-literal.

We construct a new oct-integer-literal structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-oct-integer-literal

(defmacro
    change-oct-integer-literal
    (x &rest args)
    (std::change-aggregate
         'oct-integer-literal
         x args
         '((:digits/uscores . oct-integer-literal->digits/uscores)
           (:prefix-upcase-p . oct-integer-literal->prefix-upcase-p)
           (:suffix? . oct-integer-literal->suffix?))
         'change-oct-integer-literal
         'nil))
