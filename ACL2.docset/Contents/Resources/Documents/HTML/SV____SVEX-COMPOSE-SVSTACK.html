Compose an svex with a substitution alist. Variables not in the substitution are left in place.

Signature
(svex-compose-svstack x a) → xa
Arguments
x — Guard (svex-p x).
a — Guard (svstack-p a).
Returns
xa — Type (equal xa (svex-compose x (svstack-to-svex-alist a))).
