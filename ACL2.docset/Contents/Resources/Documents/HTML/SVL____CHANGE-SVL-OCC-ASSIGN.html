Modifying constructor for svl-occ-assign structures.

Syntax
(change-svl-occ-assign x 
                       [:output <output>] 
                       [:svex <svex>])

This is an often useful alternative to make-svl-occ-assign.

We construct a new svl-occ-assign structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-svl-occ-assign

(defmacro
     change-svl-occ-assign (x &rest args)
     (std::change-aggregate 'svl-occ-assign
                            x args
                            '((:output . svl-occ-assign->output)
                              (:svex . svl-occ-assign->svex))
                            'change-svl-occ-assign
                            'nil))
