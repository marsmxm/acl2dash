Modifying constructor for symbol-nonterminal structures.

Syntax
(change-symbol-nonterminal x 
                           [:get <get>])

This is an often useful alternative to make-symbol-nonterminal.

We construct a new symbol-nonterminal structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-symbol-nonterminal

(defmacro
     change-symbol-nonterminal (x &rest args)
     (std::change-aggregate 'symbol-nonterminal
                            x
                            args '((:get . symbol-nonterminal->get))
                            'change-symbol-nonterminal
                            'nil))
