This is identical to vl-lintresult, except that we hons the structure we are creating.

Definition

This is an ordinary honsing constructor introduced by defaggregate.

Function: honsed-vl-lintresult

(defun
 honsed-vl-lintresult
 (design design0 reportcard sd-probs)
 (declare (xargs :guard (and (vl-design-p design)
                             (vl-design-p design0)
                             (vl-reportcard-p reportcard)
                             (sd-problemlist-p sd-probs))))
 (mbe
  :logic (vl-lintresult design design0 reportcard sd-probs)
  :exec
  (hons
    :vl-lintresult (hons (hons 'design design)
                         (hons (hons 'design0 design0)
                               (hons (hons 'reportcard reportcard)
                                     (hons (hons 'sd-probs sd-probs)
                                           nil)))))))
