This is a product type, introduced by deftagsum in support of vl-genelement.

Fields
test vl-expr
The expression to test against the cases, e.g., mode in case (mode) ....
cases vl-gencaselist
The match expressions and corresponding blocks.
default vl-genblock
The default block. May be an empty vl-genblock.
loc vl-location
Where this case came from in the Verilog source code.
