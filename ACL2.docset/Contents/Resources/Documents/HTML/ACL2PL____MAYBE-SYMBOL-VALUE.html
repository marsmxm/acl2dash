Fixtype of symbol values and nil.

This is an option type introduced by fty::defoption. Note that defoption is just a wrapper for fty::defflexsum, so there are :none and :some member types, a case macro, and so forth.

Member Types
:none maybe-symbol-value-none
Represents that no maybe-symbol-value is available, i.e., Nothing or None.
:some maybe-symbol-value-some
An available maybe-symbol-value, i.e., Just val or Some val.
