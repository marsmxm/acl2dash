Modifying constructor for expression-call structures.

Syntax
(change-expression-call x 
                        [:function <function>] 
                        [:types <types>] 
                        [:arguments <arguments>])

This is an often useful alternative to make-expression-call.

We construct a new expression-call structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-expression-call

(defmacro
 change-expression-call (x &rest args)
 (std::change-aggregate 'expression-call
                        x args
                        '((:function . expression-call->function)
                          (:types . expression-call->types)
                          (:arguments . expression-call->arguments))
                        'change-expression-call
                        'nil))
