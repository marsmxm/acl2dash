Helper macro for writing optimized 4vec-operations.

(if-2vec-p vars 2vec-body 4vec-body) is a macro for writing 4vec operations. Most of these operations can be optimized in the case where their arguments are two-valued. In the logic, this simply expands to 4vec-body. In the execution, it checks whether all of the arguments are 2vecs (which is fast), if so executes 2vec-body, and otherwise executes 4vec-body.
