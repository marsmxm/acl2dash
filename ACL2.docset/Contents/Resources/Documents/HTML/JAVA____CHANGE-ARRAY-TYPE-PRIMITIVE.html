Modifying constructor for array-type-primitive structures.

Syntax
(change-array-type-primitive x 
                             [:element <element>] 
                             [:dimensions <dimensions>])

This is an often useful alternative to make-array-type-primitive.

We construct a new array-type-primitive structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-array-type-primitive

(defmacro change-array-type-primitive
          (x &rest args)
          (std::change-aggregate
               'array-type-primitive
               x args
               '((:element . array-type-primitive->element)
                 (:dimensions . array-type-primitive->dimensions))
               'change-array-type-primitive
               'nil))
