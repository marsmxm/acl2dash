Modifying constructor for expr-binary structures.

Syntax
(change-expr-binary x 
                    [:op <op>] 
                    [:arg1 <arg1>] 
                    [:arg2 <arg2>])

This is an often useful alternative to make-expr-binary.

We construct a new expr-binary structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-expr-binary

(defmacro change-expr-binary (x &rest args)
          (std::change-aggregate 'expr-binary
                                 x args
                                 '((:op . expr-binary->op)
                                   (:arg1 . expr-binary->arg1)
                                   (:arg2 . expr-binary->arg2))
                                 'change-expr-binary
                                 'nil))
