Override the type that is associated with a guard function for purposes of determining automatic congruences with deffixequiv.

The form:

(set-fixequiv-guard-override my-guard my-type)

makes it so that deffixequiv, deffixequiv-mutual, and fixequiv-hook will prove congruences appropriate for my-type for define formals guarded with my-guard.
