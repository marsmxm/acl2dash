Modifying constructor for vl-tname structures.

Syntax
(change-vl-tname x 
                 [:base <base>] 
                 [:model <model>])

This is an often useful alternative to make-vl-tname.

We construct a new vl-tname structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-tname

(defmacro change-vl-tname (x &rest args)
          (std::change-aggregate 'vl-tname
                                 x args
                                 '((:base . vl-tname->base)
                                   (:model . vl-tname->model))
                                 'change-vl-tname
                                 'nil))
