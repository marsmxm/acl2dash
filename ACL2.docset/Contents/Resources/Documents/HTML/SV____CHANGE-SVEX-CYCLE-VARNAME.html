Modifying constructor for svex-cycle-varname structures.

Syntax
(change-svex-cycle-varname x 
                           [:name <name>] 
                           [:cycle <cycle>])

This is an often useful alternative to make-svex-cycle-varname.

We construct a new svex-cycle-varname structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.
