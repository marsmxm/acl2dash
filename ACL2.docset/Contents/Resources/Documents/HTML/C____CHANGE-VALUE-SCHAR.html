Modifying constructor for value-schar structures.

Syntax
(change-value-schar x 
                    [:get <get>])

This is an often useful alternative to make-value-schar.

We construct a new value-schar structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-value-schar

(defmacro change-value-schar (x &rest args)
          (std::change-aggregate 'value-schar
                                 x args '((:get . value-schar->get))
                                 'change-value-schar
                                 'nil))
