Modifying constructor for fun-adeclor-pointer structures.

Syntax
(change-fun-adeclor-pointer x 
                            [:decl <decl>])

This is an often useful alternative to make-fun-adeclor-pointer.

We construct a new fun-adeclor-pointer structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-fun-adeclor-pointer

(defmacro
     change-fun-adeclor-pointer
