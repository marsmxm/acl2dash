Rules about type-of-value-option.

These rules reduce type-of-value-option to type-of-value when the argument is a value, and to void when the argument is nil. During execution, the argument is always either nil or a term that is easily proved to be a value; so these rules suffice to eliminate type-of-value-option.

Definitions and Theorems

Theorem: type-of-value-option-when-valuep

(defthm type-of-value-option-when-valuep
        (implies (valuep x)
                 (equal (type-of-value-option x)
                        (type-of-value x))))

Theorem: type-of-value-option-of-nil

(defthm type-of-value-option-of-nil
        (equal (type-of-value-option nil)
               (type-void)))
