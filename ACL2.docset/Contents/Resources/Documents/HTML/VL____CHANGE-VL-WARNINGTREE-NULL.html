Modifying constructor for vl-warningtree-null structures.

Syntax
(change-vl-warningtree-null x 
                            )

This is an often useful alternative to make-vl-warningtree-null.

We construct a new vl-warningtree-null structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
