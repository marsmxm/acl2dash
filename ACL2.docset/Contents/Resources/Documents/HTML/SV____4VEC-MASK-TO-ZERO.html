Reduce a constant 4vec using a 4vmask; any irrelevant bits become 0s.

Signature
(4vec-mask-to-zero mask value) → masked-value
Arguments
mask — Mask of bits that we care about.
    Guard (4vmask-p mask).
value — Original value to be masked.
    Guard (4vec-p value).
Returns
masked-value — value with irrelevant bits replaced by 0s.
    Type (4vec-p masked-value).

Function: 4vec-mask-to-zero

(defun 4vec-mask-to-zero (mask value)
       (declare (xargs :guard (and (4vmask-p mask) (4vec-p value))))
