Modifying constructor for ushort structures.

Syntax
(change-ushort x 
               [:get <get>])

This is an often useful alternative to make-ushort.

We construct a new ushort structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-ushort

(defmacro change-ushort (x &rest args)
          (std::change-aggregate 'ushort
                                 x args '((:get . ushort->get))
                                 'change-ushort
                                 'nil))
