Modifying constructor for vl-cycledelayrange structures.

Syntax
(change-vl-cycledelayrange x 
                           [:left <left>] 
                           [:right <right>])

This is an often useful alternative to make-vl-cycledelayrange.

We construct a new vl-cycledelayrange structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.
