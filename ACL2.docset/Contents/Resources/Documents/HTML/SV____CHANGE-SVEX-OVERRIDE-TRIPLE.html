Modifying constructor for svex-override-triple structures.

Syntax
(change-svex-override-triple x 
                             [:testvar <testvar>] 
                             [:valvar <valvar>] 
                             [:valexpr <valexpr>])

This is an often useful alternative to make-svex-override-triple.

We construct a new svex-override-triple structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-svex-override-triple

(defmacro change-svex-override-triple
          (x &rest args)
          (std::change-aggregate
               'svex-override-triple
               x args
               '((:testvar . svex-override-triple->testvar)
                 (:valvar . svex-override-triple->valvar)
                 (:valexpr . svex-override-triple->valexpr))
               'change-svex-override-triple
               'nil))
