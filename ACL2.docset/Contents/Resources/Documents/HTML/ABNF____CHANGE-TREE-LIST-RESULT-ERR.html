Modifying constructor for tree-list-result-err structures.

Syntax
(change-tree-list-result-err x 
                             [:get <get>])

This is an often useful alternative to make-tree-list-result-err.

We construct a new tree-list-result-err structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
