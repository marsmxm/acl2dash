Basic constructor macro for expr-call structures.

Syntax
(make-expr-call [:fun <fun>] 
                [:args <args>])

This is the usual way to construct expr-call structures. It simply conses together a structure with the specified fields.

This macro generates a new expr-call structure from scratch. See also change-expr-call, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by fty::defprod.

Macro: make-expr-call

(defmacro make-expr-call (&rest args)
          (std::make-aggregate 'expr-call
                               args '((:fun) (:args))
                               'make-expr-call
                               nil))

(defun expr-call (fun args)
       (declare (xargs :guard (and (identp fun) (expr-listp args))))
       (declare (xargs :guard t))
       (let ((__function__ 'expr-call))
            (declare (ignorable __function__))
            (b* ((fun (mbe :logic (ident-fix fun) :exec fun))
                 (args (mbe :logic (expr-list-fix args)
                            :exec args)))
                (cons :call (list fun args)))))
