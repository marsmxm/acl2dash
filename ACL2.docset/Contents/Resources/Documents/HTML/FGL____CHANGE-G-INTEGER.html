Modifying constructor for g-integer structures.

Syntax
(change-g-integer x 
                  [:bits <bits>])

This is an often useful alternative to make-g-integer.

We construct a new g-integer structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-g-integer

(defmacro change-g-integer (x &rest args)
          (std::change-aggregate 'g-integer
                                 x args '((:bits . g-integer->bits))
                                 'change-g-integer
                                 'nil))
