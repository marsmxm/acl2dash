Basic constructor macro for jexpr-newarray-init structures.

Syntax
(make-jexpr-newarray-init [:type <type>]

This is the usual way to construct jexpr-newarray-init structures. It simply conses together a structure with the specified fields.

This macro generates a new jexpr-newarray-init structure from scratch. See also change-jexpr-newarray-init, which can "change" an existing structure, instead.
