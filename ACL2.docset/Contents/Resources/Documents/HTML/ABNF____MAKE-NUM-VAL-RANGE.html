Basic constructor macro for num-val-range structures.

Syntax
(make-num-val-range [:min <min>] 
                    [:max <max>])

This is the usual way to construct num-val-range structures. It simply conses together a structure with the specified fields.

This macro generates a new num-val-range structure from scratch. See also change-num-val-range, which can "change" an existing structure, instead.

This is an ordinary make- macro introduced by fty::defprod.
