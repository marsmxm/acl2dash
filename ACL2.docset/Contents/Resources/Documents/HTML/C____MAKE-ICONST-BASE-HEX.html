Basic constructor macro for iconst-base-hex structures.

Syntax
(make-iconst-base-hex )

This is the usual way to construct iconst-base-hex structures. It simply conses together a structure with the specified fields.

This macro generates a new iconst-base-hex structure from scratch. See also change-iconst-base-hex, which can "change" an existing structure, instead.
