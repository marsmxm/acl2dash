Modifying constructor for rlp-error-subtree structures.

Syntax
(change-rlp-error-subtree x 
                          [:error <error>])

This is an often useful alternative to make-rlp-error-subtree.

We construct a new rlp-error-subtree structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-rlp-error-subtree

(defmacro
     change-rlp-error-subtree (x &rest args)
     (std::change-aggregate 'rlp-error-subtree
                            x args
                            '((:error . rlp-error-subtree->error))
                            'change-rlp-error-subtree
                            'nil))
