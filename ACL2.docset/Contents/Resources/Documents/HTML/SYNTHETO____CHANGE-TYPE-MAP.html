Modifying constructor for type-map structures.

Syntax
(change-type-map x 
                 [:domain <domain>] 
                 [:range <range>])

This is an often useful alternative to make-type-map.

We construct a new type-map structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-type-map

(defmacro change-type-map (x &rest args)
          (std::change-aggregate 'type-map
                                 x args
                                 '((:domain . type-map->domain)
                                   (:range . type-map->range))
                                 'change-type-map
                                 'nil))
