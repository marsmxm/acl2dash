Modifying constructor for vl-always structures.

Syntax
(change-vl-always x 
                  [:type <type>] 
                  [:stmt <stmt>] 
                  [:atts <atts>] 
                  [:loc <loc>])

This is an often useful alternative to make-vl-always.

We construct a new vl-always structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-always

(defmacro change-vl-always (x &rest args)
          (std::change-aggregate 'vl-always
                                 x args
                                 '((:type . vl-always->type)
                                   (:stmt . vl-always->stmt)
                                   (:atts . vl-always->atts)
                                   (:loc . vl-always->loc))
                                 'change-vl-always
                                 'remake-vl-always))
