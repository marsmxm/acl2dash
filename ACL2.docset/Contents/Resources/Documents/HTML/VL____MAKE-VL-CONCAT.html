Basic constructor macro for vl-concat structures.

Syntax
(make-vl-concat [:parts <parts>] 
                [:atts <atts>])

This is the usual way to construct vl-concat structures. It simply conses together a structure with the specified fields.

This macro generates a new vl-concat structure from scratch. See also change-vl-concat, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by defprod.

Macro: make-vl-concat

(defmacro make-vl-concat (&rest args)
          (std::make-aggregate 'vl-concat
                               args '((:parts) (:atts))
                               'make-vl-concat
                               nil))

Function: vl-concat

(defun vl-concat (parts atts)
       (declare (xargs :guard (and (vl-exprlist-p parts)
                                   (vl-atts-p atts))))
       (declare (xargs :guard t))
       (let ((__function__ 'vl-concat))
            (declare (ignorable __function__))
            (b* ((parts (mbe :logic (vl-exprlist-fix parts)
                             :exec parts))
                 (atts (mbe :logic (vl-atts-fix atts)
                            :exec atts)))
                (cons :vl-concat (std::prod-cons parts atts)))))
