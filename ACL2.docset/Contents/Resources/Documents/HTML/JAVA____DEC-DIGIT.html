Fixtype of Java decimal digits [JLS:3.10.1].

A Java decimal digit is a Java ASCII character between `0' and `9'. See the grammar rule digit.

This is a type introduced by fty::deffixtype.

(defun dec-digitp (x)
       (declare (xargs :guard t))
       (let ((__function__ 'dec-digitp))
            (declare (ignorable __function__))
