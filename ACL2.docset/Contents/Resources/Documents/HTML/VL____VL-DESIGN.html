Top level representation of all modules, interfaces, programs, etc., resulting from parsing some Verilog source code.

This is a product type introduced by defprod.

Fields
version — vl-syntaxversion
Version of VL syntax being used.
mods — vl-modulelist
List of all modules.
udps — vl-udplist
List of user defined primtives.
interfaces — vl-interfacelist
List of interfaces.
programs — vl-programlist
List of all programs.
classes — vl-classlist
List of all classes.
packages — vl-packagelist
List of all packages.
configs — vl-configlist
List of configurations.
vardecls — vl-vardecllist
Top-level variable declarations.
taskdecls — vl-taskdecllist
Top-level task declarations.
fundecls — vl-fundecllist
Top-level function declarations.
paramdecls — vl-paramdecllist
Top-level (local and non-local) parameter declarations.
imports — vl-importlist
Top-level package import statements.
dpiimports — vl-dpiimportlist
Top-level DPI imports.
dpiexports — vl-dpiexportlist
Top-level DPI exports.
fwdtypes — vl-fwdtypedeflist
Forward (incomplete) typedefs.
typedefs — vl-typedeflist
Regular (non-forward, complete) typedefs.
binds — vl-bindlist
Top-level bind directives.
properties — vl-propertylist
Top-level property declarations.
sequences — vl-sequencelist
Top-level sequence declarations.
warnings — vl-warninglist
So-called "floating" warnings.
comments — vl-commentmap
So-called "floating" comments.
plusargs — string-listp
List of plusargs (without + characters).
