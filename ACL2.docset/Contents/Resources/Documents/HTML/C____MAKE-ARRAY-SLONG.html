Basic constructor macro for array-slong structures.

Syntax
(make-array-slong [:get <get>])

This is the usual way to construct array-slong structures. It simply conses together a structure with the specified fields.

This macro generates a new array-slong structure from scratch. See also change-array-slong, which can "change" an existing structure, instead.

This is an ordinary make- macro introduced by fty::defprod.
