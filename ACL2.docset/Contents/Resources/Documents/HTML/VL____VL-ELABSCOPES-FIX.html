(vl-elabscopes-fix x) is an fty alist fixing function that follows the fix-keys strategy.

Signature
(vl-elabscopes-fix x) → fty::newx
Arguments
x — Guard (vl-elabscopes-p x).
Returns
fty::newx — Type (vl-elabscopes-p fty::newx).

Note that in the execution this is just an inline identity function.
