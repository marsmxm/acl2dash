(stv-spec-p x) is a std::defaggregate of the following fields.

  • mod — Name of the top-level Verilog module to simulate.
        Invariant (stringp mod).
  • inputs — Input lines, if any.
  • outputs — Output lines, if any.
  • internals — Internal lines, if any.
  • overrides — Override lines, if any.
