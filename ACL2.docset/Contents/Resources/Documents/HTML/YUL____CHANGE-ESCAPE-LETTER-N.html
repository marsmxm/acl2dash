Modifying constructor for escape-letter-n structures.

Syntax
(change-escape-letter-n x 
                        )

This is an often useful alternative to make-escape-letter-n.

We construct a new escape-letter-n structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
