Generate the theorem saying that looking up a certain C function in the function environment yields the information for that function.

Signature
(atc-gen-cfun-fun-env-thm fn prog-const finfo? 
                          init-fun-env-thm names-to-avoid wrld) 
 
  → 
(mv local-events name updated-names-to-avoid)
Arguments
fn — Guard (symbolp fn).
prog-const — Guard (symbolp prog-const).
finfo? — Guard (fun-info-optionp finfo?).
init-fun-env-thm — Guard (symbolp init-fun-env-thm).
names-to-avoid — Guard (symbol-listp names-to-avoid).
wrld — Guard (plist-worldp wrld).
Returns
local-events — A pseudo-event-form-listp.
name — A symbolp.
updated-names-to-avoid — A symbol-listp.

This serves to speed up the proofs when there is a large number of functions involved. A previous version of ATC was generating proofs that were executing function lookups, which worked fine for small programs, but not for larger programs.

Function: atc-gen-cfun-fun-env-thm

(defun
 atc-gen-cfun-fun-env-thm
 (fn prog-const finfo?
     init-fun-env-thm names-to-avoid wrld)
 (declare (xargs :guard (and (symbolp fn)
                             (symbolp prog-const)
                             (fun-info-optionp finfo?)
                             (symbolp init-fun-env-thm)
                             (symbol-listp names-to-avoid)
                             (plist-worldp wrld))))
 (let
  ((__function__ 'atc-gen-cfun-fun-env-thm))
  (declare (ignorable __function__))
  (b*
   (((unless (fun-infop finfo?))
     (mv nil nil names-to-avoid))
    (thm-name (add-suffix fn "-FUN-ENV"))
    ((mv thm-name names-to-avoid)
     (fresh-logical-name-with-$s-suffix
          thm-name nil names-to-avoid wrld))
    (fn-name (symbol-name fn))
    (formula
         (cons 'equal
               (cons (cons 'fun-env-lookup
                           (cons (cons 'ident (cons fn-name 'nil))
                                 (cons (cons 'init-fun-env
                                             (cons prog-const 'nil))
                                       'nil)))
                     (cons (cons 'quote (cons finfo? 'nil))
                           'nil))))
    (hints
     (cons
      (cons
       '"Goal"
       (cons
        ':in-theory
        (cons
            (cons 'quote
                  (cons (cons (cons ':e (cons 'fun-env-lookup 'nil))
                              (cons (cons ':e (cons 'ident 'nil))
                                    (cons init-fun-env-thm 'nil)))
                        'nil))
            'nil)))
      'nil))
    ((mv event &)
     (evmac-generate-defthm thm-name
                            :formula formula
                            :hints hints
                            :enable nil)))
   (mv (list event)
       thm-name names-to-avoid))))
