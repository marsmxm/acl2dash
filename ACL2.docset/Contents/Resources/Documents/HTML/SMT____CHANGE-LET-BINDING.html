Modifying constructor for let-binding structures.

Syntax
(change-let-binding x 
                    [:bindings <bindings>] 
                    [:hypotheses <hypotheses>])

This is an often useful alternative to make-let-binding.

We construct a new let-binding structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-let-binding

(defmacro
   change-let-binding (x &rest args)
   (std::change-aggregate 'let-binding
                          x args
                          '((:bindings . let-binding->bindings)
                            (:hypotheses . let-binding->hypotheses))
                          'change-let-binding
                          'nil))
