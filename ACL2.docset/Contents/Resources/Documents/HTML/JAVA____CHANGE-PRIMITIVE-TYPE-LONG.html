Modifying constructor for primitive-type-long structures.

Syntax
(change-primitive-type-long x 
                            )

This is an often useful alternative to make-primitive-type-long.

We construct a new primitive-type-long structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
