Modifying constructor for natoption/natoptionlist-one structures.

Syntax
(change-natoption/natoptionlist-one x 
                                    [:get <get>])

This is an often useful alternative to make-natoption/natoptionlist-one.

We construct a new natoption/natoptionlist-one structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
