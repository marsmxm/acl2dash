Option type; vl-direction-p or nil.

This is an option type introduced by fty::defoption. Note that defoption is just a wrapper for fty::defflexsum, so there are :none and :some member types, a case macro, and so forth.

Member Types
:none → vl-maybe-direction-none
Represents that no vl-maybe-direction is available, i.e., Nothing or None.
:some → vl-maybe-direction-some
An available vl-maybe-direction, i.e., Just val or Some val.
