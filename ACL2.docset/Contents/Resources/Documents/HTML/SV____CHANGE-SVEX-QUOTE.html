Modifying constructor for svex-quote structures.

Syntax
(change-svex-quote x 
                   [:val <val>])

This is an often useful alternative to make-svex-quote.

We construct a new svex-quote structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-svex-quote

(defmacro change-svex-quote (x &rest args)
          (std::change-aggregate 'svex-quote
                                 x args '((:val . svex-quote->val))
                                 'change-svex-quote
                                 'nil))
