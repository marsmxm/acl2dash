Modifying constructor for soutcome-result-ok structures.

Syntax
(change-soutcome-result-ok x 
                           [:get <get>])

This is an often useful alternative to make-soutcome-result-ok.

We construct a new soutcome-result-ok structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
