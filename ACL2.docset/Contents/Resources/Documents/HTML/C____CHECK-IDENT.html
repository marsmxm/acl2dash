Check an identifier.

Signature
(check-ident id) → wf?
Arguments
id — Guard (identp id).
Returns
wf? — Type (wellformed-resultp wf?).

We check whether the underlying ACL2 string satisfies the conditions described in Section `Portable ASCII C identifiers' of atc.

Definitions and Theorems

Function: check-ident

(defun
 check-ident (id)
 (declare (xargs :guard (identp id)))
 (let
     ((__function__ 'check-ident))
     (declare (ignorable __function__))
     (if (atc-ident-stringp (ident->name id))
         :wellformed
         (error (list :illegal/unsupported-ident (ident-fix id))))))

Theorem: wellformed-resultp-of-check-ident

(defthm wellformed-resultp-of-check-ident
