Raw constructor for honsed vcd-multivector-p structures.

Syntax:

(honsed-vcd-multivector bits idcodes)

This is identical to vcd-multivector, except that we hons the structure we are creating.

Definition

This is an ordinary honsing constructor introduced by defaggregate.
