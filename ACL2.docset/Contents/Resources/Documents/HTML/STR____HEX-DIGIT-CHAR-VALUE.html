Coerces a hex-digit-char-p character into a number.

Signature
(hex-digit-char-value x) → val
Arguments
x — Guard (hex-digit-char-p x).
Returns
val — Type (natp val).

For instance, (hex-digit-char-value #\a) is 10. For any non-hex-digit-char-p, 0 is returned.
