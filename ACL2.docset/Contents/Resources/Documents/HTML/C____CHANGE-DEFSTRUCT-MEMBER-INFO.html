Modifying constructor for defstruct-member-info structures.

Syntax
(change-defstruct-member-info x 
                              [:memtype <memtype>] 
                              [:readers <readers>] 
                              [:writers <writers>] 
                              [:checkers <checkers>] 
                              [:reader-return-thms <reader-return-thms>] 
                              [:writer-return-thms <writer-return-thms>])

This is an often useful alternative to make-defstruct-member-info.

We construct a new defstruct-member-info structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-defstruct-member-info

(defmacro

    (:writer-return-thms .
                         defstruct-member-info->writer-return-thms))
  'change-defstruct-member-info
  'nil))
