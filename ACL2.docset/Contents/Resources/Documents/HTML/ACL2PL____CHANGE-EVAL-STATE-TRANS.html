Modifying constructor for eval-state-trans structures.

Syntax
(change-eval-state-trans x 
                         [:stack <stack>])

This is an often useful alternative to make-eval-state-trans.

We construct a new eval-state-trans structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-eval-state-trans

(defmacro
     change-eval-state-trans (x &rest args)
     (std::change-aggregate 'eval-state-trans

                            'nil))
