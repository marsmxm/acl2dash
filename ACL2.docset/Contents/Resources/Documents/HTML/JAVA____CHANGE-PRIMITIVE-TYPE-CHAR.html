Modifying constructor for primitive-type-char structures.

Syntax
(change-primitive-type-char x 
                            )

This is an often useful alternative to make-primitive-type-char.

We construct a new primitive-type-char structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
