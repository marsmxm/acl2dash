Modifying constructor for expression-option-some structures.

Syntax
(change-expression-option-some x 
                               [:val <val>])

This is an often useful alternative to make-expression-option-some.

We construct a new expression-option-some structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-expression-option-some

(defmacro
     change-expression-option-some
     (x &rest args)
     (std::change-aggregate 'expression-option-some
                            x args
                            '((:val . expression-option-some->val))
                            'change-expression-option-some
                            'nil))
