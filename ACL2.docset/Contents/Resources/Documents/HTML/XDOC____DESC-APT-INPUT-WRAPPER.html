Build a description of the :wrapper input for the user documentation of an APT transformation.

Macro: desc-apt-input-wrapper

(defmacro
 xdoc::desc-apt-input-wrapper
 (&rest additional)
 (cons
  'xdoc::desc
  (cons
   '"@(':wrapper') &mdash; default @('nil')"
   (cons
    '(xdoc::p
          "Determines whether the wrapper function is generated.")
    (cons
        '(xdoc::p "It must be one of the following:")
        (cons '(xdoc::ul (xdoc::li "@('t'), to generate it.")
                         (xdoc::li "@('nil'), to not generate it."))
              additional))))))
