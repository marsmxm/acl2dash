A simple binary operator applied to some arguments, like a + b or a & b.

This is a product type, introduced by deftagsum in support of vl-expr.

Fields
op — vl-binaryop-p
The operator being applied.
left — vl-expr
The left-hand side argument, e.g., a in a + b.
right — vl-expr
The right-hand side argument, e.g., b in a + b.
atts — vl-atts
Any (* foo = bar, baz *) style attributes.
