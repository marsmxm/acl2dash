Modifying constructor for candidate-assign structures.

Syntax
(change-candidate-assign x 
                         [:edge <edge>] 
                         [:val <val>])

This is an often useful alternative to make-candidate-assign.

We construct a new candidate-assign structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-candidate-assign

(defmacro change-candidate-assign (x &rest args)
          (std::change-aggregate 'candidate-assign
                                 x args
                                 '((:edge . candidate-assign->edge)
                                   (:val . candidate-assign->val))
                                 'change-candidate-assign
                                 'remake-candidate-assign))
