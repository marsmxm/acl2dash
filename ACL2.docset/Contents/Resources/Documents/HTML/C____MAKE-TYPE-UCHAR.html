Basic constructor macro for type-uchar structures.

Syntax
(make-type-uchar )

This is the usual way to construct type-uchar structures. It simply conses together a structure with the specified fields.

This macro generates a new type-uchar structure from scratch. See also change-type-uchar, which can "change" an existing structure, instead.
