Modifying constructor for jexpr-binary structures.

Syntax
(change-jexpr-binary x 
                     [:op <op>] 
                     [:left <left>] 
                     [:right <right>])

This is an often useful alternative to make-jexpr-binary.

We construct a new jexpr-binary structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-jexpr-binary

(defmacro change-jexpr-binary (x &rest args)
          (std::change-aggregate 'jexpr-binary
                                 x args
                                 '((:op . jexpr-binary->op)
                                   (:left . jexpr-binary->left)
                                   (:right . jexpr-binary->right))
                                 'change-jexpr-binary
                                 'nil))
