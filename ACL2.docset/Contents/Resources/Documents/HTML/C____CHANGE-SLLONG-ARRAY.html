Modifying constructor for sllong-array structures.

Syntax
(change-sllong-array x 
                     [:elements <elements>])

This is an often useful alternative to make-sllong-array.

We construct a new sllong-array structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-sllong-array

(defmacro
     change-sllong-array (x &rest args)
     (std::change-aggregate 'sllong-array
                            x args
                            '((:elements . sllong-array->elements))
                            'change-sllong-array
                            'nil))
