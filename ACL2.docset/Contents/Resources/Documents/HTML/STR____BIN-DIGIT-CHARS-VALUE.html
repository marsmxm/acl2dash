Coerces a list of bit digits into a natural number.

Signature
(bin-digit-chars-value x)  value
Arguments
x Guard (bin-digit-char-listp x).
Returns
value Type (natp value).

For instance, (bin-digit-chars-value '(#1 #0 #0 #0)) is 8. See also parse-bits-from-charlist for a more flexible function that can tolerate non-bit digits after the number.
