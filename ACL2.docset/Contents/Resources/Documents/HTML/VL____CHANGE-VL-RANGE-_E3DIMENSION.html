Modifying constructor for vl-dimension-range structures.

Syntax
(change-vl-range->dimension x 
                            [:range <range>])

This is an often useful alternative to make-vl-range->dimension.

We construct a new vl-dimension-range structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-range->dimension

(defmacro
     change-vl-range->dimension
     (x &rest args)
     (std::change-aggregate 'vl-range->dimension
                            x args
                            '((:range . vl-range->dimension->range))
                            'change-vl-range->dimension
                            'nil))
