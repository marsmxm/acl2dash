Fixtype of unary operators [C:6.5.3].

This is a tagged union type, introduced by fty::deftagsum.

Member Tags → Types
:plus → unop-plus
:minus → unop-minus
:bitnot → unop-bitnot
:lognot → unop-lognot

For now we only capture the ones that take and return integers (along with values of other types). These are unary plus, unary minus, bitwise negation/complement, and logical negation/complement.
