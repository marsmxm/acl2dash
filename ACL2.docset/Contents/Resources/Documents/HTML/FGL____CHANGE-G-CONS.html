Modifying constructor for g-cons structures.

Syntax
(change-g-cons x 
               [:car <car>] 
               [:cdr <cdr>])

This is an often useful alternative to make-g-cons.

We construct a new g-cons structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-g-cons

(defmacro change-g-cons (x &rest args)
          (std::change-aggregate 'g-cons
                                 x args
                                 '((:car . g-cons->car)
                                   (:cdr . g-cons->cdr))
                                 'change-g-cons
                                 'nil))
