Modifying constructor for scope-result-ok structures.

Syntax
(change-scope-result-ok x 
                        [:get <get>])

This is an often useful alternative to make-scope-result-ok.

We construct a new scope-result-ok structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-scope-result-ok

(defmacro
     change-scope-result-ok (x &rest args)
     (std::change-aggregate 'scope-result-ok
                            x args '((:get . scope-result-ok->get))
                            'change-scope-result-ok
                            'nil))
