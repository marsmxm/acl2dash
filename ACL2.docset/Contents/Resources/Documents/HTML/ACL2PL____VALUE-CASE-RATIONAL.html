Check if a value is a rational.

Signature
(value-case-rational x) → yes/no
Arguments
x — Guard (valuep x).
Returns
yes/no — Type (booleanp yes/no).

The name of this function is analogous to the notation (value-case ... :number) of the fixtype of values.

Function: value-case-rational

(defun value-case-rational (x)
       (declare (xargs :guard (valuep x)))
       (let ((__function__ 'value-case-rational))
            (declare (ignorable __function__))
            (and (value-case x :number)
                 (rationalp (value-number->get x)))))
