Modifying constructor for jliteral-string structures.

Syntax
(change-jliteral-string x 
                        [:value <value>])

This is an often useful alternative to make-jliteral-string.

We construct a new jliteral-string structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-jliteral-string

(defmacro
     change-jliteral-string (x &rest args)
     (std::change-aggregate 'jliteral-string
                            x args
                            '((:value . jliteral-string->value))
                            'change-jliteral-string
                            'nil))
