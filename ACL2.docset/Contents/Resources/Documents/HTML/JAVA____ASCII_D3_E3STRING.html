Convert a Java ASCII character list to an ACL2 string.

Signature
(ascii=>string ascii)  string
Arguments
ascii Guard (ascii-listp ascii).
Returns
string Type (stringp string).

This is the inverse of string=>unicode for the ASCII subset of Unicode. It converts lists of ASCII codes to Java strings, using the library function nats=>string.

This function could be extended to the ISO-8859-1 subset of Unicode, but for now this is not needed in our Java formalization.

Definitions and Theorems

Function: ascii=>string

(defun ascii=>string (ascii)
       (declare (xargs :guard (ascii-listp ascii)))
       (let ((__function__ 'ascii=>string))
            (declare (ignorable __function__))
            (b* ((ascii (mbe :logic (ascii-list-fix ascii)
                             :exec ascii)))
                (nats=>string ascii))))
