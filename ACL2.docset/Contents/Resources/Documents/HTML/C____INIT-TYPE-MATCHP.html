Check if an initializer type matches a type.

Signature
(init-type-matchp itype type) → wf?
Arguments
itype — Guard (init-typep itype).
type — Guard (typep type).
Returns
wf? — Type (wellformed-resultp wf?).

This is used when comparing the type of an initializer with the declared type of the object.

Fow now we require the initializer type to be a single one, and to be equal to the declared type after array-to-pointer conversion. Thus, for now we only support initialization of scalars: if the declared type was an array type, it would never be equal to the array-to-pointer conversion of the initializer type, which turns an array type into a pointer type. Even though [C:6.7.9/11] allows scalars to be initialized with singleton lists of expressions (i.e. single expressions between braces), we are more strict here and require a single expression.

We will extend this to declared array types and to list initializer types.

Function: init-type-matchp

(defun
 init-type-matchp (itype type)
 (declare (xargs :guard (and (init-typep itype) (typep type))))
 (let
  ((__function__ 'init-type-matchp))
  (declare (ignorable __function__))
  (init-type-case
     itype
     :single
     (if (type-equiv type (apconvert-type itype.get))
         :wellformed (error (list :init-type-mismatch
                                  :required (type-fix type)
                                  :supplied (init-type-fix itype))))
     :list
     (error (list :init-type-unsupported (init-type-fix itype))))))
