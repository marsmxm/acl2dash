Modifying constructor for lhbit-var structures.

Syntax
(change-lhbit-var x 
                  [:name <name>] 
                  [:idx <idx>])

This is an often useful alternative to make-lhbit-var.

We construct a new lhbit-var structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-lhbit-var

(defmacro change-lhbit-var (x &rest args)
          (std::change-aggregate 'lhbit-var
                                 x args
                                 '((:name . lhbit-var->name)
                                   (:idx . lhbit-var->idx))
                                 'change-lhbit-var
                                 'nil))
