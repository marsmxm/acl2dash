Fixtye of optional blocks.

This is an option type introduced by fty::defoption. Note that defoption is just a wrapper for fty::defflexsum, so there are :none and :some member types, a case macro, and so forth.

Member Types
:none → block-option-none
Represents that no block-option is available, i.e., Nothing or None.
:some → block-option-some
An available block-option, i.e., Just val or Some val.
