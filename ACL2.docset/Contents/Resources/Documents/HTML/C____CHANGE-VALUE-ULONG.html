Modifying constructor for value-ulong structures.

Syntax
(change-value-ulong x 
                    [:get <get>])

This is an often useful alternative to make-value-ulong.

We construct a new value-ulong structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-value-ulong

(defmacro change-value-ulong (x &rest args)
          (std::change-aggregate 'value-ulong
                                 x args '((:get . value-ulong->get))
                                 'change-value-ulong
                                 'nil))
