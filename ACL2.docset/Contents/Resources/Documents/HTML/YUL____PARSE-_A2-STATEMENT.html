Eats as many statements as possible (zero or more).

Signature
(parse-*-statement tokens) 
  → 
(mv result-asts tokens-after-statements)
Arguments
tokens — Guard (abnf::tree-listp tokens).
Returns
result-asts — Type (statement-listp result-asts).
tokens-after-statements — Type (abnf::tree-listp tokens-after-statements).

In Yul, there is no statement separator. There is enough syntax on each statement rule to make it reasonably easy to disambiguate the statements.
