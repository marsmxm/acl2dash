Modifying constructor for vl-dostmt structures.

Syntax
(change-vl-dostmt x 
                  [:body <body>] 
                  [:condition <condition>] 
                  [:atts <atts>])

This is an often useful alternative to make-vl-dostmt.

We construct a new vl-dostmt structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-dostmt

(defmacro
     change-vl-dostmt (x &rest args)
     (std::change-aggregate 'vl-dostmt
                            x args
                            '((:body . vl-dostmt->body)
                              (:condition . vl-dostmt->condition)
                              (:atts . vl-dostmt->atts))
                            'change-vl-dostmt
                            'remake-vl-dostmt))
