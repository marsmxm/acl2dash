Modifying constructor for iconst-length-llong structures.

Syntax
(change-iconst-length-llong x 
                            )

This is an often useful alternative to make-iconst-length-llong.

We construct a new iconst-length-llong structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
