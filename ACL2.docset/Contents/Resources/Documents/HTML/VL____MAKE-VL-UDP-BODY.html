Constructor macro for vl-udp-body-p structures.

Syntax:

(make-vl-udp-body [:init <init>] 
                  [:table <table>])

This is our preferred way to construct vl-udp-body-p structures. It simply conses together a structure with the specified fields.

This macro generates a new vl-udp-body-p structure from scratch. See also change-vl-udp-body, which can "change" an existing structure, instead.

The vl-udp-body-p structures we create here are just constructed with ordinary cons. If you want to create honsed structures, see make-honsed-vl-udp-body instead.

Definition

This is an ordinary make- macro introduced by defaggregate.

Macro: make-vl-udp-body

(defmacro make-vl-udp-body (&rest args)
          (std::make-aggregate 'vl-udp-body
                               args '((:init) (:table))
                               'make-vl-udp-body
                               nil))
