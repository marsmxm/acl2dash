(vl-echarlist-fix x) is a usual ACL2::fty list fixing function.

Signature
(vl-echarlist-fix x) → fty::newx
Arguments
x — Guard (vl-echarlist-p x).
Returns
fty::newx — Type (vl-echarlist-p fty::newx).

In the logic, we apply vl-echar-fix to each member of the x. In the execution, none of that is actually necessary and this is just an inlined identity function.
