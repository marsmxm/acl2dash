A copying macro that lets you create new hyp-tuple-p structures, based on existing structures.

Syntax:

(change-hyp-tuple x 
                  [:name <name>] 
                  [:term <term>] 
                  [:alist <alist>])

This is a sometimes useful alternative to make-hyp-tuple. It constructs a new hyp-tuple-p structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by std::defaggregate.

Macro: change-hyp-tuple

(defmacro change-hyp-tuple (x &rest args)
          (std::change-aggregate 'hyp-tuple
                                 x args
                                 '((:name . hyp-tuple->name)
                                   (:term . hyp-tuple->term)
                                   (:alist . hyp-tuple->alist))
                                 'change-hyp-tuple
                                 'nil))
