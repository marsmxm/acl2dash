Modifying constructor for fun-declor-pointer structures.

Syntax
(change-fun-declor-pointer x 
                           [:decl <decl>])

This is an often useful alternative to make-fun-declor-pointer.

We construct a new fun-declor-pointer structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-fun-declor-pointer

(defmacro
     change-fun-declor-pointer (x &rest args)
     (std::change-aggregate 'fun-declor-pointer
                            x args
                            '((:decl . fun-declor-pointer->decl))
                            'change-fun-declor-pointer
                            'nil))
