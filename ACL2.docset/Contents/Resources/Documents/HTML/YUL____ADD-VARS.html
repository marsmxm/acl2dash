Add (a list of) variables to a variable table.

Signature
(add-vars vars vartab) → vartab?
Arguments
vars — Guard (identifier-listp vars).
vartab — Guard (vartablep vartab).
Returns
vartab? — Type (vartable-resultp vartab?).

The variables are added, one after the other. Duplicates in the list will cause an error.

This lifts add-var to lists.

Definitions and Theorems

Function: add-vars

(defun add-vars (vars vartab)
       (declare (xargs :guard (and (identifier-listp vars)
                                   (vartablep vartab))))
       (let ((__function__ 'add-vars))
            (declare (ignorable __function__))
            (b* (((when (endp vars))
                  (vartable-fix vartab))
                 ((ok vartab)
                  (add-var (car vars) vartab)))
                (add-vars (cdr vars) vartab))))

(defthm vartable-resultp-of-add-vars
        (b* ((vartab? (add-vars vars vartab)))
            (vartable-resultp vartab?))
        :rule-classes :rewrite)
