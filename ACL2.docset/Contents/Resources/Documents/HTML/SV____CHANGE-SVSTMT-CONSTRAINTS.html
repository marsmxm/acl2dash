Modifying constructor for svstmt-constraints structures.

Syntax
(change-svstmt-constraints x 
                           [:constraints <constraints>])

This is an often useful alternative to make-svstmt-constraints.

We construct a new svstmt-constraints structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-svstmt-constraints

(defmacro change-svstmt-constraints (x &rest args)
          (std::change-aggregate
               'svstmt-constraints
               x args
               '((:constraints . svstmt-constraints->constraints))
               'change-svstmt-constraints
               'nil))
