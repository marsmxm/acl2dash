Modifying constructor for vl-scopecontext-interface structures.

Syntax
(change-vl-scopecontext-interface x 
                                  [:iface <iface>])

This is an often useful alternative to make-vl-scopecontext-interface.

We construct a new vl-scopecontext-interface structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-scopecontext-interface

(defmacro change-vl-scopecontext-interface
          (x &rest args)
          (std::change-aggregate
               'vl-scopecontext-interface
               x args
               '((:iface . vl-scopecontext-interface->iface))
               'change-vl-scopecontext-interface
               'remake-vl-scopecontext-interface))
