Basic constructor macro for vl-elabscope structures.

Syntax
(make-vl-elabscope [:subscopes <subscopes>] 
                   [:members <members>])

This is the usual way to construct vl-elabscope structures. It simply conses together a structure with the specified fields.

This macro generates a new vl-elabscope structure from scratch. See also change-vl-elabscope, which can "change" an existing structure, instead.
