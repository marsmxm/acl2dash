Rewrite a list of contextual assumptions

See rewrite$ for relevant background. The macro rewrite$-context is similar to rewrite$-hyps, but it takes a rewrite$-record (``rrec'') as an input to provide any forward-chaining and linear-arithmetic assumptions.
