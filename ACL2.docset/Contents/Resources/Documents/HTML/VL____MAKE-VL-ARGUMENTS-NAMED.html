Basic constructor macro for vl-arguments-named structures.

Syntax
(make-vl-arguments-named [:args <args>] 
                         [:starp <starp>])

This is the usual way to construct vl-arguments-named structures. It simply conses together a structure with the specified fields.

This macro generates a new vl-arguments-named structure from scratch. See also change-vl-arguments-named, which can "change" an existing structure, instead.
