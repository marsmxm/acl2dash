Build a Java long literal in base 10 without underscores.

Signature
(jliteral-long-dec-nouscores value) → lit
Arguments
value — Guard (natp value).
Returns
lit — Type (jliteralp lit).

This is an integer decimal literal with the type suffix L. We use the uppercase type suffix, and not the lowercase one, as recommended in [JLS:3.10.1].
