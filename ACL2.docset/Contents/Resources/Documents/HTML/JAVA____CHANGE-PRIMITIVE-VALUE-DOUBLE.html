Modifying constructor for primitive-value-double structures.

Syntax
(change-primitive-value-double x 
                               [:get <get>])

This is an often useful alternative to make-primitive-value-double.

We construct a new primitive-value-double structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-primitive-value-double

(defmacro
     change-primitive-value-double
     (x &rest args)
     (std::change-aggregate 'primitive-value-double
                            x args
                            '((:get . primitive-value-double->get))
                            'change-primitive-value-double
                            'nil))
