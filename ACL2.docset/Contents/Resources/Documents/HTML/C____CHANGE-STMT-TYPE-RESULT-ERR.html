Modifying constructor for stmt-type-result-err structures.

Syntax
(change-stmt-type-result-err x 
                             [:get <get>])

This is an often useful alternative to make-stmt-type-result-err.

We construct a new stmt-type-result-err structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
