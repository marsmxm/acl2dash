Modifying constructor for vl-arrayrange-plusminus structures.

Syntax
(change-vl-plusminus->arrayrange x 
                                 [:plusminus <plusminus>])

This is an often useful alternative to make-vl-plusminus->arrayrange.

We construct a new vl-arrayrange-plusminus structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-plusminus->arrayrange

(defmacro change-vl-plusminus->arrayrange
          (x &rest args)

               'change-vl-plusminus->arrayrange
