Basic constructor macro for maybe-4vec-some structures.

Syntax
(make-maybe-4vec-some [:val <val>])

This is the usual way to construct maybe-4vec-some structures. It simply conses together a structure with the specified fields.

This macro generates a new maybe-4vec-some structure from scratch. See also change-maybe-4vec-some, which can "change" an existing structure, instead.
