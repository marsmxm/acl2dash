Modifying constructor for alias structures.

Syntax
(change-alias x 
              [:name <name>] 
              [:val <val>])

This is an often useful alternative to make-alias.

We construct a new alias structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-alias

(defmacro change-alias (x &rest args)
          (std::change-aggregate 'alias
                                 x args
                                 '((:name . alias->name)
                                   (:val . alias->val))
                                 'change-alias
                                 'nil))
