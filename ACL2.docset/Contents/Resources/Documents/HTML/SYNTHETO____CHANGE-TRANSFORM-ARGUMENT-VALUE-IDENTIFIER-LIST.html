Modifying constructor for transform-argument-value-identifier-list structures.

Syntax
(change-transform-argument-value-identifier-list x 
                                                 [:names <names>])

This is an often useful alternative to make-transform-argument-value-identifier-list.

We construct a new transform-argument-value-identifier-list structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
