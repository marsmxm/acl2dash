Modifying constructor for outcome-specification-success structures.

Syntax
(change-outcome-specification-success x 
                                      [:message <message>])

This is an often useful alternative to make-outcome-specification-success.

We construct a new outcome-specification-success structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-outcome-specification-success

(defmacro change-outcome-specification-success
