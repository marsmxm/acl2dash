Modifying constructor for init-type-list structures.

Syntax
(change-init-type-list x 
                       [:get <get>])

This is an often useful alternative to make-init-type-list.

We construct a new init-type-list structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-init-type-list

(defmacro
     change-init-type-list (x &rest args)
     (std::change-aggregate 'init-type-list
                            x args '((:get . init-type-list->get))
                            'change-init-type-list
                            'nil))
