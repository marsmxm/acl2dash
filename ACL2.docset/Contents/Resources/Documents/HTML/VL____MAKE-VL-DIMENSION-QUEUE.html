Basic constructor macro for vl-dimension-queue structures.

Syntax
(make-vl-dimension-queue [:maxsize <maxsize>])

This is the usual way to construct vl-dimension-queue structures. It simply conses together a structure with the specified fields.

This macro generates a new vl-dimension-queue structure from scratch. See also change-vl-dimension-queue, which can "change" an existing structure, instead.
