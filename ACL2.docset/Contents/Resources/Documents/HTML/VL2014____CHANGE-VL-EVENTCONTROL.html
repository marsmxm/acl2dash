Modifying constructor for vl-eventcontrol structures.

Syntax
(change-vl-eventcontrol x 
                        [:starp <starp>] 
                        [:atoms <atoms>])

This is an often useful alternative to make-vl-eventcontrol.

We construct a new vl-eventcontrol structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-eventcontrol

(defmacro
     change-vl-eventcontrol (x &rest args)
     (std::change-aggregate 'vl-eventcontrol
                            x args
                            '((:starp . vl-eventcontrol->starp)
                              (:atoms . vl-eventcontrol->atoms))
                            'change-vl-eventcontrol
                            'remake-vl-eventcontrol))
