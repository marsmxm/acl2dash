Modifying constructor for expr-grade-postfix structures.

Syntax
(change-expr-grade-postfix x 
                           )

This is an often useful alternative to make-expr-grade-postfix.

We construct a new expr-grade-postfix structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

This is an ordinary change- macro introduced by fty::defprod.
