Implements svex-argmasks for ZEROX.

Signature
(svmask-for-zerox mask args)  argmasks
Arguments
mask Care mask for the full expression.
Guard (4vmask-p mask).
args Arguments to this ZEROX operator.
Guard (svexlist-p args).
Returns
argmasks The new care masks inferred for the args.
Type (4vmasklist-p argmasks).

We are considering a (zerox n x) expression and we know that we only care about the bits mentioned in mask. We need to figure out which bits of n and x we care about.

For n. If we care about ANY bit of (zerox n x), then we care about all of n. But if we don't care about any bit of (zerox n x), then n is completely irrelevant.

For x. The result of (zerox n x) is just the low n bits of x. So when n is a constant, we can reduce the outer mask by chopping off any bits beyond position n. Otherwise, we don't know how

Function: svmask-for-zerox

(defun svmask-for-zerox (mask args)
       (declare (xargs :guard (and (4vmask-p mask)
                                   (svexlist-p args))))
       (let ((__function__ 'svmask-for-zerox))
            (declare (ignorable __function__))
            (b* (((svex-nths n x) args)
                 (mask (4vmask-fix mask)))
                (b* ((nmask (4vmask-all-or-none mask))
                     (nval (svex-s4xeval n))
                     ((unless (s4vec-2vec-p nval))
                      (list nmask mask))
                     (nv (s4vec->upper nval))
                     ((when (sparseint-< nv 0))
                      (list nmask 0)))
                    (list nmask
                          (sparseint-concatenate (sparseint-val nv)
                                                 mask 0))))))
