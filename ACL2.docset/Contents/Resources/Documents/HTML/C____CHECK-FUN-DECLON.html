Check a function declaration.

Signature
(check-fun-declon declon tagenv) → wf
Arguments
declon — Guard (fun-declonp declon).
tagenv — Guard (tag-envp tagenv).
Returns
wf — Type (wellformed-resultp wf).

We check the type specifier sequence and the declarator. We do not return anything, because a function declaration, unlike a function definition, does not have a body to check (for which we need the variable table that contains the parameters).

Definitions and Theorems

Function: check-fun-declon

(defun check-fun-declon (declon tagenv)
       (declare (xargs :guard (and (fun-declonp declon)
                                   (tag-envp tagenv))))
       (let ((__function__ 'check-fun-declon))
            (declare (ignorable __function__))
            (b* (((fun-declon declon) declon)
                 (wf (check-tyspecseq declon.tyspec tagenv))
                 ((when (errorp wf)) wf)
                 (vartab (check-fun-declor declon.declor tagenv))
                 ((when (errorp vartab)) vartab))
                :wellformed)))
