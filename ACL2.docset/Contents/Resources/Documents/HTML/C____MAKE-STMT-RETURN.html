Basic constructor macro for stmt-return structures.

Syntax
(make-stmt-return [:value <value>])

This is the usual way to construct stmt-return structures. It simply conses together a structure with the specified fields.

This macro generates a new stmt-return structure from scratch. See also change-stmt-return, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by fty::defprod.

Macro: make-stmt-return

(defmacro make-stmt-return (&rest args)
          (std::make-aggregate 'stmt-return
                               args '((:value))
                               'make-stmt-return
                               nil))
