Events to generate the ACL2 models of the C integer operations that involve an integer type as first operand and each of a list of integer types as second operand.

Signature
(atc-def-integer-operations-2-loop-inner type types) → events
Arguments
type — Guard (typep type).
types — Guard (type-listp types).
Returns
events — Type (pseudo-event-form-listp events).

We skip the event if the two types are equal, because that case is already covered by atc-def-integer-operations-2-loop-same.

This is the inner loop for generating operations for all combinations of different operand types.
