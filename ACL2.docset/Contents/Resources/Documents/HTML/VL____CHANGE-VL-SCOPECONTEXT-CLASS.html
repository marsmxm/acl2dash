Modifying constructor for vl-scopecontext-class structures.

Syntax
(change-vl-scopecontext-class x 
                              [:class <class>])

This is an often useful alternative to make-vl-scopecontext-class.

We construct a new vl-scopecontext-class structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.
