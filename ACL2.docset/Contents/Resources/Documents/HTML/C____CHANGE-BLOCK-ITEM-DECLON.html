Modifying constructor for block-item-declon structures.

Syntax
(change-block-item-declon x 
                          [:get <get>])

This is an often useful alternative to make-block-item-declon.

We construct a new block-item-declon structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-block-item-declon

(defmacro
     change-block-item-declon (x &rest args)
     (std::change-aggregate 'block-item-declon
                            x
                            args '((:get . block-item-declon->get))
                            'change-block-item-declon
                            'nil))
