Basic constructor macro for g-var structures.

Syntax
(make-g-var [:name <name>])

This is the usual way to construct g-var structures. It simply conses together a structure with the specified fields.

This macro generates a new g-var structure from scratch. See also change-g-var, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by defprod.

Macro: make-g-var

(defmacro make-g-var (&rest args)
          (std::make-aggregate 'g-var
                               args '((:name))
                               'make-g-var
                               nil))
