Basic constructor macro for vl-bind structures.

Syntax
(make-vl-bind [:scope <scope>] 
              [:addto <addto>] 
              [:modinsts <modinsts>] 
              [:loc <loc>] 
              [:atts <atts>])

This is the usual way to construct vl-bind structures. It simply conses together a structure with the specified fields.

This macro generates a new vl-bind structure from scratch. See also change-vl-bind, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by defprod.

Macro: make-vl-bind

(defmacro make-vl-bind (&rest args)

Function: vl-bind

(defun
 vl-bind (scope addto modinsts loc atts)
 (declare (xargs :guard (and (maybe-stringp scope)
                             (vl-exprlist-p addto)
                             (vl-modinstlist-p modinsts)
                             (vl-location-p loc)
                             (vl-atts-p atts))))
 (declare (xargs :guard t))
 (let
  ((__function__ 'vl-bind))
  (declare (ignorable __function__))
  (b*
   ((scope (mbe :logic (maybe-string-fix scope)
                :exec scope))
    (addto (mbe :logic (vl-exprlist-fix addto)
                :exec addto))
    (modinsts (mbe :logic (vl-modinstlist-fix modinsts)
                   :exec modinsts))
    (loc (mbe :logic (vl-location-fix loc)
              :exec loc))
    (atts (mbe :logic (vl-atts-fix atts)
               :exec atts)))
   (cons
       :vl-bind
       (std::prod-cons
            (std::prod-cons scope addto)
            (std::prod-cons modinsts (std::prod-cons loc atts)))))))
