Basic constructor macro for function-recursion structures.

Syntax
(make-function-recursion [:definitions <definitions>])

This is the usual way to construct function-recursion structures. It simply conses together a structure with the specified fields.

This macro generates a new function-recursion structure from scratch. See also change-function-recursion, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by fty::defprod.

Macro: make-function-recursion

(defmacro make-function-recursion (&rest args)
          (std::make-aggregate 'function-recursion
                               args '((:definitions))
                               'make-function-recursion
                               nil))
