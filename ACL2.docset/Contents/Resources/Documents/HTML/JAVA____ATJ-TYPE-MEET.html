Greatest lower bound of two ATJ types.

Signature
(atj-type-meet x y) → glb
Arguments
x — Guard (atj-typep x).
y — Guard (atj-typep y).
Returns
glb — Type (atj-maybe-typep glb).

We have defined atj-type-bottom-meet in order to exhibit and prove the semilattice structure, but we always want to use ATJ types as arguments, never nil. So we introduce this function, which operates on types but may return nil, which can be also interpreted as saying that the two ATJ types have no (greatest) lower bound w.r.t atj-type-<=.
