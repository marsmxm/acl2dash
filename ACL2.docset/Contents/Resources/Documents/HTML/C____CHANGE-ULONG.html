Modifying constructor for ulong structures.

Syntax
(change-ulong x 
              [:get <get>])

This is an often useful alternative to make-ulong.

We construct a new ulong structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-ulong

(defmacro change-ulong (x &rest args)
          (std::change-aggregate 'ulong
                                 x args '((:get . ulong->get))
                                 'change-ulong
                                 'nil))
