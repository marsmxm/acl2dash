Modifying constructor for tterm-call structures.

Syntax
(change-tterm-call x 
                   [:function <function>] 
                   [:arguments <arguments>])

This is an often useful alternative to make-tterm-call.

We construct a new tterm-call structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-tterm-call

(defmacro
     change-tterm-call (x &rest args)
     (std::change-aggregate 'tterm-call
                            x args
                            '((:function . tterm-call->function)
                              (:arguments . tterm-call->arguments))
                            'change-tterm-call
                            'nil))
