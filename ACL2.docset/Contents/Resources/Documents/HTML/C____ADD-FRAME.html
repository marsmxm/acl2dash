Add a frame to a canonical representation of computation states.

Signature
(add-frame fun compst) → new-compst
Arguments
fun — Guard (identp fun).
compst — Guard (compustatep compst).
Returns

This adds a frame with an empty scope.

Definitions and Theorems

Function: add-frame

(defun add-frame (fun compst)
       (declare (xargs :guard (and (identp fun)
                                   (compustatep compst))))
       (let ((__function__ 'add-frame))
            (declare (ignorable __function__))
            (push-frame (make-frame :function fun
                                    :scopes (list nil))
                        compst)))
