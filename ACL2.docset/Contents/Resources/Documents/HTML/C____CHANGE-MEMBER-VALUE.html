Modifying constructor for member-value structures.

Syntax
(change-member-value x 
                     [:name <name>] 
                     [:value <value>])

This is an often useful alternative to make-member-value.

We construct a new member-value structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-member-value

(defmacro change-member-value (x &rest args)
          (std::change-aggregate 'member-value
                                 x args
                                 '((:name . member-value->name)
                                   (:value . member-value->value))
                                 'change-member-value
                                 'nil))
