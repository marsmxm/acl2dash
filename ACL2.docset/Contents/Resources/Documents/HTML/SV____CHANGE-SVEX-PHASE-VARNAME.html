Modifying constructor for svex-phase-varname structures.

Syntax
(change-svex-phase-varname x 
                           [:name <name>] 
                           [:phase <phase>])

This is an often useful alternative to make-svex-phase-varname.

We construct a new svex-phase-varname structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-svex-phase-varname

(defmacro
     change-svex-phase-varname (x &rest args)
     (std::change-aggregate 'svex-phase-varname
