Basic constructor macro for uint-array structures.

Syntax
(make-uint-array [:elements <elements>])

This is the usual way to construct uint-array structures. It simply conses together a structure with the specified fields.

This macro generates a new uint-array structure from scratch. See also change-uint-array, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by fty::defprod.

Macro: make-uint-array

(defmacro make-uint-array (&rest args)
          (std::make-aggregate 'uint-array

                               'make-uint-array
                               nil))
