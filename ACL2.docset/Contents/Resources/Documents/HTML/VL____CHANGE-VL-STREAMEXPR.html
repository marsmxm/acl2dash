Modifying constructor for vl-streamexpr structures.

Syntax
(change-vl-streamexpr x 
                      [:expr <expr>] 
                      [:part <part>])

This is an often useful alternative to make-vl-streamexpr.

We construct a new vl-streamexpr structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-streamexpr

(defmacro change-vl-streamexpr (x &rest args)
          (std::change-aggregate 'vl-streamexpr
                                 x args
                                 '((:expr . vl-streamexpr->expr)
                                   (:part . vl-streamexpr->part))
                                 'change-vl-streamexpr
                                 'remake-vl-streamexpr))
