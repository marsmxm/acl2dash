Modifying constructor for jexpr-unary structures.

Syntax
(change-jexpr-unary x 
                    [:op <op>] 
                    [:arg <arg>])

This is an often useful alternative to make-jexpr-unary.

We construct a new jexpr-unary structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-jexpr-unary

(defmacro change-jexpr-unary (x &rest args)
          (std::change-aggregate 'jexpr-unary
                                 x args
                                 '((:op . jexpr-unary->op)
                                   (:arg . jexpr-unary->arg))
                                 'change-jexpr-unary
                                 'nil))
