(vl-simple-id-tail-list-p x) recognizes lists where every element satisfies vl-simple-id-tail-p.

Signature
(vl-simple-id-tail-list-p x)  std::bool

This is an ordinary deflist. It is "loose" in that it does not care whether x is nil-terminated.
