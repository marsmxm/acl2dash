Modifying constructor for maybe-simpcode-none structures.

Syntax
(change-maybe-simpcode-none x 
                            )

This is an often useful alternative to make-maybe-simpcode-none.

We construct a new maybe-simpcode-none structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
