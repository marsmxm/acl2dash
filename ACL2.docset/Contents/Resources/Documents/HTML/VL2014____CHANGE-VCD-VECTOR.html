A copying macro that lets you create new vcd-vector-p structures, based on existing structures.

Syntax:

(change-vcd-vector x 
                   [:bits <bits>] 
                   [:idcode <idcode>])

This is a sometimes useful alternative to make-vcd-vector. It constructs a new vcd-vector-p structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defaggregate.

Macro: change-vcd-vector

(defmacro change-vcd-vector (x &rest args)
          (std::change-aggregate 'vcd-vector
                                 x args
                                 '((:bits . vcd-vector->bits)
                                   (:idcode . vcd-vector->idcode))
                                 'change-vcd-vector
                                 'remake-vcd-vector))
