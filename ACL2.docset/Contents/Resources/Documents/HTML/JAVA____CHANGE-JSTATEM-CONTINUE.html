Modifying constructor for jstatem-continue structures.

Syntax
(change-jstatem-continue x 
                         )

This is an often useful alternative to make-jstatem-continue.

We construct a new jstatem-continue structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
