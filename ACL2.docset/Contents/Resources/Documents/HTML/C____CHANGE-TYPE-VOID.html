Modifying constructor for type-void structures.

Syntax
(change-type-void x 
                  )

This is an often useful alternative to make-type-void.

We construct a new type-void structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-type-void

(defmacro change-type-void (x &rest args)
          (std::change-aggregate 'type-void
                                 x args 'nil
                                 'change-type-void
                                 'nil))
