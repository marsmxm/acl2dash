Modifying constructor for literal-option-none structures.

Syntax
(change-literal-option-none x 
                            )

This is an often useful alternative to make-literal-option-none.

We construct a new literal-option-none structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-literal-option-none

(defmacro change-literal-option-none
          (x &rest args)
          (std::change-aggregate 'literal-option-none

                                 'nil))
