Raw constructor for honsed wcp-lit-actions-p structures.

Syntax:

(honsed-wcp-lit-actions witnesses examples)

This is identical to wcp-lit-actions, except that we hons the structure we are creating.

Definition

This is an ordinary honsing constructor introduced by std::defaggregate.

Function: honsed-wcp-lit-actions

(defun honsed-wcp-lit-actions
       (witnesses examples)

                                   (wcp-example-appsp examples))))
