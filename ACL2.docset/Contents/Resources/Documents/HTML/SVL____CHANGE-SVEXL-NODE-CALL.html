Modifying constructor for svexl-node-call structures.

Syntax
(change-svexl-node-call x 
                        [:fn <fn>] 
                        [:args <args>])

This is an often useful alternative to make-svexl-node-call.

We construct a new svexl-node-call structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-svexl-node-call

(defmacro change-svexl-node-call (x &rest args)
          (std::change-aggregate 'svexl-node-call
                                 x args
                                 '((:fn . svexl-node-call->fn)
                                   (:args . svexl-node-call->args))
                                 'change-svexl-node-call
                                 'nil))
