Modifying constructor for iconst-length-long structures.

Syntax
(change-iconst-length-long x 
                           )

This is an often useful alternative to make-iconst-length-long.

We construct a new iconst-length-long structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
