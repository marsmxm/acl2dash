Basic constructor macro for fgl-rune-definition structures.

Syntax
(make-fgl-rune-definition [:name <name>])

This is the usual way to construct fgl-rune-definition structures. It simply conses together a structure with the specified fields.

This macro generates a new fgl-rune-definition structure from scratch. See also change-fgl-rune-definition, which can "change" an existing structure, instead.
