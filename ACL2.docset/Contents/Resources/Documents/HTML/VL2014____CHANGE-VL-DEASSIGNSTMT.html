Modifying constructor for vl-deassignstmt structures.

Syntax
(change-vl-deassignstmt x 
                        [:type <type>] 
                        [:lvalue <lvalue>] 
                        [:atts <atts>])

This is an often useful alternative to make-vl-deassignstmt.

We construct a new vl-deassignstmt structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-deassignstmt

(defmacro
     change-vl-deassignstmt (x &rest args)
     (std::change-aggregate 'vl-deassignstmt
                            x args
                            '((:type . vl-deassignstmt->type)
                              (:lvalue . vl-deassignstmt->lvalue)
                              (:atts . vl-deassignstmt->atts))
                            'change-vl-deassignstmt
                            'remake-vl-deassignstmt))
