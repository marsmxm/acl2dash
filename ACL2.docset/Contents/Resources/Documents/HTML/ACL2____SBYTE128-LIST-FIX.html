(sbyte128-list-fix x) is a usual fty list fixing function.

Signature
(sbyte128-list-fix x) → fty::newx
Arguments
x — Guard (sbyte128-listp x).
Returns
fty::newx — Type (sbyte128-listp fty::newx).

In the logic, we apply sbyte128-fix to each member of the x. In the execution, none of that is actually necessary and this is just an inlined identity function.
