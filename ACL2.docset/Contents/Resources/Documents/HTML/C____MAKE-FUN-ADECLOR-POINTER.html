Basic constructor macro for fun-adeclor-pointer structures.

Syntax
(make-fun-adeclor-pointer [:decl <decl>])

This is the usual way to construct fun-adeclor-pointer structures. It simply conses together a structure with the specified fields.

This macro generates a new fun-adeclor-pointer structure from scratch. See also change-fun-adeclor-pointer, which can "change" an existing structure, instead.
