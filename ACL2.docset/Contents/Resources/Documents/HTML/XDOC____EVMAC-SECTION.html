Construct a section of the user documentation of an event macro.

This provides more abstraction than using a specific heading.

Macro: evmac-section

(defmacro xdoc::evmac-section
          (title &rest content)
          (cons 'xdoc::&&
                (cons (cons 'xdoc::h3 (cons title 'nil))
                      content)))
