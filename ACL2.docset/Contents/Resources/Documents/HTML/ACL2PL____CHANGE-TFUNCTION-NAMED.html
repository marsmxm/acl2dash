Modifying constructor for tfunction-named structures.

Syntax
(change-tfunction-named x 
                        [:name <name>])

This is an often useful alternative to make-tfunction-named.

We construct a new tfunction-named structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-tfunction-named

(defmacro
     change-tfunction-named (x &rest args)
     (std::change-aggregate 'tfunction-named
                            x
                            args '((:name . tfunction-named->name))
                            'change-tfunction-named
                            'nil))
