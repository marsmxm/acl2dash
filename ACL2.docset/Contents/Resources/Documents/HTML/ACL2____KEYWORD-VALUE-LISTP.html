Recognizer for true lists whose even-position elements are keywords

(Keyword-value-listp l) is true if and only if l is a list of even length of the form (k1 a1 k2 a2 ... kn an), where each ki is a keyword. To list the keys ki and values ai of l evaluate (evens l) and (odds l), respectively.

Function: keyword-value-listp

(defun keyword-value-listp (l)
       (declare (xargs :guard t))
       (cond ((atom l) (null l))
             (t (and (keywordp (car l))
                     (consp (cdr l))
                     (keyword-value-listp (cddr l))))))
