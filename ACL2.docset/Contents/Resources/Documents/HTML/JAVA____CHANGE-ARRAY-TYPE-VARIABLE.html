Modifying constructor for array-type-variable structures.

Syntax
(change-array-type-variable x 
                            [:element <element>] 
                            [:dimensions <dimensions>])

This is an often useful alternative to make-array-type-variable.

We construct a new array-type-variable structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-array-type-variable

(defmacro change-array-type-variable
          (x &rest args)
          (std::change-aggregate
               'array-type-variable
               x args
               '((:element . array-type-variable->element)
                 (:dimensions . array-type-variable->dimensions))
               'change-array-type-variable
               'nil))
