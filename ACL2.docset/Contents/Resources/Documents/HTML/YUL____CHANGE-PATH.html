Modifying constructor for path structures.

Syntax
(change-path x 
             [:get <get>])

This is an often useful alternative to make-path.

We construct a new path structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-path

(defmacro change-path (x &rest args)
          (std::change-aggregate 'path
                                 x args '((:get . path->get))
                                 'change-path
                                 'nil))
