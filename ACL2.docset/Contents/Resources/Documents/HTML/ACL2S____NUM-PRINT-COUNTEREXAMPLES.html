Number of Counterexamples to be printed

Set the number of counterexamples desired to be printed. By default this parameter is set to 3. Can be set to any natural number n. Setting this number to 0 implies the user is not interested in seeing counterexamples, and thus none will be printed in the testing output. 
Usage: 
(acl2s-defaults :set num-print-counterexamples 3) 
(acl2s-defaults :get num-print-counterexamples)
