Modifying constructor for data-value-hex structures.

Syntax
(change-data-value-hex x 
                       [:get <get>])

This is an often useful alternative to make-data-value-hex.

We construct a new data-value-hex structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-data-value-hex

(defmacro
     change-data-value-hex (x &rest args)
     (std::change-aggregate 'data-value-hex
                            x args '((:get . data-value-hex->get))
                            'change-data-value-hex
                            'nil))
