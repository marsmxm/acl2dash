Basic constructor macro for iconst structures.

Syntax
(make-iconst [:value <value>] 
             [:base <base>] 
             [:unsignedp <unsignedp>] 
             [:type <type>])

This is the usual way to construct iconst structures. It simply conses together a structure with the specified fields.

This macro generates a new iconst structure from scratch. See also change-iconst, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by fty::defprod.

Macro: make-iconst

(defmacro
     make-iconst (&rest args)
     (std::make-aggregate 'iconst

                          'make-iconst
                          nil))

Function: iconst

(defun iconst (value base unsignedp type)
       (declare (xargs :guard (and (natp value)
                                   (iconst-basep base)
                                   (booleanp unsignedp)
                                   (iconst-tysuffixp type))))
       (declare (xargs :guard t))
       (let ((__function__ 'iconst))
            (declare (ignorable __function__))
            (b* ((value (mbe :logic (nfix value) :exec value))
                 (base (mbe :logic (iconst-base-fix base)
                            :exec base))
                 (unsignedp (mbe :logic (acl2::bool-fix unsignedp)
                                 :exec unsignedp))
                 (type (mbe :logic (iconst-tysuffix-fix type)
                            :exec type)))
                (cons :iconst (list (cons 'value value)
                                    (cons 'base base)
                                    (cons 'unsignedp unsignedp)
                                    (cons 'type type))))))
