Interpret a string as a octal number.

Signature
(strval8 x) → value?
Arguments
x — Guard (stringp x).
Returns
value? — Type (or (natp value?) (not value?)).

For example, (strval8 "17") is 15. If the string is empty or has any non octal digit characters (0-7), we return nil.

Function: strval8

(defun strval8 (x)
       (declare (xargs :guard (stringp x)))
       (declare (type string x))
       (declare (xargs :split-types t))
       (let ((acl2::__function__ 'strval8))
            (declare (ignorable acl2::__function__))
            (mbe :logic (let ((chars (explode x)))
                             (and (consp chars)
                                  (oct-digit-char-listp chars)
                                  (oct-digit-chars-value chars)))
                 :exec (b* (((the unsigned-byte xl) (length x))
                            ((mv (the unsigned-byte val)
                                 (the unsigned-byte len))
                             (parse-octal-from-string x 0 0 0 xl)))
                           (and (not (eql 0 len))
                                (eql len xl)
                                val)))))
