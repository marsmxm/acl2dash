Basic constructor macro for svexllist structures.

Syntax
(make-svexllist [:top-nodelist <top-nodelist>] 
                [:node-array <node-array>])

This is the usual way to construct svexllist structures. It simply conses together a structure with the specified fields.

This macro generates a new svexllist structure from scratch. See also change-svexllist, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by fty::defprod.
