Basic constructor macro for fun-type structures.

Syntax
(make-fun-type [:inputs <inputs>] 
               [:output <output>])

This is the usual way to construct fun-type structures. It simply conses together a structure with the specified fields.

This macro generates a new fun-type structure from scratch. See also change-fun-type, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by fty::defprod.

Macro: make-fun-type

(defmacro make-fun-type (&rest args)
          (std::make-aggregate 'fun-type
                               args '((:inputs) (:output))
                               'make-fun-type
                               nil))
