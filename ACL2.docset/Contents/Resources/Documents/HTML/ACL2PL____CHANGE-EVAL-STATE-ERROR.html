Modifying constructor for eval-state-error structures.

Syntax
(change-eval-state-error x 
                         )

This is an often useful alternative to make-eval-state-error.

We construct a new eval-state-error structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
