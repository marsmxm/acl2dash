(vl-filemap-fix x) is an ACL2::fty alist fixing function that follows the fix-keys strategy.

Signature
(vl-filemap-fix x) → fty::newx
Arguments
x — Guard (vl-filemap-p x).
Returns
fty::newx — Type (vl-filemap-p fty::newx).

Note that in the execution this is just an inline identity function.
