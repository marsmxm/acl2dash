Constructor macro for honsed vl-opinfo-p structures.

Syntax:

(make-honsed-vl-opinfo [:arity <arity>] 
                       [:text <text>])

This is identical to make-vl-opinfo, except that we hons the structure we are creating.

Definition

This is an ordinary honsing make- macro introduced by defaggregate.

Macro: make-honsed-vl-opinfo

(defmacro make-honsed-vl-opinfo (&rest args)
