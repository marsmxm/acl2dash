Modifying constructor for command-error-wrong-command structures.

Syntax
(change-command-error-wrong-command x 
                                    [:command <command>])

This is an often useful alternative to make-command-error-wrong-command.

We construct a new command-error-wrong-command structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
