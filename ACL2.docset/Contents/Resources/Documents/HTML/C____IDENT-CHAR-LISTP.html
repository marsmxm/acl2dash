Check if a list of ACL2 characters is not empty, consists only of ASCII letters, digits, and underscores, and does not start with a digit.

Signature
(ident-char-listp chs) → yes/no
Arguments
chs — Guard (character-listp chs).
Returns
yes/no — Type (booleanp yes/no).

Sequences of characters satisfying these conditions may be portable ASCII identifiers, provided they are distinct from C keywords.

Definitions and Theorems

Function: ident-char-listp

(defun ident-char-listp (chs)
       (declare (xargs :guard (character-listp chs)))
       (let ((__function__ 'ident-char-listp))
            (declare (ignorable __function__))
            (and (consp chs)
                 (str::letter/digit/uscore-charlist-p chs)
                 (not (str::dec-digit-char-p (car chs))))))

Theorem: booleanp-of-ident-char-listp

(defthm booleanp-of-ident-char-listp
        (b* ((yes/no (ident-char-listp chs)))
            (booleanp yes/no))
