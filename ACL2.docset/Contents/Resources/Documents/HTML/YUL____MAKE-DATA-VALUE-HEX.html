Basic constructor macro for data-value-hex structures.

Syntax
(make-data-value-hex [:get <get>])

This is the usual way to construct data-value-hex structures. It simply conses together a structure with the specified fields.

This macro generates a new data-value-hex structure from scratch. See also change-data-value-hex, which can "change" an existing structure, instead.
