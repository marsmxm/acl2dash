A SystemVerilog union datatype, or an array of unions.

This is a product type, introduced by deftagsum in support of vl-datatype.

Fields
members — vl-structmemberlist
The list of union members.
packedp — booleanp
Roughly: says whether this union is packed or not, but warning! this should normally not be used as it has the same problems as packedp for structs; see vl-struct.
pdims — vl-dimensionlist
Packed dimensions for this union type.
udims — vl-dimensionlist
Unpacked dimensions for the union type. See also vl-struct and the notes about unpacked dimensions there.
signedp — booleanp
Roughly: says whether this union is signed or not, but warning! this should normally not be used as it has the same problems as signedp for structs; see vl-struct.
taggedp — booleanp
Says whether this union is 'tagged' or not.
