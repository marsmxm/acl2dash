Modifying constructor for num-val-range structures.

Syntax
(change-num-val-range x 
                      [:min <min>] 
                      [:max <max>])

This is an often useful alternative to make-num-val-range.

We construct a new num-val-range structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-num-val-range

(defmacro change-num-val-range (x &rest args)
          (std::change-aggregate 'num-val-range
                                 x args
                                 '((:min . num-val-range->min)
                                   (:max . num-val-range->max))
                                 'change-num-val-range
                                 'nil))
