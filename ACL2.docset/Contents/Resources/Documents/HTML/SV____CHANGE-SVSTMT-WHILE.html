Modifying constructor for svstmt-while structures.

Syntax
(change-svstmt-while x 
                     [:cond <cond>] 
                     [:body <body>] 
                     [:next <next>])

This is an often useful alternative to make-svstmt-while.

We construct a new svstmt-while structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-svstmt-while

(defmacro change-svstmt-while (x &rest args)
          (std::change-aggregate 'svstmt-while
                                 x args
                                 '((:cond . svstmt-while->cond)
                                   (:body . svstmt-while->body)
                                   (:next . svstmt-while->next))
                                 'change-svstmt-while
                                 'nil))
