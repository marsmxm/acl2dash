Basic constructor macro for vl-plusminus structures.

Syntax
(make-vl-plusminus [:base <base>] 
                   [:width <width>] 
                   [:minusp <minusp>])

This is the usual way to construct vl-plusminus structures. It simply conses together a structure with the specified fields.

This macro generates a new vl-plusminus structure from scratch. See also change-vl-plusminus, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by defprod.
