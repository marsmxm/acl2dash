Modifying constructor for type-struct structures.

Syntax
(change-type-struct x 
                    [:tag <tag>])

This is an often useful alternative to make-type-struct.

We construct a new type-struct structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-type-struct

(defmacro change-type-struct (x &rest args)
          (std::change-aggregate 'type-struct
                                 x args '((:tag . type-struct->tag))
                                 'change-type-struct
                                 'nil))
