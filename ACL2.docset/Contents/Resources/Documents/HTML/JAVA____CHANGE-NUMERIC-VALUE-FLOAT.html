Modifying constructor for numeric-value-float structures.

Syntax
(change-numeric-value-float x 
                            [:get <get>])

This is an often useful alternative to make-numeric-value-float.

We construct a new numeric-value-float structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-numeric-value-float

(defmacro
     change-numeric-value-float
     (x &rest args)
     (std::change-aggregate 'numeric-value-float
                            x args
                            '((:get . numeric-value-float->get))
                            'change-numeric-value-float
                            'nil))
