Modifying constructor for statement-break structures.

Syntax
(change-statement-break x 
                        )

This is an often useful alternative to make-statement-break.

We construct a new statement-break structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-statement-break

(defmacro change-statement-break (x &rest args)
          (std::change-aggregate 'statement-break
                                 x args 'nil

                                 'nil))
