Integer negation of a 4vec.

Signature
(4vec-uminus x)  -x
Arguments
x Guard (4vec-p x).
Returns
-x Type (4vec-p -x).

If the input has X or Z bits, the result is all X bits. Otherwise, we return the signed negation of the input.
