Basic constructor macro for aexp-var structures.

Syntax
(make-aexp-var [:name <name>])

This is the usual way to construct aexp-var structures. It simply conses together a structure with the specified fields.

This macro generates a new aexp-var structure from scratch. See also change-aexp-var, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by fty::defprod.

Macro: make-aexp-var

(defmacro make-aexp-var (&rest args)
