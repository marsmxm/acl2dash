Modifying constructor for g-boolean structures.

Syntax
(change-g-boolean x 
                  [:bool <bool>])

This is an often useful alternative to make-g-boolean.

We construct a new g-boolean structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-g-boolean

(defmacro change-g-boolean (x &rest args)
          (std::change-aggregate 'g-boolean
                                 x args '((:bool . g-boolean->bool))
                                 'change-g-boolean
                                 'nil))
