Modifying constructor for bip32-ext-pub-key structures.

Syntax
(change-bip32-ext-pub-key x 
                          [:key <key>] 
                          [:chain-code <chain-code>])

This is an often useful alternative to make-bip32-ext-pub-key.

We construct a new bip32-ext-pub-key structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-bip32-ext-pub-key

(defmacro change-bip32-ext-pub-key (x &rest args)
          (std::change-aggregate
               'bip32-ext-pub-key
               x args
               '((:key . bip32-ext-pub-key->key)
                 (:chain-code . bip32-ext-pub-key->chain-code))
               'change-bip32-ext-pub-key
               'nil))
