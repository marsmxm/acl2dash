Modifying constructor for vl-hidpiece structures.

Syntax
(change-vl-hidpiece x 
                    [:name <name>])

This is an often useful alternative to make-vl-hidpiece.

We construct a new vl-hidpiece structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-hidpiece

(defmacro
     change-vl-hidpiece (x &rest args)
     (std::change-aggregate 'vl-hidpiece
                            x args '((:name . vl-hidpiece->name))
                            'change-vl-hidpiece
                            'remake-vl-hidpiece))
