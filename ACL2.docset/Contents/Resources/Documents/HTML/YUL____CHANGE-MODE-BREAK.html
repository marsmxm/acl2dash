Modifying constructor for mode-break structures.

Syntax
(change-mode-break x 
                   )

This is an often useful alternative to make-mode-break.

We construct a new mode-break structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-mode-break

(defmacro change-mode-break (x &rest args)
          (std::change-aggregate 'mode-break
                                 x args 'nil
                                 'change-mode-break
                                 'nil))
