Modifying constructor for vl-casttype-const structures.

Syntax
(change-vl-casttype-const x 
                          )

This is an often useful alternative to make-vl-casttype-const.

We construct a new vl-casttype-const structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.
