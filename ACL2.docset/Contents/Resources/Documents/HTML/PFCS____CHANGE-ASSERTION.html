Modifying constructor for assertion structures.

Syntax
(change-assertion x 
                  [:asg <asg>] 
                  [:constr <constr>])

This is an often useful alternative to make-assertion.

We construct a new assertion structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-assertion

(defmacro change-assertion (x &rest args)
          (std::change-aggregate 'assertion
                                 x args
                                 '((:asg . assertion->asg)
                                   (:constr . assertion->constr))
                                 'change-assertion
                                 'nil))
