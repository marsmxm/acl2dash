Modifying constructor for stmt-compound structures.

Syntax
(change-stmt-compound x 
                      [:items <items>])

This is an often useful alternative to make-stmt-compound.

We construct a new stmt-compound structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-stmt-compound

(defmacro
     change-stmt-compound (x &rest args)
     (std::change-aggregate 'stmt-compound
                            x
                            args '((:items . stmt-compound->items))
                            'change-stmt-compound
                            'nil))
