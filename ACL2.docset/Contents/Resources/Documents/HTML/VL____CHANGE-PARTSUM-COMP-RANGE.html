Modifying constructor for partsum-comp-range structures.

Syntax
(change-partsum-comp-range x 
                           [:msb <msb>] 
                           [:lsb <lsb>])

This is an often useful alternative to make-partsum-comp-range.

We construct a new partsum-comp-range structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-partsum-comp-range

(defmacro change-partsum-comp-range (x &rest args)
          (std::change-aggregate 'partsum-comp-range
                                 x args
                                 '((:msb . partsum-comp-range->msb)
                                   (:lsb . partsum-comp-range->lsb))
                                 'change-partsum-comp-range
                                 'nil))
