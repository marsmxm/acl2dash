Modifying constructor for member structures.

Syntax
(change-member x 
               [:name <name>] 
               [:value <value>])

This is an often useful alternative to make-member.

We construct a new member structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-member

(defmacro change-member (x &rest args)
          (std::change-aggregate 'member
                                 x args
                                 '((:name . member->name)
                                   (:value . member->value))
                                 'change-member
                                 'nil))
