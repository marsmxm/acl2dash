Modifying constructor for sint structures.

Syntax
(change-sint x 
             [:get <get>])

This is an often useful alternative to make-sint.

We construct a new sint structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-sint

(defmacro change-sint (x &rest args)
          (std::change-aggregate 'sint
                                 x args '((:get . sint->get))
                                 'change-sint
                                 'nil))
