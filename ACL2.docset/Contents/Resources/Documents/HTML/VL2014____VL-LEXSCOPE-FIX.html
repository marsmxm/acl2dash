(vl-lexscope-fix x) is an ACL2::fty alist fixing function that follows the fix-keys strategy.

Signature
(vl-lexscope-fix x) → fty::newx
Arguments
x — Guard (vl-lexscope-p x).
Returns
fty::newx — Type (vl-lexscope-p fty::newx).

Note that in the execution this is just an inline identity function.
