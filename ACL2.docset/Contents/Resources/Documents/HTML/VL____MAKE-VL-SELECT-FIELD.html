Basic constructor macro for vl-select-field structures.

Syntax
(make-vl-select-field [:name <name>])

This is the usual way to construct vl-select-field structures. It simply conses together a structure with the specified fields.

This macro generates a new vl-select-field structure from scratch. See also change-vl-select-field, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by defprod.
