Modifying constructor for jexpr-cast structures.

Syntax
(change-jexpr-cast x 
                   [:type <type>] 
                   [:arg <arg>])

This is an often useful alternative to make-jexpr-cast.

We construct a new jexpr-cast structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-jexpr-cast

(defmacro change-jexpr-cast (x &rest args)
          (std::change-aggregate 'jexpr-cast
                                 x args
                                 '((:type . jexpr-cast->type)
                                   (:arg . jexpr-cast->arg))
                                 'change-jexpr-cast
                                 'nil))
