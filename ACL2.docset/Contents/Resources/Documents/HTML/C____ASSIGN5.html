Wrapper to indicate a C local variable assignment in an mv-let for 5 bound variables.

See atc-let-designations.

Macro: assign5

(defmacro assign5 (x)
          (cons 'mv-let
                (cons '(*1 *2 *3 *4 *5)
                      (cons x '((mv (assign *1) *2 *3 *4 *5))))))
