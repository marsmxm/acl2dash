Modifying constructor for jvalue-reference structures.

Syntax
(change-jvalue-reference x 
                         [:get <get>])

This is an often useful alternative to make-jvalue-reference.

We construct a new jvalue-reference structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-jvalue-reference

(defmacro
     change-jvalue-reference (x &rest args)
     (std::change-aggregate 'jvalue-reference
                            x args '((:get . jvalue-reference->get))
                            'change-jvalue-reference
                            'nil))
