Modifying constructor for jcmember-method structures.

Syntax
(change-jcmember-method x 
                        [:get <get>])

This is an often useful alternative to make-jcmember-method.

We construct a new jcmember-method structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-jcmember-method

(defmacro
     change-jcmember-method (x &rest args)
     (std::change-aggregate 'jcmember-method
                            x args '((:get . jcmember-method->get))
                            'change-jcmember-method
                            'nil))
