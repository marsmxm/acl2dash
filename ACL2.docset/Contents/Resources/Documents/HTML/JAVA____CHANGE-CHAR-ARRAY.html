Modifying constructor for char-array structures.

Syntax
(change-char-array x 
                   [:components <components>])

This is an often useful alternative to make-char-array.

We construct a new char-array structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-char-array

(defmacro
    change-char-array (x &rest args)
    (std::change-aggregate 'char-array
                           x args
                           '((:components . char-array->components))
                           'change-char-array
                           'nil))
