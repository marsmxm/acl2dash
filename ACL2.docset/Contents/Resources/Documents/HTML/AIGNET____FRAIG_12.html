Like fraig, but overwrites the original network instead of returning a new one.

Signature
(fraig! aignet config state) → (mv new-aignet new-state)
Arguments
aignet — Input aignet -- will be replaced with transformation result.
config — Guard (fraig-config-p config).

Definitions and Theorems

Function: fraig!

(defun
 fraig! (aignet config state)
 (declare (xargs :stobjs (aignet state)))
 (declare (xargs :guard (fraig-config-p config)))
 (let
  ((__function__ 'fraig!))
  (declare (ignorable __function__))
  (b* (((local-stobjs aignet-tmp)
        (mv aignet aignet-tmp state))
       ((mv aignet-tmp state)
        (fraig-core aignet aignet-tmp config state))
       (aignet (aignet-prune-comb aignet-tmp aignet
                                  (fraig-config->gatesimp config))))
      (mv aignet aignet-tmp state))))
