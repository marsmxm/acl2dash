Fixtype of (ATC's model of) arrays of type signed long.

This is a product type introduced by fty::defprod.

Fields
elemtype — type
elements — slong-list
Additional Requirements

The following invariant is enforced on the fields:

(and (type-case elemtype :slong) (consp elements))

Theorem: slong-arrayp-alt-def

(defthm slong-arrayp-alt-def
        (equal (slong-arrayp x)
               (and (valuep x)
                    (value-case x :array)
                    (equal (value-array->elemtype x)
                           (type-slong))
                    (slong-listp (value-array->elements x)))))

Theorem: slong-array->elements-alt-def

(defthm slong-array->elements-alt-def
        (implies (slong-arrayp array)
                 (equal (slong-array->elements array)
                        (value-array->elements array))))
