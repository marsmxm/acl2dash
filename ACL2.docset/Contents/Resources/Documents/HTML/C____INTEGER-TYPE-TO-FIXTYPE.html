Name of the fixtype of the values of a C integer type.

Signature
(integer-type-to-fixtype type) → fixtype
Arguments
type — Guard (typep type).
Returns
fixtype — Type (symbolp fixtype).

This is the symbol in the "C" package with the same name as the keyword kind of the type (e.g. it is uchar for (type-uchar)).
