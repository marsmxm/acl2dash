Basic constructor macro for alias structures.

Syntax
(make-alias [:name <name>] 
            [:val <val>])

This is the usual way to construct alias structures. It simply conses together a structure with the specified fields.

This macro generates a new alias structure from scratch. See also change-alias, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by fty::defprod.

Macro: make-alias

(defmacro make-alias (&rest args)
          (std::make-aggregate 'alias
                               args '((:name . "") (:val . 0))
                               'make-alias
                               nil))

(defun alias (name val)
       (declare (xargs :guard (and (sv::svar-p name) (svex-p val))))
       (declare (xargs :guard t))
       (let ((acl2::__function__ 'alias))
            (declare (ignorable acl2::__function__))
            (b* ((name (mbe :logic (sv::svar-fix name)
                            :exec name))
                 (val (mbe :logic (svex-fix val) :exec val)))
                (list (cons 'name name)
                      (cons 'val val)))))
