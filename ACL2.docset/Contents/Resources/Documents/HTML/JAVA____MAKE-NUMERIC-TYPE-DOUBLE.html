Basic constructor macro for numeric-type-double structures.

Syntax
(make-numeric-type-double )

This is the usual way to construct numeric-type-double structures. It simply conses together a structure with the specified fields.

This macro generates a new numeric-type-double structure from scratch. See also change-numeric-type-double, which can "change" an existing structure, instead.
