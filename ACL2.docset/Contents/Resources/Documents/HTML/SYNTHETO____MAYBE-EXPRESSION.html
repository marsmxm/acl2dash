Fixtype of optional Syntheto expressions.

This is an option type introduced by fty::defoption. Note that defoption is just a wrapper for fty::defflexsum, so there are :none and :some member types, a case macro, and so forth.

Member Types
:none → maybe-expression-none
Represents that no maybe-expression is available, i.e., Nothing or None.
:some → maybe-expression-some
An available maybe-expression, i.e., Just val or Some val.
