Modifying constructor for element-group structures.

Syntax
(change-element-group x 
                      [:get <get>])

This is an often useful alternative to make-element-group.

We construct a new element-group structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-element-group

(defmacro
     change-element-group (x &rest args)
     (std::change-aggregate 'element-group
                            x args '((:get . element-group->get))
                            'change-element-group
                            'nil))
