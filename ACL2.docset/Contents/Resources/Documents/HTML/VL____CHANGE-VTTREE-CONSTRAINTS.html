Modifying constructor for vttree-constraints structures.

Syntax
(change-vttree-constraints x 
                           [:constraints <constraints>])

This is an often useful alternative to make-vttree-constraints.

We construct a new vttree-constraints structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vttree-constraints

(defmacro change-vttree-constraints (x &rest args)
          (std::change-aggregate
               'vttree-constraints
               x args
               '((:constraints . vttree-constraints->constraints))
               'change-vttree-constraints
               'nil))
