Modifying constructor for jbinop-ior structures.

Syntax
(change-jbinop-ior x 
                   )

This is an often useful alternative to make-jbinop-ior.

We construct a new jbinop-ior structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
