Basic constructor macro for char-literal-escape structures.

Syntax
(make-char-literal-escape [:get <get>])

This is the usual way to construct char-literal-escape structures. It simply conses together a structure with the specified fields.

This macro generates a new char-literal-escape structure from scratch. See also change-char-literal-escape, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by fty::defprod.
