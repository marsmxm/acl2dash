Basic constructor macro for vl-enumbasetype structures.

Syntax
(make-vl-enumbasetype [:kind <kind>] 
                      [:signedp <signedp>] 
                      [:dim <dim>])

This is the usual way to construct vl-enumbasetype structures. It simply conses together a structure with the specified fields.

This macro generates a new vl-enumbasetype structure from scratch. See also change-vl-enumbasetype, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by defprod.
