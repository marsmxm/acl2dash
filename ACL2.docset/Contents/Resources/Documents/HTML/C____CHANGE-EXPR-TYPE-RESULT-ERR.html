Modifying constructor for expr-type-result-err structures.

Syntax
(change-expr-type-result-err x 
                             [:get <get>])

This is an often useful alternative to make-expr-type-result-err.

We construct a new expr-type-result-err structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
