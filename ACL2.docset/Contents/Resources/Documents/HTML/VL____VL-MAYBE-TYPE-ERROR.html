Option type; vl-type-error or nil.

This is an option type introduced by defoption. Note that defoption is just a wrapper for defflexsum, so there are :none and :some member types, a case macro, and so forth.

Member Types
:none → vl-maybe-type-error-none
Represents that no vl-maybe-type-error is available, i.e., Nothing or None.
:some → vl-maybe-type-error-some
An available vl-maybe-type-error, i.e., Just val or Some val.
