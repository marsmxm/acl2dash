Modifying constructor for vl-arrayrange-range structures.

Syntax
(change-vl-range->arrayrange x 
                             [:range <range>])

This is an often useful alternative to make-vl-range->arrayrange.

We construct a new vl-arrayrange-range structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-range->arrayrange

(defmacro
    change-vl-range->arrayrange
    (x &rest args)
    (std::change-aggregate 'vl-range->arrayrange
                           x args
                           '((:range . vl-range->arrayrange->range))
                           'change-vl-range->arrayrange
                           'nil))
