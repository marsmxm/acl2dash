Modifying constructor for numeric-type-float structures.

Syntax
(change-numeric-type-float x 
                           )

This is an often useful alternative to make-numeric-type-float.

We construct a new numeric-type-float structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
