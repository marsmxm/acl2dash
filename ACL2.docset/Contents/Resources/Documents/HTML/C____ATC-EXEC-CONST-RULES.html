Rules for executing constants.

To symbolically execute a constant, we simply expand the definitions of a number of functions, starting with exec-const and including all the functions called by it (directly or indirectly) except for the fixtype constructors of the integer values (i.e. sint etc.). The argument of exec-const is a quoted constant during symbolic execution, because it is taken from the ASTs being executed; thus, for certain functions we only need to enable the executable counterpart.
