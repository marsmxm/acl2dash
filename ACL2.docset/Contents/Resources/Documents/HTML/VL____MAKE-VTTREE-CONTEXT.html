Basic constructor macro for vttree-context structures.

Syntax
(make-vttree-context [:ctx <ctx>] 
                     [:subtree <subtree>])

This is the usual way to construct vttree-context structures. It simply conses together a structure with the specified fields.

This macro generates a new vttree-context structure from scratch. See also change-vttree-context, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by defprod.

Macro: make-vttree-context

(defmacro make-vttree-context (&rest args)
          (std::make-aggregate 'vttree-context
                               args '((:ctx) (:subtree))
                               'make-vttree-context
                               nil))

(defun vttree-context (ctx subtree)
       (declare (xargs :guard (vttree-p subtree)))
       (declare (xargs :guard t))
       (let ((__function__ 'vttree-context))
            (declare (ignorable __function__))
            (b* ((subtree (mbe :logic (vttree-fix subtree)
                               :exec subtree)))
                (cons :context (cons ctx subtree)))))
