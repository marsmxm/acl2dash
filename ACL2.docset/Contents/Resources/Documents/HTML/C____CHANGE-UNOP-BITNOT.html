Modifying constructor for unop-bitnot structures.

Syntax
(change-unop-bitnot x 
                    )

This is an often useful alternative to make-unop-bitnot.

We construct a new unop-bitnot structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-unop-bitnot

(defmacro change-unop-bitnot (x &rest args)
          (std::change-aggregate 'unop-bitnot
