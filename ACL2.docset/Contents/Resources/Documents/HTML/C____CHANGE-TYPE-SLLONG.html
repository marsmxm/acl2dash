Modifying constructor for type-sllong structures.

Syntax
(change-type-sllong x 
                    )

This is an often useful alternative to make-type-sllong.

We construct a new type-sllong structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

(defmacro change-type-sllong (x &rest args)
