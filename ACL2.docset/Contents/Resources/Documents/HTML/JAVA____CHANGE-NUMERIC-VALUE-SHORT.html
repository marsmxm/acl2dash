Modifying constructor for numeric-value-short structures.

Syntax
(change-numeric-value-short x 
                            [:get <get>])

This is an often useful alternative to make-numeric-value-short.

We construct a new numeric-value-short structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-numeric-value-short

(defmacro
     change-numeric-value-short
     (x &rest args)
     (std::change-aggregate 'numeric-value-short
                            x args
                            '((:get . numeric-value-short->get))
                            'change-numeric-value-short
                            'nil))
