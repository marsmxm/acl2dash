x — Guard (vl-package-p x).

The -aux function is memoized. In a single-threaded context, there would be no need to call make-fast-alist again. But for multi-threaded code, e.g., the vl-server, the alist may only be fast in some threads. Calling make-fast-alist again corrects for this and should be very cheap in the single-threaded case.
