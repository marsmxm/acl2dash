Split a list of cases into default and non-default cases.

Signature
(vl-filter-parsed-caseitemlist x) 
  → 
(mv default-stmts normal-cases)
Arguments
x — Guard (vl-caselist-p x).
Returns
default-stmts — All statements in x whose expressions are NIL, i.e., all of the default statements.
    Type (vl-stmtlist-p default-stmts).
normal-cases — All of the cases that are not default statements.
    Type (vl-caselist-p normal-cases).

Function: vl-filter-parsed-caseitemlist

(defun vl-filter-parsed-caseitemlist (x)
       (declare (xargs :guard (vl-caselist-p x)))
       (let ((__function__ 'vl-filter-parsed-caseitemlist))
            (declare (ignorable __function__))
            (b* ((x (vl-caselist-fix x))
                 ((when (atom x)) (mv nil nil))
                 ((mv default-stmts normal-cases)
                  (vl-filter-parsed-caseitemlist (cdr x)))
                 ((cons exprs1 stmt1) (car x))
                 ((when exprs1)
                  (mv default-stmts
                      (cons (car x) normal-cases))))
                (mv (cons stmt1 default-stmts)
                    normal-cases))))
