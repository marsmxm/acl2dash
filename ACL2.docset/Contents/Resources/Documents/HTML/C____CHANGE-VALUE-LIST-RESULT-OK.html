Modifying constructor for value-list-result-ok structures.

Syntax
(change-value-list-result-ok x 
                             [:get <get>])

This is an often useful alternative to make-value-list-result-ok.

We construct a new value-list-result-ok structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
