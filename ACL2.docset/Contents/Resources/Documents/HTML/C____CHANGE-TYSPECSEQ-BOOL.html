Modifying constructor for tyspecseq-bool structures.

Syntax
(change-tyspecseq-bool x 
                       )

This is an often useful alternative to make-tyspecseq-bool.

We construct a new tyspecseq-bool structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-tyspecseq-bool

(defmacro change-tyspecseq-bool (x &rest args)
