A library for JSON.

The JavaScript Object Notation (JSON) is described by:

  • The JSON web site.
  • The ECMA-404 Standard (2nd Edition).
  • The ISO/IEC 21778:2017 Standard.
  • The RFC 8259.

These are all meant to be consistent, at least in defining the JSON format, although they may contain slightly different recommendations.
