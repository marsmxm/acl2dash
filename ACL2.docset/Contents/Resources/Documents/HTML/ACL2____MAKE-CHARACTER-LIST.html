coerce to a list of characters

Non-characters in the given list are coerced to the character with code 0.

Function: make-character-list

(defun make-character-list (x)
       (declare (xargs :guard t))
       (cond ((atom x) nil)
             ((characterp (car x))
              (cons (car x)
                    (make-character-list (cdr x))))
             (t (cons (code-char 0)
                      (make-character-list (cdr x))))))
