Modifying constructor for svexl-node-quote structures.

Syntax
(change-svexl-node-quote x 
                         [:val <val>])

This is an often useful alternative to make-svexl-node-quote.

We construct a new svexl-node-quote structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
