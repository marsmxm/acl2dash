Modifying constructor for binary-op-eq structures.

Syntax
(change-binary-op-eq x 
                     )

This is an often useful alternative to make-binary-op-eq.

We construct a new binary-op-eq structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
