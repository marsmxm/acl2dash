Modifying constructor for nat-option-none structures.

Syntax
(change-nat-option-none x 
                        )

This is an often useful alternative to make-nat-option-none.

We construct a new nat-option-none structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
