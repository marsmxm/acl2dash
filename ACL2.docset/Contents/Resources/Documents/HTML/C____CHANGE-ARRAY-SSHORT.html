Modifying constructor for array-sshort structures.

Syntax
(change-array-sshort x 
                     [:get <get>])

This is an often useful alternative to make-array-sshort.

We construct a new array-sshort structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-array-sshort

(defmacro
     change-array-sshort (x &rest args)
     (std::change-aggregate 'array-sshort

                            'change-array-sshort
                            'nil))
