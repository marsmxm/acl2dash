This is only used for repetitions whose repeat prefix is *, i.e. zero or more. The name is retrieved from the alist for repetition parsing functions. We generate a recursive function that repeatedly attempts to parse the underlying element. Note that we set the check-error-p flag to nil here, because we don't want the error to be returned, we just want to stop the recursion.
