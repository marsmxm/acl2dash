Modifying constructor for vl-final structures.

Syntax
(change-vl-final x 
                 [:stmt <stmt>] 
                 [:atts <atts>] 
                 [:loc <loc>])

This is an often useful alternative to make-vl-final.

We construct a new vl-final structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-final

(defmacro change-vl-final (x &rest args)
          (std::change-aggregate 'vl-final
                                 x args
                                 '((:stmt . vl-final->stmt)
                                   (:atts . vl-final->atts)
                                   (:loc . vl-final->loc))
                                 'change-vl-final
                                 'remake-vl-final))
