Modifying constructor for binop-add structures.

Syntax
(change-binop-add x 
                  )

This is an often useful alternative to make-binop-add.

We construct a new binop-add structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-binop-add

(defmacro change-binop-add (x &rest args)
          (std::change-aggregate 'binop-add
