Pretty-print an octal constant.

Signature
(pprint-oct-const n) → part
Arguments
n — Guard (natp n).
Returns
part — Type (msgp part).

According to the grammar [C:6.4.4.1], an `octal constant' is a sequence of octal digits starting with 0; with the optional suffixes, this becomes an integer constant.

Definitions and Theorems

Function: pprint-oct-const

(defun pprint-oct-const (n)
       (declare (xargs :guard (natp n)))
       (let ((__function__ 'pprint-oct-const))
            (declare (ignorable __function__))
            (if (zp n)
                "0"
                (str::cat "0" (str::nat-to-oct-string n)))))
