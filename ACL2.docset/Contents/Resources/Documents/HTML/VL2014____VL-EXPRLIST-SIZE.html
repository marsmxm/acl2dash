Self-determine the sizes of a list of expressions.

Signature
(vl-exprlist-size x ss ctx warnings) 
  → 
(mv successp warnings new-x)
Arguments
x — Should be a list of self-determined expressions.
    Guard (vl-exprlist-p x).
ss — Guard (vl-scopestack-p ss).
ctx — Guard (vl-context-p ctx).
warnings — Guard (vl-warninglist-p warnings).
Returns
successp — Type (booleanp successp).
warnings — Type (vl-warninglist-p warnings).
new-x — Type (and (vl-exprlist-p new-x) (equal (len new-x) (len x))).

We just use vl-expr-size (with lhs-size = nil) to size each of the expressions in x.
