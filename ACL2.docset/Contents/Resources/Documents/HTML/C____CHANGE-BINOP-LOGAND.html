Modifying constructor for binop-logand structures.

Syntax
(change-binop-logand x 
                     )

This is an often useful alternative to make-binop-logand.

We construct a new binop-logand structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
