Modifying constructor for nat/natlist-list structures.

Syntax
(change-nat/natlist-list x 
                         [:get <get>])

This is an often useful alternative to make-nat/natlist-list.

We construct a new nat/natlist-list structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
