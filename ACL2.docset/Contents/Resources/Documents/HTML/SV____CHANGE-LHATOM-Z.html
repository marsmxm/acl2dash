Modifying constructor for lhatom-z structures.

Syntax
(change-lhatom-z x 
                 )

This is an often useful alternative to make-lhatom-z.

We construct a new lhatom-z structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-lhatom-z

(defmacro change-lhatom-z (x &rest args)
          (std::change-aggregate 'lhatom-z
