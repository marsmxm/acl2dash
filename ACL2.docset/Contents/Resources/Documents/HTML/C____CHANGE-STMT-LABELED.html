Modifying constructor for stmt-labeled structures.

Syntax
(change-stmt-labeled x 
                     [:label <label>] 
                     [:body <body>])

This is an often useful alternative to make-stmt-labeled.

We construct a new stmt-labeled structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-stmt-labeled

(defmacro change-stmt-labeled (x &rest args)
          (std::change-aggregate 'stmt-labeled
                                 x args
                                 '((:label . stmt-labeled->label)
                                   (:body . stmt-labeled->body))
                                 'change-stmt-labeled
                                 'nil))
