Modifying constructor for svar-override-triple structures.

Syntax
(change-svar-override-triple x 
                             [:testvar <testvar>] 
                             [:valvar <valvar>] 
                             [:refvar <refvar>])

This is an often useful alternative to make-svar-override-triple.

We construct a new svar-override-triple structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.
