Modifying constructor for value-true structures.

Syntax
(change-value-true x 
                   )

This is an often useful alternative to make-value-true.

We construct a new value-true structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-value-true

(defmacro change-value-true (x &rest args)
          (std::change-aggregate 'value-true
                                 x args 'nil
                                 'change-value-true
                                 'nil))
