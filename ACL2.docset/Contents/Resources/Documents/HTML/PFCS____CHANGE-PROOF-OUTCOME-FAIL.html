Modifying constructor for proof-outcome-fail structures.

Syntax
(change-proof-outcome-fail x 
                           )

This is an often useful alternative to make-proof-outcome-fail.

We construct a new proof-outcome-fail structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
