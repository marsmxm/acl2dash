Wrapper to indicate a C local variable declaration in an mv-let for 2 bound variables.

See atc-let-designations.

Macro: declar2

(defmacro
  declar2 (x)
  (cons 'mv-let
        (cons (cons '*1 (cons '*2 'nil))
              (cons x
                    (cons (cons 'mv
                                (cons (cons 'declar (cons '*1 'nil))
                                      (cons '*2 'nil)))
                          'nil)))))
