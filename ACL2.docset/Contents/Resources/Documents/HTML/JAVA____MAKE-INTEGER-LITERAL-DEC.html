Basic constructor macro for integer-literal-dec structures.

Syntax
(make-integer-literal-dec [:get <get>])

This is the usual way to construct integer-literal-dec structures. It simply conses together a structure with the specified fields.

This macro generates a new integer-literal-dec structure from scratch. See also change-integer-literal-dec, which can "change" an existing structure, instead.
