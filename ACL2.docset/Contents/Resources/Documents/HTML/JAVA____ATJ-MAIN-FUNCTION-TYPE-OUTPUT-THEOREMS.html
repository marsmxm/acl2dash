Theorems generated by atj-main-function-type for all the outputs of a multi-valued ACL2 function.

Signature
(atj-main-function-type-output-theorems 
     fn guard formals 
     nresults rev-out-types hints wrld) 
 
  → 
events
Arguments
fn — Guard (symbolp fn).
guard — Guard (pseudo-termp guard).
formals — Guard (symbol-listp formals).
nresults — Guard (natp nresults).
rev-out-types — Guard (atj-type-listp rev-out-types).
hints — Guard (true-listp hints).
wrld — Guard (plist-worldp wrld).
Returns
events — A ACL2::pseudo-event-form-listp.

This is only used when fn returns multiple values. We generate an output type theorem for each, counting down from nresults - 1 to 0.

The output types are reversed (as suggested by the rev- prefix, so that they ``match'' the counting down mentioned just above.

Function: atj-main-function-type-output-theorems

(defun atj-main-function-type-output-theorems
       (fn guard formals
           nresults rev-out-types hints wrld)
       (declare (xargs :guard (and (symbolp fn)
                                   (pseudo-termp guard)
                                   (symbol-listp formals)
                                   (natp nresults)
                                   (atj-type-listp rev-out-types)
                                   (true-listp hints)
                                   (plist-worldp wrld))))
       (declare (xargs :guard (= nresults (len rev-out-types))))
       (let ((__function__ 'atj-main-function-type-output-theorems))
            (declare (ignorable __function__))
            (if (zp nresults)
                nil
                (cons (atj-main-function-type-output-theorem
                           fn guard formals (1- nresults)
                           (car rev-out-types)
                           hints wrld)
                      (atj-main-function-type-output-theorems
                           fn guard formals (1- nresults)
                           (cdr rev-out-types)
                           hints wrld)))))
