Modifying constructor for tyspecseq-slong structures.

Syntax
(change-tyspecseq-slong x 
                        )

This is an often useful alternative to make-tyspecseq-slong.

We construct a new tyspecseq-slong structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
