Modifying constructor for sd-key structures.

Syntax
(change-sd-key x 
               [:pat <pat>] 
               [:index <index>] 
               [:orig <orig>])

This is an often useful alternative to make-sd-key.

We construct a new sd-key structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-sd-key

(defmacro change-sd-key (x &rest args)
          (std::change-aggregate 'sd-key
                                 x args
                                 '((:pat . sd-key->pat)
                                   (:index . sd-key->index)
                                   (:orig . sd-key->orig))
                                 'change-sd-key
                                 'remake-sd-key))
