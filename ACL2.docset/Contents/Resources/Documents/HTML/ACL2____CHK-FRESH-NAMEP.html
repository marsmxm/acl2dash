Check whether name is a legal new name.

Signature
(chk-fresh-namep name type ctx wrld state) 
  → 
(mv erp val state)
Arguments
name — Guard (symbolp name).
wrld — Guard (plist-worldp wrld).

Returns an error triple (mv erp val state) where erp is nil if and only if name is a legal new name, and val is irrelevant. If erp is not nil, then an explanatory error message is printed.

For more information about legality of new names see fresh-namep-msg, which returns an error triple, (mv nil msg/nil state). When non-nil, the value msg/nil provides the message printed by chk-fresh-namep.
