Modifying constructor for neteval-sigordering-remainder structures.

Syntax
(change-neteval-sigordering-remainder x 
                                      [:ord <ord>])

This is an often useful alternative to make-neteval-sigordering-remainder.

We construct a new neteval-sigordering-remainder structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
