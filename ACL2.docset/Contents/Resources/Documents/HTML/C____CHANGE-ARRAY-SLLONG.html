Modifying constructor for array-sllong structures.

Syntax
(change-array-sllong x 
                     [:get <get>])

This is an often useful alternative to make-array-sllong.

We construct a new array-sllong structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-array-sllong

(defmacro
     change-array-sllong (x &rest args)
     (std::change-aggregate 'array-sllong
                            x args '((:get . array-sllong->get))
                            'change-array-sllong
                            'nil))
