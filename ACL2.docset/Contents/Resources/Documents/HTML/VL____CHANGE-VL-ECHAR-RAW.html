Modifying constructor for vl-echar-raw structures.

Syntax
(change-vl-echar-raw x 
                     [:filename <filename>] 
                     [:pack <pack>])

This is an often useful alternative to make-vl-echar-raw.

We construct a new vl-echar-raw structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-echar-raw

(defmacro
     change-vl-echar-raw (x &rest args)
     (std::change-aggregate 'vl-echar-raw
                            x args
                            '((:filename . vl-echar-raw->filename)
                              (:pack . vl-echar-raw->pack))
                            'change-vl-echar-raw
                            'remake-vl-echar-raw))
