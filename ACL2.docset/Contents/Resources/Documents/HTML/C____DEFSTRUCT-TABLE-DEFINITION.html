Definition of the table of shallowly embedded C structures.

The keys are strings that are symbol-names of symbols that represent the tags of the structures. The name of each such symbol is a portable ASCII C identifier, but this constraint is not enforced in the table's guard. The keys in the table are unique.

The values are the information about the structures. See defstruct-info.
