Basic constructor macro for fun-declor-pointer structures.

Syntax
(make-fun-declor-pointer [:decl <decl>])

This is the usual way to construct fun-declor-pointer structures. It simply conses together a structure with the specified fields.

This macro generates a new fun-declor-pointer structure from scratch. See also change-fun-declor-pointer, which can "change" an existing structure, instead.

This is an ordinary make- macro introduced by fty::defprod.
