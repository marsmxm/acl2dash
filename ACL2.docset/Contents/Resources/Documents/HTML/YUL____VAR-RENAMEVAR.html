Check if two variables are related by variable renaming.

Signature
(var-renamevar old new ren) → _
Arguments
old — Guard (identifierp old).
new — Guard (identifierp new).
ren — Guard (renamingp ren).
Returns
_ — Type (resulterr-optionp _).

We check if the two variables form a pair in the renaming list.

Definitions and Theorems

Function: var-renamevar

(defun var-renamevar (old new ren)
       (declare (xargs :guard (and (identifierp old)
                                   (identifierp new)
                                   (renamingp ren))))
       (let ((__function__ 'var-renamevar))
            (declare (ignorable __function__))
            (b* ((old (identifier-fix old))
                 (new (identifier-fix new)))
                (if (member-equal (cons old new)
                                  (renaming->list ren))
                    nil
                    (err (list :mismatch
                               old new (renaming-fix ren)))))))
