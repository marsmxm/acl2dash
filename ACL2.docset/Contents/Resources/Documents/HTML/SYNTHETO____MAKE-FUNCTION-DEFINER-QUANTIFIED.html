Basic constructor macro for function-definer-quantified structures.

Syntax
(make-function-definer-quantified [:quantifier <quantifier>] 
                                  [:variables <variables>] 
                                  [:matrix <matrix>])

This is the usual way to construct function-definer-quantified structures. It simply conses together a structure with the specified fields.

This macro generates a new function-definer-quantified structure from scratch. See also change-function-definer-quantified, which can "change" an existing structure, instead.

This is an ordinary make- macro introduced by fty::defprod.
