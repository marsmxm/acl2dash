Signature
(vl-exprlist-selfsize x ss ctx warnings) 
  → 
(mv warnings size-list)
Arguments
x — Expressions whose sizes we are to compute.
    Guard (vl-exprlist-p x).
ss — Scope where these expressions occur.
    Guard (vl-scopestack-p ss).
ctx — Context for warnings.
    Guard (vl-context-p ctx).
warnings — Ordinary warnings accumulator.
    Guard (vl-warninglist-p warnings).
Returns
warnings — Type (vl-warninglist-p warnings).
size-list — Type (and (vl-maybe-nat-listp size-list) (equal (len size-list) (len x))).
