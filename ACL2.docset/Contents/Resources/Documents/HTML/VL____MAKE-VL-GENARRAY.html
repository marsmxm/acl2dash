Basic constructor macro for vl-genarray structures.

Syntax
(make-vl-genarray [:name <name>] 
                  [:var <var>] 
                  [:genvarp <genvarp>] 
                  [:blocks <blocks>] 
                  [:loc <loc>])

This is the usual way to construct vl-genarray structures. It simply conses together a structure with the specified fields.

This macro generates a new vl-genarray structure from scratch. See also change-vl-genarray, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by defprod.

(defun
 vl-genarray
 (name var genvarp blocks loc)
 (declare (xargs :guard (and (maybe-stringp name)
                             (stringp var)
                             (booleanp genvarp)
                             (vl-genblocklist-p blocks)
                             (vl-location-p loc))))
 (declare (xargs :guard t))
 (let
    ((__function__ 'vl-genarray))
    (declare (ignorable __function__))
    (b* ((name (mbe :logic (maybe-string-fix name)
                    :exec name))
         (var (mbe :logic (str-fix var) :exec var))
         (genvarp (mbe :logic (acl2::bool-fix genvarp)
                       :exec genvarp))
         (blocks (mbe :logic (vl-genblocklist-fix blocks)
                      :exec blocks))
         (loc (mbe :logic (vl-location-fix loc)
                   :exec loc)))
        (cons :vl-genarray
              (std::prod-cons
                   (std::prod-cons name var)
                   (std::prod-cons genvarp
                                   (std::prod-cons blocks loc)))))))
