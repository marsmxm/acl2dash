Check if a constraint is well-formed.

Signature
(constraint-wfp constr sys) → yes/no
Arguments
constr — Guard (constraintp constr).
sys — Guard (systemp sys).
Returns
yes/no — Type (booleanp yes/no).

An equality is always well-formed. An application of a named relation to some expressions is well-formed iff the relation is defined in the system and the number of argument expressions is the same as the number of formal parameters.

Definitions and Theorems

Function: constraint-wfp

(defun
 constraint-wfp (constr sys)
 (declare (xargs :guard (and (constraintp constr)
                             (systemp sys))))
 (let
     ((__function__ 'constraint-wfp))
     (declare (ignorable __function__))
     (constraint-case
          constr
          :equal t
          :relation (b* ((def? (lookup-definition constr.name sys)))
                        (and (definitionp def?)
                             (= (len constr.args)
                                (len (definition->para def?))))))))
