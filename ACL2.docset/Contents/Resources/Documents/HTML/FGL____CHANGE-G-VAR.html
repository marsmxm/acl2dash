Modifying constructor for g-var structures.

Syntax
(change-g-var x 
              [:name <name>])

This is an often useful alternative to make-g-var.

We construct a new g-var structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-g-var

(defmacro change-g-var (x &rest args)
          (std::change-aggregate 'g-var
                                 x args '((:name . g-var->name))
                                 'change-g-var
                                 'nil))
