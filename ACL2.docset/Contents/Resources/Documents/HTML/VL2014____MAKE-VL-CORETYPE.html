Basic constructor macro for vl-coretype structures.

Syntax
(make-vl-coretype [:name <name>] 
                  [:signedp <signedp>] 
                  [:pdims <pdims>] 
                  [:udims <udims>])

This is the usual way to construct vl-coretype structures. It simply conses together a structure with the specified fields.

This macro generates a new vl-coretype structure from scratch. See also change-vl-coretype, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by defprod.

(defun
 vl-coretype (name signedp pdims udims)
 (declare (xargs :guard (and (vl-coretypename-p name)
                             (booleanp signedp)
                             (vl-packeddimensionlist-p pdims)
                             (vl-packeddimensionlist-p udims))))
 (declare (xargs :guard t))
 (let
  ((__function__ 'vl-coretype))
  (declare (ignorable __function__))
  (b*
   ((name (mbe :logic (vl-coretypename-fix name)
               :exec name))
    (signedp (mbe :logic (acl2::bool-fix signedp)
                  :exec signedp))
    (pdims (mbe :logic (vl-packeddimensionlist-fix pdims)
                :exec pdims))
    (udims (mbe :logic (vl-packeddimensionlist-fix udims)
                :exec udims)))
   (cons
      :vl-coretype (std::prod-cons (std::prod-cons name signedp)
                                   (std::prod-cons pdims udims))))))
