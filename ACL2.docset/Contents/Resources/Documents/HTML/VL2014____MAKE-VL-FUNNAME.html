Basic constructor macro for vl-funname structures.

Syntax
(make-vl-funname [:name <name>])

This is the usual way to construct vl-funname structures. It simply conses together a structure with the specified fields.

This macro generates a new vl-funname structure from scratch. See also change-vl-funname, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by defprod.

(defmacro make-vl-funname (&rest args)
