Restrict an svex-env to only particular variables. Variables that are present in keys but not env will be bound to X.

Signature
(svex-env-extract keys env) → sub-env
Arguments
keys — Variables to keep.
    Guard (svarlist-p keys).
env — Original environment to filter. Need not be fast.
    Guard (svex-env-p env).
Returns
sub-env — Restriction of env to keys. Slow alist.
    Type (svex-env-p sub-env).

Function: svex-env-extract

(defun
 svex-env-extract (keys env)
 (declare (xargs :guard (and (svarlist-p keys)
                             (svex-env-p env))))
 (let
  ((__function__ 'svex-env-extract))
  (declare (ignorable __function__))
  (mbe
      :logic (if (atom keys)
                 nil
                 (cons (cons (svar-fix (car keys))
                             (svex-env-fastlookup (car keys) env))
                       (svex-env-extract (cdr keys) env)))
      :exec (with-fast-alist env (svex-env-extract-aux keys env)))))
