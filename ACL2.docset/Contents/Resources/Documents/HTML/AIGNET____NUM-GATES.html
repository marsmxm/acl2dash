(num-gates aignet) returns the number of AND gate nodes in an aignet.

Logically this is just (stype-count :gate aignet)

In the execution this is just a stobj field access.

Function: num-gates

(defun
     aignet$a::num-gates (aignet)
     (declare (xargs :guard (aignet$a::aignet-well-formedp aignet)))
     (let ((__function__ 'aignet$a::num-gates))
          (declare (ignorable __function__))
          (+ (stype-count (and-stype) aignet)
             (stype-count (xor-stype) aignet))))
