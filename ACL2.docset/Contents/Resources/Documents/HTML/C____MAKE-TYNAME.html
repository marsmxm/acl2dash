Basic constructor macro for tyname structures.

Syntax
(make-tyname [:specs <specs>] 
             [:pointerp <pointerp>])

This is the usual way to construct tyname structures. It simply conses together a structure with the specified fields.

This macro generates a new tyname structure from scratch. See also change-tyname, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by fty::defprod.

Macro: make-tyname

(defmacro make-tyname (&rest args)
          (std::make-aggregate 'tyname
                               args '((:specs) (:pointerp))
                               'make-tyname
                               nil))

(defun tyname (specs pointerp)
       (declare (xargs :guard (and (tyspecseqp specs)
                                   (booleanp pointerp))))
       (declare (xargs :guard t))
       (let ((__function__ 'tyname))
            (declare (ignorable __function__))
            (b* ((specs (mbe :logic (tyspecseq-fix specs)
                             :exec specs))
                 (pointerp (mbe :logic (acl2::bool-fix pointerp)
                                :exec pointerp)))
                (cons :tyname (list (cons 'specs specs)
                                    (cons 'pointerp pointerp))))))
