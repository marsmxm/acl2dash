Modifying constructor for expression-multi structures.

Syntax
(change-expression-multi x 
                         [:arguments <arguments>])

This is an often useful alternative to make-expression-multi.

We construct a new expression-multi structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-expression-multi

(defmacro change-expression-multi (x &rest args)
          (std::change-aggregate
               'expression-multi
               x args
               '((:arguments . expression-multi->arguments))
               'change-expression-multi
               'nil))
