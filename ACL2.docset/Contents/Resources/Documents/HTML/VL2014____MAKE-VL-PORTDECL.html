Basic constructor macro for vl-portdecl structures.

Syntax
(make-vl-portdecl [:name <name>] 
                  [:dir <dir>] 
                  [:nettype <nettype>] 
                  [:type <type>] 
                  [:atts <atts>] 
                  [:loc <loc>])

This is the usual way to construct vl-portdecl structures. It simply conses together a structure with the specified fields.

This macro generates a new vl-portdecl structure from scratch. See also change-vl-portdecl, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by defprod.

(defun
 vl-portdecl
 (name dir nettype type atts loc)
 (declare (xargs :guard (and (stringp name)
                             (vl-direction-p dir)
                             (vl-maybe-nettypename-p nettype)
                             (vl-datatype-p type)
                             (vl-atts-p atts)
                             (vl-location-p loc))))
 (declare (xargs :guard t))
 (let
  ((__function__ 'vl-portdecl))
  (declare (ignorable __function__))
  (b*
     ((name (mbe :logic (str-fix name) :exec name))
      (dir (mbe :logic (vl-direction-fix dir)
                :exec dir))
      (nettype (mbe :logic (vl-maybe-nettypename-fix nettype)
                    :exec nettype))
      (type (mbe :logic (vl-datatype-fix type)
                 :exec type))
      (atts (mbe :logic (vl-atts-fix atts)
                 :exec atts))
      (loc (mbe :logic (vl-location-fix loc)
                :exec loc)))
     (cons :vl-portdecl
           (std::prod-cons
                (std::prod-cons name (std::prod-cons dir nettype))
                (std::prod-cons type (std::prod-cons atts loc)))))))
