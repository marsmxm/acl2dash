Modifying constructor for vl-hidexpr-end structures.

Syntax
(change-vl-hidexpr-end x 
                       [:name <name>])

This is an often useful alternative to make-vl-hidexpr-end.

We construct a new vl-hidexpr-end structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-hidexpr-end

(defmacro
     change-vl-hidexpr-end (x &rest args)
     (std::change-aggregate 'vl-hidexpr-end
                            x args '((:name . vl-hidexpr-end->name))
                            'change-vl-hidexpr-end
                            'nil))
