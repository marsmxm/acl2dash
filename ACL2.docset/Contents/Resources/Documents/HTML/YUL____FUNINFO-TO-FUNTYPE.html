Turn function information into a function type.

Signature
(funinfo-to-funtype funinfo) → ftype
Arguments
funinfo — Guard (funinfop funinfo).
Returns
ftype — Type (funtypep ftype).

A function type is the static counterpart of function information. We extract the number of inputs and outputs from the function information's input and output lists.

Definitions and Theorems

Function: funinfo-to-funtype

(defun funinfo-to-funtype (funinfo)
       (declare (xargs :guard (funinfop funinfo)))
       (let ((__function__ 'funinfo-to-funtype))
            (declare (ignorable __function__))
            (b* (((funinfo funinfo) funinfo))
                (make-funtype :in (len funinfo.inputs)
                              :out (len funinfo.outputs)))))

Theorem: funtypep-of-funinfo-to-funtype

(defthm funtypep-of-funinfo-to-funtype
        (b* ((ftype (funinfo-to-funtype funinfo)))
            (funtypep ftype))
        :rule-classes :rewrite)
