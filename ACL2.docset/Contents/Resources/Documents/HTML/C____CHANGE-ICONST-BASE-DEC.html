Modifying constructor for iconst-base-dec structures.

Syntax
(change-iconst-base-dec x 
                        )

This is an often useful alternative to make-iconst-base-dec.

We construct a new iconst-base-dec structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

This is an ordinary change- macro introduced by fty::defprod.
