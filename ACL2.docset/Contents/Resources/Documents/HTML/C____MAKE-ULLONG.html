Basic constructor macro for ullong structures.

Syntax
(make-ullong [:get <get>])

This is the usual way to construct ullong structures. It simply conses together a structure with the specified fields.

This macro generates a new ullong structure from scratch. See also change-ullong, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by fty::defprod.

Macro: make-ullong

(defmacro make-ullong (&rest args)
          (std::make-aggregate 'ullong
                               args '((:get))
                               'make-ullong
                               nil))
