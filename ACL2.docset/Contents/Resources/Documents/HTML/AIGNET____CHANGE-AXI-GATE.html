Modifying constructor for axi-gate structures.

Syntax
(change-axi-gate x 
                 [:op <op>] 
                 [:left <left>] 
                 [:right <right>])

This is an often useful alternative to make-axi-gate.

We construct a new axi-gate structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-axi-gate

(defmacro change-axi-gate (x &rest args)
          (std::change-aggregate 'axi-gate
                                 x args
                                 '((:op . axi-gate->op)
                                   (:left . axi-gate->left)
                                   (:right . axi-gate->right))
                                 'change-axi-gate
                                 'nil))
