Fixtype of specifications of parsing functions.

This is a tagged union type, introduced by fty::deftagsum.

Member Tags → Types
:rulename → def-parse-function-spec-rulename
:group → def-parse-function-spec-group
:option → def-parse-function-spec-option
:repetition → def-parse-function-spec-repetition

A value of this type descriptively specifies a parsing function:

  • A parsing function for a rule name, with a possible reordering of the alternatives (or just the list (1 2 3 ...) if there is no reordering.
  • A parsing function for a group, with a possible reordering of the alternatives (or just the list (1 2 3 ...) if there is no reordering.
  • A parsing function for an option, with a possible reordering of the alternatives (or just the list (1 2 3 ...) if there is no reordering.
  • A parsing function for a repetition.
