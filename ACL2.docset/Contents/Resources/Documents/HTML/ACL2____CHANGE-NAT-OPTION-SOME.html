Modifying constructor for nat-option-some structures.

Syntax
(change-nat-option-some x 
                        [:val <val>])

This is an often useful alternative to make-nat-option-some.

We construct a new nat-option-some structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
