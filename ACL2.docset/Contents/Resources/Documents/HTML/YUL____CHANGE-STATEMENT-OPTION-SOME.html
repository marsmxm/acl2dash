Modifying constructor for statement-option-some structures.

Syntax
(change-statement-option-some x 
                              [:val <val>])

This is an often useful alternative to make-statement-option-some.

We construct a new statement-option-some structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-statement-option-some

(defmacro
     change-statement-option-some
     (x &rest args)
     (std::change-aggregate 'statement-option-some
                            x args
                            '((:val . statement-option-some->val))
                            'change-statement-option-some
                            'nil))
