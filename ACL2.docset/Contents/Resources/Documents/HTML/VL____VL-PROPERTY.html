Represents a single property...endproperty declaration.

This is a product type introduced by defprod.

Fields
name — stringp
E.g., this is foo for property foo ... endproperty.
ports — vl-propportlist
The ports for this property declaration.
decls — vl-vardecllist
Variable declarations for this property.
spec — vl-propspec
The main content of the property declaration. The property spec contains its clocking information, disable condition, and the core property expression.
loc — vl-location
