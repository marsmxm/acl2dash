Modifying constructor for regex-concat structures.

Syntax
(change-regex-concat x 
                     [:lst <lst>])

This is an often useful alternative to make-regex-concat.

We construct a new regex-concat structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-regex-concat

(defmacro
     change-regex-concat (x &rest args)
     (std::change-aggregate 'regex-concat
                            x args '((:lst . regex-concat->lst))
                            'change-regex-concat
                            'nil))
