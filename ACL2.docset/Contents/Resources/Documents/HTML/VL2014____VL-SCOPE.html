Recognizer for a syntactic element that can have named elements within it.

This is a ``transparent'' sum type introduced using deftranssum. It is simply any one of the following types:

  • vl-interface
  • vl-module
  • vl-genblob
  • vl-blockscope
  • vl-design
  • vl-package
  • vl-scopeinfo
