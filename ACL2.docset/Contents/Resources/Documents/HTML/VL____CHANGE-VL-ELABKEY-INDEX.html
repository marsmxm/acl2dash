Modifying constructor for vl-elabkey-index structures.

Syntax
(change-vl-elabkey-index x 
                         [:val <val>])

This is an often useful alternative to make-vl-elabkey-index.

We construct a new vl-elabkey-index structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-elabkey-index

(defmacro
     change-vl-elabkey-index (x &rest args)
     (std::change-aggregate 'vl-elabkey-index
                            x args '((:val . vl-elabkey-index->val))
                            'change-vl-elabkey-index
                            'nil))
