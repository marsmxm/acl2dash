Modifying constructor for resulterr structures.

Syntax
(change-resulterr x 
                  [:info <info>])

This is an often useful alternative to make-resulterr.

We construct a new resulterr structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-resulterr

(defmacro change-resulterr (x &rest args)
          (std::change-aggregate 'resulterr
                                 x args '((:info . resulterr->info))
                                 'change-resulterr
                                 'nil))
