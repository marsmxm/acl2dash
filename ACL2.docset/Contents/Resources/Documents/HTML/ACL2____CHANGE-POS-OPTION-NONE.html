Modifying constructor for pos-option-none structures.

Syntax
(change-pos-option-none x 
                        )

This is an often useful alternative to make-pos-option-none.

We construct a new pos-option-none structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
