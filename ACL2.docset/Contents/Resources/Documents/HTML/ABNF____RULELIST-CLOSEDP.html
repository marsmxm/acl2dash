A rule list is closed iff it defines all the rules that it calls.

Signature
(rulelist-closedp rules) → yes/no
Arguments
rules — Guard (rulelistp rules).
Returns
yes/no — Type (booleanp yes/no).

Definitions and Theorems

Function: rulelist-closedp

(defun rulelist-closedp (rules)
       (declare (xargs :guard (rulelistp rules)))
       (subset (rulelist-called-rules rules)
               (rulelist-defined-rules rules)))

Theorem: booleanp-of-rulelist-closedp

(defthm booleanp-of-rulelist-closedp
        (b* ((yes/no (rulelist-closedp rules)))
            (booleanp yes/no))
        :rule-classes :rewrite)
