Safely add a dependency onto a definition (i.e., an interface, module, user-defined primitive, etc. If there is no such definition, add a warning instead of a dependency.

Signature
(vl-immdeps-add-definition name ans &key (ss 'ss) (ctx 'ctx)) 
  → 
new-ans
Arguments
name — Guard (stringp name).
ans — Guard (vl-immdeps-p ans).
ss — Guard (vl-scopestack-p ss).
ctx — Guard (acl2::any-p ctx).
Returns
new-ans — Type (vl-immdeps-p new-ans).

Definitions and Theorems

Function: vl-immdeps-add-definition-fn

(defun
    vl-immdeps-add-definition-fn
    (name ans ss ctx)
    (declare (xargs :guard (and (stringp name)
                                (vl-immdeps-p ans)
                                (vl-scopestack-p ss)
                                (acl2::any-p ctx))))
    (let ((__function__ 'vl-immdeps-add-definition))
         (declare (ignorable __function__))
         (b* ((name (string-fix name))
              (item (vl-scopestack-find-definition name ss))
              ((unless item)
               (vl-immdeps-add-error
                    ans
                    :type :vl-missing-definition
                    :msg "~a0: reference to unknown definition ~a1."
                    :args (list ctx name)
                    :fatalp t))
              (ans (vl-immdeps-add-raw-dependency name ans)))
             ans)))
