Generate a list of fresh names.

Signature
(vl-namedb-plain-names names db) → (mv fresh-names new-db)
Arguments
names — Guard (string-listp names).
db — Guard (vl-namedb-p db).
Returns
fresh-names — Type (string-listp fresh-names).
new-db — Type (vl-namedb-p new-db).

When possible, fresh-names are just names. When this is not possible due to name collisions, some of the fresh_names may have additional indexes as in vl-namedb-indexed-name, and notes may be printed.

(defun vl-namedb-plain-names (names db)
       (declare (xargs :guard (and (string-listp names)
                                   (vl-namedb-p db))))
       (let ((__function__ 'vl-namedb-plain-names))
            (declare (ignorable __function__))

                  (mv nil (vl-namedb-fix db)))
