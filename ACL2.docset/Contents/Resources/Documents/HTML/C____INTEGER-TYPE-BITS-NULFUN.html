Name of the nullary function that defines the size in bits of a C integer type.

Signature
(integer-type-bits-nulfun type) → bits
Arguments
type — Guard (typep type).
Returns
bits — Type (symbolp bits).

This is the name of one of the nullary functions introduced in integer-formats.

We take the name of the kind, remove the initial s or u, and add -bits at the end.
