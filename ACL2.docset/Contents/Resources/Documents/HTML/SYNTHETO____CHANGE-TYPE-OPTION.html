Modifying constructor for type-option structures.

Syntax
(change-type-option x 
                    [:base <base>])

This is an often useful alternative to make-type-option.

We construct a new type-option structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-type-option

(defmacro
     change-type-option (x &rest args)
     (std::change-aggregate 'type-option
                            x args '((:base . type-option->base))
                            'change-type-option
                            'nil))
