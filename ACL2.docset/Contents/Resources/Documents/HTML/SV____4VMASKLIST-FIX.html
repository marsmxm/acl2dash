(4vmasklist-fix x) is a usual fty list fixing function.

Signature
(4vmasklist-fix x) → fty::newx
Arguments
x — Guard (4vmasklist-p x).
Returns
fty::newx — Type (4vmasklist-p fty::newx).

In the logic, we apply 4vmask-fix to each member of the x. In the execution, none of that is actually necessary and this is just an inlined identity function.
