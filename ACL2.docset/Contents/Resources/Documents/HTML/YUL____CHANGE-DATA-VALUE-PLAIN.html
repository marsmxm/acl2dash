Modifying constructor for data-value-plain structures.

Syntax
(change-data-value-plain x 
                         [:get <get>])

This is an often useful alternative to make-data-value-plain.

We construct a new data-value-plain structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-data-value-plain

(defmacro
     change-data-value-plain (x &rest args)
     (std::change-aggregate 'data-value-plain

                            'nil))
