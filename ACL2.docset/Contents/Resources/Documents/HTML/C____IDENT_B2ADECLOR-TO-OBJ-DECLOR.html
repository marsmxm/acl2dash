Compose an identifier and an abstract object declarator into an object declarator.

Signature
(ident+adeclor-to-obj-declor id adeclor) → declor
Arguments
id — Guard (identp id).
adeclor — Guard (obj-adeclorp adeclor).
Returns
declor — Type (obj-declorp declor).

This is the inverse of obj-declor-to-ident+adeclor.

 ident+adeclor-to-obj-declor (id adeclor)
 (declare (xargs :guard (and (identp id)
                             (obj-adeclorp adeclor))))
 (let
  ((__function__ 'ident+adeclor-to-obj-declor))
  (declare (ignorable __function__))
  (obj-adeclor-case
     adeclor

                 :decl (ident+adeclor-to-obj-declor id adeclor.decl)
                 :size adeclor.size))))
