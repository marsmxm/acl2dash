Modifying constructor for character-result-err structures.

Syntax
(change-character-result-err x 
                             [:get <get>])

This is an often useful alternative to make-character-result-err.

We construct a new character-result-err structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-character-result-err

(defmacro
