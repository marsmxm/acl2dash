Modifying constructor for function-specifier-regular structures.

Syntax
(change-function-specifier-regular x 
                                   [:body <body>])

This is an often useful alternative to make-function-specifier-regular.

We construct a new function-specifier-regular structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-function-specifier-regular

(defmacro change-function-specifier-regular
          (x &rest args)
          (std::change-aggregate
               'function-specifier-regular
               x args
               '((:body . function-specifier-regular->body))
               'change-function-specifier-regular
               'nil))
