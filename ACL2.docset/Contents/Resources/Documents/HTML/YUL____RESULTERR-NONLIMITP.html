Recognize non-limit errors.

Signature
(resulterr-nonlimitp x) → yes/no
Returns
yes/no — Type (booleanp yes/no).

This recognizes all the errors that are not recognized by resulterr-limitp. See that recognizer's documentation.

       (let ((__function__ 'resulterr-nonlimitp))
