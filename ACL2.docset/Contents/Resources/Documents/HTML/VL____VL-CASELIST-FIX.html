(vl-caselist-fix x) is an fty alist fixing function that follows the fix-keys strategy.

Signature
(vl-caselist-fix x) → fty::newx
Arguments
x — Guard (vl-caselist-p x).
Returns
fty::newx — Type (vl-caselist-p fty::newx).

Note that in the execution this is just an inline identity function.
