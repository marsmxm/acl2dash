Constructor macro for vl-plaintoken-p structures.

Syntax:

(make-vl-plaintoken [:type <type>] 
                    [:etext <etext>])

This is our preferred way to construct vl-plaintoken-p structures. It simply conses together a structure with the specified fields.

This macro generates a new vl-plaintoken-p structure from scratch. See also change-vl-plaintoken, which can "change" an existing structure, instead.

The vl-plaintoken-p structures we create here are just constructed with ordinary cons. If you want to create honsed structures, see make-honsed-vl-plaintoken instead.

Definition

This is an ordinary make- macro introduced by defaggregate.

Macro: make-vl-plaintoken

(defmacro make-vl-plaintoken (&rest args)
          (std::make-aggregate 'vl-plaintoken
                               args '((:type) (:etext))
                               'make-vl-plaintoken
                               nil))
