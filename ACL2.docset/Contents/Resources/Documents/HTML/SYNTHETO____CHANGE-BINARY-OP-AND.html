Modifying constructor for binary-op-and structures.

Syntax
(change-binary-op-and x 
                      )

This is an often useful alternative to make-binary-op-and.

We construct a new binary-op-and structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-binary-op-and

(defmacro change-binary-op-and (x &rest args)
          (std::change-aggregate 'binary-op-and
