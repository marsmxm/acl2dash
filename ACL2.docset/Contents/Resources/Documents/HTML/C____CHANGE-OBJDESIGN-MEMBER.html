Modifying constructor for objdesign-member structures.

Syntax
(change-objdesign-member x 
                         [:super <super>] 
                         [:name <name>])

This is an often useful alternative to make-objdesign-member.

We construct a new objdesign-member structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-objdesign-member

(defmacro
     change-objdesign-member (x &rest args)
     (std::change-aggregate 'objdesign-member
                            x args
                            '((:super . objdesign-member->super)
                              (:name . objdesign-member->name))
                            'change-objdesign-member
