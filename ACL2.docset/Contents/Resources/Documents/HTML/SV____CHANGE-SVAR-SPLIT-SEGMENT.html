Modifying constructor for svar-split-segment structures.

Syntax
(change-svar-split-segment x 
                           [:width <width>] 
                           [:var <var>] 
                           [:rest <rest>])

This is an often useful alternative to make-svar-split-segment.

We construct a new svar-split-segment structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.
