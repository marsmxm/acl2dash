Modifying constructor for type-result-ok structures.

Syntax
(change-type-result-ok x 
                       [:types <types>] 
                       [:obligations <obligations>])

This is an often useful alternative to make-type-result-ok.

We construct a new type-result-ok structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-type-result-ok

(defmacro change-type-result-ok (x &rest args)
          (std::change-aggregate
               'type-result-ok

                 (:obligations . type-result-ok->obligations))
