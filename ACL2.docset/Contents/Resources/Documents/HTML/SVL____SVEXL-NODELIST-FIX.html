(svexl-nodelist-fix x) is a usual ACL2::fty list fixing function.

Signature
(svexl-nodelist-fix x)  fty::newx
Arguments
x Guard (svexl-nodelist-p x).
Returns
fty::newx Type (svexl-nodelist-p fty::newx).

In the logic, we apply svexl-node-fix to each member of the x. In the execution, none of that is actually necessary and this is just an inlined identity function.
