Number of Counterexamples to be searched

Set the number of counterexamples desired to be searched. By default this parameter is set to 3. Can be set to any natural number n. Setting this number to 0 implies the user is not interested in searching for counterexamples. 
Usage: 
(acl2s-defaults :set num-counterexamples 3) 
(acl2s-defaults :get num-counterexamples)
