Basic constructor macro for resulterr structures.

Syntax
(make-resulterr [:info <info>] 
                [:stack <stack>])

This is the usual way to construct resulterr structures. It simply conses together a structure with the specified fields.

This macro generates a new resulterr structure from scratch. See also change-resulterr, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by defprod.

Macro: make-resulterr

(defmacro make-resulterr (&rest args)
          (std::make-aggregate 'resulterr
                               args '((:info) (:stack))
                               'make-resulterr
                               nil))
