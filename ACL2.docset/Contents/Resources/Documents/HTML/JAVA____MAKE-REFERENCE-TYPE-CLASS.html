Basic constructor macro for reference-type-class structures.

Syntax
(make-reference-type-class [:get <get>])

This is the usual way to construct reference-type-class structures. It simply conses together a structure with the specified fields.

This macro generates a new reference-type-class structure from scratch. See also change-reference-type-class, which can "change" an existing structure, instead.
