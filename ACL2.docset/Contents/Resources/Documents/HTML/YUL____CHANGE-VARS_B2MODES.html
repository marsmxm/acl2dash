Modifying constructor for vars+modes structures.

Syntax
(change-vars+modes x 
                   [:vars <vars>] 
                   [:modes <modes>])

This is an often useful alternative to make-vars+modes.

We construct a new vars+modes structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-vars+modes

(defmacro change-vars+modes (x &rest args)
          (std::change-aggregate 'vars+modes
                                 x args
                                 '((:vars . vars+modes->vars)
                                   (:modes . vars+modes->modes))
                                 'change-vars+modes
                                 'nil))
