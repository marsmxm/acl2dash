Modifying constructor for atj-atype-symbol structures.

Syntax
(change-atj-atype-symbol x 
                         )

This is an often useful alternative to make-atj-atype-symbol.

We construct a new atj-atype-symbol structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
