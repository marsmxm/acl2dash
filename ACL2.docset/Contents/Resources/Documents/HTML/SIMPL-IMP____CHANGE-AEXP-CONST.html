Modifying constructor for aexp-const structures.

Syntax
(change-aexp-const x 
                   [:value <value>])

This is an often useful alternative to make-aexp-const.

We construct a new aexp-const structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-aexp-const

(defmacro
     change-aexp-const (x &rest args)
     (std::change-aggregate 'aexp-const
                            x args '((:value . aexp-const->value))
                            'change-aexp-const
                            'nil))
