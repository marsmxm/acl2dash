Modifying constructor for tag-env-result-ok structures.

Syntax
(change-tag-env-result-ok x 
                          [:get <get>])

This is an often useful alternative to make-tag-env-result-ok.

We construct a new tag-env-result-ok structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
