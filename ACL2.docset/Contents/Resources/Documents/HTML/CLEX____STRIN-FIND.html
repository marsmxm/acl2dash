Find the first occurrence of the string literal in the input stream, and return its position.

Signature
(strin-find lit x) → pos
Arguments
lit — Guard (stringp lit).
x — Guard (strin-p x).
Returns
pos — Starting index of the first match, or nil if there are no matches.

Function: strin-find

(defun strin-find (lit x)
       (declare (xargs :guard (and (stringp lit) (strin-p x))))
       (let ((__function__ 'strin-find))
            (declare (ignorable __function__))
            (listpos (coerce lit 'list)
                     (strin-left x))))
