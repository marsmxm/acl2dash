Identity, except Z bits become 0.

Signature
(4vec-onset x)  res
Arguments
x Guard (4vec-p x).
Returns
res Type (4vec-p res).

The closest monotonic approximation of the onset of each of the bits of the input. It differs from the actual onset in that the 4vec-onset of an X bit is X.

Function: 4vec-onset

(defun 4vec-onset (x)
       (declare (xargs :guard (4vec-p x)))
       (let ((__function__ '4vec-onset))
            (declare (ignorable __function__))
            (b* (((4vec x)))
                (4vec x.upper (logand x.upper x.lower)))))
