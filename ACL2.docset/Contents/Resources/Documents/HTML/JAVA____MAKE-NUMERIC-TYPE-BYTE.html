Basic constructor macro for numeric-type-byte structures.

This is the usual way to construct numeric-type-byte structures. It simply conses together a structure with the specified fields.

This macro generates a new numeric-type-byte structure from scratch. See also change-numeric-type-byte, which can "change" an existing structure, instead.
