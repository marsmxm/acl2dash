Modifying constructor for vl-returnstmt structures.

Syntax
(change-vl-returnstmt x 
                      [:val <val>] 
                      [:atts <atts>] 
                      [:loc <loc>])

This is an often useful alternative to make-vl-returnstmt.

We construct a new vl-returnstmt structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-returnstmt

(defmacro change-vl-returnstmt (x &rest args)
          (std::change-aggregate 'vl-returnstmt
                                 x args
                                 '((:val . vl-returnstmt->val)
                                   (:atts . vl-returnstmt->atts)
                                   (:loc . vl-returnstmt->loc))
                                 'change-vl-returnstmt
                                 'remake-vl-returnstmt))
