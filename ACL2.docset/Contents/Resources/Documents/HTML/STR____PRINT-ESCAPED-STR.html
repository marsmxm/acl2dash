Print a string, escaping backslashes and some other character.

Signature
(print-escaped-str x slash-char acc) → new-acc
Arguments
x — String or symbol name to be printed.
    Guard (stringp x).
slash-char — Extra character to escape. When printing escaped strings this is typically the " character; for printing symbol names it would typically be the | character.
    Guard (characterp slash-char).
Returns
new-acc — Type (character-listp new-acc), given (character-listp acc).
