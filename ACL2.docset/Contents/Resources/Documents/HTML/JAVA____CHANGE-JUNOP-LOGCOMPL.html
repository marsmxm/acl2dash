Modifying constructor for junop-logcompl structures.

Syntax
(change-junop-logcompl x 
                       )

This is an often useful alternative to make-junop-logcompl.

We construct a new junop-logcompl structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
