Match { attribute_instance }, collecting all attributes in the order they were seen, without proper duplicity checking.

Signature
(vl-parse-0+-attribute-instances-aux &key (tokstream 'tokstream) 
                                     (config 'config)) 
 
  → 
(mv errmsg? value new-tokstream)
Arguments
config — Guard (vl-loadconfig-p config).

We convert each individual attribute_instance into an vl-atts-p alist, and then merge these together using append, so that for instance:

(* foo *) (* bar *)

Is treated in just the same way as:

(* foo, bar *)
