Convert an alist to a list of two-element lists

The call (alist-to-doublets alist) returns the result of replacing each pair (x . y) in the given alist by the two-element list (x y). The order is preserved, i.e., the following is a theorem.

(implies (and (natp i) (< i (len alist)))
         (equal (nth i (alist-to-doublets alist))
                (let ((pair (nth i alist)))
                  (list (car pair) (cdr pair)))))
