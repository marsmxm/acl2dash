Modifying constructor for atj-function-type-info structures.

Syntax
(change-atj-function-type-info x 
                               [:main <main>] 
                               [:others <others>])

This is an often useful alternative to make-atj-function-type-info.

We construct a new atj-function-type-info structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-atj-function-type-info

(defmacro change-atj-function-type-info
          (x &rest args)
          (std::change-aggregate
               'atj-function-type-info
               x args
               '((:main . atj-function-type-info->main)
                 (:others . atj-function-type-info->others))
               'change-atj-function-type-info
               'nil))
