Modifying constructor for comm-while structures.

Syntax
(change-comm-while x 
                   [:cond <cond>] 
                   [:body <body>])

This is an often useful alternative to make-comm-while.

We construct a new comm-while structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-comm-while

(defmacro change-comm-while (x &rest args)
          (std::change-aggregate 'comm-while
                                 x args
                                 '((:cond . comm-while->cond)
                                   (:body . comm-while->body))
                                 'change-comm-while
                                 'nil))
