Fallback names in case our preferred names aren't available. Only called if there is a name conflict that prevents us from using good names.

Signature
(vl-bad-replicate-names n basename nf) → (mv names nf)
Arguments
n — Guard (natp n).
basename — Guard (stringp basename).
nf — Guard (vl-namefactory-p nf).
Returns
names — Type (string-listp names).
nf — Type (vl-namefactory-p nf).

Function: vl-bad-replicate-names

(defun vl-bad-replicate-names (n basename nf)
       (declare (xargs :guard (and (natp n)
                                   (stringp basename)
                                   (vl-namefactory-p nf))))
       (let ((__function__ 'vl-bad-replicate-names))
            (declare (ignorable __function__))
            (b* (((when (zp n))
                  (mv nil (vl-namefactory-fix nf)))
                 ((mv name nf)
                  (vl-namefactory-indexed-name basename nf))
                 ((mv others nf)
                  (vl-bad-replicate-names (- n 1)
                                          basename nf)))
                (mv (cons name others) nf))))
