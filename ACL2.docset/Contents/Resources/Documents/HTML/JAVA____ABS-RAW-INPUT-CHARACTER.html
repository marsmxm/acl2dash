Abstract a raw-input-character tree to a Unicode character.

Signature
(abs-raw-input-character tree) → unicode
Arguments
tree — Guard (abnf-tree-with-root-p tree "raw-input-character").
Returns
unicode — Type (unicodep unicode), given the guard.

A raw-input-character tree is essentially a single Unicode character, which is unchanged by Java's first lexical translation step. In other words, we abstract this kind of tree to its single Unicode character.

A raw-input-character has a single subtree, accessible via caar. That subtree is a terminal leaf tree with a single terminal (natural number) in the range of Java Unicode characters. That is the result.
