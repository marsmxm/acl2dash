Modifying constructor for floating-point-type-double structures.

Syntax
(change-floating-point-type-double x 
                                   )

This is an often useful alternative to make-floating-point-type-double.

We construct a new floating-point-type-double structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
