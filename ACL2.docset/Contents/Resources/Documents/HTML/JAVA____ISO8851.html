Fixtype of ISO 8851-1 characters.

The ISO 8851-1 characters are the first 256 Unicode characters.

Since we model Java Unicode characters as unsigned 16-bit integers, we model ISO 8851-1 characters as unsigned 8-bit integers.

The names of this fixtype and of its recognizer, fixer, and equivalence omit the last `1' in `ISO 8851-1' for brevity. In the context of our Java language formalization, `ISO 8859' only refers to Part 1 of the ISO 8859 standard.

Note that ACL2 characters are consistent with ISO 8851-1. Thus, this fixtype is isomorphic to the ACL2 characters.
