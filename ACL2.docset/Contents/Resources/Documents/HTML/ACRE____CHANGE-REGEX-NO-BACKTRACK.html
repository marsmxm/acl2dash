Modifying constructor for regex-no-backtrack structures.

Syntax
(change-regex-no-backtrack x 
                           [:pat <pat>])

This is an often useful alternative to make-regex-no-backtrack.

We construct a new regex-no-backtrack structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-regex-no-backtrack

(defmacro
     change-regex-no-backtrack (x &rest args)
     (std::change-aggregate 'regex-no-backtrack
                            x
                            args '((:pat . regex-no-backtrack->pat))
                            'change-regex-no-backtrack
                            'nil))
