Modifying constructor for def-parse-function-spec-option structures.

Syntax
(change-def-parse-function-spec-option x 
                                       [:get <get>] 
                                       [:order <order>])

This is an often useful alternative to make-def-parse-function-spec-option.

We construct a new def-parse-function-spec-option structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
