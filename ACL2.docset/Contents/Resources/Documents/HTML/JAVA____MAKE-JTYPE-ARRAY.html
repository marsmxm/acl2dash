Basic constructor macro for jtype-array structures.

Syntax
(make-jtype-array [:comp <comp>])

This is the usual way to construct jtype-array structures. It simply conses together a structure with the specified fields.

This macro generates a new jtype-array structure from scratch. See also change-jtype-array, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by fty::defprod.

(defun jtype-array (comp)
       (declare (xargs :guard (jtypep comp)))
       (declare (xargs :guard t))
       (let ((__function__ 'jtype-array))
            (declare (ignorable __function__))
            (b* ((comp (mbe :logic (jtype-fix comp)
                            :exec comp)))
                (cons :array (list comp)))))
