Modifying constructor for type-uint structures.

Syntax
(change-type-uint x 
                  )

This is an often useful alternative to make-type-uint.

We construct a new type-uint structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-type-uint

(defmacro change-type-uint (x &rest args)
          (std::change-aggregate 'type-uint
