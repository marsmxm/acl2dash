Modifying constructor for lhbit-z structures.

Syntax
(change-lhbit-z x 
                )

This is an often useful alternative to make-lhbit-z.

We construct a new lhbit-z structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-lhbit-z

(defmacro change-lhbit-z (x &rest args)
          (std::change-aggregate 'lhbit-z
                                 x args 'nil
                                 'change-lhbit-z
                                 'nil))
