(assigns-fix x) is an fty alist fixing function that follows the fix-keys strategy.

Signature
(assigns-fix x) → fty::newx
Arguments
x — Guard (assigns-p x).
Returns
fty::newx — Type (assigns-p fty::newx).

Note that in the execution this is just an inline identity function.

(defun assigns-fix$inline (x)

       (let ((__function__ 'assigns-fix))
