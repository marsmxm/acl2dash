Modifying constructor for constraint structures.

Syntax
(change-constraint x 
                   [:name <name>] 
                   [:cond <cond>])

This is an often useful alternative to make-constraint.

We construct a new constraint structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-constraint

(defmacro change-constraint (x &rest args)
          (std::change-aggregate 'constraint
                                 x args
                                 '((:name . constraint->name)
                                   (:cond . constraint->cond))
                                 'change-constraint
                                 'remake-constraint))
