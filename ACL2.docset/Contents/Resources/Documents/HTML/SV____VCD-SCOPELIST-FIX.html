(vcd-scopelist-fix x) is a usual fty list fixing function.

Signature
(vcd-scopelist-fix x)  fty::newx
Arguments
x Guard (vcd-scopelist-p x).
Returns
fty::newx Type (vcd-scopelist-p fty::newx).

In the logic, we apply vcd-scope-fix to each member of the x. In the execution, none of that is actually necessary and this is just an inlined identity function.
