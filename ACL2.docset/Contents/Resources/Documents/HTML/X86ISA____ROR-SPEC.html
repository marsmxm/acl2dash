Specification for the ROR instruction

Signature

Source: Intel Manual, Volume 2B, Instruction Set Reference (N-Z).

For the ROR and ROR instructions, the orig- inal value of the CF flag is not a part of the result, but the CF flag receives a copy of the bit that was shifted from one end to the other. ... The OF flag is defined only for the 1-bit rotates; it is undefined in all other cases (except RCL and RCR instructions only: a zero-bit rotate does nothing, that is affects no flags). For right rotates, the OF flag is set to the exclusive OR of the two most-significant bits of the result.

Function: ror-spec$inline

(defun ror-spec$inline
       (size dst src input-rflags)
       (declare (type (member 1 2 4 8) size)
                (type (unsigned-byte 32) input-rflags))
       (declare (xargs :guard (and (n06p src)
                                   (case size (1 (n08p dst))
                                         (2 (n16p dst))
                                         (4 (n32p dst))
                                         (8 (n64p dst))
                                         (otherwise nil)))))
       (case size
             (1 (ror-spec-8 dst src input-rflags))
             (2 (ror-spec-16 dst src input-rflags))
             (4 (ror-spec-32 dst src input-rflags))
             (8 (ror-spec-64 dst src input-rflags))
             (otherwise (mv 0 0 0))))
