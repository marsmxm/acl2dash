Process the target functions fn1, ..., fnp.

Signature
(atc-process-fn1...fnp fn1...fnp ctx state) 
  → 
(mv erp nothing state)
Arguments
fn1...fnp — Guard (true-listp fn1...fnp).
ctx — Guard (ctxp ctx).
Returns
nothing — Type (null nothing).

We initialize the list of uncalled recursive functions to be empty, and we ensure that the list is empty after processing all functions.

Definitions and Theorems

Function: atc-process-fn1...fnp

(defun
 atc-process-fn1...fnp
 (fn1...fnp ctx state)
 (declare (xargs :stobjs (state)))
 (declare (xargs :guard (and (true-listp fn1...fnp)
                             (ctxp ctx))))
 (let
  ((__function__ 'atc-process-fn1...fnp))
  (declare (ignorable __function__))
  (b*
   (((er uncalled-fns)
     (atc-process-function-list fn1...fnp nil ctx state))
    ((unless (consp fn1...fnp))
     (er-soft+ ctx t nil
               "At least one target function must be supplied."))
    ((er &)
     (ensure-list-has-no-duplicates$
          fn1...fnp
          (msg "The list of target functions ~x0"
               fn1...fnp)
          t nil))
    ((unless (endp uncalled-fns))
     (er-soft+
      ctx t nil
      "The recursive target functions ~x0 ~
                   are not called by any other target function."
      uncalled-fns)))
   (acl2::value nil))))
