Modifying constructor for exec-outcome-error structures.

Syntax
(change-exec-outcome-error x 
                           )

This is an often useful alternative to make-exec-outcome-error.

We construct a new exec-outcome-error structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
