Modifying constructor for vttree-context structures.

Syntax
(change-vttree-context x 
                       [:ctx <ctx>] 
                       [:subtree <subtree>])

This is an often useful alternative to make-vttree-context.

We construct a new vttree-context structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vttree-context

(defmacro
     change-vttree-context (x &rest args)
     (std::change-aggregate 'vttree-context
