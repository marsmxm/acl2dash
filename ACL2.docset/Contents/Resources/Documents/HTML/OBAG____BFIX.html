Fixing function for obags.

Signature
(bfix x) → fixed-x
Arguments
x — Guard (bagp x).
Returns
fixed-x — Type (bagp fixed-x).

This is similar to sfix for osets.

Function: bfix

(defun bfix (x)
       (declare (xargs :guard (bagp x)))
       (let ((__function__ 'bfix))

            (mbe :logic (if (bagp x) x nil)
