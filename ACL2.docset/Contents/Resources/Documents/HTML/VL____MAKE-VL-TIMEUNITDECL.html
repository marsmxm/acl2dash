Basic constructor macro for vl-timeunitdecl structures.

Syntax
(make-vl-timeunitdecl [:numerator <numerator>] 
                      [:denominator <denominator>] 
                      [:loc <loc>])

This is the usual way to construct vl-timeunitdecl structures. It simply conses together a structure with the specified fields.

This macro generates a new vl-timeunitdecl structure from scratch. See also change-vl-timeunitdecl, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by defprod.

(defun
 vl-timeunitdecl
 (numerator denominator loc)
 (declare (xargs :guard (and (vl-timeliteral-p numerator)
                             (vl-maybe-timeliteral-p denominator)
                             (vl-location-p loc))))
 (declare (xargs :guard t))
 (let
  ((__function__ 'vl-timeunitdecl))
  (declare (ignorable __function__))
  (b*
   ((numerator (mbe :logic (vl-timeliteral-fix numerator)
                    :exec numerator))
    (denominator (mbe :logic (vl-maybe-timeliteral-fix denominator)
                      :exec denominator))
    (loc (mbe :logic (vl-location-fix loc)
              :exec loc)))
   (cons
        :vl-timeunitdecl (cons numerator (cons denominator loc))))))
