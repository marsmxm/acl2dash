Modifying constructor for vl-id structures.

Syntax
(change-vl-id x 
              [:name <name>])

This is an often useful alternative to make-vl-id.

We construct a new vl-id structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-id

(defmacro change-vl-id (x &rest args)
          (std::change-aggregate 'vl-id
                                 x args '((:name . vl-id->name))
                                 'change-vl-id
                                 'nil))
