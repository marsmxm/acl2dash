Check if two function definitions are related by function renaming.

Signature
(fundef-renamefun old new ren) → _
Arguments
old — Guard (fundefp old).
new — Guard (fundefp new).
ren — Guard (renamingp ren).
Returns
_ — Type (resulterr-optionp _).

When we process two function definitions, their association pair has already been added to the function renaming: see the treatment of blocks. The inputs and outputs must be the same. The bodies must be related by function renaming.
