Modifying constructor for expr-grade-shift structures.

Syntax
(change-expr-grade-shift x 
                         )

This is an often useful alternative to make-expr-grade-shift.

We construct a new expr-grade-shift structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
