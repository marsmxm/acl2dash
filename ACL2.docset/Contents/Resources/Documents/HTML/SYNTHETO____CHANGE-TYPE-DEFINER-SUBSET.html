Modifying constructor for type-definer-subset structures.

Syntax
(change-type-definer-subset x 
                            [:get <get>])

This is an often useful alternative to make-type-definer-subset.

We construct a new type-definer-subset structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
