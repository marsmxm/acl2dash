Modifying constructor for const-int structures.

Syntax
(change-const-int x 
                  [:get <get>])

This is an often useful alternative to make-const-int.

We construct a new const-int structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-const-int

(defmacro change-const-int (x &rest args)
          (std::change-aggregate 'const-int
                                 x args '((:get . const-int->get))
                                 'change-const-int
                                 'nil))
