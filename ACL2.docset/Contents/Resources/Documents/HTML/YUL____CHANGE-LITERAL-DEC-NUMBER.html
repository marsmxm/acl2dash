Modifying constructor for literal-dec-number structures.

Syntax
(change-literal-dec-number x 
                           [:get <get>])

This is an often useful alternative to make-literal-dec-number.

We construct a new literal-dec-number structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-literal-dec-number

(defmacro
     change-literal-dec-number (x &rest args)
     (std::change-aggregate 'literal-dec-number
                            x
                            args '((:get . literal-dec-number->get))
                            'change-literal-dec-number
                            'nil))
