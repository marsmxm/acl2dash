Modifying constructor for maybe-stat-some structures.

Syntax
(change-maybe-stat-some x 
                        [:val <val>])

This is an often useful alternative to make-maybe-stat-some.

We construct a new maybe-stat-some structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-maybe-stat-some

(defmacro
