Modifying constructor for type-defined structures.

Syntax
(change-type-defined x 
                     [:name <name>])

This is an often useful alternative to make-type-defined.

We construct a new type-defined structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-type-defined

(defmacro
     change-type-defined (x &rest args)
     (std::change-aggregate 'type-defined
                            x args '((:name . type-defined->name))
                            'change-type-defined
                            'nil))
