Recognizer for strings and nil.

This is like an option type; when x satisfies maybe-stringp, then either it is a string or nothing.

Definitions and Theorems

Function: maybe-stringp$inline

(defun maybe-stringp$inline (x)

Theorem: maybe-stringp-compound-recognizer

(defthm maybe-stringp-compound-recognizer

               (or (not x) (stringp x)))
