Basic constructor macro for type-sum structures.

Syntax
(make-type-sum [:alternatives <alternatives>])

This is the usual way to construct type-sum structures. It simply conses together a structure with the specified fields.

This macro generates a new type-sum structure from scratch. See also change-type-sum, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by fty::defprod.
