Modifying constructor for string-list-result-ok structures.

Syntax
(change-string-list-result-ok x 
                              [:get <get>])

This is an often useful alternative to make-string-list-result-ok.

We construct a new string-list-result-ok structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
