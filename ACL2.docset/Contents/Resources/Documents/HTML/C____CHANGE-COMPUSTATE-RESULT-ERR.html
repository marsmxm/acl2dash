Modifying constructor for compustate-result-err structures.

Syntax
(change-compustate-result-err x 
                              [:get <get>])

This is an often useful alternative to make-compustate-result-err.

We construct a new compustate-result-err structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
