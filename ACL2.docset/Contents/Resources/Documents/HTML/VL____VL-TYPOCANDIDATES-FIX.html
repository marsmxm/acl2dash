(vl-typocandidates-fix x) is an fty alist fixing function that follows the fix-keys strategy.

Signature
(vl-typocandidates-fix x) → fty::newx
Arguments
x — Guard (vl-typocandidates-p x).
Returns
fty::newx — Type (vl-typocandidates-p fty::newx).

Note that in the execution this is just an inline identity function.
