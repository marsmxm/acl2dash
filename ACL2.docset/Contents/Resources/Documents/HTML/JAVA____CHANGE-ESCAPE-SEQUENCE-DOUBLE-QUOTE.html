Modifying constructor for escape-sequence-double-quote structures.

Syntax
(change-escape-sequence-double-quote x 
                                     )

This is an often useful alternative to make-escape-sequence-double-quote.

We construct a new escape-sequence-double-quote structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
