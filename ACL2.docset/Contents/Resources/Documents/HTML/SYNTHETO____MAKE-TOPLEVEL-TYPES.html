Basic constructor macro for toplevel-types structures.

Syntax
(make-toplevel-types [:get <get>])

This is the usual way to construct toplevel-types structures. It simply conses together a structure with the specified fields.

This macro generates a new toplevel-types structure from scratch. See also change-toplevel-types, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by fty::defprod.

Macro: make-toplevel-types

(defmacro make-toplevel-types (&rest args)
