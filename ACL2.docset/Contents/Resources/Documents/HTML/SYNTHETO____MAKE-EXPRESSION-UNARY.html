Basic constructor macro for expression-unary structures.

Syntax
(make-expression-unary [:operator <operator>] 
                       [:operand <operand>])

This is the usual way to construct expression-unary structures. It simply conses together a structure with the specified fields.

This macro generates a new expression-unary structure from scratch. See also change-expression-unary, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by fty::defprod.

(defun expression-unary (operator operand)
       (declare (xargs :guard (and (unary-opp operator)
                                   (expressionp operand))))
       (declare (xargs :guard t))
       (let ((__function__ 'expression-unary))
            (declare (ignorable __function__))
            (b* ((operator (mbe :logic (unary-op-fix operator)
                                :exec operator))
                 (operand (mbe :logic (expression-fix operand)
                               :exec operand)))
                (cons :unary (list operator operand)))))
