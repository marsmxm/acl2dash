Places where a bind statement may occur.

This is a ``transparent'' sum type introduced using deftranssum. It is simply any one of the following types:

  • vl-interface
  • vl-module
  • vl-design

This is only intended for error reporting / attribute creation. In particular you should not use this to do scope lookups, because you need real scopestacks for that.
