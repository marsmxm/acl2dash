An expression range is a temporary internal representation of the ranges for select-like expressions (bit selects, array indexes, part selects, +: and -: expressions.

(vl-erange-p x) is a defaggregate of the following fields.

  • type Invariant (vl-erangetype-p type).
  • left Invariant (vl-expr-p left).
  • right Invariant (vl-expr-p right).

Source link: vl-erange-p

For single-bit selects, we just set left and right to the same expression.
