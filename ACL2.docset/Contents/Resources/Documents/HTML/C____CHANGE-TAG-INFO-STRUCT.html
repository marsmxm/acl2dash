Modifying constructor for tag-info-struct structures.

Syntax
(change-tag-info-struct x 
                        [:members <members>])

This is an often useful alternative to make-tag-info-struct.

We construct a new tag-info-struct structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-tag-info-struct

(defmacro
     change-tag-info-struct (x &rest args)
     (std::change-aggregate 'tag-info-struct
                            x args
                            '((:members . tag-info-struct->members))
                            'change-tag-info-struct
                            'nil))
