Cgen/Testing timeout (in seconds)

Maximum allowed time (in seconds) for Cgen to search for counterexamples to a particular form/subgoal. The default timeout limit is set to 10 sec. Setting this parameter to 0 amounts to disabling the timeout mechanism, i.e. its a no-op. Guard : Timeout should be a non-negative rational. 
Usage: 
(acl2s-defaults :set cgen-local-timeout 10) 
(acl2s-defaults :get cgen-local-timeout)
