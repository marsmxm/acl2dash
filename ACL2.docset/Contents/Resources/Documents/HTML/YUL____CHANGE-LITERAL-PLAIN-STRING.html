Modifying constructor for literal-plain-string structures.

Syntax
(change-literal-plain-string x 
                             [:get <get>])

This is an often useful alternative to make-literal-plain-string.

We construct a new literal-plain-string structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-literal-plain-string

(defmacro
     change-literal-plain-string
     (x &rest args)
     (std::change-aggregate 'literal-plain-string
                            x args
                            '((:get . literal-plain-string->get))
                            'change-literal-plain-string
                            'nil))
