Basic constructor macro for expr-cast structures.

Syntax
(make-expr-cast [:type <type>] 
                [:arg <arg>])

This is the usual way to construct expr-cast structures. It simply conses together a structure with the specified fields.

This macro generates a new expr-cast structure from scratch. See also change-expr-cast, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by fty::defprod.

Macro: make-expr-cast

(defmacro make-expr-cast (&rest args)
          (std::make-aggregate 'expr-cast
                               args '((:type) (:arg))
                               'make-expr-cast
                               nil))

(defun expr-cast (type arg)
       (declare (xargs :guard (and (tynamep type) (exprp arg))))
       (declare (xargs :guard t))
       (let ((__function__ 'expr-cast))
            (declare (ignorable __function__))
            (b* ((type (mbe :logic (tyname-fix type)
                            :exec type))
                 (arg (mbe :logic (expr-fix arg) :exec arg)))
