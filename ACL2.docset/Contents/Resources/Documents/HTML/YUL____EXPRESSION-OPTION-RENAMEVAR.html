Check if two optional expressions are related by variable renaming.

Signature
(expression-option-renamevar old new ren) → _
Arguments
old — Guard (expression-optionp old).
new — Guard (expression-optionp new).
ren — Guard (renamingp ren).
Returns
_ — Type (resulterr-optionp _).

Either both expressions must be present or both must be absent. If present, they must be related.

Function: expression-option-renamevar

(defun
 expression-option-renamevar
 (old new ren)
 (declare (xargs :guard (and (expression-optionp old)
                             (expression-optionp new)
                             (renamingp ren))))
 (let
  ((__function__ 'expression-option-renamevar))
  (declare (ignorable __function__))
  (expression-option-case
   old
   :none (if (expression-option-case new :none)
             nil
             (err (list :mismatch (expression-option-fix old)
                        (expression-option-fix new))))
   :some
   (expression-option-case
       new
       :none (err (list :mismatch (expression-option-fix old)
                        (expression-option-fix new)))
       :some (expression-renamevar (expression-option-some->val old)
                                   (expression-option-some->val new)
                                   ren)))))
