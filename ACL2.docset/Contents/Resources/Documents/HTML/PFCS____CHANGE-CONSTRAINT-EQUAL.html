Modifying constructor for constraint-equal structures.

Syntax
(change-constraint-equal x 
                         [:left <left>] 
                         [:right <right>])

This is an often useful alternative to make-constraint-equal.

We construct a new constraint-equal structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-constraint-equal

(defmacro
     change-constraint-equal (x &rest args)
     (std::change-aggregate 'constraint-equal
                            x args
                            '((:left . constraint-equal->left)
                              (:right . constraint-equal->right))
                            'change-constraint-equal
                            'nil))
