Modifying constructor for vl-patternkey-type structures.

Syntax
(change-vl-patternkey-type x 
                           [:type <type>])

This is an often useful alternative to make-vl-patternkey-type.

We construct a new vl-patternkey-type structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
