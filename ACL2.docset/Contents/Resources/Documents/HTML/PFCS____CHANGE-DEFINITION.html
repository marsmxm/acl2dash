Modifying constructor for definition structures.

Syntax
(change-definition x 
                   [:name <name>] 
                   [:para <para>] 
                   [:body <body>])

This is an often useful alternative to make-definition.

We construct a new definition structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-definition

(defmacro change-definition (x &rest args)
          (std::change-aggregate 'definition
                                 x args
                                 '((:name . definition->name)
                                   (:para . definition->para)
                                   (:body . definition->body))
                                 'change-definition
                                 'nil))
