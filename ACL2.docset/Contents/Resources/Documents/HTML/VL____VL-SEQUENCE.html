Represents a single sequence...endsequence declaration.

This is a product type introduced by defprod.

Fields
name — stringp
E.g., this is foo for sequence foo ... endsequence.
ports — vl-propportlist
The ports for this sequence declaration.
decls — vl-vardecllist
Variable declarations for this sequence.
expr — vl-propexpr
The main content of the sequence declaration. The expression should be a valid sequence expression. We represent it as a vl-propexpr, which is overly permissive but sufficient to capture any sequence expression.
loc — vl-location
