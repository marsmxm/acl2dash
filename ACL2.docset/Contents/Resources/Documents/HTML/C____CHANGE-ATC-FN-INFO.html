Modifying constructor for atc-fn-info structures.

Syntax
(change-atc-fn-info x 
                    [:out-type <out-type>] 
                    [:in-types <in-types>] 
                    [:loop? <loop?>] 
                    [:affect <affect>] 
                    [:result-thm <result-thm>] 
                    [:correct-thm <correct-thm>] 
                    [:measure-nat-thm <measure-nat-thm>] 
                    [:fun-env-thm <fun-env-thm>] 
                    [:limit <limit>])

This is an often useful alternative to make-atc-fn-info.

We construct a new atc-fn-info structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
