Kinds of rewrite rules associated to defun-sk functions with the universal quantifier.

These correspond to the values of the :rewrite option of defun-sk, with :custom standing for anything but :default or :direct.

This is an ordinary std::defenum.

Function: defun-sk-rewrite-kind-p

(defun defun-sk-rewrite-kind-p (x)
       (declare (xargs :guard t))
       (or (eq x ':default)
           (eq x ':direct)
           (eq x ':custom)))
