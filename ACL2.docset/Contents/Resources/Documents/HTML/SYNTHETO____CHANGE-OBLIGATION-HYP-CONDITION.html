Modifying constructor for obligation-hyp-condition structures.

Syntax
(change-obligation-hyp-condition x 
                                 [:get <get>])

This is an often useful alternative to make-obligation-hyp-condition.

We construct a new obligation-hyp-condition structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-obligation-hyp-condition

(defmacro
    change-obligation-hyp-condition
    (x &rest args)
    (std::change-aggregate 'obligation-hyp-condition
                           x args
                           '((:get . obligation-hyp-condition->get))
                           'change-obligation-hyp-condition
                           'nil))
