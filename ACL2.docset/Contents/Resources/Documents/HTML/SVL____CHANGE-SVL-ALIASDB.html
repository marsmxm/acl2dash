Modifying constructor for svl-aliasdb structures.

Syntax
(change-svl-aliasdb x 
                    [:this <this>] 
                    [:sub <sub>])

This is an often useful alternative to make-svl-aliasdb.

We construct a new svl-aliasdb structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-svl-aliasdb

(defmacro change-svl-aliasdb (x &rest args)
          (std::change-aggregate 'svl-aliasdb
                                 x args
                                 '((:this . svl-aliasdb->this)
                                   (:sub . svl-aliasdb->sub))
                                 'change-svl-aliasdb
                                 'nil))
