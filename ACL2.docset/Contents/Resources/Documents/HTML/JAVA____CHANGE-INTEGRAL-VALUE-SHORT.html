Modifying constructor for integral-value-short structures.

Syntax
(change-integral-value-short x 
                             [:get <get>])

This is an often useful alternative to make-integral-value-short.

We construct a new integral-value-short structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-integral-value-short

(defmacro
     change-integral-value-short
     (x &rest args)
     (std::change-aggregate 'integral-value-short
                            x args
                            '((:get . integral-value-short->get))
                            'change-integral-value-short
                            'nil))
