Modifying constructor for pflat structures.

Syntax
(change-pflat x 
              [:width <width>] 
              [:what <what>])

This is an often useful alternative to make-pflat.

We construct a new pflat structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-pflat

(defmacro change-pflat (x &rest args)
          (std::change-aggregate 'pflat
                                 x args
                                 '((:width . pflat->width)
                                   (:what . pflat->what))
                                 'change-pflat
                                 'remake-pflat))
