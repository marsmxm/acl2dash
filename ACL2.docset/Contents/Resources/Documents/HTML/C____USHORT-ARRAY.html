Fixtype of (ATC's model of) arrays of type unsigned short.

This is a product type introduced by fty::defprod.

Fields

elements — ushort-list
Additional Requirements

The following invariant is enforced on the fields:

(and (type-case elemtype :ushort) (consp elements))

Theorem: ushort-arrayp-alt-def

(defthm ushort-arrayp-alt-def
        (equal (ushort-arrayp x)
               (and (valuep x)
                    (value-case x :array)
                    (equal (value-array->elemtype x)
                           (type-ushort))
                    (ushort-listp (value-array->elements x)))))

Theorem: ushort-array->elements-alt-def

(defthm ushort-array->elements-alt-def
        (implies (ushort-arrayp array)
                 (equal (ushort-array->elements array)
                        (value-array->elements array))))
