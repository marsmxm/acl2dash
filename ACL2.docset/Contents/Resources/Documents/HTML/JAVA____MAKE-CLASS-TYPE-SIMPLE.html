Basic constructor macro for class-type-simple structures.

Syntax
(make-class-type-simple [:name <name>] 
                        [:arguments <arguments>])

This is the usual way to construct class-type-simple structures. It simply conses together a structure with the specified fields.

This macro generates a new class-type-simple structure from scratch. See also change-class-type-simple, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by fty::defprod.
