Modifying constructor for vl-context1 structures.

Syntax
(change-vl-context1 x 
                    [:mod <mod>] 
                    [:elem <elem>])

This is an often useful alternative to make-vl-context1.

We construct a new vl-context1 structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-context1

(defmacro change-vl-context1 (x &rest args)
          (std::change-aggregate 'vl-context1
                                 x args
