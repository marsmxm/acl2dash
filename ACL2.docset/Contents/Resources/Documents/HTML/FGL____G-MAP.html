FGL object constructor for fast alists and arrays.

This is a product type, introduced by defflexsum in support of fgl-object.

Fields
tag — g-map-tag
alist — fgl-object-alist

An object constructed as (g-map tag alist) evaluates to the evaluation of alist as a FGL object alist -- that is, a list of pairs where the keys are concrete objects which are not evaluated and the values are FGL objects. The g-map kind is used in fast-alist and fgarray primitives to offer constant-time lookups in fast alists and arrays.
