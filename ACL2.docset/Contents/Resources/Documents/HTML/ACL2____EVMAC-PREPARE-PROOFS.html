Events to prepare proof generation.

We disable the default and override hints; these are implicitly local events. We also add an explicitly local event to prevent mv-nth from being expanded, which is accomplished via a system attachment.

Macro: evmac-prepare-proofs

(defmacro
  evmac-prepare-proofs nil
  '(progn (set-default-hints nil)
          (set-override-hints nil)
          (local (defattach-system simplifiable-mv-nth-p
                                   constant-nil-function-arity-0))))
