Modifying constructor for aexp-mul structures.

Syntax
(change-aexp-mul x 
                 [:left <left>] 
                 [:right <right>])

This is an often useful alternative to make-aexp-mul.

We construct a new aexp-mul structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-aexp-mul

(defmacro change-aexp-mul (x &rest args)
          (std::change-aggregate 'aexp-mul
                                 x args
                                 '((:left . aexp-mul->left)
                                   (:right . aexp-mul->right))
                                 'change-aexp-mul
                                 'nil))
