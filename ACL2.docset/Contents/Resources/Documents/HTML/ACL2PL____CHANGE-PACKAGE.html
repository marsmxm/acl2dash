Modifying constructor for package structures.

Syntax
(change-package x 
                [:name <name>] 
                [:imports <imports>])

This is an often useful alternative to make-package.

We construct a new package structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-package

(defmacro change-package (x &rest args)
          (std::change-aggregate 'package
                                 x args
                                 '((:name . package->name)
                                   (:imports . package->imports))
                                 'change-package
                                 'nil))
