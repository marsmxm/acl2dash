Modifying constructor for struct-declon structures.

Syntax
(change-struct-declon x 
                      [:tyspec <tyspec>] 
                      [:declor <declor>])

This is an often useful alternative to make-struct-declon.

We construct a new struct-declon structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-struct-declon

(defmacro
     change-struct-declon (x &rest args)
     (std::change-aggregate 'struct-declon
                            x args
                            '((:tyspec . struct-declon->tyspec)
                              (:declor . struct-declon->declor))
                            'change-struct-declon
                            'nil))
