Set the separator for paired names.

This macro generates an event to override the default, or the previously set value.

Macro: set-paired-name-separator

(defmacro set-paired-name-separator (separator)
          (cons 'table
                (cons 'paired-name-separator
                      (cons ''separator
                            (cons separator 'nil)))))
