Modifying constructor for type-product structures.

Syntax
(change-type-product x 
                     [:fields <fields>] 
                     [:invariant <invariant>])

This is an often useful alternative to make-type-product.

We construct a new type-product structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-type-product

(defmacro
    change-type-product (x &rest args)
    (std::change-aggregate 'type-product
                           x args
                           '((:fields . type-product->fields)
                             (:invariant . type-product->invariant))
                           'change-type-product
                           'nil))
