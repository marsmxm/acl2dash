Modifying constructor for vl-define structures.

Syntax
(change-vl-define x 
                  [:formals <formals>] 
                  [:body <body>] 
                  [:loc <loc>] 
                  [:stickyp <stickyp>])

This is an often useful alternative to make-vl-define.

We construct a new vl-define structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-define

(defmacro change-vl-define (x &rest args)
          (std::change-aggregate 'vl-define
                                 x args
                                 '((:formals . vl-define->formals)
                                   (:body . vl-define->body)
                                   (:loc . vl-define->loc)
                                   (:stickyp . vl-define->stickyp))
                                 'change-vl-define
                                 'nil))
