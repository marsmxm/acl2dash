Fixtype of Java octal digits [JLS:3.10.1].

A Java octal digit is a Java ASCII character between `0' and `7'. See the grammar rule octal-digit.

This is a type introduced by fty::deffixtype.

Function: oct-digitp

(defun oct-digitp (x)
       (declare (xargs :guard t))
       (let ((__function__ 'oct-digitp))
            (declare (ignorable __function__))
            (and (integerp x)
                 (<= (char-code #\0) x)
                 (<= x (char-code #\7)))))
