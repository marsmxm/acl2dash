Modifying constructor for scratchobj-fgl-obj structures.

Syntax
(change-scratchobj-fgl-obj x 
                           [:val <val>])

This is an often useful alternative to make-scratchobj-fgl-obj.

We construct a new scratchobj-fgl-obj structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-scratchobj-fgl-obj

(defmacro
     change-scratchobj-fgl-obj (x &rest args)
