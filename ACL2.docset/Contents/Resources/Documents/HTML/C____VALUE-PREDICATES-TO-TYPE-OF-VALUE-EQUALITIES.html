Rules that rewrite predicates for values to equalities of the types of the values.

Definitions and Theorems

Theorem: ucharp-to-type-of-value

(defthm ucharp-to-type-of-value
        (implies (valuep x)
                 (equal (ucharp x)
                        (equal (type-of-value x)
                               (type-uchar)))))
