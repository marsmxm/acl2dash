Modifying constructor for outcome-nonterminating structures.

Syntax
(change-outcome-nonterminating x 
                               )

This is an often useful alternative to make-outcome-nonterminating.

We construct a new outcome-nonterminating structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
