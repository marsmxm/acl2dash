Modifying constructor for funtype structures.

Syntax
(change-funtype x 
                [:in <in>] 
                [:out <out>])

This is an often useful alternative to make-funtype.

We construct a new funtype structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-funtype

(defmacro change-funtype (x &rest args)
          (std::change-aggregate 'funtype
                                 x args
                                 '((:in . funtype->in)
                                   (:out . funtype->out))
                                 'change-funtype
                                 'nil))
