A simple unary operator applied to some argument, like &a or -b.

This is a product type, introduced by deftagsum in support of vl-expr.

Fields
op — vl-unaryop-p
The operator being applied.
arg — vl-expr
The argument to the operator.
atts — vl-atts
Any (* foo = bar, baz *) style attributes.
