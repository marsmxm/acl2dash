Modifying constructor for expression-mul structures.

Syntax
(change-expression-mul x 
                       [:arg1 <arg1>] 
                       [:arg2 <arg2>])

This is an often useful alternative to make-expression-mul.

We construct a new expression-mul structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-expression-mul

(defmacro change-expression-mul (x &rest args)
          (std::change-aggregate 'expression-mul
                                 x args
                                 '((:arg1 . expression-mul->arg1)
                                   (:arg2 . expression-mul->arg2))
                                 'change-expression-mul
                                 'nil))
