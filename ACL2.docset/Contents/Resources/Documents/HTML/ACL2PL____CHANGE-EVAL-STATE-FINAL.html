Modifying constructor for eval-state-final structures.

Syntax
(change-eval-state-final x 
                         [:result <result>])

This is an often useful alternative to make-eval-state-final.

We construct a new eval-state-final structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-eval-state-final

(defmacro
