Recognizer for strings whose characters are octal digits.

Signature
(oct-digit-string-p x)  bool

Corner case: this accepts the empty string since all of its characters are octal digits.

Logically this is defined in terms of oct-digit-char-listp. But in the execution, we use a char-based function that avoids exploding the string. This provides much better performance, e.g., on an AMD FX-8350 with CCL:

;; 0.13 seconds, no garbage
(let ((x "deadbeef"))
  (time$ (loop for i fixnum from 1 to 10000000 do
               (str::oct-digit-string-p x))))

;; 1.36 seconds, 1.28 GB allocated
(let ((x "deadbeef"))
  (time$ (loop for i fixnum from 1 to 10000000 do
               (str::oct-digit-char-listp (explode x)))))
