Check if a prime field element is a square of an odd field element.

Same as pfield-squarep except restricts the root to be odd.

Definitions and Theorems

Theorem: pfield-odd-squarep-suff

(defthm pfield-odd-squarep-suff
        (implies (and (fep r p)
                      (oddp r)
                      (equal (mul r r p) x))
                 (pfield-odd-squarep x p)))

Theorem: booleanp-of-pfield-odd-squarep

(defthm booleanp-of-pfield-odd-squarep
        (b* ((yes/no (pfield-odd-squarep x p)))
            (booleanp yes/no))
        :rule-classes :rewrite)
