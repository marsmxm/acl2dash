Modifying constructor for nat/natlist-one structures.

Syntax
(change-nat/natlist-one x 
                        [:get <get>])

This is an often useful alternative to make-nat/natlist-one.

We construct a new nat/natlist-one structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-nat/natlist-one

(defmacro
