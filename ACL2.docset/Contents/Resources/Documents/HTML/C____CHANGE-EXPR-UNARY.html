Modifying constructor for expr-unary structures.

Syntax
(change-expr-unary x 
                   [:op <op>] 
                   [:arg <arg>])

This is an often useful alternative to make-expr-unary.

We construct a new expr-unary structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-expr-unary

(defmacro change-expr-unary (x &rest args)
          (std::change-aggregate 'expr-unary
                                 x args
                                 '((:op . expr-unary->op)
                                   (:arg . expr-unary->arg))
                                 'change-expr-unary
                                 'nil))
