Turn an ACL2 rational into a Java identifier part.

Signature
(atj-gen-shallow-rational-id-part rational) → core
Arguments
rational — Guard (rationalp rational).
Returns
core — Type (stringp core).

This is part of the names of the static final fields for quoted numbers.

If the rational is an integer, we use atj-gen-shallow-integer-id-part. Otherwise, we generate the integer numerator, followed by _over_ to denote the fraction, followed by the integer denominator (always greater than 1).

Definitions and Theorems

Function: atj-gen-shallow-rational-id-part

(defun
 atj-gen-shallow-rational-id-part
 (rational)
 (declare (xargs :guard (rationalp rational)))
 (let
  ((__function__ 'atj-gen-shallow-rational-id-part))
  (declare (ignorable __function__))
  (if
   (integerp rational)
   (atj-gen-shallow-integer-id-part rational)
   (str::cat
        (atj-gen-shallow-integer-id-part (numerator rational))
        "_over_"
        (atj-gen-shallow-integer-id-part (denominator rational))))))
