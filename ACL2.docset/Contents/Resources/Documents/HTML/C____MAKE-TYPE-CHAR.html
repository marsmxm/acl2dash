Basic constructor macro for type-char structures.

Syntax
(make-type-char )

This is the usual way to construct type-char structures. It simply conses together a structure with the specified fields.

This macro generates a new type-char structure from scratch. See also change-type-char, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by fty::defprod.

Macro: make-type-char

(defmacro make-type-char (&rest args)
