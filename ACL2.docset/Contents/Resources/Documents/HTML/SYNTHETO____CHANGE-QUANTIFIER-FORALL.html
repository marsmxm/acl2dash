Modifying constructor for quantifier-forall structures.

Syntax
(change-quantifier-forall x 
                          )

This is an often useful alternative to make-quantifier-forall.

We construct a new quantifier-forall structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-quantifier-forall

(defmacro change-quantifier-forall (x &rest args)
          (std::change-aggregate 'quantifier-forall
                                 x args 'nil
                                 'change-quantifier-forall
                                 'nil))
