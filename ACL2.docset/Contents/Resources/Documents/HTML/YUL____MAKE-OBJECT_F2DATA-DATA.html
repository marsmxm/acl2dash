Basic constructor macro for object/data-data structures.

Syntax
(make-object/data-data [:get <get>])

This is the usual way to construct object/data-data structures. It simply conses together a structure with the specified fields.

This macro generates a new object/data-data structure from scratch. See also change-object/data-data, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by fty::defprod.
