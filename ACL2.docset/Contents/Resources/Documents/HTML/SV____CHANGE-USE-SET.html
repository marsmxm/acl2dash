Modifying constructor for use-set structures.

Syntax
(change-use-set x 
                [:uses <uses>] 
                [:sets <sets>])

This is an often useful alternative to make-use-set.

We construct a new use-set structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-use-set

(defmacro change-use-set (x &rest args)
          (std::change-aggregate 'use-set
                                 x args
                                 '((:uses . use-set->uses)
                                   (:sets . use-set->sets))
                                 'change-use-set
                                 'remake-use-set))
