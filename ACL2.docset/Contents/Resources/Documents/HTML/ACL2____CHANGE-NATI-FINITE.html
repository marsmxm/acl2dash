Modifying constructor for nati-finite structures.

Syntax
(change-nati-finite x 
                    [:get <get>])

This is an often useful alternative to make-nati-finite.

We construct a new nati-finite structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-nati-finite

(defmacro change-nati-finite (x &rest args)
          (std::change-aggregate 'nati-finite
                                 x args '((:get . nati-finite->get))
                                 'change-nati-finite
                                 'nil))
