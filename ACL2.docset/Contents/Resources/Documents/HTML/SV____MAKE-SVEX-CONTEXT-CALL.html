Basic constructor macro for svex-context-call structures.

Syntax
(make-svex-context-call [:argnum <argnum>] 
                        [:fn <fn>] 
                        [:args <args>])

This is the usual way to construct svex-context-call structures. It simply conses together a structure with the specified fields.

This macro generates a new svex-context-call structure from scratch. See also change-svex-context-call, which can "change" an existing structure, instead.

This is an ordinary make- macro introduced by defprod.
