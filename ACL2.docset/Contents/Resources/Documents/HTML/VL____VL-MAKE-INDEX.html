Safely create a constant integer atom whose value is n.

Signature
(vl-make-index n) → index
Arguments
n — Guard (integerp n).
Returns
index — Type (vl-expr-p index).

The expression we create doesn't have a type, because it would depend on the context. In many cases it might suffice to use an integer type; in this case see vl-make-integer.

Definitions and Theorems

Function: vl-make-index

(defun
 vl-make-index (n)
 (declare (xargs :guard (integerp n)))
 (let
  ((__function__ 'vl-make-index))
  (declare (ignorable __function__))
  (let*
   ((value (lifix n))
    (width (+ 1 (integer-length value))))
   (if
    (<= width 31)
    (hons-copy
      (make-vl-literal
           :val (make-vl-constint :origwidth 32
                                  :origsign :vl-signed
                                  :wasunsized t
                                  :value (acl2::loghead 32 value))))
    (hons-copy
     (make-vl-literal
         :val
         (make-vl-constint :origwidth width
                           :origsign :vl-signed
                           :value (acl2::loghead width value))))))))
