The new type/size/signedness/constness to cast an expression to.

This is a tagged union type, introduced by deftagsum.

Member Tags → Types
:type → vl-casttype-type
A cast to a datatype, like int'(foo).
:size → vl-casttype-size
A cast to a size, like mywidth'(foo).
:signedness → vl-casttype-signedness
A cast to a signedness, like signed'(foo) or unsigned'(foo).
:const → vl-casttype-const
A cast to a constant, like const'(foo).
