Modifying constructor for double-array structures.

Syntax
(change-double-array x 
                     [:components <components>])

This is an often useful alternative to make-double-array.

We construct a new double-array structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-double-array

(defmacro
  change-double-array (x &rest args)
  (std::change-aggregate 'double-array
                         x args
                         '((:components . double-array->components))
                         'change-double-array
                         'nil))
