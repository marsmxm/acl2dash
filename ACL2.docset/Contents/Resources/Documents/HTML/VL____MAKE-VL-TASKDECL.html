Basic constructor macro for vl-taskdecl structures.

Syntax
(make-vl-taskdecl [:name <name>] 
                  [:body <body>] 
                  [:loc <loc>] 
                  [:portdecls <portdecls>] 
                  [:vardecls <vardecls>] 
                  [:paramdecls <paramdecls>] 
                  [:typedefs <typedefs>] 
                  [:imports <imports>] 
                  [:atts <atts>] 
                  [:lifetime <lifetime>] 
                  [:loaditems <loaditems>])

This is the usual way to construct vl-taskdecl structures. It simply conses together a structure with the specified fields.

This macro generates a new vl-taskdecl structure from scratch. See also change-vl-taskdecl, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by defprod.

Function: vl-taskdecl

(defun
 vl-taskdecl
 (name body loc
       portdecls vardecls paramdecls typedefs
       imports atts lifetime loaditems)
 (declare
   (xargs :guard (and (stringp name)
                      (vl-stmt-p body)
                      (vl-location-p loc)
                      (vl-portdecllist-p portdecls)
                      (vl-vardecllist-p vardecls)
                      (vl-paramdecllist-p paramdecls)
                      (vl-typedeflist-p typedefs)
                      (vl-importlist-p imports)
                      (vl-atts-p atts)
                      (vl-lifetime-p lifetime)
                      (vl-portdecl-or-blockitem-list-p loaditems))))
 (declare (xargs :guard t))
 (let
  ((__function__ 'vl-taskdecl))
  (declare (ignorable __function__))
  (b*
   ((name (mbe :logic (str-fix name) :exec name))
    (body (mbe :logic (vl-stmt-fix body)
               :exec body))
    (loc (mbe :logic (vl-location-fix loc)
              :exec loc))
    (portdecls (mbe :logic (vl-portdecllist-fix portdecls)
                    :exec portdecls))
    (vardecls (mbe :logic (vl-vardecllist-fix vardecls)
                   :exec vardecls))
    (paramdecls (mbe :logic (vl-paramdecllist-fix paramdecls)
                     :exec paramdecls))
    (typedefs (mbe :logic (vl-typedeflist-fix typedefs)
                   :exec typedefs))
    (imports (mbe :logic (vl-importlist-fix imports)
                  :exec imports))
    (atts (mbe :logic (vl-atts-fix atts)
               :exec atts))
    (lifetime (mbe :logic (vl-lifetime-fix lifetime)
                   :exec lifetime))
    (loaditems
         (mbe :logic (vl-portdecl-or-blockitem-list-fix loaditems)
              :exec loaditems)))
   (cons
    :vl-taskdecl
    (std::prod-cons
     (std::prod-cons
          (std::prod-cons name body)
          (std::prod-cons loc
                          (std::prod-cons portdecls vardecls)))
     (std::prod-cons
          (std::prod-cons paramdecls
                          (std::prod-cons typedefs imports))
          (std::prod-cons atts
                          (std::prod-cons lifetime loaditems))))))))
