Modifying constructor for command-error-malformed-to structures.

Syntax
(change-command-error-malformed-to x 
                                   [:to <to>])

This is an often useful alternative to make-command-error-malformed-to.

We construct a new command-error-malformed-to structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
