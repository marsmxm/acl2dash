Fixtype of optional sequences of type specifiers.

This is an option type introduced by fty::defoption. Note that defoption is just a wrapper for fty::defflexsum, so there are :none and :some member types, a case macro, and so forth.

Member Types
:none tyspecseq-option-none
Represents that no tyspecseq-option is available, i.e., Nothing or None.
:some tyspecseq-option-some
An available tyspecseq-option, i.e., Just val or Some val.
