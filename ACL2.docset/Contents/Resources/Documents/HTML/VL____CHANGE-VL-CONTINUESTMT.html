Modifying constructor for vl-continuestmt structures.

Syntax
(change-vl-continuestmt x 
                        [:atts <atts>])

This is an often useful alternative to make-vl-continuestmt.

We construct a new vl-continuestmt structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-continuestmt

(defmacro
     change-vl-continuestmt (x &rest args)
     (std::change-aggregate 'vl-continuestmt
                            x
                            args '((:atts . vl-continuestmt->atts))
                            'change-vl-continuestmt
                            'remake-vl-continuestmt))
