Modifying constructor for jexpr-name structures.

Syntax
(change-jexpr-name x 
                   [:get <get>])

This is an often useful alternative to make-jexpr-name.

We construct a new jexpr-name structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-jexpr-name

(defmacro change-jexpr-name (x &rest args)
          (std::change-aggregate 'jexpr-name
                                 x args '((:get . jexpr-name->get))
                                 'change-jexpr-name
                                 'nil))
