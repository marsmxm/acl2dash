(vl-modinstlist-fix x) is a usual fty list fixing function.

Signature
(vl-modinstlist-fix x) → fty::newx
Arguments
x — Guard (vl-modinstlist-p x).
Returns
fty::newx — Type (vl-modinstlist-p fty::newx).

In the logic, we apply vl-modinst-fix to each member of the x. In the execution, none of that is actually necessary and this is just an inlined identity function.
