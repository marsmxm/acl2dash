Set the starting marker of the numeric index of numbered names.

This macro generates an event to override the default, or the previously set value.

Macro: set-numbered-name-index-start

(defmacro set-numbered-name-index-start (start)
          (cons 'table
                (cons 'numbered-name-index-start
                      (cons (cons 'quote (cons 'start 'nil))
                            (cons start 'nil)))))
