Modifying constructor for value-string structures.

Syntax
(change-value-string x 
                     [:get <get>])

This is an often useful alternative to make-value-string.

We construct a new value-string structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-value-string

(defmacro
     change-value-string (x &rest args)
     (std::change-aggregate 'value-string
                            x args '((:get . value-string->get))
                            'change-value-string
                            'nil))
