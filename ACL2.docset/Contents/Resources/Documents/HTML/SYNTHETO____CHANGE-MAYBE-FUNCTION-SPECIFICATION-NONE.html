Modifying constructor for maybe-function-specification-none structures.

Syntax
(change-maybe-function-specification-none x 
                                          )

This is an often useful alternative to make-maybe-function-specification-none.

We construct a new maybe-function-specification-none structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
