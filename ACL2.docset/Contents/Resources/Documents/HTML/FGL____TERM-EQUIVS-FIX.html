(term-equivs-fix x) is an ACL2::fty alist fixing function that follows the drop-keys strategy.

Signature
(term-equivs-fix x) → fty::newx
Arguments
x — Guard (term-equivs-p x).
Returns
fty::newx — Type (term-equivs-p fty::newx).

Note that in the execution this is just an inline identity function.

Function: term-equivs-fix$inline

(defun
 term-equivs-fix$inline (x)
 (declare (xargs :guard (term-equivs-p x)))
 (let
  ((__function__ 'term-equivs-fix))
  (declare (ignorable __function__))
  (mbe
   :logic
   (if
      (atom x)
      nil
      (let ((rest (term-equivs-fix (cdr x))))
           (if (and (consp (car x))
                    (fgl-object-p (caar x)))
               (let ((fty::first-key (caar x))
                     (fty::first-val (acl2::nat-list-fix (cdar x))))
                    (cons (cons fty::first-key fty::first-val)
                          rest))
               rest)))
   :exec x)))
