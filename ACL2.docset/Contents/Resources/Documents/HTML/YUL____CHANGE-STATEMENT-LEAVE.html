Modifying constructor for statement-leave structures.

Syntax
(change-statement-leave x 
                        )

This is an often useful alternative to make-statement-leave.

We construct a new statement-leave structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-statement-leave

(defmacro change-statement-leave (x &rest args)
          (std::change-aggregate 'statement-leave
