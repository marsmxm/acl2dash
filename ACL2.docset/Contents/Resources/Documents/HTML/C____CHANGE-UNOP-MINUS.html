Modifying constructor for unop-minus structures.

Syntax
(change-unop-minus x 
                   )

This is an often useful alternative to make-unop-minus.

We construct a new unop-minus structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-unop-minus

(defmacro change-unop-minus (x &rest args)
          (std::change-aggregate 'unop-minus
