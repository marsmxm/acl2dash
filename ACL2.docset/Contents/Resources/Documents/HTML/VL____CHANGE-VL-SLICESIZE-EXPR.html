Modifying constructor for vl-slicesize-expr structures.

Syntax
(change-vl-slicesize-expr x 
                          [:expr <expr>])

This is an often useful alternative to make-vl-slicesize-expr.

We construct a new vl-slicesize-expr structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-slicesize-expr

(defmacro
     change-vl-slicesize-expr (x &rest args)
     (std::change-aggregate 'vl-slicesize-expr
                            x args
                            '((:expr . vl-slicesize-expr->expr))
                            'change-vl-slicesize-expr
                            'remake-vl-slicesize-expr))
