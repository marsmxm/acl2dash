Modifying constructor for class-type-simple structures.

Syntax
(change-class-type-simple x 
                          [:name <name>] 
                          [:arguments <arguments>])

This is an often useful alternative to make-class-type-simple.

We construct a new class-type-simple structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-class-type-simple

(defmacro change-class-type-simple (x &rest args)
          (std::change-aggregate
               'class-type-simple

                 (:arguments . class-type-simple->arguments))
