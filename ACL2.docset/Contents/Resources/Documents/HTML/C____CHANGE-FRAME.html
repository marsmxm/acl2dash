Modifying constructor for frame structures.

Syntax
(change-frame x 
              [:function <function>] 
              [:scopes <scopes>])

This is an often useful alternative to make-frame.

We construct a new frame structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-frame

(defmacro change-frame (x &rest args)
          (std::change-aggregate 'frame
                                 x args
                                 '((:function . frame->function)
                                   (:scopes . frame->scopes))
                                 'change-frame
                                 'nil))
