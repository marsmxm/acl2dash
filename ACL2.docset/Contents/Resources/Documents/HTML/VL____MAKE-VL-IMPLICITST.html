Basic constructor macro for vl-implicitst structures.

Syntax
(make-vl-implicitst [:portdecls <portdecls>] 
                    [:decls <decls>] 
                    [:imports <imports>] 
                    [:ss <ss>])

This is the usual way to construct vl-implicitst structures. It simply conses together a structure with the specified fields.

This macro generates a new vl-implicitst structure from scratch. See also change-vl-implicitst, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by defprod.

(defun
  vl-implicitst
  (portdecls decls imports ss)
  (declare (xargs :guard (and (vl-portdecl-alist-p portdecls)
                              (any-p decls)
                              (any-p imports)
                              (vl-scopestack-p ss))))
  (declare (xargs :guard t))
  (let ((__function__ 'vl-implicitst))
       (declare (ignorable __function__))
       (b* ((portdecls (mbe :logic (vl-portdecl-alist-fix portdecls)
                            :exec portdecls))
            (decls (mbe :logic (identity decls)
                        :exec decls))
            (imports (mbe :logic (identity imports)
                          :exec imports))
            (ss (mbe :logic (vl-scopestack-fix ss)
                     :exec ss)))
           (std::prod-cons (std::prod-cons portdecls decls)
                           (std::prod-cons imports ss)))))
