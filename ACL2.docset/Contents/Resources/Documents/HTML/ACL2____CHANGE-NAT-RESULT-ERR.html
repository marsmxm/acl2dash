Modifying constructor for nat-result-err structures.

Syntax
(change-nat-result-err x 
                       [:get <get>])

This is an often useful alternative to make-nat-result-err.

We construct a new nat-result-err structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
