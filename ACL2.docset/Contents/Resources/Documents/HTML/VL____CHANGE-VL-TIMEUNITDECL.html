Modifying constructor for vl-timeunitdecl structures.

Syntax
(change-vl-timeunitdecl x 
                        [:numerator <numerator>] 
                        [:denominator <denominator>] 
                        [:loc <loc>])

This is an often useful alternative to make-vl-timeunitdecl.

We construct a new vl-timeunitdecl structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-timeunitdecl

(defmacro change-vl-timeunitdecl (x &rest args)
          (std::change-aggregate
               'vl-timeunitdecl
               x args
               '((:numerator . vl-timeunitdecl->numerator)
                 (:denominator . vl-timeunitdecl->denominator)
                 (:loc . vl-timeunitdecl->loc))
               'change-vl-timeunitdecl
               'remake-vl-timeunitdecl))
