Modifying constructor for num-val-direct structures.

Syntax
(change-num-val-direct x 
                       [:get <get>])

This is an often useful alternative to make-num-val-direct.

We construct a new num-val-direct structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-num-val-direct

(defmacro
     change-num-val-direct (x &rest args)
     (std::change-aggregate 'num-val-direct
                            x args '((:get . num-val-direct->get))
                            'change-num-val-direct
                            'nil))
