Extend a value in the sneaky store.

Example:

(sneaky-save 'foo '(3))
(sneaky-push 'foo 4)
(sneaky-load 'foo state) --> (mv '(4 3) state)

In the logic, sneaky-push just returns nil. Under the hood, we update the sneaky store, updating name by "pushing" a new value onto its current value. The accumulated values can later be accessed with sneaky-load. See also sneaky-save.
