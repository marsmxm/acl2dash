Wrapper to indicate a C local variable assignment in an mv-let for 8 bound variables.

See atc-let-designations.

Macro: assign9

(defmacro assign9 (x)
          (cons 'mv-let
                (cons '(*1 *2 *3 *4 *5 *6 *7 *8 *9)
                      (cons x
                            '((mv (assign *1)
                                  *2 *3 *4 *5 *6 *7 *8 *9))))))
