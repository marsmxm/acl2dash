Modifying constructor for scratchobj-fgl-objlist structures.

Syntax
(change-scratchobj-fgl-objlist x 
                               [:val <val>])

This is an often useful alternative to make-scratchobj-fgl-objlist.

We construct a new scratchobj-fgl-objlist structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
