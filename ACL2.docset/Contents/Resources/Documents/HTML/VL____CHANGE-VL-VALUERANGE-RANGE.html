Modifying constructor for vl-valuerange-range structures.

Syntax
(change-vl-valuerange-range x 
                            [:low <low>] 
                            [:high <high>])

This is an often useful alternative to make-vl-valuerange-range.

We construct a new vl-valuerange-range structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.
