Return an error result with a singleton stack.

This macro constructs an error result of fixtype resulterr with the specified information info, accompanied by the name of the current function fn, as a doublet (fn info). A singleton list with this doublet is returned. This is a singleton stack, in the sense that err-push pushes onto the stack by consing additional pairs with the same form.

This assumes that __function__ is bound to the function name, which happens automatically with define.

Macro: err

(defmacro
 err (info)
 (cons
  'make-resulterr
  (cons
      ':info
      (cons (cons 'list
                  (cons (cons 'list
                              (cons '__function__ (cons info 'nil)))
                        'nil))
            'nil))))
