Return an error result with a singleton stack.

This macro constructs an error result of fixtype resulterr with the specified information and with the current function name as singleton stack.

This assumes that __function__ is bound to the function name, which happens automatically with define.

Macro: err

(defmacro
 err (x)
 (cons
      'make-resulterr
      (cons ':info
            (cons x
                  (cons ':stack
                        (cons (cons 'list (cons '__function__ 'nil))
                              'nil))))))
