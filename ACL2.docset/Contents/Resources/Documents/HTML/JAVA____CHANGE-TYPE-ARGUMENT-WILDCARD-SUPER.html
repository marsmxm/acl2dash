Modifying constructor for type-argument-wildcard-super structures.

Syntax
(change-type-argument-wildcard-super x 
                                     [:bound <bound>])

This is an often useful alternative to make-type-argument-wildcard-super.

We construct a new type-argument-wildcard-super structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
