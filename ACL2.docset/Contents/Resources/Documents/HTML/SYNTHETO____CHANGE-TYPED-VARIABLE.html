Modifying constructor for typed-variable structures.

Syntax
(change-typed-variable x 
                       [:name <name>] 
                       [:type <type>])

This is an often useful alternative to make-typed-variable.

We construct a new typed-variable structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-typed-variable

(defmacro change-typed-variable (x &rest args)
          (std::change-aggregate 'typed-variable
                                 x args
                                 '((:name . typed-variable->name)
                                   (:type . typed-variable->type))
                                 'change-typed-variable
                                 'nil))
