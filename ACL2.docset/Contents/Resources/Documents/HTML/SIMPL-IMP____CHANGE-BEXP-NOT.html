Modifying constructor for bexp-not structures.

Syntax
(change-bexp-not x 
                 [:arg <arg>])

This is an often useful alternative to make-bexp-not.

We construct a new bexp-not structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-bexp-not

(defmacro change-bexp-not (x &rest args)
          (std::change-aggregate 'bexp-not
                                 x args '((:arg . bexp-not->arg))
                                 'change-bexp-not
                                 'nil))
