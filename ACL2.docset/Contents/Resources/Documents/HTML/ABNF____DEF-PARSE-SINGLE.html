Define a single parsing function for a specification.

This is an auxiliary macro, used by the others, which are the ones that should be used by the user.

Here we hardwire the names of the named constants for the grammar and the group, option, and repetition alists.

Macro: def-parse-single

(defmacro
 def-parse-single (spec)
 (cons
  'make-event
  (cons
   (cons
     'def-parse-gen-function-for-spec
     (cons spec
           (cons '*def-parse-grammar*
                 (cons '*def-parse-fn-name*
                       (cons '*def-parse-group-fns*
                             (cons '*def-parse-option-fns*
                                   (cons '*def-parse-repetition-fns*
                                         'nil)))))))
   'nil)))
