Modifying constructor for jcbody-element-member structures.

Syntax
(change-jcbody-element-member x 
                              [:get <get>])

This is an often useful alternative to make-jcbody-element-member.

We construct a new jcbody-element-member structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
