Fixtype of type names [C:6.7.7].

This is a product type introduced by fty::defprod.

Fields
specs — tyspecseq
pointerp — booleanp

For now we only capture type names consisting of the type specifier sequences captured by tyspecseq, and pointers thereof (only single pointers, not pointers to pointers). We capture the presence or absence of * (for pointer) via a boolean flag.
