Modifying constructor for vl-proprepeat structures.

Syntax
(change-vl-proprepeat x 
                      [:seq <seq>] 
                      [:reps <reps>])

This is an often useful alternative to make-vl-proprepeat.

We construct a new vl-proprepeat structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-proprepeat

(defmacro change-vl-proprepeat (x &rest args)
          (std::change-aggregate 'vl-proprepeat
                                 x args
                                 '((:seq . vl-proprepeat->seq)
                                   (:reps . vl-proprepeat->reps))
                                 'change-vl-proprepeat
                                 'nil))
