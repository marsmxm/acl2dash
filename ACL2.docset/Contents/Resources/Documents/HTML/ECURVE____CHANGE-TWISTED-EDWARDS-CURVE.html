Modifying constructor for twisted-edwards-curve structures.

Syntax
(change-twisted-edwards-curve x 
                              [:p <p>] 
                              [:a <a>] 
                              [:d <d>])

This is an often useful alternative to make-twisted-edwards-curve.

We construct a new twisted-edwards-curve structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-twisted-edwards-curve

(defmacro change-twisted-edwards-curve
          (x &rest args)
          (std::change-aggregate 'twisted-edwards-curve
                                 x args
                                 '((:p . twisted-edwards-curve->p)
                                   (:a . twisted-edwards-curve->a)
                                   (:d . twisted-edwards-curve->d))
                                 'change-twisted-edwards-curve
                                 'nil))
