Modifying constructor for regex-disjunct structures.

Syntax
(change-regex-disjunct x 
                       [:lst <lst>])

This is an often useful alternative to make-regex-disjunct.

We construct a new regex-disjunct structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-regex-disjunct

(defmacro
     change-regex-disjunct (x &rest args)
     (std::change-aggregate 'regex-disjunct
                            x args '((:lst . regex-disjunct->lst))
                            'change-regex-disjunct
                            'nil))
