Modifying constructor for svex-context-call structures.

Syntax
(change-svex-context-call x 
                          [:argnum <argnum>] 
                          [:fn <fn>] 
                          [:args <args>])

This is an often useful alternative to make-svex-context-call.

We construct a new svex-context-call structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-svex-context-call
