Modifying constructor for decdig/uscore-digit structures.

Syntax
(change-decdig/uscore-digit x 
                            [:get <get>])

This is an often useful alternative to make-decdig/uscore-digit.

We construct a new decdig/uscore-digit structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-decdig/uscore-digit

(defmacro
