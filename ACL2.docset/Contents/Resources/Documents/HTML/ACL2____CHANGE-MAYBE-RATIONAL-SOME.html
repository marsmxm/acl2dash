Modifying constructor for maybe-rational-some structures.

Syntax
(change-maybe-rational-some x 
                            [:val <val>])

This is an often useful alternative to make-maybe-rational-some.

We construct a new maybe-rational-some structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
