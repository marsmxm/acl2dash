Basic constructor macro for array-ushort structures.

Syntax
(make-array-ushort [:get <get>])

This is the usual way to construct array-ushort structures. It simply conses together a structure with the specified fields.

This macro generates a new array-ushort structure from scratch. See also change-array-ushort, which can "change" an existing structure, instead.

This is an ordinary make- macro introduced by fty::defprod.
