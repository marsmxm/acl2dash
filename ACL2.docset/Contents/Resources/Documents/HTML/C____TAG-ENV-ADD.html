Add tag information to a tag environment.

Signature
(tag-env-add tag info env) → new-env
Arguments
tag — Guard (identp tag).
info — Guard (tag-infop info).
env — Guard (tag-envp env).
Returns
new-env — Type (tag-env-optionp new-env).

We return the updated environment if the tag is not already present, otherwise we return no tag environment.

Function: tag-env-add

(defun
  tag-env-add (tag info env)
  (declare (xargs :guard (and (identp tag)
                              (tag-infop info)
                              (tag-envp env))))
  (let ((__function__ 'tag-env-add))
       (declare (ignorable __function__))
       (b* ((tag (ident-fix tag))
            (info (tag-info-fix info))
            (env (tag-env-fix env)))
           (if (omap::in tag env)
               (tag-env-option-none)
               (tag-env-option-some (omap::update tag info env))))))
