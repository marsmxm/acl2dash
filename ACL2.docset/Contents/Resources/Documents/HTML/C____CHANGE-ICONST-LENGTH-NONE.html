Modifying constructor for iconst-length-none structures.

Syntax
(change-iconst-length-none x 
                           )

This is an often useful alternative to make-iconst-length-none.

We construct a new iconst-length-none structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
