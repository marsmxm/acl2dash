Modifying constructor for command-error-malformed-value structures.

Syntax
(change-command-error-malformed-value x 
                                      [:value <value>])

This is an often useful alternative to make-command-error-malformed-value.

We construct a new command-error-malformed-value structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
