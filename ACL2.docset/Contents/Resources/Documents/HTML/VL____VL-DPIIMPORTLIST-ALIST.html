Extend an alist by binding the names of vl-dpiimports to their definitions.

Signature
(vl-dpiimportlist-alist x acc) → alist
Arguments
x — Guard (vl-dpiimportlist-p x).
Returns
alist — Type (equal (vl-dpiimport-alist-p alist) (vl-dpiimport-alist-p acc)).

This can be used as an alternative to vl-find-dpiimport when you need to perform a lot of lookups.

Definitions and Theorems

Function: vl-dpiimportlist-alist

(defun vl-dpiimportlist-alist (x acc)
       (declare (xargs :guard (vl-dpiimportlist-p x)))
       (let ((__function__ 'vl-dpiimportlist-alist))
            (declare (ignorable __function__))
            (if (atom x)
                acc
                (cons (cons (vl-dpiimport->name (car x))
                            (vl-dpiimport-fix (car x)))
                      (vl-dpiimportlist-alist (cdr x) acc)))))

Theorem: lookup-in-vl-dpiimportlist-alist-fast

(defthm
  lookup-in-vl-dpiimportlist-alist-fast
  (implies
       (stringp name)
       (equal (hons-assoc-equal name (vl-dpiimportlist-alist x nil))
              (let ((val (vl-find-dpiimport name x)))
                   (and val (cons name val))))))
