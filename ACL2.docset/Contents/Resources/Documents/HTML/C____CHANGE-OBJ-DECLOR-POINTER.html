Modifying constructor for obj-declor-pointer structures.

Syntax
(change-obj-declor-pointer x 
                           [:decl <decl>])

This is an often useful alternative to make-obj-declor-pointer.

We construct a new obj-declor-pointer structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-obj-declor-pointer

(defmacro
     change-obj-declor-pointer (x &rest args)
     (std::change-aggregate 'obj-declor-pointer
                            x args
                            '((:decl . obj-declor-pointer->decl))
                            'change-obj-declor-pointer
                            'nil))
