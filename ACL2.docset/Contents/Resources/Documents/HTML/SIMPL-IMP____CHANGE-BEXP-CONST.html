Modifying constructor for bexp-const structures.

Syntax
(change-bexp-const x 
                   [:value <value>])

This is an often useful alternative to make-bexp-const.

We construct a new bexp-const structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-bexp-const

(defmacro
     change-bexp-const (x &rest args)
     (std::change-aggregate 'bexp-const
                            x args '((:value . bexp-const->value))
                            'change-bexp-const
                            'nil))
