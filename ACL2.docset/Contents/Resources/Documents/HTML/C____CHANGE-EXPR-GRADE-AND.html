Modifying constructor for expr-grade-and structures.

Syntax
(change-expr-grade-and x 
                       )

This is an often useful alternative to make-expr-grade-and.

We construct a new expr-grade-and structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
