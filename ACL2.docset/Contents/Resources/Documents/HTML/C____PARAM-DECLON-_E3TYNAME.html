Turn a parameter declaration into the corresponding type name.

Signature
(param-declon->tyname param) → tyname
Arguments
param — Guard (param-declonp param).
Returns
tyname — Type (tynamep tyname).

This is obtained by removing the identifier.

Definitions and Theorems

Function: param-declon->tyname

(defun
 param-declon->tyname (param)
 (declare (xargs :guard (param-declonp param)))
 (let
   ((__function__ 'param-declon->tyname))
   (declare (ignorable __function__))
   (make-tyname
        :specs (param-declon->type param)
        :pointerp (declor->pointerp (param-declon->declor param)))))

(defthm tynamep-of-param-declon->tyname
