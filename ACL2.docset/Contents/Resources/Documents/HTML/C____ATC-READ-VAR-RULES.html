Rules about read-var.

The theorems below about read-var are a bit different because read-var does not return a state, but a value instead. The first theorem skips over enter-scope. The second theorem either returns the value of the encountered variable or skips over it, based on whether the names coincide or not. There is no theorem for add-frame because this situation never happens during the symbolic execution. The third theorem serves for variables read in loops that are declared outside the scope of the loop, i.e. that are represented as update-vars: if the two variables are the same, the value is returned; otherwise, we skip over the update-var in search for the variable. The fourth and fifth theorems serve to move past object updates.

Definitions and Theorems

Theorem: read-var-of-enter-scope

(defthm read-var-of-enter-scope
        (implies (not (equal (compustate-frames-number compst)
                             0))
                 (equal (read-var var (enter-scope compst))
                        (read-var var compst))))

Theorem: read-var-of-add-var

(defthm read-var-of-add-var
        (implies (not (equal (compustate-frames-number compst)
                             0))
                 (equal (read-var var (add-var var2 val compst))
                        (if (equal (ident-fix var) (ident-fix var2))
                            (value-fix val)
                            (read-var var compst)))))

Theorem: read-var-of-update-var

(defthm read-var-of-update-var
        (implies (not (equal (compustate-frames-number compst)
                             0))
                 (equal (read-var var (update-var var2 val2 compst))
                        (if (equal (ident-fix var) (ident-fix var2))
                            (value-fix val2)
                            (read-var var compst)))))

Theorem: read-var-of-update-object

(defthm
    read-var-of-update-object
    (implies (not (equal (compustate-frames-number compst)
                         0))
             (equal (read-var var (update-object objdes obj compst))
                    (read-var var compst))))
