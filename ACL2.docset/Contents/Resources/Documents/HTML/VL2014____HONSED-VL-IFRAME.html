Raw constructor for honsed vl-iframe-p structures.

Syntax:

(honsed-vl-iframe initially-activep 
                  some-thing-satisfiedp already-saw-elsep)

This is identical to vl-iframe, except that we hons the structure we are creating.

Definition

This is an ordinary honsing constructor introduced by defaggregate.

Function: honsed-vl-iframe

(defun
 honsed-vl-iframe
 (initially-activep some-thing-satisfiedp already-saw-elsep)
 (declare (xargs :guard (and (booleanp initially-activep)
                             (booleanp some-thing-satisfiedp)
                             (booleanp already-saw-elsep))))
 (mbe
   :logic (vl-iframe initially-activep
                     some-thing-satisfiedp already-saw-elsep)
   :exec (hons :vl-iframe (hons (hons 'initially-activep
                                      initially-activep)
                                (hons (hons 'some-thing-satisfiedp
                                            some-thing-satisfiedp)
                                      (hons (hons 'already-saw-elsep
                                                  already-saw-elsep)
                                            nil))))))
