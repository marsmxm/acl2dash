A single step along the way of a hierarchical identifier.

This is a product type introduced by defprod.

Fields
name — stringp
Name from the hid
index — vl-maybe-expr
Instance array/genarray index, if present
item — vl-scopeitem
The item encountered along the path of the HID.
ss — vl-scopestack
The scope where this item was found.
elabpath — vl-elabtraversal
Reversed instructions to get to this scope from the original scope.

See hid-tools. Some routines for following hids produce traces of the items they encounter along the way. A hidstep structure represents a single step along a HID.
