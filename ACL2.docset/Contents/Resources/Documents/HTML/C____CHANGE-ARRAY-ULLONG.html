Modifying constructor for array-ullong structures.

Syntax
(change-array-ullong x 
                     [:get <get>])

This is an often useful alternative to make-array-ullong.

We construct a new array-ullong structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-array-ullong

(defmacro
     change-array-ullong (x &rest args)
     (std::change-aggregate 'array-ullong
                            x args '((:get . array-ullong->get))
                            'change-array-ullong
                            'nil))
