Primitive resistive pmos transistor.

The Verilog meaning of this module is:

module VL_1_BIT_RPMOS (out, data, ctrl);
  output out ;
  input data ;
  input ctrl ;
  rpmos gate (out, data, ctrl) ;
endmodule

VL takes this as a primitive. The gate-elim transform converts certain rpmos gates into instances of this module.

ESIM has no notion of strengths and very little support for transistors, but this may be a convenient target for other back-end tools. The corresponding esim primitive is ACL2::*esim-nmos*.
