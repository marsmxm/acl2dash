Modifying constructor for svexl-alist structures.

Syntax
(change-svexl-alist x 
                    [:top-node-alist <top-node-alist>] 
                    [:node-array <node-array>])

This is an often useful alternative to make-svexl-alist.

We construct a new svexl-alist structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-svexl-alist

(defmacro change-svexl-alist (x &rest args)
          (std::change-aggregate
               'svexl-alist
               x args
               '((:top-node-alist . svexl-alist->top-node-alist)
                 (:node-array . svexl-alist->node-array))
               'change-svexl-alist
               'nil))
