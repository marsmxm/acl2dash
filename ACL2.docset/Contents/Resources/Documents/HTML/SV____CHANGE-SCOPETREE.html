Modifying constructor for scopetree structures.

Syntax
(change-scopetree x 
                  [:leaves <leaves>] 
                  [:subscopes <subscopes>])

This is an often useful alternative to make-scopetree.

We construct a new scopetree structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-scopetree

(defmacro
     change-scopetree (x &rest args)
     (std::change-aggregate 'scopetree
                            x args
                            '((:leaves . scopetree->leaves)
                              (:subscopes . scopetree->subscopes))
                            'change-scopetree
                            'remake-scopetree))
