Modifying constructor for value-character structures.

Syntax
(change-value-character x 
                        [:get <get>])

This is an often useful alternative to make-value-character.

We construct a new value-character structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-value-character

(defmacro
     change-value-character (x &rest args)
     (std::change-aggregate 'value-character
                            x args '((:get . value-character->get))
                            'change-value-character
                            'nil))
