Check that a function definition has no nested function definitions.

Signature
(fundef-nofunp fundef) → yes/no
Arguments
fundef — Guard (fundefp fundef).
Returns
yes/no — Type (booleanp yes/no).

The function definition itself is okay: we just check that its body has no function definitions. This is the condition of all the function definitions after the FunctionHoister transformation.
