Basic constructor macro for numeric-value-double structures.

Syntax
(make-numeric-value-double [:get <get>])

This is the usual way to construct numeric-value-double structures. It simply conses together a structure with the specified fields.

This macro generates a new numeric-value-double structure from scratch. See also change-numeric-value-double, which can "change" an existing structure, instead.
