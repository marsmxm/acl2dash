Modifying constructor for nat/natlist-result-ok structures.

Syntax
(change-nat/natlist-result-ok x 
                              [:get <get>])

This is an often useful alternative to make-nat/natlist-result-ok.

We construct a new nat/natlist-result-ok structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
