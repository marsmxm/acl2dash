(vl-luciddb-fix x) is an ACL2::fty alist fixing function that follows the fix-keys strategy.

Signature

Arguments
x — Guard (vl-luciddb-p x).
Returns
fty::newx — Type (vl-luciddb-p fty::newx).

Note that in the execution this is just an inline identity function.
