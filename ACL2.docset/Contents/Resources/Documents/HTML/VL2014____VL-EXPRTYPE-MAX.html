(vl-exprtype-max x y &rest rst) is given vl-exprtype-ps as arguments; it returns :vl-unsigned if any argument is unsigned, or :vl-signed when all arguments are signed.

Function: vl-exprtype-max-fn

(defun vl-exprtype-max-fn (x y)
       (declare (xargs :guard (and (vl-exprtype-p x)
                                   (vl-exprtype-p y))))
       (let ((x-fix (mbe :logic (case (vl-exprtype-fix x)
                                      (:vl-signed :vl-signed)
                                      (otherwise :vl-unsigned))
                         :exec x))
             (y-fix (mbe :logic (case (vl-exprtype-fix y)
                                      (:vl-signed :vl-signed)
                                      (otherwise :vl-unsigned))
                         :exec y)))
            (if (or (eq x-fix :vl-unsigned)
                    (eq y-fix :vl-unsigned))
                :vl-unsigned :vl-signed)))
