Specify the search strategy to be used.

Specify which of the following strategies to use for instantiating free variables of the conjecture under test: :simple or :incremental. :incremental uses a dpll-like algorithm to search for counterexamples (and is currently much slower). By default this parameter is set to :simple. 
Usage: 
(acl2s-defaults :set search-strategy :simple) 
(acl2s-defaults :get search-strategy)
