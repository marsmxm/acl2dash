Weaker version of element equivalence for four-valued alists.

This is a universal equivalence, introduced using def-universal-equiv.

Function: 4v-cdr-equiv

(defun 4v-cdr-equiv (x y)
       (and (4v-equiv (cdr x) (cdr y))))
