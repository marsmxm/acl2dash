Modifying constructor for matchstate structures.

Syntax
(change-matchstate x 
                   [:index <index>] 
                   [:backrefs <backrefs>])

This is an often useful alternative to make-matchstate.

We construct a new matchstate structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-matchstate

(defmacro
     change-matchstate (x &rest args)
     (std::change-aggregate 'matchstate
                            x args
                            '((:index . matchstate->index)
                              (:backrefs . matchstate->backrefs))
                            'change-matchstate
                            'nil))
