Modifying constructor for vl-propalways structures.

Syntax
(change-vl-propalways x 
                      [:strongp <strongp>] 
                      [:range <range>] 
                      [:prop <prop>])

This is an often useful alternative to make-vl-propalways.

We construct a new vl-propalways structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-propalways

(defmacro
     change-vl-propalways (x &rest args)
     (std::change-aggregate 'vl-propalways
                            x args
                            '((:strongp . vl-propalways->strongp)
                              (:range . vl-propalways->range)
                              (:prop . vl-propalways->prop))
                            'change-vl-propalways
                            'nil))
