Modifying constructor for pipeline-setup structures.

Syntax
(change-pipeline-setup x 
                       [:probes <probes>] 
                       [:inputs <inputs>] 
                       [:overrides <overrides>] 
                       [:initst <initst>])

This is an often useful alternative to make-pipeline-setup.

We construct a new pipeline-setup structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-pipeline-setup

(defmacro
   change-pipeline-setup (x &rest args)
   (std::change-aggregate 'pipeline-setup
                          x args
                          '((:probes . pipeline-setup->probes)
                            (:inputs . pipeline-setup->inputs)
                            (:overrides . pipeline-setup->overrides)
                            (:initst . pipeline-setup->initst))
                          'change-pipeline-setup
                          'nil))
