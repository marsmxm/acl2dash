Basic constructor macro for base-fsm structures.

Syntax
(make-base-fsm [:values <values>] 
               [:nextstate <nextstate>])

This is the usual way to construct base-fsm structures. It simply conses together a structure with the specified fields.

This macro generates a new base-fsm structure from scratch. See also change-base-fsm, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by defprod.

(defmacro make-base-fsm (&rest args)
