Check if address is valid Bech32 or Bech32m.

Signature
(valid-bech32-or-bech32m address) → y/n
Arguments
address — Guard (stringp address).
Returns
y/n — Type (booleanp y/n).

Verifies that address is a valid Bech32 or Bech32m string. Does not indicate which kind it is.

Disallows a mixed-case string; then downcases the string before converting the data characters to bytes and verifying the checksum.

Function: valid-bech32-or-bech32m

(defun valid-bech32-or-bech32m (address)
       (declare (xargs :guard (stringp address)))
       (let ((__function__ 'valid-bech32-or-bech32m))
            (declare (ignorable __function__))
            (b* (((when (mixed-case-stringp address))
                  nil)
                 (address (str::downcase-string address))
                 ((mv erp hrp data)
                  (bech32-split-address address))
                 ((when erp) nil))
                (bech32-or-bech32m-verify-checksum hrp data))))
