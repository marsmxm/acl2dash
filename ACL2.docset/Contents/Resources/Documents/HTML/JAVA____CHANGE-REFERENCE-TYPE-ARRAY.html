Modifying constructor for reference-type-array structures.

Syntax
(change-reference-type-array x 
                             [:get <get>])

This is an often useful alternative to make-reference-type-array.

We construct a new reference-type-array structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-reference-type-array

(defmacro
     change-reference-type-array
     (x &rest args)
     (std::change-aggregate 'reference-type-array
                            x args
                            '((:get . reference-type-array->get))
                            'change-reference-type-array
                            'nil))
