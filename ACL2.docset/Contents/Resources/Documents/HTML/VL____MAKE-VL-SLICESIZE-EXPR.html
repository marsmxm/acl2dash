Basic constructor macro for vl-slicesize-expr structures.

Syntax
(make-vl-slicesize-expr [:expr <expr>])

This is the usual way to construct vl-slicesize-expr structures. It simply conses together a structure with the specified fields.

This macro generates a new vl-slicesize-expr structure from scratch. See also change-vl-slicesize-expr, which can "change" an existing structure, instead.

This is an ordinary make- macro introduced by defprod.
