Modifying constructor for function-recursion structures.

Syntax
(change-function-recursion x 
                           [:definitions <definitions>])

This is an often useful alternative to make-function-recursion.

We construct a new function-recursion structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-function-recursion

(defmacro change-function-recursion (x &rest args)
          (std::change-aggregate
               'function-recursion
               x args
               '((:definitions . function-recursion->definitions))
               'change-function-recursion
               'nil))
