Modifying constructor for object structures.

Syntax
(change-object x 
               [:name <name>] 
               [:code <code>] 
               [:sub/data <sub/data>])

This is an often useful alternative to make-object.

We construct a new object structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-object

(defmacro change-object (x &rest args)
          (std::change-aggregate 'object
                                 x args
                                 '((:name . object->name)
                                   (:code . object->code)
                                   (:sub/data . object->sub/data))
                                 'change-object
                                 'nil))
