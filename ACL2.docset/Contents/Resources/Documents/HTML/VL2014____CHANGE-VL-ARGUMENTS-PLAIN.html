Modifying constructor for vl-arguments-plain structures.

Syntax
(change-vl-arguments-plain x 
                           [:args <args>])

This is an often useful alternative to make-vl-arguments-plain.

We construct a new vl-arguments-plain structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-arguments-plain

(defmacro
     change-vl-arguments-plain (x &rest args)
     (std::change-aggregate 'vl-arguments-plain
                            x args
                            '((:args . vl-arguments-plain->args))
                            'change-vl-arguments-plain
                            'nil))
