Modifying constructor for vl-importresult structures.

Syntax
(change-vl-importresult x 
                        [:item <item>] 
                        [:pkg-name <pkg-name>] 
                        [:loc <loc>])

This is an often useful alternative to make-vl-importresult.

We construct a new vl-importresult structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-importresult

(defmacro
    change-vl-importresult (x &rest args)
    (std::change-aggregate 'vl-importresult
                           x args
                           '((:item . vl-importresult->item)
                             (:pkg-name . vl-importresult->pkg-name)
                             (:loc . vl-importresult->loc))
                           'change-vl-importresult
                           'remake-vl-importresult))
