Modifying constructor for vl-funname structures.

Syntax
(change-vl-funname x 
                   [:name <name>])

This is an often useful alternative to make-vl-funname.

We construct a new vl-funname structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-funname

(defmacro
     change-vl-funname (x &rest args)
     (std::change-aggregate 'vl-funname
                            x args '((:name . vl-funname->name))
                            'change-vl-funname
                            'remake-vl-funname))
