Representation of a single property or sequence port.

This is a product type introduced by defprod.

Fields
name — stringp
Name of this port.
localp — booleanp
True if the local keyword was provided.
dir — vl-direction-p
The direction for this port. Note that the ports for a sequence can be either input, output, or inout ports. The ports for a property are always input ports.
type — vl-datatype
The type declared for this port, if any. Note that the special keywords property, sequence, and untyped can also be used here; we represent them as ordinary vl-datatypes, see vl-coretypename-p. This type includes any array dimensions associated with the port.
arg — vl-propactual
The default property or sequence actual assigned to this port. If there is no such assignment, this will just be a blank vl-propactual.
atts — vl-atts
Any (* foo = bar, baz *) style attributes.
loc — vl-location
