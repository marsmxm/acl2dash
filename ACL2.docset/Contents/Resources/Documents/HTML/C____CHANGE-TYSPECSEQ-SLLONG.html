Modifying constructor for tyspecseq-sllong structures.

Syntax
(change-tyspecseq-sllong x 
                         [:signed <signed>] 
                         [:int <int>])

This is an often useful alternative to make-tyspecseq-sllong.

We construct a new tyspecseq-sllong structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-tyspecseq-sllong

(defmacro
     change-tyspecseq-sllong (x &rest args)
     (std::change-aggregate 'tyspecseq-sllong
                            x args
                            '((:signed . tyspecseq-sllong->signed)
                              (:int . tyspecseq-sllong->int))
                            'change-tyspecseq-sllong
                            'nil))
