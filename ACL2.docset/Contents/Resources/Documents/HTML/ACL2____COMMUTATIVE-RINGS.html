Axiomatization of two associative and commutative operations, one distributes over the other, while the other has an identity and an unary inverse operation, developed by John Cowles.

Theory of Commutative Rings

ACL2-CRG::plus and ACL2-CRG::times are associative and commutative binary operations on the set (of equivalence classes formed by the equivalence relation, ACL2-CRG::equiv, on the set)

RG = { x | (ACL2-CRG::pred x) != nil }

with ACL2-CRG::times distributing over ACL2-CRG::plus.

ACL2-CRG::zero is a constant in the set RG which acts as an unit for ACL2-CRG::plus.

ACL2-CRG::minus is an unary operation on the set (of equivalence classes formed by the equivalence relation, ACL2-CRG::equiv, on the set) RG which acts as an ACL2-CRG::plus-inverse for ACL2-CRG::zero.

For example, let

  • ACL2-CRG::pred = Booleanp,
  • ACL2-CRG::plus = exclusive-or,
  • ACL2-CRG::times = and,
  • ACL2-CRG::zero = nil, and
  • ACL2-CRG::minus = identity function.

Axioms of the theory of Commutative Rings

Using encapsulate, we introduce constrained functions:

  • (equiv x y)
  • (pred x)
  • (plus x y)
  • (times x y)
  • (zero)
  • (minus x)

with the following, constraining axioms:

Theorem: equiv-is-an-equivalence

(defthm
 equiv-is-an-equivalence
 (and (booleanp (equiv x y))
      (equiv x x)
      (implies (equiv x y) (equiv y x))
      (implies (and (equiv x y) (equiv y z))
               (equiv x z)))
 :rule-classes
 (:equivalence
      (:type-prescription :corollary (or (equal (equiv x y) t)
                                         (equal (equiv x y) nil)))))

Theorem: equiv-implies-iff-pred-1

(defthm equiv-implies-iff-pred-1
        (implies (equiv x x-equiv)
                 (iff (pred x) (pred x-equiv)))
        :rule-classes (:congruence))

Theorem: equiv-implies-equiv-plus-1

(defthm equiv-implies-equiv-plus-1
        (implies (equiv x x-equiv)
                 (equiv (plus x y) (plus x-equiv y)))
        :rule-classes (:congruence))

Theorem: equiv-implies-equiv-plus-2

(defthm equiv-implies-equiv-plus-2
        (implies (equiv y y-equiv)
                 (equiv (plus x y) (plus x y-equiv)))
        :rule-classes (:congruence))

Theorem: equiv-implies-equiv-times-1

(defthm equiv-implies-equiv-times-1
        (implies (equiv x x-equiv)
                 (equiv (times x y) (times x-equiv y)))
        :rule-classes (:congruence))

Theorem: equiv-implies-equiv-times-2

(defthm equiv-implies-equiv-times-2
        (implies (equiv y y-equiv)
                 (equiv (times x y) (times x y-equiv)))
        :rule-classes (:congruence))

Theorem: equiv-implies-equiv-minus-1

(defthm equiv-implies-equiv-minus-1
        (implies (equiv x x-equiv)
                 (equiv (minus x) (minus x-equiv)))
        :rule-classes (:congruence))

Theorem: closure-of-plus-for-pred

(defthm closure-of-plus-for-pred
        (implies (and (pred x) (pred y))
                 (pred (plus x y))))

Theorem: closure-of-times-for-pred

(defthm closure-of-times-for-pred
        (implies (and (pred x) (pred y))
                 (pred (times x y))))

Theorem: closure-of-zero-for-pred

(defthm closure-of-zero-for-pred (pred (zero)))

Theorem: closure-of-minus-for-pred

(defthm closure-of-minus-for-pred
        (implies (pred x) (pred (minus x))))

Theorem: commutativity-of-plus

(defthm commutativity-of-plus
        (implies (and (pred x) (pred y))
                 (equiv (plus x y) (plus y x))))

Theorem: commutativity-of-times

(defthm commutativity-of-times
        (implies (and (pred x) (pred y))
                 (equiv (times x y) (times y x))))

Theorem: associativity-of-plus

(defthm associativity-of-plus
        (implies (and (pred x) (pred y) (pred z))
                 (equiv (plus (plus x y) z)
                        (plus x (plus y z)))))

Theorem: associativity-of-times

(defthm associativity-of-times
        (implies (and (pred x) (pred y) (pred z))
                 (equiv (times (times x y) z)
                        (times x (times y z)))))

Theorem: left-distributivity-of-times-over-plus

(defthm left-distributivity-of-times-over-plus
        (implies (and (pred x) (pred y) (pred z))
                 (equiv (times x (plus y z))
                        (plus (times x y) (times x z)))))

Theorem: left-unicity-of-zero-for-plus

(defthm left-unicity-of-zero-for-plus
        (implies (pred x)
                 (equiv (plus (zero) x) x)))

Theorem: right-inverse-for-plus

(defthm right-inverse-for-plus
        (implies (pred x)
                 (equiv (plus x (minus x)) (zero))))

Theorems of the theory of Commutative Rings

Given the above constraints, we prove the following generic theorems.

Besides the theorems below, note that <RG, ACL2-CRG::plus> and <RG, ACL2-CRG::times> are both semigroups, and <RG, ACL2-CRG::plus, ACL2-CRG::minus, ACL2-CRG::zero> is an Abelian Group. Thus, additional theorems of the theory of Commutative Rings may be obtained as instances of the theorems of the theories of abelian-semigroups and abelian-groups.

Theorem: right-distributivity-of-times-over-plus

(defthm right-distributivity-of-times-over-plus
        (implies (and (pred x) (pred y) (pred z))
                 (equiv (times (plus x y) z)
                        (plus (times x z) (times y z)))))

Theorem: left-nullity-of-zero-for-times

(defthm left-nullity-of-zero-for-times
        (implies (pred x)
                 (equiv (times (zero) x) (zero))))

Theorem: right-nullity-of-zero-for-times

(defthm right-nullity-of-zero-for-times
        (implies (pred x)
                 (equiv (times x (zero)) (zero))))

Theorem: functional-commutativity-of-minus-times-right

(defthm functional-commutativity-of-minus-times-right
        (implies (and (pred x) (pred y))
                 (equiv (times x (minus y))
                        (minus (times x y)))))

Theorem: functional-commutativity-of-minus-times-left

(defthm functional-commutativity-of-minus-times-left
        (implies (and (pred x) (pred y))
                 (equiv (times (minus x) y)
                        (minus (times x y)))))
