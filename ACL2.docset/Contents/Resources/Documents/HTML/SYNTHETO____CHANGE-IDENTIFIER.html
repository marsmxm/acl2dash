Modifying constructor for identifier structures.

Syntax
(change-identifier x 
                   [:name <name>])

This is an often useful alternative to make-identifier.

We construct a new identifier structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-identifier

(defmacro
     change-identifier (x &rest args)
     (std::change-aggregate 'identifier
                            x args '((:name . identifier->name))
                            'change-identifier
                            'nil))
