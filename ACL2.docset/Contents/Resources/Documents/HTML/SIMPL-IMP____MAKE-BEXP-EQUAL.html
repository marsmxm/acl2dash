Basic constructor macro for bexp-equal structures.

Syntax
(make-bexp-equal [:left <left>] 
                 [:right <right>])

This is the usual way to construct bexp-equal structures. It simply conses together a structure with the specified fields.

This macro generates a new bexp-equal structure from scratch. See also change-bexp-equal, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by fty::defprod.

(defun bexp-equal (left right)
       (declare (xargs :guard (and (aexpp left) (aexpp right))))
       (declare (xargs :guard t))
       (let ((__function__ 'bexp-equal))
            (declare (ignorable __function__))
            (b* ((left (mbe :logic (aexp-fix left) :exec left))
                 (right (mbe :logic (aexp-fix right)
                             :exec right)))
                (cons :equal (list left right)))))
