Basic constructor macro for numeric-value-byte structures.

Syntax
(make-numeric-value-byte [:get <get>])

This is the usual way to construct numeric-value-byte structures. It simply conses together a structure with the specified fields.

This macro generates a new numeric-value-byte structure from scratch. See also change-numeric-value-byte, which can "change" an existing structure, instead.
