Get the starting location for a token.

Signature
(vl-token->loc x) → loc
Arguments
x — Guard (vl-token-p x).
Returns
loc — Type (vl-location-p loc), given the guard.

Because etext is always required to be a non-empty, we can say that each token has a location, namely the location of its first character.

Definitions and Theorems

Function: vl-token->loc$inline

(defun vl-token->loc$inline (x)
       (declare (xargs :guard (vl-token-p x)))
       (let ((__function__ 'vl-token->loc))
            (declare (ignorable __function__))
            (vl-echar->loc (car (vl-token->etext x)))))
