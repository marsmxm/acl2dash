Basic constructor macro for type-ulong structures.

Syntax
(make-type-ulong )

This is the usual way to construct type-ulong structures. It simply conses together a structure with the specified fields.

This macro generates a new type-ulong structure from scratch. See also change-type-ulong, which can "change" an existing structure, instead.
