Modifying constructor for vl-hidexpr-dot structures.

Syntax
(change-vl-hidexpr-dot x 
                       [:first <first>] 
                       [:rest <rest>])

This is an often useful alternative to make-vl-hidexpr-dot.

We construct a new vl-hidexpr-dot structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-hidexpr-dot

(defmacro change-vl-hidexpr-dot (x &rest args)
          (std::change-aggregate 'vl-hidexpr-dot
                                 x args
                                 '((:first . vl-hidexpr-dot->first)
                                   (:rest . vl-hidexpr-dot->rest))
                                 'change-vl-hidexpr-dot
                                 'nil))
