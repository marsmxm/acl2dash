Modifying constructor for plain-string structures.

Syntax
(change-plain-string x 
                     [:content <content>] 
                     [:double-quote-p <double-quote-p>])

This is an often useful alternative to make-plain-string.

We construct a new plain-string structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-plain-string

(defmacro change-plain-string (x &rest args)
          (std::change-aggregate
               'plain-string
               x args
               '((:content . plain-string->content)
                 (:double-quote-p . plain-string->double-quote-p))
               'change-plain-string
               'nil))
