Modifying constructor for expr-cast structures.

Syntax
(change-expr-cast x 
                  [:type <type>] 
                  [:arg <arg>])

This is an often useful alternative to make-expr-cast.

We construct a new expr-cast structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-expr-cast

(defmacro change-expr-cast (x &rest args)
          (std::change-aggregate 'expr-cast
                                 x args
                                 '((:type . expr-cast->type)
                                   (:arg . expr-cast->arg))
                                 'change-expr-cast
                                 'nil))
