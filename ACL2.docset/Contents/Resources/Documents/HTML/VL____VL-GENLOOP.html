A loop generate construct, before elaboration.

This is a product type, introduced by deftagsum in support of vl-genelement.

Fields
var — stringp
Iterator variable for this generate loop.
genvarp — booleanp
Is the variable declared using the genvar keyword, locally
initval — vl-expr
Initial value of the variable.
continue — vl-expr
Continue the loop until this expression is false.
nextval — vl-expr
Next value expression for the variable.
body — vl-genblock
Body of the loop.
loc — vl-location
Where this loop came from in the Verilog source code.

This structure captures something like the ``just parsed'' form of a generate loop. Elaboration should convert these into the more regular vl-genarray form.
