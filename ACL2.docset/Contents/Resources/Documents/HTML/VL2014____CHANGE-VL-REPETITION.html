Modifying constructor for vl-repetition structures.

Syntax
(change-vl-repetition x 
                      [:type <type>] 
                      [:left <left>] 
                      [:right <right>])

This is an often useful alternative to make-vl-repetition.

We construct a new vl-repetition structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-repetition

(defmacro change-vl-repetition (x &rest args)
          (std::change-aggregate 'vl-repetition
                                 x args
                                 '((:type . vl-repetition->type)
                                   (:left . vl-repetition->left)
                                   (:right . vl-repetition->right))
                                 'change-vl-repetition
                                 'remake-vl-repetition))
