Basic constructor macro for ulong structures.

Syntax
(make-ulong [:get <get>])

This is the usual way to construct ulong structures. It simply conses together a structure with the specified fields.

This macro generates a new ulong structure from scratch. See also change-ulong, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by fty::defprod.

Macro: make-ulong

(defmacro make-ulong (&rest args)
          (std::make-aggregate 'ulong
                               args '((:get))
                               'make-ulong
                               nil))
