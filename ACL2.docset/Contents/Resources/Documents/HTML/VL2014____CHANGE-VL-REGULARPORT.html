Modifying constructor for vl-regularport structures.

Syntax
(change-vl-regularport x 
                       [:name <name>] 
                       [:expr <expr>] 
                       [:loc <loc>])

This is an often useful alternative to make-vl-regularport.

We construct a new vl-regularport structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-regularport

(defmacro change-vl-regularport (x &rest args)
          (std::change-aggregate 'vl-regularport
                                 x args
                                 '((:name . vl-regularport->name)
                                   (:expr . vl-regularport->expr)
                                   (:loc . vl-regularport->loc))
                                 'change-vl-regularport
                                 'remake-vl-regularport))
