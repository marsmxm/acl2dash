Fixtype of information about members of shallowly embedded C structures.

This is a product type introduced by fty::defprod.

Fields
memtype — member-type
readers — symbol-list
writers — symbol-list
checkers — symbol-list
reader-return-thms — symbol-list
writer-return-thms — symbol-list

These are part of defstruct-info.

For each member, we store:

  • The member type, which consists of the name and type of the member. See member-type in the deep embedding.
  • The named of the readers of the member. For an integer member, it is a singleton list of the one reader. For an array member, it is a list of ten readers, one for each supported integer type for the index.
  • The names of the writers of the member. For an integer member, it is a singleton list of the one writer. For an array member, it is a list of ten writers, one for each supported integer type for the index.
  • The names of the checkers of the member. This is the empty list for an integer member, while it consists of ten checkers for an array member, one for each supported integer type for the index.
  • The names of the return type theorems for all the readers, in the same order as the list of readers.
  • The names of the return type theorems for all the writers, in the same order as the list of writers.
