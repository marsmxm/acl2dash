Option type; vl-elabkey or nil.

This is an option type introduced by defoption. Note that defoption is just a wrapper for defflexsum, so there are :none and :some member types, a case macro, and so forth.

Member Types
:none → vl-maybe-elabkey-none
Represents that no vl-maybe-elabkey is available, i.e., Nothing or None.
:some → vl-maybe-elabkey-some
An available vl-maybe-elabkey, i.e., Just val or Some val.
