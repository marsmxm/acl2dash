Basic constructor macro for pipeline-setup structures.

Syntax
(make-pipeline-setup [:probes <probes>] 
                     [:inputs <inputs>] 
                     [:overrides <overrides>] 
                     [:initst <initst>])

This is the usual way to construct pipeline-setup structures. It simply conses together a structure with the specified fields.

This macro generates a new pipeline-setup structure from scratch. See also change-pipeline-setup, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by defprod.

Macro: make-pipeline-setup

(defmacro make-pipeline-setup (&rest args)
          (std::make-aggregate 'pipeline-setup
                               args
                               '((:probes)
                                 (:inputs)
                                 (:overrides)
                                 (:initst))
                               'make-pipeline-setup
                               nil))

Function: pipeline-setup

(defun
     pipeline-setup
     (probes inputs overrides initst)
     (declare (xargs :guard (and (svtv-probealist-p probes)
                                 (svex-alistlist-p inputs)
                                 (svex-alistlist-p overrides)
                                 (svex-alist-p initst))))
     (declare (xargs :guard t))
     (let ((__function__ 'pipeline-setup))
          (declare (ignorable __function__))
          (b* ((probes (mbe :logic (svtv-probealist-fix probes)
                            :exec probes))
               (inputs (mbe :logic (svex-alistlist-fix inputs)
                            :exec inputs))
               (overrides (mbe :logic (svex-alistlist-fix overrides)
                               :exec overrides))
               (initst (mbe :logic (svex-alist-fix initst)
                            :exec initst)))
              (list (cons 'probes probes)
                    (cons 'inputs inputs)
                    (cons 'overrides overrides)
                    (cons 'initst initst)))))
