(vl-print-str x) is an optimized version of vl-print for strings.

Macro: vl-print-str

(defmacro
 vl-print-str (x &key (ps 'ps))
 (cond ((equal x "") ps)
       ((and (stringp x)
             (not (vl-string-needs-html-encoding-p x 0 (length x))))
        (cons 'vl-print-raw-fast
              (cons x
                    (cons (length x)
                          (cons ':ps (cons ps 'nil))))))
       (t (cons 'vl-print-str-main
                (cons x (cons ':ps (cons ps 'nil)))))))
