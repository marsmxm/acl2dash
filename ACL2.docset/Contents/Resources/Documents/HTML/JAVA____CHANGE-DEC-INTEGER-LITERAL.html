Modifying constructor for dec-integer-literal structures.

Syntax
(change-dec-integer-literal x 
                            [:digits/uscores <digits/uscores>] 
                            [:suffix? <suffix?>])

This is an often useful alternative to make-dec-integer-literal.

We construct a new dec-integer-literal structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-dec-integer-literal

(defmacro
     change-dec-integer-literal
     (x &rest args)
     (std::change-aggregate
          'dec-integer-literal
          x args
          '((:digits/uscores . dec-integer-literal->digits/uscores)
            (:suffix? . dec-integer-literal->suffix?))
          'change-dec-integer-literal
          'nil))
