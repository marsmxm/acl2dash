Extend an alist by binding the names of vl-paramdecls to their definitions.

Signature
(vl-paramdecllist-alist x acc)  alist
Arguments
x Guard (vl-paramdecllist-p x).
Returns
alist Type (equal (vl-paramdecl-alist-p alist) (vl-paramdecl-alist-p acc)).

This can be used as an alternative to vl-find-paramdecl when you need to perform a lot of lookups.

Definitions and Theorems

Function: vl-paramdecllist-alist

(defun vl-paramdecllist-alist (x acc)
       (declare (xargs :guard (vl-paramdecllist-p x)))
       (let ((__function__ 'vl-paramdecllist-alist))
            (declare (ignorable __function__))
            (if (atom x)
                acc
                (cons (cons (vl-paramdecl->name (car x))
                            (vl-paramdecl-fix (car x)))
                      (vl-paramdecllist-alist (cdr x) acc)))))

Theorem: lookup-in-vl-paramdecllist-alist-fast

(defthm
  lookup-in-vl-paramdecllist-alist-fast
  (implies
       (stringp name)
       (equal (hons-assoc-equal name (vl-paramdecllist-alist x nil))
              (let ((val (vl-find-paramdecl name x)))
                   (and val (cons name val))))))
