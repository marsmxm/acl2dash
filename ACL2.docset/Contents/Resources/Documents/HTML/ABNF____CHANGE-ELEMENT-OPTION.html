Modifying constructor for element-option structures.

Syntax
(change-element-option x 
                       [:get <get>])

This is an often useful alternative to make-element-option.

We construct a new element-option structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-element-option

(defmacro
     change-element-option (x &rest args)
     (std::change-aggregate 'element-option
                            x args '((:get . element-option->get))
                            'change-element-option
                            'nil))
