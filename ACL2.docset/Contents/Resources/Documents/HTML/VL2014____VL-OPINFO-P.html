(vl-opinfo-p x) is a defaggregate of the following fields.

  • arity — For operators that have a fixed number of arguments this is the associated arity. For operations that do not have fixed arities (e.g., concatenation, function calls, ...), we just say the arity is nil.
        Invariant (maybe-natp arity).
  • text — Printed representation of this operator like "+", if applicable. This is intended for use in lightweight pretty printing, i.e., if you are writing a warning about a particular expression, you might want to refer to a "+ operator" or similar.
        Invariant (maybe-stringp text).
