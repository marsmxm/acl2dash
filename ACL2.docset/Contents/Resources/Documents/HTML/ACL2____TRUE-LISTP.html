Recognizer for proper (nil-terminated) lists

True-listp is the function that checks whether its argument is a list that ends in, or equals, nil.

For a discussion of whether requiring lists to be nil-terminated is right for you, see std::strict-list-recognizers.

Function: true-listp

(defun true-listp (x)
       (declare (xargs :guard t))
       (if (consp x)
           (true-listp (cdr x))
           (eq x nil)))
