Modifying constructor for element-num-val structures.

Syntax
(change-element-num-val x 
                        [:get <get>])

This is an often useful alternative to make-element-num-val.

We construct a new element-num-val structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-element-num-val

(defmacro
