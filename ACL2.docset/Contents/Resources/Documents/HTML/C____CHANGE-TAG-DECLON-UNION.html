Modifying constructor for tag-declon-union structures.

Syntax
(change-tag-declon-union x 
                         [:tag <tag>] 
                         [:members <members>])

This is an often useful alternative to make-tag-declon-union.

We construct a new tag-declon-union structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-tag-declon-union

(defmacro
    change-tag-declon-union (x &rest args)
    (std::change-aggregate 'tag-declon-union
                           x args
                           '((:tag . tag-declon-union->tag)
                             (:members . tag-declon-union->members))
                           'change-tag-declon-union
                           'nil))
