Modifying constructor for escape-sequence-n structures.

Syntax
(change-escape-sequence-n x 
                          )

This is an often useful alternative to make-escape-sequence-n.

We construct a new escape-sequence-n structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
