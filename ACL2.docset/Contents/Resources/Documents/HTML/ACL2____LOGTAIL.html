(logtail pos i) returns the high-order part of i, starting at bit position pos.

Signature
(logtail pos i) → int
Arguments
pos — Guard (and (integerp pos) (<= 0 pos)).
i — Guard (integerp i).
Returns
int — Type (integerp int).

In languages like C, this might be written as i >> pos.
