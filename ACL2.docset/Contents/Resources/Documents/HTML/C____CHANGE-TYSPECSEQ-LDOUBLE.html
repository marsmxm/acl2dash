Modifying constructor for tyspecseq-ldouble structures.

Syntax
(change-tyspecseq-ldouble x 
                          [:complex <complex>])

This is an often useful alternative to make-tyspecseq-ldouble.

We construct a new tyspecseq-ldouble structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-tyspecseq-ldouble

(defmacro
   change-tyspecseq-ldouble (x &rest args)
