Modifying constructor for outcome-theorem-success structures.

Syntax
(change-outcome-theorem-success x 
                                [:message <message>])

This is an often useful alternative to make-outcome-theorem-success.

We construct a new outcome-theorem-success structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
