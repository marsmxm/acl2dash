Modifying constructor for identifier-list-result-err structures.

Syntax
(change-identifier-list-result-err x 
                                   [:get <get>])

This is an often useful alternative to make-identifier-list-result-err.

We construct a new identifier-list-result-err structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
