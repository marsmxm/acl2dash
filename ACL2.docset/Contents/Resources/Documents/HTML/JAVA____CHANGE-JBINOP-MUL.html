Modifying constructor for jbinop-mul structures.

Syntax
(change-jbinop-mul x 
                   )

This is an often useful alternative to make-jbinop-mul.

We construct a new jbinop-mul structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

This is an ordinary change- macro introduced by fty::defprod.
