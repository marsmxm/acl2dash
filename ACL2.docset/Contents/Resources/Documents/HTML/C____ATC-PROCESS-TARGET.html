Process a target among t1, ..., tp.

Signature
(atc-process-target target previous-structs 
                    previous-fns uncalled-fns ctx state) 
 
  → 
(mv erp val state)
Arguments
previous-structs — Guard (symbol-listp previous-structs).
previous-fns — Guard (symbol-listp previous-fns).
uncalled-fns — Guard (symbol-listp uncalled-fns).
ctx — Guard (ctxp ctx).
Returns
val — Type (tuple (new-previous-structs symbol-listp) (new-previous-fns symbol-listp) (new-uncalled-fns symbol-listp) val) , given (and (symbol-listp previous-structs) (symbol-listp previous-fns) (symbol-listp uncalled-fns)) .

The parameters previous-fns and uncalled-fns are explained in atc-process-function. The parameter previous-structs is analogous to previous-fns, but for the defstruct targets instead of the function targets: it lists all the defstruct targets that precede target in the list of targets (t1 ... tp). This is used to detect duplicate defstruct targets.

If the target is a function name, its processing is delegated to atc-process-function. Otherwise, it must be a defstruct name, and it is processed here. We just check that the it is in the defstruct table.

Definitions and Theorems

Function: atc-process-target

(defun
 atc-process-target
 (target previous-structs
         previous-fns uncalled-fns ctx state)
 (declare (xargs :stobjs (state)))
 (declare (xargs :guard (and (symbol-listp previous-structs)
                             (symbol-listp previous-fns)
                             (symbol-listp uncalled-fns)
                             (ctxp ctx))))
 (let
  ((__function__ 'atc-process-target))
  (declare (ignorable __function__))
  (b*
   ((irrelevant (list nil nil nil))
    ((when (acl2::function-namep target (w state)))
     (b*
       (((mv erp (list previous-fns uncalled-fns)
             state)
         (atc-process-function target
                               previous-fns uncalled-fns ctx state))
        ((when erp) (mv erp irrelevant state)))
       (acl2::value (list previous-structs
                          previous-fns uncalled-fns))))
    ((when (member-eq target previous-structs))
     (er-soft+
      ctx t irrelevant
      "The target DEFSTRUCT name ~x0 appears more than once ~
                   in the list of targets."
      target))
    ((unless (symbolp target))
     (er-soft+ ctx t irrelevant
               "The target ~x0 is not a symbol."
               target))
    ((unless (defstruct-table-lookup (symbol-name target)
                                     (w state)))
     (er-soft+
      ctx t irrelevant
      "The target ~x0 is neither a function name ~
                   nor a DEFSTRUCT name."
      target))
    (previous-structs (cons target previous-structs)))
   (acl2::value (list previous-structs
                      previous-fns uncalled-fns)))))
