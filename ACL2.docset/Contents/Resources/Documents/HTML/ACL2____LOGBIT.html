(logbit pos i) returns the bit of i at bit-position pos as a bitp, 0 or 1.

Signature
(logbit pos i) → bit
Arguments
pos — Guard (natp pos).
i — Guard (integerp i).
Returns
bit — Type (bitp bit).

This is just like the Common Lisp function (logbitp pos i), except that we return 1 or 0 (instead of t or nil).

In languages like C, this might be written as (i >> pos) & 1.
