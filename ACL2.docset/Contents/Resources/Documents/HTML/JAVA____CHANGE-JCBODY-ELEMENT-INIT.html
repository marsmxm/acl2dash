Modifying constructor for jcbody-element-init structures.

Syntax
(change-jcbody-element-init x 
                            [:get <get>])

This is an often useful alternative to make-jcbody-element-init.

We construct a new jcbody-element-init structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
