Fixtype of packages and nil.

This is an option type introduced by fty::defoption. Note that defoption is just a wrapper for fty::defflexsum, so there are :none and :some member types, a case macro, and so forth.

Member Types
:none → maybe-package-none
Represents that no maybe-package is available, i.e., Nothing or None.
:some → maybe-package-some
An available maybe-package, i.e., Just val or Some val.
