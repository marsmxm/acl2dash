Modifying constructor for maybe-function-some structures.

Syntax
(change-maybe-function-some x 
                            [:val <val>])

This is an often useful alternative to make-maybe-function-some.

We construct a new maybe-function-some structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-maybe-function-some

(defmacro
     change-maybe-function-some
     (x &rest args)
     (std::change-aggregate 'maybe-function-some
                            x args
                            '((:val . maybe-function-some->val))
                            'change-maybe-function-some
                            'nil))
