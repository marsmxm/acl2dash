Indicates the kind of an always statement.

In Verilog-2005 there are only always statements. SystemVerilog-2012 adds always_comb, always_latch, and always_ff.

This is an ordinary defenum.

Function: vl-alwaystype-p

(defun vl-alwaystype-p (x)
       (declare (xargs :guard t))
       (or (eq x ':vl-always)
           (eq x ':vl-always-comb)
           (eq x ':vl-always-latch)
           (eq x ':vl-always-ff)))

Theorem: type-when-vl-alwaystype-p

(defthm type-when-vl-alwaystype-p
        (implies (vl-alwaystype-p x)
                 (if (symbolp x)
                     (if (not (equal x 't))
                         (not (equal x 'nil))
                         'nil)
                     'nil))
        :rule-classes :compound-recognizer)
