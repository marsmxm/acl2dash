Modifying constructor for outcome-theorem-failure structures.

Syntax
(change-outcome-theorem-failure x 
                                [:message <message>])

This is an often useful alternative to make-outcome-theorem-failure.

We construct a new outcome-theorem-failure structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
