Fixtype of string elements.

This is a tagged union type, introduced by fty::deftagsum.

Member Tags Types
:char string-element-char
:escape string-element-escape

A string literal consists of a sequence of printable ASCII characters and escapes: these are the string elements we define here. We use ACL2 characters for the former, which can represent all the printable ASCII characters and more; We migh restrict the range of characters at some point.
