Gather all of the values throughout an expression.

Signature
(vl-expr-values x) → values
Arguments
x — Guard (vl-expr-p x).
Returns
values — Type (vl-valuelist-p values).

We simply gather all of the values of all of the literals throughout the expression, with repetition. The resulting list may contain any vl-value.

Theorem: return-type-of-vl-expr-values.values

(defthm return-type-of-vl-expr-values.values
        (b* ((common-lisp::?values (vl-expr-values x)))
            (vl-valuelist-p values))
        :rule-classes :rewrite)
