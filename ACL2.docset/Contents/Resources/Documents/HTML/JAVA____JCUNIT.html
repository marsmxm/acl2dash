We only capture ordinary compilation units, not modular ones. Thus, we do not capture module declarations either.

We capture the package declaration [JLS:7.4] as an ACL2 string for the name. This declaration is optional.

We do not capture interfaces, so a type declaration [JLS:7.6] is always a class declaration.
