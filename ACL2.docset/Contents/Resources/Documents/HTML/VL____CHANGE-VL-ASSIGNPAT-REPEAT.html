Modifying constructor for vl-assignpat-repeat structures.

Syntax
(change-vl-assignpat-repeat x 
                            [:reps <reps>] 
                            [:vals <vals>])

This is an often useful alternative to make-vl-assignpat-repeat.

We construct a new vl-assignpat-repeat structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-assignpat-repeat

(defmacro
