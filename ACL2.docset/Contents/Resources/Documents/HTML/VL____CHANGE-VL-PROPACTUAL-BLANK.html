Modifying constructor for vl-propactual-blank structures.

Syntax
(change-vl-propactual-blank x 
                            [:name <name>])

This is an often useful alternative to make-vl-propactual-blank.

We construct a new vl-propactual-blank structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.
