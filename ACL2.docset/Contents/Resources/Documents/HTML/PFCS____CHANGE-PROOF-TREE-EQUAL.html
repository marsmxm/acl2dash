Modifying constructor for proof-tree-equal structures.

Syntax
(change-proof-tree-equal x 
                         [:asg <asg>] 
                         [:left <left>] 
                         [:right <right>])

This is an often useful alternative to make-proof-tree-equal.

We construct a new proof-tree-equal structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-proof-tree-equal

(defmacro
