Modifying constructor for vl-genbegin structures.

Syntax
(change-vl-genbegin x 
                    [:block <block>])

This is an often useful alternative to make-vl-genbegin.

We construct a new vl-genbegin structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-genbegin

(defmacro
     change-vl-genbegin (x &rest args)
     (std::change-aggregate 'vl-genbegin
                            x args '((:block . vl-genbegin->block))
                            'change-vl-genbegin
                            'nil))
