Modifying constructor for jbinop-shl structures.

Syntax
(change-jbinop-shl x 
                   )

This is an often useful alternative to make-jbinop-shl.

We construct a new jbinop-shl structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
