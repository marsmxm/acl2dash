Modifying constructor for binary-op-implies structures.

Syntax
(change-binary-op-implies x 
                          )

This is an often useful alternative to make-binary-op-implies.

We construct a new binary-op-implies structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
