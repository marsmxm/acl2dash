Remove the underscores from a list of decimal digits and underscores.

Signature
(decdig/uscores-to-digits dus) → ds
Arguments
dus — Guard (decdig/uscore-listp dus).
Returns
ds — Type (dec-digit-listp ds).

Any underscores in a decimal-numeral in the grammar are just used for separation. This function returns the underlying sequence of decimal digits.
