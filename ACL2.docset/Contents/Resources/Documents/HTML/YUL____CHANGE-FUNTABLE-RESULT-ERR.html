Modifying constructor for funtable-result-err structures.

Syntax
(change-funtable-result-err x 
                            [:get <get>])

This is an often useful alternative to make-funtable-result-err.

We construct a new funtable-result-err structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-funtable-result-err

(defmacro
     change-funtable-result-err
     (x &rest args)
     (std::change-aggregate 'funtable-result-err
