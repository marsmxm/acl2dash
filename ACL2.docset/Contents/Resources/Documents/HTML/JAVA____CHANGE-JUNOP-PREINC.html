Modifying constructor for junop-preinc structures.

Syntax
(change-junop-preinc x 
                     )

This is an often useful alternative to make-junop-preinc.

We construct a new junop-preinc structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
