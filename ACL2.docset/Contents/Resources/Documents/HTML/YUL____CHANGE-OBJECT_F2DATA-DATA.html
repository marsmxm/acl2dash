Modifying constructor for object/data-data structures.

Syntax
(change-object/data-data x 
                         [:get <get>])

This is an often useful alternative to make-object/data-data.

We construct a new object/data-data structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-object/data-data

(defmacro
     change-object/data-data (x &rest args)
     (std::change-aggregate 'object/data-data
                            x args '((:get . object/data-data->get))
                            'change-object/data-data
                            'nil))
