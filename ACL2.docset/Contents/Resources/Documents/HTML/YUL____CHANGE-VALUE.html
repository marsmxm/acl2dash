Modifying constructor for value structures.

Syntax
(change-value x 
              [:get <get>])

This is an often useful alternative to make-value.

We construct a new value structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-value

(defmacro change-value (x &rest args)
          (std::change-aggregate 'value
                                 x args '((:get . value->get))
                                 'change-value
                                 'nil))
