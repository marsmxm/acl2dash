Gets the expressions from a variable declaration, for making implicit wires. We omit the expressions inside the datatype.

Signature
(vl-vardecl-exprs-for-implicit-wires x)  exprs
Arguments
x Guard (vl-vardecl-p x).
Returns
exprs Type (vl-exprlist-p exprs).

(defun vl-vardecl-exprs-for-implicit-wires (x)
       (declare (xargs :guard (vl-vardecl-p x)))
       (let ((__function__ 'vl-vardecl-exprs-for-implicit-wires))
