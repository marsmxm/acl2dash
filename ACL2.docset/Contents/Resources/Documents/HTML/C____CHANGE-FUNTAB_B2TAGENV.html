Modifying constructor for funtab+tagenv structures.

Syntax
(change-funtab+tagenv x 
                      [:funs <funs>] 
                      [:tags <tags>])

This is an often useful alternative to make-funtab+tagenv.

We construct a new funtab+tagenv structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
