(hex-pair-list-fix x) is a usual ACL2::fty list fixing function.

Signature
(hex-pair-list-fix x) → fty::newx
Arguments
x — Guard (hex-pair-listp x).
Returns
fty::newx — Type (hex-pair-listp fty::newx).

In the logic, we apply hex-pair-fix to each member of the x. In the execution, none of that is actually necessary and this is just an inlined identity function.
