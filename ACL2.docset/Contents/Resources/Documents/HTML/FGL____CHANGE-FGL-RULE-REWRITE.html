Modifying constructor for fgl-rule-rewrite structures.

Syntax
(change-fgl-rule-rewrite x 
                         [:rune <rune>] 
                         [:rule <rule>])

This is an often useful alternative to make-fgl-rule-rewrite.

We construct a new fgl-rule-rewrite structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-fgl-rule-rewrite
