(calist-fix x) is an ACL2::fty alist fixing function that follows the fix-keys strategy.

Signature
(calist-fix x) → fty::newx
Arguments
x — Guard (calistp x).
Returns
fty::newx — Type (calistp fty::newx).

Note that in the execution this is just an inline identity function.

Definitions and Theorems

Function: calist-fix$inline

(defun
 calist-fix$inline (x)
 (declare (xargs :guard (calistp x)))
 (let
  ((__function__ 'calist-fix))
  (declare (ignorable __function__))
  (mbe
    :logic
    (if (atom x)
        nil
        (let ((rest (calist-fix (cdr x))))
             (if (and (consp (car x)))
                 (let ((fty::first-key (caar x))
                       (fty::first-val (bfix (cdar x))))
                      (if (hons-assoc-equal fty::first-key rest)
                          rest
                          (cons (cons fty::first-key fty::first-val)
                                rest)))
                 rest)))
    :exec x)))

(defthm calist-equiv-is-an-equivalence
