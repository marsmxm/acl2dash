Fixtype of structure declarations [C:6.7.2.1].

This is a product type introduced by fty::defprod.

Fields
type — tyspecseq
declor — declor

These are used inside structure specifiers: they do not declare structure types, but rather their members.

For now we only capture structure declarations that consist of a type specifier sequence from tyspecseq, and a structure declarator that is a declarator from declor. We do not capture static assertions. We do not capture bit field sizes.
