Modifying constructor for iconst-base-hex structures.

Syntax
(change-iconst-base-hex x 
                        )

This is an often useful alternative to make-iconst-base-hex.

We construct a new iconst-base-hex structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
