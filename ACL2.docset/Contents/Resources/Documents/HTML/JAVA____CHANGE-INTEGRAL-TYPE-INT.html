Modifying constructor for integral-type-int structures.

Syntax
(change-integral-type-int x 
                          )

This is an often useful alternative to make-integral-type-int.

We construct a new integral-type-int structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

This is an ordinary change- macro introduced by fty::defprod.
