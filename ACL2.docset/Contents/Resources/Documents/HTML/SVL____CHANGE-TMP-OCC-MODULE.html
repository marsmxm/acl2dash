Modifying constructor for tmp-occ-module structures.

Syntax
(change-tmp-occ-module x 
                       [:inputs <inputs>] 
                       [:outputs <outputs>] 
                       [:name <name>])

This is an often useful alternative to make-tmp-occ-module.

We construct a new tmp-occ-module structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-tmp-occ-module

(defmacro
     change-tmp-occ-module (x &rest args)
     (std::change-aggregate 'tmp-occ-module
                            x args

                              (:outputs . tmp-occ-module->outputs)
                              (:name . tmp-occ-module->name))
                            'change-tmp-occ-module
                            'nil))
