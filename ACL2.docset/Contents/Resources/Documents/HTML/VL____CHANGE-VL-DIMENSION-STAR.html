Modifying constructor for vl-dimension-star structures.

Syntax
(change-vl-dimension-star x 
                          )

This is an often useful alternative to make-vl-dimension-star.

We construct a new vl-dimension-star structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-dimension-star

(defmacro change-vl-dimension-star (x &rest args)
          (std::change-aggregate 'vl-dimension-star
