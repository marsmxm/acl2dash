Set the wildcard for the numeric index of numbered names.

This macro generates an event to override the default, or the previously set value.

Macro: set-numbered-name-index-wildcard

(defmacro set-numbered-name-index-wildcard
          (wildcard)
          (cons 'table
                (cons 'numbered-name-index-wildcard
                      (cons ''wildcard
                            (cons wildcard 'nil)))))
