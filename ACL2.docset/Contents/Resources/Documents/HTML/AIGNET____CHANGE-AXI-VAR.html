Modifying constructor for axi-var structures.

Syntax
(change-axi-var x 
                [:name <name>])

This is an often useful alternative to make-axi-var.

We construct a new axi-var structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-axi-var

(defmacro change-axi-var (x &rest args)
          (std::change-aggregate 'axi-var
                                 x args '((:name . axi-var->name))
                                 'change-axi-var
                                 'nil))
