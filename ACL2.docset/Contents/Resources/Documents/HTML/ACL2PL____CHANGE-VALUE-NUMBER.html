Modifying constructor for value-number structures.

Syntax
(change-value-number x 
                     [:get <get>])

This is an often useful alternative to make-value-number.

We construct a new value-number structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-value-number

(defmacro
     change-value-number (x &rest args)
     (std::change-aggregate 'value-number
                            x args '((:get . value-number->get))
                            'change-value-number
                            'nil))
