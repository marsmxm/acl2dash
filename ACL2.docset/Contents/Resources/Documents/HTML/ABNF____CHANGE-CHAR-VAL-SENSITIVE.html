Modifying constructor for char-val-sensitive structures.

Syntax
(change-char-val-sensitive x 
                           [:get <get>])

This is an often useful alternative to make-char-val-sensitive.

We construct a new char-val-sensitive structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-char-val-sensitive

(defmacro
     change-char-val-sensitive (x &rest args)
     (std::change-aggregate 'char-val-sensitive
                            x
                            args '((:get . char-val-sensitive->get))
                            'change-char-val-sensitive
                            'nil))
