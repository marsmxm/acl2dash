Modifying constructor for vl-select-field structures.

Syntax
(change-vl-select-field x 
                        [:name <name>])

This is an often useful alternative to make-vl-select-field.

We construct a new vl-select-field structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-select-field

(defmacro
     change-vl-select-field (x &rest args)
     (std::change-aggregate 'vl-select-field
                            x
                            args '((:name . vl-select-field->name))
                            'change-vl-select-field
                            'remake-vl-select-field))
