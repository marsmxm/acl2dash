Name of the Java method to run one of the specified tests.

Signature
(atj-gen-test-method-name test-name) → method-name
Arguments
test-name — Guard (stringp test-name).
Returns
method-name — Type (stringp method-name).

This is generated only if the :tests input is not nil.

We generate a private static method for each test specified by the :tests input. This function generates the name of this method, which has the form test_..., where ... is the name of the test. Since ATJ checks that the tests have unique names, this scheme ensures that the method names are all distinct.

The argument of this function is the name of the test.
