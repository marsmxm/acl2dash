Modifying constructor for primitive-value-byte structures.

Syntax
(change-primitive-value-byte x 
                             [:get <get>])

This is an often useful alternative to make-primitive-value-byte.

We construct a new primitive-value-byte structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-primitive-value-byte

(defmacro
     change-primitive-value-byte
     (x &rest args)
     (std::change-aggregate 'primitive-value-byte
                            x args
                            '((:get . primitive-value-byte->get))
                            'change-primitive-value-byte
                            'nil))
