Basic constructor macro for unop-minus structures.

Syntax
(make-unop-minus )

This is the usual way to construct unop-minus structures. It simply conses together a structure with the specified fields.

This macro generates a new unop-minus structure from scratch. See also change-unop-minus, which can "change" an existing structure, instead.
