Modifying constructor for junop-uplus structures.

Syntax
(change-junop-uplus x 
                    )

This is an often useful alternative to make-junop-uplus.

We construct a new junop-uplus structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
