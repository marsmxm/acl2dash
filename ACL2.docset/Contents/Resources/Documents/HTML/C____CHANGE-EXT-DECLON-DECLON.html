Modifying constructor for ext-declon-declon structures.

Syntax
(change-ext-declon-declon x 
                          [:get <get>])

This is an often useful alternative to make-ext-declon-declon.

We construct a new ext-declon-declon structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-ext-declon-declon

(defmacro
     change-ext-declon-declon (x &rest args)
     (std::change-aggregate 'ext-declon-declon
                            x
                            args '((:get . ext-declon-declon->get))
                            'change-ext-declon-declon
                            'nil))
