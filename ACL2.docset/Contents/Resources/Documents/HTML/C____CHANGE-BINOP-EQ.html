Modifying constructor for binop-eq structures.

Syntax
(change-binop-eq x 
                 )

This is an often useful alternative to make-binop-eq.

We construct a new binop-eq structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-binop-eq

(defmacro change-binop-eq (x &rest args)
          (std::change-aggregate 'binop-eq
