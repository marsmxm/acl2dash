Modifying constructor for vl-hidindex structures.

Syntax
(change-vl-hidindex x 
                    [:name <name>] 
                    [:indices <indices>])

This is an often useful alternative to make-vl-hidindex.

We construct a new vl-hidindex structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-hidindex

(defmacro
     change-vl-hidindex (x &rest args)
     (std::change-aggregate 'vl-hidindex
                            x args
                            '((:name . vl-hidindex->name)
                              (:indices . vl-hidindex->indices))
                            'change-vl-hidindex
                            'remake-vl-hidindex))
