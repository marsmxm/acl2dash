Look up a definition in a system of constraints.

Signature
(lookup-definition name sys)  def?
Arguments
name Guard (symbolp name).
sys Guard (systemp sys).
Returns
def? Type (definition-optionp def?).

If the system has a definition for the given name, return that definition. Otherwise return nil.

We return the first definition found for that name. In a well-formed system of constraints, there is at most a definition for each name, and thus returning the first one found is also the only one.

Definitions and Theorems

Function: lookup-definition

(defun lookup-definition (name sys)
       (declare (xargs :guard (and (symbolp name) (systemp sys))))
       (let ((__function__ 'lookup-definition))
            (declare (ignorable __function__))
            (b* (((when (endp sys)) nil)
                 (def (car sys))
                 ((when (eq (definition->name def)
                            (symbol-fix name)))
                  (definition-fix def)))
                (lookup-definition name (cdr sys)))))

Theorem: definition-optionp-of-lookup-definition

(defthm definition-optionp-of-lookup-definition
        (b* ((def? (lookup-definition name sys)))
            (definition-optionp def?))
        :rule-classes :rewrite)

Theorem: lookup-definition-of-symbol-fix-name

(defthm lookup-definition-of-symbol-fix-name
        (equal (lookup-definition (symbol-fix name)

               (lookup-definition name sys)))
