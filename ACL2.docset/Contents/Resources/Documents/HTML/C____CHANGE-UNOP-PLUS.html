Modifying constructor for unop-plus structures.

Syntax
(change-unop-plus x 
                  )

This is an often useful alternative to make-unop-plus.

We construct a new unop-plus structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-unop-plus

(defmacro change-unop-plus (x &rest args)
          (std::change-aggregate 'unop-plus
                                 x args 'nil
