Option type; simpcode or nil.

This is an option type introduced by fty::defoption. Note that defoption is just a wrapper for fty::defflexsum, so there are :none and :some member types, a case macro, and so forth.

Member Types
:none → maybe-simpcode-none
Represents that no maybe-simpcode is available, i.e., Nothing or None.
:some → maybe-simpcode-some
An available maybe-simpcode, i.e., Just val or Some val.
