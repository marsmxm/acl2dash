Modifying constructor for svl-occ-module structures.

Syntax
(change-svl-occ-module x 
                       [:inputs <inputs>] 
                       [:outputs <outputs>] 
                       [:name <name>])

This is an often useful alternative to make-svl-occ-module.

We construct a new svl-occ-module structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-svl-occ-module

(defmacro
     change-svl-occ-module (x &rest args)
     (std::change-aggregate 'svl-occ-module
                            x args
                            '((:inputs . svl-occ-module->inputs)
                              (:outputs . svl-occ-module->outputs)
                              (:name . svl-occ-module->name))
                            'change-svl-occ-module
                            'nil))
