Modifying constructor for swcase structures.

Syntax
(change-swcase x 
               [:value <value>] 
               [:body <body>])

This is an often useful alternative to make-swcase.

We construct a new swcase structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-swcase

(defmacro change-swcase (x &rest args)
          (std::change-aggregate 'swcase
                                 x args
                                 '((:value . swcase->value)
                                   (:body . swcase->body))
                                 'change-swcase
                                 'nil))
