Event to generate the part of the model of arrays of an integer type that involves indices of an integer type.

Signature
(atc-def-integer-arrays-indices etype itype) → event
Arguments
etype — Guard (typep etype).
itype — Guard (typep itype).
Returns
event — Type (pseudo-event-formp event).

Here etype is the type of the array elements, while itype is the type of the array indices.
