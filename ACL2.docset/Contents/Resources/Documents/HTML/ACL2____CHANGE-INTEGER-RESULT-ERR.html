Modifying constructor for integer-result-err structures.

Syntax
(change-integer-result-err x 
                           [:get <get>])

This is an often useful alternative to make-integer-result-err.

We construct a new integer-result-err structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-integer-result-err

(defmacro
     change-integer-result-err (x &rest args)
     (std::change-aggregate 'integer-result-err
                            x
                            args '((:get . integer-result-err->get))
                            'change-integer-result-err
                            'nil))
