Modifying constructor for junop-predec structures.

Syntax
(change-junop-predec x 
                     )

This is an often useful alternative to make-junop-predec.

We construct a new junop-predec structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
