Convert an unsigned 8-bit byte to two hexadecimal digit characters.

Signature
(ubyte8=>hexchars byte) → (mv hi-char lo-char)
Arguments
byte — Guard (unsigned-byte-p 8 byte).
Returns
hi-char — Type (str::hex-digit-char-p hi-char).
lo-char — Type (str::hex-digit-char-p lo-char).

The most significant digit comes first, followed by the least significant one. If the byte is below 16, the first digit is zero. The hexadecimal digits above 9 are uppercase letters.

Function: ubyte8=>hexchars

(defun ubyte8=>hexchars (byte)
       (declare (xargs :guard (unsigned-byte-p 8 byte)))
       (let ((__function__ 'ubyte8=>hexchars))
            (declare (ignorable __function__))
            (b* ((byte (mbe :logic (unsigned-byte-fix 8 byte)
                            :exec byte)))
                (mv (str::hex-digit-to-char (floor byte 16))
                    (str::hex-digit-to-char (mod byte 16))))))
