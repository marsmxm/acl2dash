Generate a type specifier sequence for a type.

Signature
(atc-gen-tyspecseq type) → tyspecseq
Arguments
type — Guard (typep type).
Returns
tyspecseq — Type (tyspecseqp tyspecseq).

This is always called on a non-pointer type currently.

Definitions and Theorems

Function: atc-gen-tyspecseq

(defun
   atc-gen-tyspecseq (type)
   (declare (xargs :guard (typep type)))
   (declare (xargs :guard (not (type-case type :pointer))))
   (let ((__function__ 'atc-gen-tyspecseq))
        (declare (ignorable __function__))
        (type-case type
                   :void (tyspecseq-void)
                   :char (tyspecseq-char)
                   :schar (tyspecseq-schar)
                   :sshort (tyspecseq-sshort)
                   :sint (tyspecseq-sint)
                   :slong (tyspecseq-slong)
                   :sllong (tyspecseq-sllong)
                   :uchar (tyspecseq-uchar)
                   :ushort (tyspecseq-ushort)
                   :uint (tyspecseq-uint)
                   :ulong (tyspecseq-ulong)
                   :ullong (tyspecseq-ullong)
                   :pointer (prog2$ (impossible) (irr-tyspecseq)))))
