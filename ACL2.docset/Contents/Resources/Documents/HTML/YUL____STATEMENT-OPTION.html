Fixtype of optional statements.

This is an option type introduced by fty::defoption. Note that defoption is just a wrapper for fty::defflexsum, so there are :none and :some member types, a case macro, and so forth.

Member Types
:none → statement-option-none
Represents that no statement-option is available, i.e., Nothing or None.
:some → statement-option-some
An available statement-option, i.e., Just val or Some val.
