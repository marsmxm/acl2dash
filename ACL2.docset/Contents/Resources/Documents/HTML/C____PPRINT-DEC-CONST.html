Pretty-print a decimal constant.

Signature
(pprint-dec-const n) → part
Arguments
n — Guard (natp n).
Returns
part — Type (msgp part).

According to the grammar [C:6.4.4.1], a `decimal constant' is a sequence of decimal digits that does not start with 0; with the optional suffixes, this becomes an integer constant. As noted in iconst, we allow 0 here, which is turned into the single digit 0.
