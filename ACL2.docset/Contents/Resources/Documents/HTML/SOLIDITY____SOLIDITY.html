An ACL2 library for Solidity.

Currently this library contains a formalization of some aspects of the Solidity language for Ethereum smart contracts.

In the documentation of this library, the Solidity language documentation is referenced and linked as `[SD]' as a whole. Sections of it are referenced and linked by appending titles separated by colon, e.g. `[SD: Types]' for the `Types' section; note that sections are not numbered in [SD].
