Modifying constructor for vl-typeparam structures.

Syntax
(change-vl-typeparam x 
                     [:default <default>])

This is an often useful alternative to make-vl-typeparam.

We construct a new vl-typeparam structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-typeparam

(defmacro
     change-vl-typeparam (x &rest args)
     (std::change-aggregate 'vl-typeparam
                            x args
                            '((:default . vl-typeparam->default))
                            'change-vl-typeparam
                            'remake-vl-typeparam))
