Modifying constructor for maybe-rational-none structures.

Syntax
(change-maybe-rational-none x 
                            )

This is an often useful alternative to make-maybe-rational-none.

We construct a new maybe-rational-none structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
