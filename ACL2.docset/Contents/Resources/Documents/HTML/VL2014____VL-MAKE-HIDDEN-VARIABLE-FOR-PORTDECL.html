Create the hidden vl-vardecls corresponding to the vl-portdecls of a function or task.

Signature
(vl-make-hidden-variable-for-portdecl x) → var
Arguments
x — Guard (vl-portdecl-p x).
Returns
var — Type (vl-vardecl-p var).

See the description of the decls in vl-fundecl and vl-taskdecl: for each port of the function/task, we create a corresponding variable declaration, marked with a special attribute. This variable declaration plays well with scopestack and similar.

Definitions and Theorems

Function: vl-make-hidden-variable-for-portdecl

(defun
 vl-make-hidden-variable-for-portdecl (x)
 (declare (xargs :guard (vl-portdecl-p x)))
 (let
  ((__function__ 'vl-make-hidden-variable-for-portdecl))
  (declare (ignorable __function__))
  (b*
   (((vl-portdecl x)))
   (make-vl-vardecl
        :name x.name
        :type x.type
        :atts (cons (list (hons-copy "VL_HIDDEN_DECL_FOR_TASKPORT"))
                    x.atts)
        :loc x.loc))))
