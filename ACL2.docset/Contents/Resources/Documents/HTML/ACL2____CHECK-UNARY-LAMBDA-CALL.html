Check if a term is a (translated) call of a unary lambda expression.

Signature
(check-unary-lambda-call term) → (mv yes/no formal body arg)
Arguments
term — Guard (pseudo-termp term).
Returns
yes/no — Type (booleanp yes/no).
formal — Type (symbolp formal), given the guard.
body — Type (pseudo-termp body), given the guard.
arg — Type (pseudo-termp arg), given the guard.

This is a specialization of check-nary-lambda-call that returns a single formal and a single argument instead of two singleton lists.

Definitions and Theorems

Function: check-unary-lambda-call

(defun check-unary-lambda-call (term)
       (declare (xargs :guard (pseudo-termp term)))
       (let ((__function__ 'check-unary-lambda-call))
            (declare (ignorable __function__))
            (b* (((mv yes/no formals body args)
                  (check-nary-lambda-call term 1))
                 ((unless yes/no) (mv nil nil nil nil)))
                (mv t (car formals) body (car args)))))
