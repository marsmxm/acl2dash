Modifying constructor for hex-pair structures.

Syntax
(change-hex-pair x 
                 [:1st <1st>] 
                 [:2nd <2nd>])

This is an often useful alternative to make-hex-pair.

We construct a new hex-pair structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-hex-pair

(defmacro change-hex-pair (x &rest args)
          (std::change-aggregate 'hex-pair
                                 x args
                                 '((:1st . hex-pair->1st)
                                   (:2nd . hex-pair->2nd))
                                 'change-hex-pair
                                 'nil))
