Raw constructor for honsed vl-erange-p structures.

Syntax:

(honsed-vl-erange type left right)

This is identical to vl-erange, except that we hons the structure we are creating.

Definition

This is an ordinary honsing constructor introduced by defaggregate.

Function: honsed-vl-erange

(defun honsed-vl-erange (type left right)
       (declare (xargs :guard (and (vl-erangetype-p type)
                                   (vl-expr-p left)
                                   (vl-expr-p right))))
       (mbe :logic (vl-erange type left right)
            :exec (hons :vl-erange (hons type (hons left right)))))
