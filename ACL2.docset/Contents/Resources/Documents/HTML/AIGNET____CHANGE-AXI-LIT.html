Modifying constructor for axi-lit structures.

Syntax
(change-axi-lit x 
                [:negp <negp>] 
                [:abs <abs>])

This is an often useful alternative to make-axi-lit.

We construct a new axi-lit structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-axi-lit

(defmacro change-axi-lit (x &rest args)
          (std::change-aggregate 'axi-lit
                                 x args
                                 '((:negp . axi-lit->negp)
                                   (:abs . axi-lit->abs))
                                 'change-axi-lit
                                 'nil))
