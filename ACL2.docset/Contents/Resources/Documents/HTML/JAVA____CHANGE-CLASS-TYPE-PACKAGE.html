Modifying constructor for class-type-package structures.

Syntax
(change-class-type-package x 
                           [:package <package>] 
                           [:name <name>] 
                           [:arguments <arguments>])

This is an often useful alternative to make-class-type-package.

We construct a new class-type-package structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-class-type-package

(defmacro change-class-type-package (x &rest args)
          (std::change-aggregate
               'class-type-package
               x args
               '((:package . class-type-package->package)
                 (:name . class-type-package->name)
                 (:arguments . class-type-package->arguments))
               'change-class-type-package
               'nil))
