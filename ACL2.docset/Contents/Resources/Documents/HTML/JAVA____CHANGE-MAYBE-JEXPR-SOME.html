Modifying constructor for maybe-jexpr-some structures.

Syntax
(change-maybe-jexpr-some x 
                         [:val <val>])

This is an often useful alternative to make-maybe-jexpr-some.

We construct a new maybe-jexpr-some structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
