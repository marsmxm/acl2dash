Modifying constructor for tree-option-some structures.

Syntax
(change-tree-option-some x 
                         [:val <val>])

This is an often useful alternative to make-tree-option-some.

We construct a new tree-option-some structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
