Searching the documentation

The :doc command will display, at the terminal, documentation topics defined in ACL2 or in books that have already been included. But how can you find documentation for books that are not included in the current ACL2 session?

The xdoc ACL2+Books Manual includes documentation for both the ACL2 system and the community-books. For more information on this manual and how to view it, see documentation.
