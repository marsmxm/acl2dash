Modifying constructor for exec-outcome-terminating structures.

Syntax
(change-exec-outcome-terminating x 
                                 [:result <result>])

This is an often useful alternative to make-exec-outcome-terminating.

We construct a new exec-outcome-terminating structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-exec-outcome-terminating

(defmacro change-exec-outcome-terminating
          (x &rest args)
          (std::change-aggregate
               'exec-outcome-terminating
               x args
               '((:result . exec-outcome-terminating->result))
               'change-exec-outcome-terminating
               'nil))
