(identifier-set-fix x) is a usual ACL2::fty set fixing function.

Signature
(identifier-set-fix x) → *
Arguments
x — Guard (identifier-setp x).

In the logic, we apply identifier-fix to each member of the x. In the execution, none of that is actually necessary and this is just an inlined identity function.

(defun identifier-set-fix (x)
       (declare (xargs :guard (identifier-setp x)))
       (mbe :logic (if (identifier-setp x) x nil)
            :exec x))
