Modifying constructor for tag-declon-struct structures.

Syntax
(change-tag-declon-struct x 
                          [:tag <tag>] 
                          [:members <members>])

This is an often useful alternative to make-tag-declon-struct.

We construct a new tag-declon-struct structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-tag-declon-struct

(defmacro
   change-tag-declon-struct (x &rest args)
   (std::change-aggregate 'tag-declon-struct
                          x args
                          '((:tag . tag-declon-struct->tag)
                            (:members . tag-declon-struct->members))
                          'change-tag-declon-struct
                          'nil))
