Variable renaming relation over function scopes.

Signature
(funscope-renamevarp old new) → yes/no
Arguments
old — Guard (funscopep old).
new — Guard (funscopep new).
Returns
yes/no — Type (booleanp yes/no).

The two scopes must have the same function names, and the function information for the same function name must be related.

Definitions and Theorems

Function: funscope-renamevarp

(defun
 funscope-renamevarp (old new)
 (declare (xargs :guard (and (funscopep old) (funscopep new))))
 (let
  ((__function__ 'funscope-renamevarp))
  (declare (ignorable __function__))
  (b*
   ((old (funscope-fix old))
    (new (funscope-fix new)))
   (and
    (or (and (omap::empty old)
             (omap::empty new))
        (and (not (omap::empty old))
             (not (omap::empty new))
             (b* (((mv old-fun old-info) (omap::head old))
                  ((mv new-fun new-info)
                   (omap::head new)))
                 (and (equal old-fun new-fun)
                      (funinfo-renamevarp old-info new-info)
                      (funscope-renamevarp (omap::tail old)
                                           (omap::tail new))))))))))
