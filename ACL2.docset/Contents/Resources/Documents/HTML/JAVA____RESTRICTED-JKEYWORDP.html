Recognize restricted Java keywords.

Signature
(restricted-jkeywordp x) → yes/no
Returns
yes/no — Type (booleanp yes/no).

A restricted Java keyword is a list of Java Unicode characters that consist of the (ASCII) codes of some element in *restricted-jkeywords*.

Definitions and Theorems

Function: restricted-jkeywordp

(defun restricted-jkeywordp (x)
       (declare (xargs :guard t))
       (let ((__function__ 'restricted-jkeywordp))
            (declare (ignorable __function__))
            (and (ascii-listp x)
                 (member-equal (ascii=>string x)
                               *restricted-jkeywords*)
                 t)))

Theorem: booleanp-of-restricted-jkeywordp

(defthm booleanp-of-restricted-jkeywordp
        (b* ((yes/no (restricted-jkeywordp x)))
            (booleanp yes/no))
        :rule-classes :rewrite)
