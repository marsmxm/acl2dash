Modifying constructor for fun-declor-base structures.

Syntax
(change-fun-declor-base x 
                        [:name <name>] 
                        [:params <params>])

This is an often useful alternative to make-fun-declor-base.

We construct a new fun-declor-base structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-fun-declor-base

(defmacro
     change-fun-declor-base (x &rest args)
     (std::change-aggregate 'fun-declor-base
                            x args
                            '((:name . fun-declor-base->name)
                              (:params . fun-declor-base->params))
                            'change-fun-declor-base
                            'nil))
