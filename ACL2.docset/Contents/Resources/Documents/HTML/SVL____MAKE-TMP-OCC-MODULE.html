Basic constructor macro for tmp-occ-module structures.

Syntax
(make-tmp-occ-module [:inputs <inputs>]

This is the usual way to construct tmp-occ-module structures. It simply conses together a structure with the specified fields.

This macro generates a new tmp-occ-module structure from scratch. See also change-tmp-occ-module, which can "change" an existing structure, instead.
