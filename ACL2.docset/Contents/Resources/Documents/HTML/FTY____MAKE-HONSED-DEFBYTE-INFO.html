Constructor macro for honsed defbyte-infop structures.

Syntax:

(make-honsed-defbyte-info [:size <size>] 
                          [:signed <signed>])

This is identical to make-defbyte-info, except that we hons the structure we are creating.

Definition

This is an ordinary honsing make- macro introduced by std::defaggregate.

Macro: make-honsed-defbyte-info

(defmacro make-honsed-defbyte-info (&rest args)
          (std::make-aggregate 'defbyte-info
                               args '((:size) (:signed))
                               'make-honsed-defbyte-info
                               t))
