Check if all the functions in a lambda expression are guard-verified.

Signature
(lambda-guard-verified-fnsp lambd wrld) → yes/no
Arguments
lambd — Guard (pseudo-lambdap lambd).
wrld — Guard (plist-worldp wrld).
Returns
yes/no — Type (booleanp yes/no).

The name of this function is consistent with the name of guard-verified-fnsp.

Note that if any function inside the :logic component of an mbe or called via ec-call is not guard-verified, we return nil, even when lambd could otherwise be fully guard-verified. See lambda-guard-verified-exec-fnsp for a similar utility that ignores the guard verification status of functions in the :logic components of mbes or called via ec-call.
