Representation of a single Verilog bit (0, 1, X, or Z).

Verilog has four register-transfer level values, 0, 1, X, and Z. We represent these values using the keyword symbols accepted by vl-bit-p:

  • :vl-0val means 0,
  • :vl-1val means 1,
  • :vl-xval means X, and
  • :vl-zval means Z.
