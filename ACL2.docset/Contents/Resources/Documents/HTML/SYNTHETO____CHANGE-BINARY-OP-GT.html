Modifying constructor for binary-op-gt structures.

Syntax
(change-binary-op-gt x 
                     )

This is an often useful alternative to make-binary-op-gt.

We construct a new binary-op-gt structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
