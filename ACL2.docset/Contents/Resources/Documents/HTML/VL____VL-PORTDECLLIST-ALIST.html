Extend an alist by binding the names of vl-portdecls to their definitions.

Signature
(vl-portdecllist-alist x acc) → alist
Arguments
x — Guard (vl-portdecllist-p x).
Returns
alist — Type (equal (vl-portdecl-alist-p alist) (vl-portdecl-alist-p acc)).

This can be used as an alternative to vl-find-portdecl when you need to perform a lot of lookups.

Definitions and Theorems

Function: vl-portdecllist-alist

(defun vl-portdecllist-alist (x acc)
       (declare (xargs :guard (vl-portdecllist-p x)))
       (let ((__function__ 'vl-portdecllist-alist))
            (declare (ignorable __function__))
            (if (atom x)
                acc
                (cons (cons (vl-portdecl->name (car x))
                            (vl-portdecl-fix (car x)))
                      (vl-portdecllist-alist (cdr x) acc)))))

Theorem: lookup-in-vl-portdecllist-alist-fast

(defthm
   lookup-in-vl-portdecllist-alist-fast
   (implies
        (stringp name)
        (equal (hons-assoc-equal name (vl-portdecllist-alist x nil))
               (let ((val (vl-find-portdecl name x)))
                    (and val (cons name val))))))
