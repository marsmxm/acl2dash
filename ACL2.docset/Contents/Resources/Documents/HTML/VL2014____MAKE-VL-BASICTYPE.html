Basic constructor macro for vl-basictype structures.

Syntax
(make-vl-basictype [:kind <kind>])

This is the usual way to construct vl-basictype structures. It simply conses together a structure with the specified fields.

This macro generates a new vl-basictype structure from scratch. See also change-vl-basictype, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by defprod.
