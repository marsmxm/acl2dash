Extend a function environment with a function definition.

Signature
(fun-env-extend fundef fenv) → result
Arguments
fundef — Guard (fundefp fundef).
fenv — Guard (fun-envp fenv).
Returns
result — Type (fun-env-resultp result).

If the environment already has a function with the same name, this operation returns an error.

Definitions and Theorems

Function: fun-env-extend

(defun
 fun-env-extend (fundef fenv)
 (declare (xargs :guard (and (fundefp fundef) (fun-envp fenv))))
 (let
    ((__function__ 'fun-env-extend))
    (declare (ignorable __function__))
    (b* ((fenv (fun-env-fix fenv))
         ((fundef fundef) fundef)
         ((when (fun-env-lookup fundef.name fenv))
          (error (list :duplicate-function-definition fundef.name)))
         (info (fun-info-from-fundef fundef)))
        (omap::update fundef.name info fenv))))
