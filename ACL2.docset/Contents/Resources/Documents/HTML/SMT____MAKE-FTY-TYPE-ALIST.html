Basic constructor macro for fty-type-alist structures.

Syntax
(make-fty-type-alist [:key-type <key-type>] 
                     [:val-type <val-type>])

This is the usual way to construct fty-type-alist structures. It simply conses together a structure with the specified fields.

This macro generates a new fty-type-alist structure from scratch. See also change-fty-type-alist, which can "change" an existing structure, instead.
