Modifying constructor for jstatem-return structures.

Syntax
(change-jstatem-return x 
                       [:expr? <expr?>])

This is an often useful alternative to make-jstatem-return.

We construct a new jstatem-return structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-jstatem-return

(defmacro
     change-jstatem-return (x &rest args)
     (std::change-aggregate 'jstatem-return
                            x
                            args '((:expr? . jstatem-return->expr?))
                            'change-jstatem-return
                            'nil))
