An AIG that captures exactly when a list of FAIGs always evaluate to purely Boolean values.

Signature
(faig-purebool-list-aig x) → aig
Arguments
x — An FAIG List.

This is useful mainly to implement faig-purebool-list-check.

Definitions and Theorems

Function: faig-purebool-list-aig

(defun faig-purebool-list-aig (x)
       (declare (xargs :guard t))
       (let ((__function__ 'faig-purebool-list-aig))
            (declare (ignorable __function__))
            (if (atom x)
                t
                (aig-and (faig-purebool-aig (car x))
                         (faig-purebool-list-aig (cdr x))))))

(defthm faig-purebool-list-p-as-aig-eval

(defthm faig-purebool-list-p-monotonicity
        (implies (not (aig-eval (faig-purebool-list-aig x)

                 (not (aig-eval (faig-purebool-list-aig x)
