(sd-problemlist-fix x) is a usual fty list fixing function.

Signature
(sd-problemlist-fix x)  fty::newx
Arguments
x Guard (sd-problemlist-p x).
Returns
fty::newx Type (sd-problemlist-p fty::newx).

In the logic, we apply sd-problem-fix to each member of the x. In the execution, none of that is actually necessary and this is just an inlined identity function.
