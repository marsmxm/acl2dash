Modifying constructor for atj-atype-character structures.

Syntax
(change-atj-atype-character x 
                            )

This is an often useful alternative to make-atj-atype-character.

We construct a new atj-atype-character structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
