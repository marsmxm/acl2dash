(vl-genarrayblocklist-fix x) is a usual ACL2::fty list fixing function.

Signature
(vl-genarrayblocklist-fix x) → fty::newx
Arguments
x — Guard (vl-genarrayblocklist-p x).
Returns
fty::newx — Type (vl-genarrayblocklist-p fty::newx).

In the logic, we apply vl-genarrayblock-fix to each member of the x. In the execution, none of that is actually necessary and this is just an inlined identity function.
