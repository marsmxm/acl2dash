Modifying constructor for type-char structures.

Syntax
(change-type-char x 
                  )

This is an often useful alternative to make-type-char.

We construct a new type-char structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-type-char

(defmacro change-type-char (x &rest args)
          (std::change-aggregate 'type-char
                                 x args 'nil

                                 'nil))
