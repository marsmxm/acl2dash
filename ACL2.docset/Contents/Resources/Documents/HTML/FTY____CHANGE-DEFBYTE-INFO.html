A copying macro that lets you create new defbyte-infop structures, based on existing structures.

Syntax:

(change-defbyte-info x 
                     [:size <size>] 
                     [:signed <signed>])

This is a sometimes useful alternative to make-defbyte-info. It constructs a new defbyte-infop structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by std::defaggregate.

Macro: change-defbyte-info

(defmacro change-defbyte-info (x &rest args)
          (std::change-aggregate 'defbyte-info
                                 x args
                                 '((:size . defbyte-info->size)
                                   (:signed . defbyte-info->signed))
                                 'change-defbyte-info
                                 'nil))
