Modifying constructor for vl-index structures.

Syntax
(change-vl-index x 
                 [:scope <scope>] 
                 [:indices <indices>] 
                 [:part <part>] 
                 [:atts <atts>])

This is an often useful alternative to make-vl-index.

We construct a new vl-index structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-index

(defmacro change-vl-index (x &rest args)
          (std::change-aggregate 'vl-index
                                 x args
                                 '((:scope . vl-index->scope)
                                   (:indices . vl-index->indices)
                                   (:part . vl-index->part)
                                   (:atts . vl-index->atts))
                                 'change-vl-index
                                 'remake-vl-index))
