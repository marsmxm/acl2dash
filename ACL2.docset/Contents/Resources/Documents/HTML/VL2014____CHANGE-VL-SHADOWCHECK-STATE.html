Modifying constructor for vl-shadowcheck-state structures.

Syntax
(change-vl-shadowcheck-state x 
                             [:lexscopes <lexscopes>] 
                             [:ss <ss>] 
                             [:design <design>])

This is an often useful alternative to make-vl-shadowcheck-state.

We construct a new vl-shadowcheck-state structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-shadowcheck-state

(defmacro change-vl-shadowcheck-state
          (x &rest args)
          (std::change-aggregate
               'vl-shadowcheck-state
               x args
               '((:lexscopes . vl-shadowcheck-state->lexscopes)
                 (:ss . vl-shadowcheck-state->ss)
                 (:design . vl-shadowcheck-state->design))
               'change-vl-shadowcheck-state
               'remake-vl-shadowcheck-state))
