Basic constructor macro for jvaluex-reference structures.

Syntax
(make-jvaluex-reference [:get <get>])

This is the usual way to construct jvaluex-reference structures. It simply conses together a structure with the specified fields.

This macro generates a new jvaluex-reference structure from scratch. See also change-jvaluex-reference, which can "change" an existing structure, instead.

This is an ordinary make- macro introduced by fty::defprod.
