Add a variable to (the innermost block of) a variable table.

Signature
(var-table-add-var var type vartab) → new-vartab
Arguments
var — Guard (identp var).
type — Guard (typep type).
vartab — Guard (var-tablep vartab).
Returns
new-vartab — Type (var-table-resultp new-vartab).

If the block already has a variable with that name, it is an error. Otherwise, we add the variable and return the variable table.

Definitions and Theorems

Function: var-table-add-var

(defun var-table-add-var (var type vartab)
       (declare (xargs :guard (and (identp var)
                                   (typep type)
                                   (var-tablep vartab))))
       (let ((__function__ 'var-table-add-var))
            (declare (ignorable __function__))
            (b* ((var (ident-fix var))
                 (type (type-fix type))
                 (vartab (var-table-fix vartab))
                 (varscope (car vartab))
                 ((when (omap::in var varscope))
                  (error (list :duplicate-var var)))
                 (new-varscope (omap::update var type varscope)))
                (cons new-varscope (cdr vartab)))))
