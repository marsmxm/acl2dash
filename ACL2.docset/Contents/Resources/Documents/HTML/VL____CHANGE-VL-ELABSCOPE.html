Modifying constructor for vl-elabscope structures.

Syntax
(change-vl-elabscope x 
                     [:subscopes <subscopes>] 
                     [:members <members>])

This is an often useful alternative to make-vl-elabscope.

We construct a new vl-elabscope structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-elabscope

(defmacro
     change-vl-elabscope (x &rest args)
     (std::change-aggregate 'vl-elabscope
                            x args
                            '((:subscopes . vl-elabscope->subscopes)
                              (:members . vl-elabscope->members))
                            'change-vl-elabscope
                            'nil))
