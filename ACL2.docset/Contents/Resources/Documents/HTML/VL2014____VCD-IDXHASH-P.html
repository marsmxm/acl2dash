(vcd-idxhash-p x len) recognizes association lists where every key satisfies vcd-index-p and each value satisfies not.

This is an ordinary defalist.

Function: vcd-idxhash-p

(defun vcd-idxhash-p (x len)
       (declare (xargs :guard (natp len)))
       (if (consp x)
           (and (consp (car x))
                (vcd-index-p (caar x) len)
                (not (cdar x))
                (vcd-idxhash-p (cdr x) len))
           t))
