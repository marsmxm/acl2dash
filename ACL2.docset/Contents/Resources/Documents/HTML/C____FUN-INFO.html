Fixtype of information about a C function in an environment.

This is a product type introduced by fty::defprod.

Fields
params — param-declon-list
result — tyspecseq
body — block-item-list

This consists of the components of a function definition (see fundef) without the name. This is because names are used as keys in a function environment. The other components form the value associated to the key.
