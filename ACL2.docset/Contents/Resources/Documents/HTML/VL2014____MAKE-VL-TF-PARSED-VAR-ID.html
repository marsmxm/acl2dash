Constructor macro for vl-tf-parsed-var-id-p structures.

Syntax:

(make-vl-tf-parsed-var-id [:name <name>] 
                          [:udims <udims>])

This is our preferred way to construct vl-tf-parsed-var-id-p structures. It simply conses together a structure with the specified fields.

This macro generates a new vl-tf-parsed-var-id-p structure from scratch. See also change-vl-tf-parsed-var-id, which can "change" an existing structure, instead.

The vl-tf-parsed-var-id-p structures we create here are just constructed with ordinary cons. If you want to create honsed structures, see make-honsed-vl-tf-parsed-var-id instead.

Definition

This is an ordinary make- macro introduced by defaggregate.
