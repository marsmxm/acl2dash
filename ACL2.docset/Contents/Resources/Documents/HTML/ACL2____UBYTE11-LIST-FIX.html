(ubyte11-list-fix x) is a usual fty list fixing function.

Signature
(ubyte11-list-fix x) → fty::newx
Arguments
x — Guard (ubyte11-listp x).
Returns
fty::newx — Type (ubyte11-listp fty::newx).

In the logic, we apply ubyte11-fix to each member of the x. In the execution, none of that is actually necessary and this is just an inlined identity function.
