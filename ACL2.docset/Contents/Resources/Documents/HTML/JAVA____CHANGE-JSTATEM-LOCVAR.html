Modifying constructor for jstatem-locvar structures.

Syntax
(change-jstatem-locvar x 
                       [:get <get>])

This is an often useful alternative to make-jstatem-locvar.

We construct a new jstatem-locvar structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-jstatem-locvar

(defmacro
     change-jstatem-locvar (x &rest args)
     (std::change-aggregate 'jstatem-locvar
                            x args '((:get . jstatem-locvar->get))
                            'change-jstatem-locvar
                            'nil))
