Parse a string into a natural number.

Signature
(string-to-nat string)  (mv error? nat)
Arguments
string Guard (stringp string).
Returns
error? Type (booleanp error?).
nat Type (natp nat).

Certain wallet commands take natural numbers as inputs, which are supplied by the wallet's user as strings. These must be non-empty sequences of decimal digits, which are turned into their big-endian value. If the string cannot be converted to a natural number, the error result is t.
