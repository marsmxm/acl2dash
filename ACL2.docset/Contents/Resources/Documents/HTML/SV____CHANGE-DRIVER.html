Modifying constructor for driver structures.

Syntax
(change-driver x 
               [:value <value>] 
               [:strength <strength>])

This is an often useful alternative to make-driver.

We construct a new driver structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-driver

(defmacro change-driver (x &rest args)
          (std::change-aggregate 'driver
                                 x args
                                 '((:value . driver->value)
                                   (:strength . driver->strength))
                                 'change-driver
                                 'remake-driver))
