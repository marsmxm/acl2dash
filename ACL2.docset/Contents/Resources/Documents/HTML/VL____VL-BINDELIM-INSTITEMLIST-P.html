(vl-bindelim-institemlist-p x) recognizes lists where every element satisfies vl-bindelim-institem-p.

Signature
(vl-bindelim-institemlist-p x)  *

This is an ordinary deflist. It is "loose" in that it does not care whether x is nil-terminated.
