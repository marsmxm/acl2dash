Modifying constructor for jresult-type structures.

Syntax
(change-jresult-type x 
                     [:get <get>])

This is an often useful alternative to make-jresult-type.

We construct a new jresult-type structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-jresult-type

(defmacro
     change-jresult-type (x &rest args)
     (std::change-aggregate 'jresult-type
                            x args '((:get . jresult-type->get))
                            'change-jresult-type
                            'nil))
