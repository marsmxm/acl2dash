Modifying constructor for stmt-option-some structures.

Syntax
(change-stmt-option-some x 
                         [:val <val>])

This is an often useful alternative to make-stmt-option-some.

We construct a new stmt-option-some structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-stmt-option-some

(defmacro
     change-stmt-option-some (x &rest args)
     (std::change-aggregate 'stmt-option-some
