Basic constructor macro for vl-parsed-ports structures.

Syntax
(make-vl-parsed-ports [:ansi-p <ansi-p>] 
                      [:ports <ports>] 
                      [:portdecls <portdecls>] 
                      [:vardecls <vardecls>])

This is the usual way to construct vl-parsed-ports structures. It simply conses together a structure with the specified fields.

This macro generates a new vl-parsed-ports structure from scratch. See also change-vl-parsed-ports, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by defprod.
