Modifying constructor for vl-disablestmt structures.

Syntax
(change-vl-disablestmt x 
                       [:id <id>] 
                       [:atts <atts>])

This is an often useful alternative to make-vl-disablestmt.

We construct a new vl-disablestmt structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-disablestmt

(defmacro change-vl-disablestmt (x &rest args)
          (std::change-aggregate 'vl-disablestmt
                                 x args
                                 '((:id . vl-disablestmt->id)
                                   (:atts . vl-disablestmt->atts))
                                 'change-vl-disablestmt
                                 'remake-vl-disablestmt))
