Constructor macro for wcp-lit-actions-p structures.

Syntax:

(make-wcp-lit-actions [:witnesses <witnesses>] 
                      [:examples <examples>])

This is our preferred way to construct wcp-lit-actions-p structures. It simply conses together a structure with the specified fields.

This macro generates a new wcp-lit-actions-p structure from scratch. See also change-wcp-lit-actions, which can "change" an existing structure, instead.

The wcp-lit-actions-p structures we create here are just constructed with ordinary cons. If you want to create honsed structures, see make-honsed-wcp-lit-actions instead.

Definition

This is an ordinary make- macro introduced by std::defaggregate.

Macro: make-wcp-lit-actions

(defmacro make-wcp-lit-actions (&rest args)
          (std::make-aggregate 'wcp-lit-actions
                               args '((:witnesses) (:examples))
                               'make-wcp-lit-actions
                               nil))
