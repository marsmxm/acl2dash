Modifying constructor for literal-integer structures.

Syntax
(change-literal-integer x 
                        [:get <get>])

This is an often useful alternative to make-literal-integer.

We construct a new literal-integer structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-literal-integer

(defmacro
     change-literal-integer (x &rest args)
     (std::change-aggregate 'literal-integer
                            x args '((:get . literal-integer->get))
                            'change-literal-integer
                            'nil))
