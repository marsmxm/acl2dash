Modifying constructor for tfunction-lambda structures.

Syntax
(change-tfunction-lambda x 
                         [:parameters <parameters>] 
                         [:body <body>])

This is an often useful alternative to make-tfunction-lambda.

We construct a new tfunction-lambda structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-tfunction-lambda

(defmacro change-tfunction-lambda (x &rest args)
          (std::change-aggregate
               'tfunction-lambda
               x args
               '((:parameters . tfunction-lambda->parameters)
                 (:body . tfunction-lambda->body))
               'change-tfunction-lambda
               'nil))
