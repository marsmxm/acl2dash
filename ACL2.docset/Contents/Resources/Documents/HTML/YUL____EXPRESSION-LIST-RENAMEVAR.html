Check if two lists of expressions are related by variable renaming.

Signature
(expression-list-renamevar old new ren) → _
Arguments
old — Guard (expression-listp old).
new — Guard (expression-listp new).
ren — Guard (renamingp ren).
Returns
_ — Type (resulterr-optionp _).

The two lists must have the same length, and corresponding elements must be related.
