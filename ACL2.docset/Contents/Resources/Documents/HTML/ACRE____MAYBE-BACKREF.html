Option type; backref or nil.

This is an option type introduced by fty::defoption. Note that defoption is just a wrapper for fty::defflexsum, so there are :none and :some member types, a case macro, and so forth.

Member Types
:none → maybe-backref-none
Represents that no maybe-backref is available, i.e., Nothing or None.
:some → maybe-backref-some
An available maybe-backref, i.e., Just val or Some val.
