(symbol-value-list-fix x) is a usual ACL2::fty list fixing function.

Signature
(symbol-value-list-fix x) → fty::newx
Arguments
x — Guard (symbol-value-listp x).
Returns
fty::newx — Type (symbol-value-listp fty::newx).

In the logic, we apply symbol-value-fix to each member of the x. In the execution, none of that is actually necessary and this is just an inlined identity function.
