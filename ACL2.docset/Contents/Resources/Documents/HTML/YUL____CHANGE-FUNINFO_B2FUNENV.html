Modifying constructor for funinfo+funenv structures.

Syntax
(change-funinfo+funenv x 
                       [:info <info>] 
                       [:env <env>])

This is an often useful alternative to make-funinfo+funenv.

We construct a new funinfo+funenv structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-funinfo+funenv

(defmacro change-funinfo+funenv (x &rest args)
          (std::change-aggregate 'funinfo+funenv
                                 x args
                                 '((:info . funinfo+funenv->info)
                                   (:env . funinfo+funenv->env))
                                 'change-funinfo+funenv
                                 'nil))
