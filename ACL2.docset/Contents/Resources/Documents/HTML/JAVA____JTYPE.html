We capture all eight primitive types.

We use ACL2 strings to capture not only single type identifiers, but also dot-separated sequences with package names and type identifiers: the latter have dots in the ACL2 strings. This also accommodates types with type arguments, by using < and > in the ACL2 strings. Since the Java grammar defines InterfaceType as ClassType, we use :class for both class and interface types; this is consistent with similar terminology in Java (e.g. a `class file' may define an interface.)

We capture array types via a natural recursive structure, by wrapping the component type.
