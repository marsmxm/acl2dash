Modifying constructor for identifier-option-none structures.

Syntax
(change-identifier-option-none x 
                               )

This is an often useful alternative to make-identifier-option-none.

We construct a new identifier-option-none structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
