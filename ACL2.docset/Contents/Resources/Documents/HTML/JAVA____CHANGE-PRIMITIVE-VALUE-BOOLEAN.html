Modifying constructor for primitive-value-boolean structures.

Syntax
(change-primitive-value-boolean x 
                                [:get <get>])

This is an often useful alternative to make-primitive-value-boolean.

We construct a new primitive-value-boolean structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-primitive-value-boolean

(defmacro
     change-primitive-value-boolean
     (x &rest args)
     (std::change-aggregate 'primitive-value-boolean
                            x args
                            '((:get . primitive-value-boolean->get))
                            'change-primitive-value-boolean
                            'nil))
