Semantic function saying if an assignment satisfies a list of constraints, given a system of constraints and a prime field.

This is similar to constraint-satp, but for lists of constraints.

Definitions and Theorems

Theorem: constraint-list-satp-suff

(defthm
 constraint-list-satp-suff
 (implies
    (and (proof-tree-listp ptrees)
         (equal (exec-proof-tree-list ptrees sys p)
                (proof-list-outcome-assertions
                     (assertion-list-from (repeat (len constrs) asg)
                                          constrs))))
    (constraint-list-satp asg constrs sys p)))

Theorem: booleanp-of-constraint-list-satp

(defthm booleanp-of-constraint-list-satp
        (b* ((yes/no (constraint-list-satp asg constrs sys p)))
            (booleanp yes/no))
        :rule-classes :rewrite)
