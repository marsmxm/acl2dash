Construct a subsubsubsection of the user documentation of an event macro.

This provides more abstraction than using a specific heading.

Macro: evmac-subsubsubsection

(defmacro xdoc::evmac-subsubsubsection
          (title &rest content)
          (cons 'xdoc::&&
                (cons (cons 'xdoc::h6 (cons title 'nil))
                      content)))
