Irreflexive kernel (i.e. strict version) of atj-type-<=.

Signature
(atj-type-< sub sup) → yes/no
Arguments
sub — Guard (atj-typep sub).
sup — Guard (atj-typep sup).
Returns
yes/no — Type (booleanp yes/no).
Since atj-type-<= fixes its arguments, we use (in)equality modulo fixing here, so that this function fixes its arguments too.
