We say a vl-evatom-p is plain when it has no posedge or negedge, and the expression is a simple identifier. For instance, an event control list like @(a or b or c) contains only plain evatoms.

Signature
(vl-evatom-plain-p x) → plainp
Arguments
x — Guard (vl-evatom-p x).

Typically an always block that has this sort of event control contains either behavioral-style combinational logic or latches, whereas an event control with something like @(posedge clk) expresses flip-flops.
