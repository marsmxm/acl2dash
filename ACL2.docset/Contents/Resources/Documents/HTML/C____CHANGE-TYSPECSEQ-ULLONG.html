Modifying constructor for tyspecseq-ullong structures.

Syntax
(change-tyspecseq-ullong x 
                         )

This is an often useful alternative to make-tyspecseq-ullong.

We construct a new tyspecseq-ullong structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
