Modifying constructor for expression-binary structures.

Syntax
(change-expression-binary x 
                          [:operator <operator>] 
                          [:left-operand <left-operand>] 
                          [:right-operand <right-operand>])

This is an often useful alternative to make-expression-binary.

We construct a new expression-binary structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-expression-binary

(defmacro
     change-expression-binary (x &rest args)
     (std::change-aggregate
          'expression-binary
          x args
          '((:operator . expression-binary->operator)
            (:left-operand . expression-binary->left-operand)
            (:right-operand . expression-binary->right-operand))
          'change-expression-binary
          'nil))
