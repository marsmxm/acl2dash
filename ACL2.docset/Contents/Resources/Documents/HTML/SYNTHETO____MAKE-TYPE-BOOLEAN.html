Basic constructor macro for type-boolean structures.

Syntax
(make-type-boolean )

This is the usual way to construct type-boolean structures. It simply conses together a structure with the specified fields.

This macro generates a new type-boolean structure from scratch. See also change-type-boolean, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by fty::defprod.

(defmacro make-type-boolean (&rest args)
