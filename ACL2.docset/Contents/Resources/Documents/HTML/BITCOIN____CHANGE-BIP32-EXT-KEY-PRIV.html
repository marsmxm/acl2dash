Modifying constructor for bip32-ext-key-priv structures.

Syntax
(change-bip32-ext-key-priv x 
                           [:get <get>])

This is an often useful alternative to make-bip32-ext-key-priv.

We construct a new bip32-ext-key-priv structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
