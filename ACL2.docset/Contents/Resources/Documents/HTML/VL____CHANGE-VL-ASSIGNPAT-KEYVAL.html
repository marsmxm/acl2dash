Modifying constructor for vl-assignpat-keyval structures.

Syntax
(change-vl-assignpat-keyval x 
                            [:pairs <pairs>])

This is an often useful alternative to make-vl-assignpat-keyval.

We construct a new vl-assignpat-keyval structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.
