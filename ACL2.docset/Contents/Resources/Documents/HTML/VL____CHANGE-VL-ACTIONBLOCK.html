Modifying constructor for vl-actionblock structures.

Syntax
(change-vl-actionblock x 
                       [:then <then>] 
                       [:else <else>])

This is an often useful alternative to make-vl-actionblock.

We construct a new vl-actionblock structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-actionblock

(defmacro change-vl-actionblock (x &rest args)
          (std::change-aggregate 'vl-actionblock
                                 x args
                                 '((:then . vl-actionblock->then)
                                   (:else . vl-actionblock->else))
                                 'change-vl-actionblock
                                 'remake-vl-actionblock))
