Raw constructor for array-fieldinfo-p structures.

Syntax:

(array-fieldinfo tr-key size-key)

This is the lowest-level constructor for array-fieldinfo-p structures. It simply conses together a structure with the specified fields.

Note: It's generally better to use macros like make-array-fieldinfo or change-array-fieldinfo instead. These macros lead to more readable and robust code, because you don't have to remember the order of the fields.

The array-fieldinfo-p structures we create here are just constructed with ordinary cons. If you want to create honsed structures, see honsed-array-fieldinfo instead.

Definition

This is an ordinary constructor function introduced by std::defaggregate.

Function: array-fieldinfo

(defun array-fieldinfo (tr-key size-key)
       (declare (xargs :guard (and (symbolp tr-key)
                                   (symbolp size-key))))
       (cons :array (std::prod-cons tr-key size-key)))
