(wrb i bsp j) writes the (bsp-size bsp) low-order bits of i into the byte of j specified by bsp.

Signature
(wrb i bsp j) → int
Arguments
i — Guard (integerp i).
bsp — Guard (bspp bsp).
j — Guard (integerp j).
Returns
int — Type (integerp int).

This is analogous to Common Lisp's (dpb newbyte bytespec integer).
