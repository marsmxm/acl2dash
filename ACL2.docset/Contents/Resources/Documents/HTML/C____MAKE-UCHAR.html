Basic constructor macro for uchar structures.

Syntax
(make-uchar [:get <get>])

This is the usual way to construct uchar structures. It simply conses together a structure with the specified fields.

This macro generates a new uchar structure from scratch. See also change-uchar, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by fty::defprod.

Macro: make-uchar

(defmacro make-uchar (&rest args)
          (std::make-aggregate 'uchar
                               args '((:get))
                               'make-uchar
                               nil))
