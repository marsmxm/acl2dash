Signature
(vl-stmtlist-ifmerge x outer-cond warnings elem) 
  → 
(mv warnings flat-stmts)
Arguments
x — List of statements from a simple begin/end block.
    Guard (vl-stmtlist-p x).
outer-cond — Guard (or (not outer-cond) (and (vl-expr-p outer-cond) (posp (vl-expr->finalwidth outer-cond)) (equal (vl-expr->finaltype outer-cond) :vl-unsigned))) .
warnings — Guard (vl-warninglist-p warnings).
elem — Guard (vl-modelement-p elem).
Returns
warnings — Type (vl-warninglist-p warnings).
flat-stmts — Type (vl-stmtlist-p flat-stmts).
