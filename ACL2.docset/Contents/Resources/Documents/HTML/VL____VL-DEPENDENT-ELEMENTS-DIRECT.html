Gathers the names of all design elements that directly depend on particular descriptions.

Signature
(vl-dependent-elements-direct subs design)  superiors
Arguments
subs Names of the sub design elements we're interested in.
Guard (string-listp subs).
design The design.
Guard (vl-design-p design).
Returns
superiors Names of the superior design elements that directly depend on these sub elements.
Type (and (string-listp superiors) (setp superiors)).

Definitions and Theorems

Function: vl-dependent-elements-direct

(defun vl-dependent-elements-direct
       (subs design)
       (declare (xargs :guard (and (string-listp subs)
                                   (vl-design-p design))))
       (let ((__function__ 'vl-dependent-elements-direct))
            (declare (ignorable __function__))
            (vl-collect-dependencies (string-list-fix subs)
                                     (vl-design-upgraph design))))
