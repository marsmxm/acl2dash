Modifying constructor for object/data-object structures.

Syntax
(change-object/data-object x 
                           [:get <get>])

This is an often useful alternative to make-object/data-object.

We construct a new object/data-object structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-object/data-object

(defmacro
     change-object/data-object (x &rest args)
     (std::change-aggregate 'object/data-object
                            x
                            args '((:get . object/data-object->get))
                            'change-object/data-object
                            'nil))
