Basic constructor macro for integer-literal-bin structures.

Syntax
(make-integer-literal-bin [:get <get>])

This is the usual way to construct integer-literal-bin structures. It simply conses together a structure with the specified fields.

This macro generates a new integer-literal-bin structure from scratch. See also change-integer-literal-bin, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by fty::defprod.
