Raw constructor for wcp-lit-actions-p structures.

Syntax:

(wcp-lit-actions witnesses examples)

This is the lowest-level constructor for wcp-lit-actions-p structures. It simply conses together a structure with the specified fields.

Note: It's generally better to use macros like make-wcp-lit-actions or change-wcp-lit-actions instead. These macros lead to more readable and robust code, because you don't have to remember the order of the fields.

The wcp-lit-actions-p structures we create here are just constructed with ordinary cons. If you want to create honsed structures, see honsed-wcp-lit-actions instead.

Definition

This is an ordinary constructor function introduced by std::defaggregate.

Function: wcp-lit-actions

(defun wcp-lit-actions (witnesses examples)
       (declare (xargs :guard (and (wcp-witness-rulesp witnesses)
                                   (wcp-example-appsp examples))))
       (cons (cons 'witnesses witnesses)
             (cons (cons 'examples examples) nil)))
