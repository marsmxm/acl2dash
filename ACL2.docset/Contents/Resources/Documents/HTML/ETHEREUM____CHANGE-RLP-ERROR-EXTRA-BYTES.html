Modifying constructor for rlp-error-extra-bytes structures.

Syntax
(change-rlp-error-extra-bytes x 
                              [:bytes <bytes>])

This is an often useful alternative to make-rlp-error-extra-bytes.

We construct a new rlp-error-extra-bytes structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-rlp-error-extra-bytes

(defmacro
   change-rlp-error-extra-bytes
   (x &rest args)
   (std::change-aggregate 'rlp-error-extra-bytes
                          x args
                          '((:bytes . rlp-error-extra-bytes->bytes))
                          'change-rlp-error-extra-bytes
                          'nil))
