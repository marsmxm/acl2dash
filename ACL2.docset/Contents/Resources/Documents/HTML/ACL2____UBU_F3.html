Undo commands, with queries as appropriate

See undo for an introduction to undoing commands.

Example:
:ubu? :x-4

The keyword command :ubu? is the same as :ubu, but with appropriate queries made. :Oops will undo the last :ubu?. Also see ubu!, ubu, ubt!, ubt, ubt?, and u.
