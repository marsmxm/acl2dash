Find the maximal and leftmost subterm of term-to-try that satisfies the conditions for nonrec<x1,...,xn> described in the documentation.

Signature
(tailrec-find-nonrec-term combine-nonrec term-to-try r q) 
   
(mv success nonrec combine)
Arguments
combine-nonrec Guard (pseudo-termp combine-nonrec).
term-to-try Subterm of combine<nonrec<x1,...,xn>,r> to examine next.
Guard (pseudo-termp term-to-try).
r Guard (symbolp r).
q Guard (symbolp q).
Returns
success A booleanp.
nonrec The pseudo-termp nonrec<x1,...,xn> described in the documentation, if success is t; otherwise nil.
combine The pseudo-termp combine<q,r> described in the documentation, if success is t; otherwise nil.

When initially invoked on combine<nonrec<x1,...,xn>,r> as term-to-try, attempt to recursively finds nonrec<x1,...,xn> as described in the documentation.
