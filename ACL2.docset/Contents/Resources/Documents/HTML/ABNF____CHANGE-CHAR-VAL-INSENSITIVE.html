Modifying constructor for char-val-insensitive structures.

Syntax
(change-char-val-insensitive x 
                             [:get <get>])

This is an often useful alternative to make-char-val-insensitive.

We construct a new char-val-insensitive structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-char-val-insensitive

(defmacro
