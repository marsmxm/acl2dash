Modifying constructor for maybe-4vec-some structures.

Syntax
(change-maybe-4vec-some x 
                        [:val <val>])

This is an often useful alternative to make-maybe-4vec-some.

We construct a new maybe-4vec-some structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-maybe-4vec-some

(defmacro
     change-maybe-4vec-some (x &rest args)
     (std::change-aggregate 'maybe-4vec-some

                            'change-maybe-4vec-some
                            'nil))
