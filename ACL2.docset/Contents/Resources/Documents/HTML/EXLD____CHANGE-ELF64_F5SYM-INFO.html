Modifying constructor for elf64_sym-info structures.

Syntax
(change-elf64_sym-info x 
                       [:name-str <name-str>] 
                       [:entry <entry>])

This is an often useful alternative to make-elf64_sym-info.

We construct a new elf64_sym-info structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-elf64_sym-info

(defmacro
     change-elf64_sym-info (x &rest args)
     (std::change-aggregate 'elf64_sym-info
                            x args
                            '((:name-str . elf64_sym-info->name-str)
                              (:entry . elf64_sym-info->entry))
                            'change-elf64_sym-info
                            'nil))
