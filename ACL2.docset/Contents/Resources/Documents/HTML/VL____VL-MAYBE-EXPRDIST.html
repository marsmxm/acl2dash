Option type; vl-exprdist or nil.

This is an option type introduced by defoption. Note that defoption is just a wrapper for defflexsum, so there are :none and :some member types, a case macro, and so forth.

Member Types
:none → vl-maybe-exprdist-none
Represents that no vl-maybe-exprdist is available, i.e., Nothing or None.
:some → vl-maybe-exprdist-some
An available vl-maybe-exprdist, i.e., Just val or Some val.
