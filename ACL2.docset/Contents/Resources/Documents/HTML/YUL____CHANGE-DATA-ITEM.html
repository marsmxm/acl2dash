Modifying constructor for data-item structures.

Syntax
(change-data-item x 
                  [:name <name>] 
                  [:value <value>])

This is an often useful alternative to make-data-item.

We construct a new data-item structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-data-item

(defmacro change-data-item (x &rest args)
          (std::change-aggregate 'data-item
                                 x args
                                 '((:name . data-item->name)
                                   (:value . data-item->value))
                                 'change-data-item
                                 'nil))
