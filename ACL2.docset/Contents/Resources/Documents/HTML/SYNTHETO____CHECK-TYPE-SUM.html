Check if a type sum is statically well-formed.

Signature
(check-type-sum tsum ctxt) → (mv err? obligs)
Arguments
tsum — Guard (type-sump tsum).
ctxt — Guard (contextp ctxt).
Returns
obligs — Type (proof-obligation-listp obligs).

The guard assumptions on the context are motivated as in check-type-product.

The alternatives must have distinct names and be all statically well-formed.

Definitions and Theorems

Function: check-type-sum

(defun
 check-type-sum (tsum ctxt)
 (declare (xargs :guard (and (type-sump tsum) (contextp ctxt))))
 (declare
      (xargs :guard (and (null (context->functions ctxt))
                         (omap::empty (context->variables ctxt))
                         (null (context->obligation-vars ctxt))
                         (null (context->obligation-hyps ctxt)))))
 (let
   ((__function__ 'check-type-sum))
   (declare (ignorable __function__))
   (b* ((alternatives (type-sum->alternatives tsum))
        ((unless (no-duplicatesp-equal
                      (alternative-list->name-list alternatives)))
         (mv (list :duplicate-alternative-names (type-sum-fix tsum))
             nil)))
       (check-alternative-list alternatives ctxt))))
