Basic constructor macro for vl-fwdtypedef structures.

Syntax
(make-vl-fwdtypedef [:atts <atts>] 
                    [:kind <kind>] 
                    [:name <name>] 
                    [:loc <loc>])

This is the usual way to construct vl-fwdtypedef structures. It simply conses together a structure with the specified fields.

This macro generates a new vl-fwdtypedef structure from scratch. See also change-vl-fwdtypedef, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by defprod.

(defun vl-fwdtypedef (atts kind name loc)
       (declare (xargs :guard (and (vl-atts-p atts)
                                   (vl-fwdtypedefkind-p kind)
                                   (stringp name)
                                   (vl-location-p loc))))
       (declare (xargs :guard t))
       (let ((__function__ 'vl-fwdtypedef))
            (declare (ignorable __function__))
            (b* ((atts (mbe :logic (vl-atts-fix atts)
                            :exec atts))
                 (kind (mbe :logic (vl-fwdtypedefkind-fix kind)
                            :exec kind))
                 (name (mbe :logic (str-fix name) :exec name))
                 (loc (mbe :logic (vl-location-fix loc)
                           :exec loc)))
                (cons :vl-fwdtypedef (list (cons 'atts atts)
                                           (cons 'kind kind)
                                           (cons 'name name)
                                           (cons 'loc loc))))))
