Modifying constructor for maybe-stat-none structures.

Syntax
(change-maybe-stat-none x 
                        )

This is an often useful alternative to make-maybe-stat-none.

We construct a new maybe-stat-none structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
