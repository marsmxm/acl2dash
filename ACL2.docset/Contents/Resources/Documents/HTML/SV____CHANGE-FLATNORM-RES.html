Modifying constructor for flatnorm-res structures.

Syntax
(change-flatnorm-res x 
                     [:assigns <assigns>] 
                     [:delays <delays>] 
                     [:constraints <constraints>])

This is an often useful alternative to make-flatnorm-res.

We construct a new flatnorm-res structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-flatnorm-res

(defmacro change-flatnorm-res (x &rest args)
          (std::change-aggregate
               'flatnorm-res
               x args
               '((:assigns . flatnorm-res->assigns)
                 (:delays . flatnorm-res->delays)
                 (:constraints . flatnorm-res->constraints))
               'change-flatnorm-res
               'nil))
