Modifying constructor for regex-case-sens structures.

Syntax
(change-regex-case-sens x 
                        [:pat <pat>] 
                        [:case-insens <case-insens>])

This is an often useful alternative to make-regex-case-sens.

We construct a new regex-case-sens structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-regex-case-sens

(defmacro change-regex-case-sens (x &rest args)
          (std::change-aggregate
               'regex-case-sens
               x args
               '((:pat . regex-case-sens->pat)
                 (:case-insens . regex-case-sens->case-insens))
               'change-regex-case-sens
               'nil))
