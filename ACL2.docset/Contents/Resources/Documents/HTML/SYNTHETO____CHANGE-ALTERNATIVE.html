Modifying constructor for alternative structures.

Syntax
(change-alternative x 
                    [:name <name>] 
                    [:product <product>])

This is an often useful alternative to make-alternative.

We construct a new alternative structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-alternative

(defmacro
     change-alternative (x &rest args)
     (std::change-aggregate 'alternative
                            x args
                            '((:name . alternative->name)
                              (:product . alternative->product))
                            'change-alternative
                            'nil))
