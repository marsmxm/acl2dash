Modifying constructor for primitive-type-short structures.

Syntax
(change-primitive-type-short x 
                             )

This is an often useful alternative to make-primitive-type-short.

We construct a new primitive-type-short structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
