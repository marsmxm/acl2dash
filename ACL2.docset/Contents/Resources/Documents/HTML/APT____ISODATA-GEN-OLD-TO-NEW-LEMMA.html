Generate the lemma used to prove old-to-new when old is multi-valued and some result is transformed.

Signature
(isodata-gen-old-to-new-lemma 
     old$ arg-isomaps res-isomaps new$ 
     appcond-thm-names new-to-old-lemma-name 
     names-to-avoid wrld) 
 
  → 
(mv old-to-new-lemma-name 
    old-to-new-lemma-event 
    updated-names-to-avoid)
Arguments
old$ — Guard (symbolp old$).
arg-isomaps — Guard (isodata-symbol-isomap-alistp arg-isomaps).
res-isomaps — Guard (isodata-pos-isomap-alistp res-isomaps).
new$ — Guard (symbolp new$).
appcond-thm-names — Guard (symbol-symbol-alistp appcond-thm-names).
new-to-old-lemma-name — Guard (symbolp new-to-old-lemma-name).
names-to-avoid — Guard (symbol-listp names-to-avoid).
wrld — Guard (plist-worldp wrld).
Returns
old-to-new-lemma-name — A symbolp.
old-to-new-lemma-event — A pseudo-event-formp.
updated-names-to-avoid — A symbol-listp.

This is generated only if m is 2 or more. This is a form of old-to-new that uses mv-nth; it seems (based on some earlier proof generation attempts) easier to prove than old-to-new, which is then easily proved from this lemma, with the help of another lemma.

Definitions and Theorems

Function: isodata-gen-old-to-new-lemma

(defun
 isodata-gen-old-to-new-lemma
 (old$ arg-isomaps res-isomaps new$
       appcond-thm-names new-to-old-lemma-name
       names-to-avoid wrld)
 (declare
      (xargs :guard (and (symbolp old$)
                         (isodata-symbol-isomap-alistp arg-isomaps)
                         (isodata-pos-isomap-alistp res-isomaps)
                         (symbolp new$)
                         (symbol-symbol-alistp appcond-thm-names)
                         (symbolp new-to-old-lemma-name)
                         (symbol-listp names-to-avoid)
                         (plist-worldp wrld))))
 (let
  ((__function__ 'isodata-gen-old-to-new-lemma))
  (declare (ignorable __function__))
  (b* (((mv name names-to-avoid)
        (fresh-logical-name-with-$s-suffix 'old-to-new-lemma
                                           nil names-to-avoid wrld))
       (formula (isodata-gen-old-to-new-lemma-formula
                     old$ arg-isomaps res-isomaps new$ wrld))
       (hints (isodata-gen-old-to-new-lemma-hints
                   appcond-thm-names old$ arg-isomaps
                   res-isomaps new-to-old-lemma-name wrld))
       ((mv event &)
        (evmac-generate-defthm name
                               :formula formula
                               :hints hints
                               :enable nil)))
      (mv name event names-to-avoid))))
