Modifying constructor for declon-struct structures.

Syntax
(change-declon-struct x 
                      [:tag <tag>] 
                      [:members <members>])

This is an often useful alternative to make-declon-struct.

We construct a new declon-struct structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-declon-struct

(defmacro
     change-declon-struct (x &rest args)
     (std::change-aggregate 'declon-struct
                            x args
                            '((:tag . declon-struct->tag)
                              (:members . declon-struct->members))
                            'change-declon-struct
                            'nil))
