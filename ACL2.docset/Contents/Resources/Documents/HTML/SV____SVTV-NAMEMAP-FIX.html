(svtv-namemap-fix x) is an fty alist fixing function that follows the drop-keys strategy.

Signature
(svtv-namemap-fix x) → fty::newx
Arguments
x — Guard (svtv-namemap-p x).
Returns
fty::newx — Type (svtv-namemap-p fty::newx).

Note that in the execution this is just an inline identity function.

 svtv-namemap-fix$inline (x)
 (declare (xargs :guard (svtv-namemap-p x)))
 (let
   ((__function__ 'svtv-namemap-fix))
   (declare (ignorable __function__))
   (mbe :logic
        (if (atom x)
            nil
            (let ((rest (svtv-namemap-fix (cdr x))))
                 (if (and (consp (car x)) (svar-p (caar x)))
                     (let ((fty::first-key (caar x))
                           (fty::first-val (str-fix (cdar x))))
                          (cons (cons fty::first-key fty::first-val)
                                rest))
                     rest)))
        :exec x)))
