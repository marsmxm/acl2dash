Execute an initializer.

Signature
(exec-initer initer compst fenv limit) 
  → 
(mv result new-compst)
Arguments
initer — Guard (initerp initer).
compst — Guard (compustatep compst).
fenv — Guard (fun-envp fenv).
limit — Guard (natp limit).
Returns
result — Type (value-resultp result).
new-compst — Type (compustatep new-compst).

For now we only accept single expressions. The single expression must be a function call or a pure expression. If it is a function call, it must return a value (not nil).
