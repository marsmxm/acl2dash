Modifying constructor for reference-value-null structures.

Syntax
(change-reference-value-null x 
                             )

This is an often useful alternative to make-reference-value-null.

We construct a new reference-value-null structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-reference-value-null

(defmacro change-reference-value-null
          (x &rest args)
          (std::change-aggregate 'reference-value-null
                                 x args 'nil
                                 'change-reference-value-null
                                 'nil))
