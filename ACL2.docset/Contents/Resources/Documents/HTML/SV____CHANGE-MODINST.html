Modifying constructor for modinst structures.

Syntax
(change-modinst x 
                [:instname <instname>] 
                [:modname <modname>])

This is an often useful alternative to make-modinst.

We construct a new modinst structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-modinst

(defmacro change-modinst (x &rest args)
          (std::change-aggregate 'modinst
                                 x args
                                 '((:instname . modinst->instname)
                                   (:modname . modinst->modname))
                                 'change-modinst
                                 'remake-modinst))
