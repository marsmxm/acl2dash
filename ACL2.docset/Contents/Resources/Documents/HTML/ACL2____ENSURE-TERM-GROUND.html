Cause an error if a term is not ground (i.e. it has free variables).

Signature
(ensure-term-ground term description error-erp error-val ctx state) 
  → 
(mv erp val state)
Arguments
term — Term to check.
    Guard (pseudo-termp term).
description — Guard (msgp description).
error-erp — Flag to return in case of error.
error-val — Value to return in case of error.
ctx — Context for errors.
Returns
erp — Type (implies erp (equal erp error-erp)).
val — Type (and (implies erp (equal val error-val)) (implies (and (not erp) error-erp) (not val))) .

Function: ensure-term-ground

(defun
 ensure-term-ground
 (term description
       error-erp error-val ctx state)
 (declare (xargs :stobjs (state)))
 (declare (xargs :guard (and (pseudo-termp term)
                             (msgp description))))
 (b*
  (((unless (null (all-vars term)))
    (er-soft+
      ctx error-erp error-val
      "~@0 must contain no free variables, but it contains the ~@1."
      description
      (let ((vars (all-vars term)))
           (if (= (len vars) 1)
               (msg "variable ~x0" (car vars))
               (msg "variables ~&0" vars))))))
  (value nil)))
