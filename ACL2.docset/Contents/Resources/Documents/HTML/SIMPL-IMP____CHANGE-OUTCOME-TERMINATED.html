Modifying constructor for outcome-terminated structures.

Syntax
(change-outcome-terminated x 
                           [:env <env>])

This is an often useful alternative to make-outcome-terminated.

We construct a new outcome-terminated structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-outcome-terminated

(defmacro
     change-outcome-terminated (x &rest args)
     (std::change-aggregate 'outcome-terminated
                            x
                            args '((:env . outcome-terminated->env))
                            'change-outcome-terminated
                            'nil))
