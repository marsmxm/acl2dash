Modifying constructor for escape-sequence-t structures.

Syntax
(change-escape-sequence-t x 
                          )

This is an often useful alternative to make-escape-sequence-t.

We construct a new escape-sequence-t structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
