A copying macro that lets you create new vl-extinttoken-p structures, based on existing structures.

Syntax:

(change-vl-extinttoken x 
                       [:etext <etext>] 
                       [:value <value>])

This is a sometimes useful alternative to make-vl-extinttoken. It constructs a new vl-extinttoken-p structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defaggregate.

Macro: change-vl-extinttoken

(defmacro change-vl-extinttoken (x &rest args)
          (std::change-aggregate 'vl-extinttoken
                                 x args
                                 '((:etext . vl-extinttoken->etext)
                                   (:value . vl-extinttoken->value))
                                 'change-vl-extinttoken
                                 'remake-vl-extinttoken))
