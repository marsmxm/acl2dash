Fixtype of (ATC's model of) arrays of type unsigned char.

This is a product type introduced by fty::defprod.

Fields
elemtype — type
elements — uchar-list
Additional Requirements

The following invariant is enforced on the fields:

(and (type-case elemtype :uchar) (consp elements))

Theorem: uchar-arrayp-alt-def

(defthm uchar-arrayp-alt-def
        (equal (uchar-arrayp x)
               (and (valuep x)
                    (value-case x :array)
                    (equal (value-array->elemtype x)
                           (type-uchar))
                    (uchar-listp (value-array->elements x)))))

Theorem: uchar-array->elements-alt-def

(defthm uchar-array->elements-alt-def
        (implies (uchar-arrayp array)
                 (equal (uchar-array->elements array)
                        (value-array->elements array))))
