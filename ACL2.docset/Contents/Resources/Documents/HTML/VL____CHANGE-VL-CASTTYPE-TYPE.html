Modifying constructor for vl-casttype-type structures.

Syntax
(change-vl-casttype-type x 
                         [:type <type>])

This is an often useful alternative to make-vl-casttype-type.

We construct a new vl-casttype-type structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-casttype-type

(defmacro
     change-vl-casttype-type (x &rest args)
     (std::change-aggregate 'vl-casttype-type
                            x
                            args '((:type . vl-casttype-type->type))
                            'change-vl-casttype-type
                            'remake-vl-casttype-type))
