Construct the expression for a single variable.

Signature
(vl-stmt-cblock-varexpr varname x curr) → expr?
Arguments
varname — Variable we're considering.
    Guard (stringp varname).
x — Statement we're descending through.
    Guard (vl-stmt-p x).
curr — Expression we've built up for varname so far.
    Guard (vl-expr-p curr).
Returns
expr? — New expression for varname, if any.
    Type (vl-expr-p expr?), given the guard.
