Fixtype of Syntheto function definitions.

This is a product type introduced by fty::defprod.

Fields
header — function-header
precondition — maybe-expression
postcondition — maybe-expression
definer — function-definer

A function definition consists of a function header and a function definer. It also has an optional precondition and an optional postcondition. Both must be boolean expressions.
