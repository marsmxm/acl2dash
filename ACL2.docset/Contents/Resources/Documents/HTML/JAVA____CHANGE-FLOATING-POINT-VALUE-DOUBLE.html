Modifying constructor for floating-point-value-double structures.

Syntax
(change-floating-point-value-double x 
                                    [:get <get>])

This is an often useful alternative to make-floating-point-value-double.

We construct a new floating-point-value-double structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
