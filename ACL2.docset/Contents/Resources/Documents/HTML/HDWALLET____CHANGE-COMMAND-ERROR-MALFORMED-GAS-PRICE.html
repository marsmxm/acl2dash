Modifying constructor for command-error-malformed-gas-price structures.

Syntax
(change-command-error-malformed-gas-price x 
                                          [:gas-price <gas-price>])

This is an often useful alternative to make-command-error-malformed-gas-price.

We construct a new command-error-malformed-gas-price structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
