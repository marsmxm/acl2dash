A specialization of must-fail to ensure that an error occurs.

Evaluation of (must-fail-with-error <form>) returns without error exactly when evaluation of <form> causes an error.

See must-fail for more details, as must-fail-with-error abbreviates must-fail as follows.

Macro: must-fail-with-error

(defmacro must-fail-with-error (form &rest args)
          (list* 'must-fail
                 form :expected
                 :any args))

Also see must-fail-with-soft-error and must-fail-with-hard-error.
