Modifying constructor for expression-sum-test structures.

Syntax
(change-expression-sum-test x 
                            [:type <type>] 
                            [:target <target>] 
                            [:alternative <alternative>])

This is an often useful alternative to make-expression-sum-test.

We construct a new expression-sum-test structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
