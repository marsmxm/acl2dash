Modifying constructor for base-fsm structures.

Syntax
(change-base-fsm x 
                 [:values <values>] 
                 [:nextstate <nextstate>])

This is an often useful alternative to make-base-fsm.

We construct a new base-fsm structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-base-fsm

(defmacro
     change-base-fsm (x &rest args)
     (std::change-aggregate 'base-fsm
                            x args
                            '((:values . base-fsm->values)
                              (:nextstate . base-fsm->nextstate))
                            'change-base-fsm
                            'nil))
