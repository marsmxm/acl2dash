Modifying constructor for vl-typename structures.

Syntax
(change-vl-typename x 
                    [:name <name>])

This is an often useful alternative to make-vl-typename.

We construct a new vl-typename structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-typename

(defmacro
     change-vl-typename (x &rest args)
     (std::change-aggregate 'vl-typename
                            x args '((:name . vl-typename->name))
                            'change-vl-typename
                            'remake-vl-typename))
