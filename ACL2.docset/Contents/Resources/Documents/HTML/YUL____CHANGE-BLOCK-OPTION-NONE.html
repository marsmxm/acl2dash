Modifying constructor for block-option-none structures.

Syntax
(change-block-option-none x 
                          )

This is an often useful alternative to make-block-option-none.

We construct a new block-option-none structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
