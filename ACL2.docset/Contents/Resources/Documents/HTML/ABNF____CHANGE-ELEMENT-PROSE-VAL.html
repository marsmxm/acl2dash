Modifying constructor for element-prose-val structures.

Syntax
(change-element-prose-val x 
                          [:get <get>])

This is an often useful alternative to make-element-prose-val.

We construct a new element-prose-val structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-element-prose-val

(defmacro
     change-element-prose-val (x &rest args)
     (std::change-aggregate 'element-prose-val
                            x
                            args '((:get . element-prose-val->get))
                            'change-element-prose-val
                            'nil))
