A copying macro that lets you create new vcd-multivector-p structures, based on existing structures.

Syntax:

(change-vcd-multivector x 
                        [:bits <bits>] 
                        [:idcodes <idcodes>])

This is a sometimes useful alternative to make-vcd-multivector. It constructs a new vcd-multivector-p structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defaggregate.

Macro: change-vcd-multivector

(defmacro
     change-vcd-multivector (x &rest args)
     (std::change-aggregate 'vcd-multivector
                            x args
                            '((:bits . vcd-multivector->bits)
                              (:idcodes . vcd-multivector->idcodes))
                            'change-vcd-multivector
                            'remake-vcd-multivector))
