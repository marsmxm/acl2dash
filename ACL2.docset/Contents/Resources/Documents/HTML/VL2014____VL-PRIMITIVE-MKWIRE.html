Convenient way to generate an internal wire for a primitive.

Signature
(vl-primitive-mkwire name) → (mv expr vardecl)
Arguments
name — Guard (stringp name).
Returns
expr — Type (vl-expr-p expr).
vardecl — Type (vl-vardecl-p vardecl).

This is similar to vl-occform-mkwire, but our primitives are all one-bit things so we leave out the ranges to make them prettier.

Definitions and Theorems

Function: vl-primitive-mkwire

(defun
  vl-primitive-mkwire (name)
  (declare (xargs :guard (stringp name)))
  (let ((__function__ 'vl-primitive-mkwire))
       (declare (ignorable __function__))
       (b* ((name (hons-copy (string-fix name)))
            (expr (vl-idexpr name 1 :vl-unsigned))
            (vardecl (make-vl-vardecl :name name
                                      :type *vl-plain-old-wire-type*
                                      :nettype :vl-wire
                                      :loc *vl-fakeloc*)))
           (mv expr vardecl))))
