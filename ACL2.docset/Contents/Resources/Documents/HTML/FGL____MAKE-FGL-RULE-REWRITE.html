Basic constructor macro for fgl-rule-rewrite structures.

Syntax
(make-fgl-rule-rewrite [:rune <rune>] 
                       [:rule <rule>])

This is the usual way to construct fgl-rule-rewrite structures. It simply conses together a structure with the specified fields.

This macro generates a new fgl-rule-rewrite structure from scratch. See also change-fgl-rule-rewrite, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by defprod.

(defun fgl-rule-rewrite (rune rule)
       (declare (xargs :guard (and (fgl-rune-p rune)
                                   (cmr::rewrite-p rule))))
       (declare (xargs :guard t))
       (let ((__function__ 'fgl-rule-rewrite))
            (declare (ignorable __function__))
            (b* ((rune (mbe :logic (fgl-rune-fix rune)
                            :exec rune))
                 (rule (mbe :logic (cmr::rewrite-fix rule)
                            :exec rule)))
