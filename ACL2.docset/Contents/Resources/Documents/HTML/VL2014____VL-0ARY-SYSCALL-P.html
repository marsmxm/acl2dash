Recognize calls of a particular zero-ary system function call.

Signature
(vl-0ary-syscall-p name x) → *
Arguments
name — Name of the particular system call, e.g., $time.
    Guard (stringp name).
x — Expression to test.
    Guard (vl-expr-p x).

For instance, to see if x is a call of $time, you could write:

(vl-0ary-syscall-p "$time" x)

Definitions and Theorems

Function: vl-0ary-syscall-p

(defun vl-0ary-syscall-p (name x)
       (declare (xargs :guard (and (stringp name) (vl-expr-p x))))
       (let ((__function__ 'vl-0ary-syscall-p))
            (declare (ignorable __function__))
            (b* (((when (vl-fast-atom-p x)) nil)
                 ((vl-nonatom x))
                 ((unless (and (eq x.op :vl-syscall)
                               (consp x.args)
                               (atom (cdr x.args))))
                  nil)
                 (fn (first x.args))
                 ((unless (vl-sysfunexpr-p fn)) nil))
                (equal (vl-sysfunexpr->name fn)
                       (string-fix name)))))

Theorem: arity-stuff-about-vl-0ary-syscall-p

(defthm arity-stuff-about-vl-0ary-syscall-p
        (implies (vl-0ary-syscall-p name x)
                 (and (not (equal (vl-expr-kind x) :atom))
                      (equal (vl-nonatom->op x) :vl-syscall)
                      (consp (vl-nonatom->args x))
                      (not (consp (cdr (vl-nonatom->args x))))))
        :rule-classes :forward-chaining)
