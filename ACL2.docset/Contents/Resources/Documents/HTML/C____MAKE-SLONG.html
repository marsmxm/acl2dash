Basic constructor macro for slong structures.

Syntax
(make-slong [:get <get>])

This is the usual way to construct slong structures. It simply conses together a structure with the specified fields.

This macro generates a new slong structure from scratch. See also change-slong, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by fty::defprod.

Macro: make-slong

(defmacro make-slong (&rest args)
          (std::make-aggregate 'slong
                               args '((:get))
                               'make-slong
                               nil))
