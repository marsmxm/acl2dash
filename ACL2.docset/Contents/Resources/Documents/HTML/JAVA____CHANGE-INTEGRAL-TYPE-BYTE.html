Modifying constructor for integral-type-byte structures.

Syntax
(change-integral-type-byte x 
                           )

This is an often useful alternative to make-integral-type-byte.

We construct a new integral-type-byte structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
