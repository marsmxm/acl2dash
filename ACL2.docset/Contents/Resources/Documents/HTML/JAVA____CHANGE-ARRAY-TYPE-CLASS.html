Modifying constructor for array-type-class structures.

Syntax
(change-array-type-class x 
                         [:element <element>] 
                         [:dimensions <dimensions>])

This is an often useful alternative to make-array-type-class.

We construct a new array-type-class structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-array-type-class

(defmacro change-array-type-class (x &rest args)
          (std::change-aggregate
               'array-type-class
               x args
               '((:element . array-type-class->element)
                 (:dimensions . array-type-class->dimensions))
               'change-array-type-class
               'nil))
