Modifying constructor for type-definer-product structures.

Syntax
(change-type-definer-product x 
                             [:get <get>])

This is an often useful alternative to make-type-definer-product.

We construct a new type-definer-product structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-type-definer-product

(defmacro
