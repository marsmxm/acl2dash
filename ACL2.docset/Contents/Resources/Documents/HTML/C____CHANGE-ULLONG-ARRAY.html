Modifying constructor for ullong-array structures.

Syntax
(change-ullong-array x 
                     [:elemtype <elemtype>] 
                     [:elements <elements>])

This is an often useful alternative to make-ullong-array.

We construct a new ullong-array structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-ullong-array

(defmacro
     change-ullong-array (x &rest args)
     (std::change-aggregate 'ullong-array
                            x args
                            '((:elemtype . ullong-array->elemtype)
                              (:elements . ullong-array->elements))
                            'change-ullong-array
                            'nil))
