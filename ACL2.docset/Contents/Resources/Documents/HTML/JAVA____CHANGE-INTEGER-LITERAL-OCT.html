Modifying constructor for integer-literal-oct structures.

Syntax
(change-integer-literal-oct x 
                            [:get <get>])

This is an often useful alternative to make-integer-literal-oct.

We construct a new integer-literal-oct structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-integer-literal-oct

(defmacro
     change-integer-literal-oct
     (x &rest args)
     (std::change-aggregate 'integer-literal-oct
                            x args
                            '((:get . integer-literal-oct->get))
                            'change-integer-literal-oct
                            'nil))
