(vl-letdecllist-fix x) is a usual fty list fixing function.

Signature
(vl-letdecllist-fix x) → fty::newx
Arguments
x — Guard (vl-letdecllist-p x).
Returns
fty::newx — Type (vl-letdecllist-p fty::newx).

In the logic, we apply vl-letdecl-fix to each member of the x. In the execution, none of that is actually necessary and this is just an inlined identity function.
