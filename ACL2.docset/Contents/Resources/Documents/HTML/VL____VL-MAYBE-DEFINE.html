Option type; vl-define or nil.

This is an option type introduced by defoption. Note that defoption is just a wrapper for defflexsum, so there are :none and :some member types, a case macro, and so forth.

Member Types
:none → vl-maybe-define-none
Represents that no vl-maybe-define is available, i.e., Nothing or None.
:some → vl-maybe-define-some
An available vl-maybe-define, i.e., Just val or Some val.
