Extend an alist by binding the names of vl-interfaces to their definitions.

Signature
(vl-interfacelist-alist x acc) → alist
Arguments
x — Guard (vl-interfacelist-p x).
Returns
alist — Type (equal (vl-interface-alist-p alist) (vl-interface-alist-p acc)).

This can be used as an alternative to vl-find-interface when you need to perform a lot of lookups.

Definitions and Theorems

Function: vl-interfacelist-alist

(defun vl-interfacelist-alist (x acc)
       (declare (xargs :guard (vl-interfacelist-p x)))
       (let ((__function__ 'vl-interfacelist-alist))
            (declare (ignorable __function__))
            (if (atom x)
                acc
                (cons (cons (vl-interface->name (car x))
                            (vl-interface-fix (car x)))
                      (vl-interfacelist-alist (cdr x) acc)))))

Theorem: lookup-in-vl-interfacelist-alist-fast

(defthm
  lookup-in-vl-interfacelist-alist-fast
  (implies
       (stringp name)
       (equal (hons-assoc-equal name (vl-interfacelist-alist x nil))
              (let ((val (vl-find-interface name x)))
                   (and val (cons name val))))))
