Modifying constructor for tyspecseq-typedef structures.

Syntax
(change-tyspecseq-typedef x 
                          [:name <name>])

This is an often useful alternative to make-tyspecseq-typedef.

We construct a new tyspecseq-typedef structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-tyspecseq-typedef

(defmacro
     change-tyspecseq-typedef (x &rest args)
     (std::change-aggregate 'tyspecseq-typedef
                            x args
                            '((:name . tyspecseq-typedef->name))
                            'change-tyspecseq-typedef
                            'nil))
