Modifying constructor for escape-sequence-r structures.

Syntax
(change-escape-sequence-r x 
                          )

This is an often useful alternative to make-escape-sequence-r.

We construct a new escape-sequence-r structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
