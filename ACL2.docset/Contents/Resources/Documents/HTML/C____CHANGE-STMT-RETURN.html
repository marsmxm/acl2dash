Modifying constructor for stmt-return structures.

Syntax
(change-stmt-return x 
                    [:value <value>])

This is an often useful alternative to make-stmt-return.

We construct a new stmt-return structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-stmt-return

(defmacro
     change-stmt-return (x &rest args)
     (std::change-aggregate 'stmt-return
                            x args '((:value . stmt-return->value))
                            'change-stmt-return
                            'nil))
