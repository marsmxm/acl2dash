Modifying constructor for function-definition structures.

Syntax
(change-function-definition x 
                            [:header <header>] 
                            [:precondition <precondition>] 
                            [:postcondition <postcondition>] 
                            [:definer <definer>])

This is an often useful alternative to make-function-definition.

We construct a new function-definition structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-function-definition

(defmacro
     change-function-definition
     (x &rest args)
     (std::change-aggregate
          'function-definition
          x args
          '((:header . function-definition->header)
            (:precondition . function-definition->precondition)
            (:postcondition . function-definition->postcondition)
            (:definer . function-definition->definer))
          'change-function-definition
          'nil))
