Modifying constructor for floating-point-value-float structures.

Syntax
(change-floating-point-value-float x 
                                   [:get <get>])

This is an often useful alternative to make-floating-point-value-float.

We construct a new floating-point-value-float structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
