Modifying constructor for g-map-tag structures.

Syntax
(change-g-map-tag x 
                  [:index <index>])

This is an often useful alternative to make-g-map-tag.

We construct a new g-map-tag structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-g-map-tag

(defmacro
     change-g-map-tag (x &rest args)
     (std::change-aggregate 'g-map-tag
                            x args '((:index . g-map-tag->index))
                            'change-g-map-tag
                            'remake-g-map-tag))
