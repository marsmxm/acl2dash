(vl-packeddimensionlist-fix x) is a usual ACL2::fty list fixing function.

Signature
(vl-packeddimensionlist-fix x) → fty::newx
Arguments
x — Guard (vl-packeddimensionlist-p x).
Returns
fty::newx — Type (vl-packeddimensionlist-p fty::newx).

In the logic, we apply vl-packeddimension-fix to each member of the x. In the execution, none of that is actually necessary and this is just an inlined identity function.
