Modifying constructor for int-value structures.

Syntax
(change-int-value x 
                  [:int <int>])

This is an often useful alternative to make-int-value.

We construct a new int-value structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-int-value

(defmacro change-int-value (x &rest args)
          (std::change-aggregate 'int-value
                                 x args '((:int . int-value->int))
                                 'change-int-value
                                 'nil))
