Modifying constructor for vl-waitstmt structures.

Syntax
(change-vl-waitstmt x 
                    [:condition <condition>] 
                    [:body <body>] 
                    [:atts <atts>])

This is an often useful alternative to make-vl-waitstmt.

We construct a new vl-waitstmt structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-waitstmt

(defmacro
     change-vl-waitstmt (x &rest args)
     (std::change-aggregate 'vl-waitstmt
                            x args
                            '((:condition . vl-waitstmt->condition)
                              (:body . vl-waitstmt->body)
                              (:atts . vl-waitstmt->atts))
                            'change-vl-waitstmt
                            'remake-vl-waitstmt))
