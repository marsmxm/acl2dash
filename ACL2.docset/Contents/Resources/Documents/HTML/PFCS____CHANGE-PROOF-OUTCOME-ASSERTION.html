Modifying constructor for proof-outcome-assertion structures.

Syntax
(change-proof-outcome-assertion x 
                                [:get <get>])

This is an often useful alternative to make-proof-outcome-assertion.

We construct a new proof-outcome-assertion structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
