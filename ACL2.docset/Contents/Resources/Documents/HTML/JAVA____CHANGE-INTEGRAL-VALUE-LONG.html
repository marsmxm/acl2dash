Modifying constructor for integral-value-long structures.

Syntax
(change-integral-value-long x 
                            [:get <get>])

This is an often useful alternative to make-integral-value-long.

We construct a new integral-value-long structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-integral-value-long

(defmacro
     change-integral-value-long
     (x &rest args)
     (std::change-aggregate 'integral-value-long
                            x args
                            '((:get . integral-value-long->get))

                            'nil))
