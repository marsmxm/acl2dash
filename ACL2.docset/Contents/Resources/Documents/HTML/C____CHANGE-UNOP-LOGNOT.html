Modifying constructor for unop-lognot structures.

Syntax
(change-unop-lognot x 
                    )

This is an often useful alternative to make-unop-lognot.

We construct a new unop-lognot structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
