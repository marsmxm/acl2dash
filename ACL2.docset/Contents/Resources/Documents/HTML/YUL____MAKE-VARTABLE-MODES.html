Basic constructor macro for vartable-modes structures.

Syntax
(make-vartable-modes [:variables <variables>] 
                     [:modes <modes>])

This is the usual way to construct vartable-modes structures. It simply conses together a structure with the specified fields.

This macro generates a new vartable-modes structure from scratch. See also change-vartable-modes, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by fty::defprod.

(defun
    vartable-modes (variables modes)
    (declare (xargs :guard (and (vartablep variables)
                                (mode-setp modes))))
    (declare (xargs :guard t))
    (let ((__function__ 'vartable-modes))
         (declare (ignorable __function__))
         (b* ((variables (mbe :logic (vartable-fix variables)
                              :exec variables))
              (modes (mbe :logic (mode-set-fix modes)
                          :exec modes)))
             (cons :vartable-modes (list (cons 'variables variables)
                                         (cons 'modes modes))))))
