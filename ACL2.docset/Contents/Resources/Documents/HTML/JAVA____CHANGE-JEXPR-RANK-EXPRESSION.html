Modifying constructor for jexpr-rank-expression structures.

Syntax
(change-jexpr-rank-expression x 
                              )

This is an often useful alternative to make-jexpr-rank-expression.

We construct a new jexpr-rank-expression structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
