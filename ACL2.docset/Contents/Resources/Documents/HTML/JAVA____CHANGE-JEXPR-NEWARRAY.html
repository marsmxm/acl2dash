Modifying constructor for jexpr-newarray structures.

Syntax
(change-jexpr-newarray x 
                       [:type <type>] 
                       [:size <size>])

This is an often useful alternative to make-jexpr-newarray.

We construct a new jexpr-newarray structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-jexpr-newarray

(defmacro change-jexpr-newarray (x &rest args)
          (std::change-aggregate 'jexpr-newarray
                                 x args
                                 '((:type . jexpr-newarray->type)
                                   (:size . jexpr-newarray->size))
                                 'change-jexpr-newarray
                                 'nil))
