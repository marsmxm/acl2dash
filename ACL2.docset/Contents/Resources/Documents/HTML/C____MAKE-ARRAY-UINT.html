Basic constructor macro for array-uint structures.

Syntax
(make-array-uint [:get <get>])

This is the usual way to construct array-uint structures. It simply conses together a structure with the specified fields.

This macro generates a new array-uint structure from scratch. See also change-array-uint, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by fty::defprod.

Macro: make-array-uint

(defmacro make-array-uint (&rest args)
