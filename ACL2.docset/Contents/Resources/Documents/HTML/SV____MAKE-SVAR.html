Basic constructor macro for svar structures.

Syntax
(make-svar [:name <name>] 
           [:delay <delay>] 
           [:nonblocking <nonblocking>] 
           [:override-test <override-test>] 
           [:override-val <override-val>])

This is the usual way to construct svar structures. It simply conses together a structure with the specified fields.

This macro generates a new svar structure from scratch. See also change-svar, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by defprod.

Macro: make-svar

(defmacro make-svar (&rest args)
          (std::make-aggregate 'svar
                               args
                               '((:name)
                                 (:delay . 0)
                                 (:nonblocking)
                                 (:override-test)
                                 (:override-val))
                               'make-svar
                               nil))

(defun
 svar
 (name delay
       nonblocking override-test override-val)
 (declare (xargs :guard (and (natp delay)
                             (booleanp nonblocking)
                             (booleanp override-test)
                             (booleanp override-val))))
 (declare (xargs :guard t))
 (let
  ((__function__ 'svar))
  (declare (ignorable __function__))
  (b*
   ((delay (mbe :logic (nfix delay) :exec delay))
    (nonblocking (mbe :logic (bool-fix nonblocking)
                      :exec nonblocking))
    (override-test (mbe :logic (bool-fix override-test)
                        :exec override-test))
    (override-val (mbe :logic (bool-fix override-val)
                       :exec override-val)))
   (if
    (and (or (stringp name)
             (and (symbolp name)
                  (not (booleanp name))))
         (not nonblocking)
         (not override-test)
         (not override-val)
         (eql delay 0))
    name
    (hons :var (hons name
                     (if (or nonblocking override-test override-val)
                         (acl2::loglist* (bool->bit override-test)
                                         (bool->bit override-val)
                                         (bool->bit nonblocking)
                                         (lognot delay))
                         delay)))))))
