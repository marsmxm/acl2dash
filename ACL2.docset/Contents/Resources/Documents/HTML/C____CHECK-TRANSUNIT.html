Check a translation unit.

Signature
(check-transunit tunit)  wf
Arguments
tunit Guard (transunitp tunit).
Returns
wf Type (wellformed-resultp wf).

Starting from the initial (empty) function table, we check all the external declarations, threading the function table through, and discarding the final one (it served its pupose).

Definitions and Theorems

Function: check-transunit

(defun
     check-transunit (tunit)
     (declare (xargs :guard (transunitp tunit)))
     (let ((__function__ 'check-transunit))
          (declare (ignorable __function__))
          (b* (((transunit tunit) tunit)
               (funtab (fun-table-init))
               (funtab (check-ext-declon-list tunit.declons funtab))
               ((when (errorp funtab))
                (error (list :transunit-error funtab))))
              :wellformed)))

Theorem: wellformed-resultp-of-check-transunit

(defthm wellformed-resultp-of-check-transunit
        (b* ((wf (check-transunit tunit)))
