Modifying constructor for lhs-override structures.

Syntax
(change-lhs-override x 
                     [:lhs <lhs>] 
                     [:test <test>] 
                     [:val <val>])

This is an often useful alternative to make-lhs-override.

We construct a new lhs-override structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-lhs-override

(defmacro change-lhs-override (x &rest args)
          (std::change-aggregate 'lhs-override
                                 x args
                                 '((:lhs . lhs-override->lhs)
                                   (:test . lhs-override->test)
                                   (:val . lhs-override->val))
                                 'change-lhs-override
                                 'remake-lhs-override))
