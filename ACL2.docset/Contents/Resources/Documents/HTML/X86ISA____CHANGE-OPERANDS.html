Modifying constructor for operands structures.

Syntax
(change-operands x 
                 [:op1 <op1>] 
                 [:op2 <op2>] 
                 [:op3 <op3>] 
                 [:op4 <op4>])

This is an often useful alternative to make-operands.

We construct a new operands structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-operands

(defmacro change-operands (x &rest args)
          (std::change-aggregate 'operands
                                 x args
                                 '((:op1 . operands->op1)
                                   (:op2 . operands->op2)
                                   (:op3 . operands->op3)
                                   (:op4 . operands->op4))
                                 'change-operands
                                 'remake-operands))
