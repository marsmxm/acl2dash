Modifying constructor for acid4 structures.

Syntax
(CHANGE-acid4 x 
              [:id <id>])

This is an often useful alternative to MAKE-acid4.

We construct a new acid4 structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-acid4

(defmacro
