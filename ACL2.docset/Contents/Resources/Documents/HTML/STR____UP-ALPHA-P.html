Determine if a character is an upper-case letter (A-Z).

ACL2 has a built-in alternative to this function, common-lisp::upper-case-p, but it is irritating to use because it has standard-char-p guards. In contrast, up-alpha-p works on arbitrary characters.

Function: up-alpha-p$inline

(defun up-alpha-p$inline (x)
       (declare (type character x))
       (let ((acl2::__function__ 'up-alpha-p))
            (declare (ignorable acl2::__function__))
            (b* (((the (unsigned-byte 8) code)
                  (char-code x)))
                (and (<= (big-a) code)
                     (<= code (big-z))))))
