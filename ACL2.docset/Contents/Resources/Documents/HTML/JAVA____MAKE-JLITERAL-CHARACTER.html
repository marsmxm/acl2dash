Basic constructor macro for jliteral-character structures.

Syntax
(make-jliteral-character [:value <value>])

This is the usual way to construct jliteral-character structures. It simply conses together a structure with the specified fields.

This macro generates a new jliteral-character structure from scratch. See also change-jliteral-character, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by fty::defprod.

(defmacro make-jliteral-character (&rest args)

(defun jliteral-character (value)
       (declare (xargs :guard (characterp value)))
       (declare (xargs :guard t))
       (let ((__function__ 'jliteral-character))
            (declare (ignorable __function__))
            (b* ((value (mbe :logic (acl2::char-fix value)
                             :exec value)))
                (cons :character (list value)))))
