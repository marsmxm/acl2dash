Modifying constructor for maybe-type-product-some structures.

Syntax
(change-maybe-type-product-some x 
                                [:val <val>])

This is an often useful alternative to make-maybe-type-product-some.

We construct a new maybe-type-product-some structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
