Modifying constructor for nat-option-result-err structures.

Syntax
(change-nat-option-result-err x 
                              [:get <get>])

This is an often useful alternative to make-nat-option-result-err.

We construct a new nat-option-result-err structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
