(4vec-driverlist-fix x) is a usual fty list fixing function.

Signature
(4vec-driverlist-fix x) → fty::newx
Arguments
x — Guard (4vec-driverlist-p x).
Returns
fty::newx — Type (4vec-driverlist-p fty::newx).

In the logic, we apply 4vec-driver-fix to each member of the x. In the execution, none of that is actually necessary and this is just an inlined identity function.
