Modifying constructor for expr-grade-relational structures.

Syntax
(change-expr-grade-relational x 
                              )

This is an often useful alternative to make-expr-grade-relational.

We construct a new expr-grade-relational structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
