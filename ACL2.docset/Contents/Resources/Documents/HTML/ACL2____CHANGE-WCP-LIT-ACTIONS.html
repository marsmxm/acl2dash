A copying macro that lets you create new wcp-lit-actions-p structures, based on existing structures.

Syntax:

(change-wcp-lit-actions x 
                        [:witnesses <witnesses>] 
                        [:examples <examples>])

This is a sometimes useful alternative to make-wcp-lit-actions. It constructs a new wcp-lit-actions-p structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by std::defaggregate.

Macro: change-wcp-lit-actions

(defmacro
  change-wcp-lit-actions (x &rest args)
  (std::change-aggregate 'wcp-lit-actions
                         x args
                         '((:witnesses . wcp-lit-actions->witnesses)
                           (:examples . wcp-lit-actions->examples))
                         'change-wcp-lit-actions
                         'nil))
