Modifying constructor for obj-declor-ident structures.

Syntax
(change-obj-declor-ident x 
                         [:get <get>])

This is an often useful alternative to make-obj-declor-ident.

We construct a new obj-declor-ident structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-obj-declor-ident

(defmacro
     change-obj-declor-ident (x &rest args)
     (std::change-aggregate 'obj-declor-ident
                            x args '((:get . obj-declor-ident->get))
                            'change-obj-declor-ident
                            'nil))
