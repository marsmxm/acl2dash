Check if two lists of variables are related by variable renaming.

Signature
(var-list-renamevar old new ren) → _
Arguments
old — Guard (identifier-listp old).
new — Guard (identifier-listp new).
ren — Guard (renamingp ren).
Returns
_ — Type (resulterr-optionp _).

The two lists must have the same length, and corresponding elements must be related.

Definitions and Theorems

Function: var-list-renamevar

(defun
 var-list-renamevar (old new ren)
 (declare (xargs :guard (and (identifier-listp old)
                             (identifier-listp new)
                             (renamingp ren))))
 (let
  ((__function__ 'var-list-renamevar))
  (declare (ignorable __function__))
  (b*
   (((when (endp old))
     (if
        (endp new)
        nil
        (err (list :mismatch-extra-new (identifier-list-fix new)))))
    ((when (endp new))
     (err (list :mismatch-extra-old (identifier-list-fix old))))
    ((ok &)
     (var-renamevar (car old)
                    (car new)
                    ren)))
   (var-list-renamevar (cdr old)
                       (cdr new)
                       ren))))
