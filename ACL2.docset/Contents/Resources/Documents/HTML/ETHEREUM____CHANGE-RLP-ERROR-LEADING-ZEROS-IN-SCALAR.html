Modifying constructor for rlp-error-leading-zeros-in-scalar structures.

Syntax
(change-rlp-error-leading-zeros-in-scalar x 
                                          [:bytes <bytes>])

This is an often useful alternative to make-rlp-error-leading-zeros-in-scalar.

We construct a new rlp-error-leading-zeros-in-scalar structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
