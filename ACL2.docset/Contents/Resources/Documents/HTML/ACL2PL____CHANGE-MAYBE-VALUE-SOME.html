Modifying constructor for maybe-value-some structures.

Syntax
(change-maybe-value-some x 
                         [:val <val>])

This is an often useful alternative to make-maybe-value-some.

We construct a new maybe-value-some structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-maybe-value-some

(defmacro
     change-maybe-value-some (x &rest args)
     (std::change-aggregate 'maybe-value-some
                            x args '((:val . maybe-value-some->val))
                            'change-maybe-value-some
                            'nil))
