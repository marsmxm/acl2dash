Modifying constructor for jparam structures.

Syntax
(change-jparam x 
               [:final? <final?>] 
               [:type <type>] 
               [:name <name>])

This is an often useful alternative to make-jparam.

We construct a new jparam structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-jparam

(defmacro change-jparam (x &rest args)
          (std::change-aggregate 'jparam
                                 x args
                                 '((:final? . jparam->final?)
                                   (:type . jparam->type)
                                   (:name . jparam->name))
                                 'change-jparam
                                 'nil))
