Basic constructor macro for funtype structures.

Syntax
(make-funtype [:in <in>] 
              [:out <out>])

This is the usual way to construct funtype structures. It simply conses together a structure with the specified fields.

This macro generates a new funtype structure from scratch. See also change-funtype, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by fty::defprod.

Macro: make-funtype

(defmacro make-funtype (&rest args)
          (std::make-aggregate 'funtype
                               args '((:in) (:out))
                               'make-funtype
                               nil))

(defun

    (declare (xargs :guard (and (natp in) (natp out))))
    (declare (xargs :guard t))
    (let ((__function__ 'funtype))
         (declare (ignorable __function__))
