Basic constructor macro for array-type-class structures.

Syntax
(make-array-type-class [:element <element>] 
                       [:dimensions <dimensions>])

This is the usual way to construct array-type-class structures. It simply conses together a structure with the specified fields.

This macro generates a new array-type-class structure from scratch. See also change-array-type-class, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by fty::defprod.

(defun array-type-class (element dimensions)

                                   (posp dimensions))))
