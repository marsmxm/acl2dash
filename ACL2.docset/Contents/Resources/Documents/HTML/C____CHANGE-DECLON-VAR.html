Modifying constructor for declon-var structures.

Syntax
(change-declon-var x 
                   [:type <type>] 
                   [:declor <declor>] 
                   [:init <init>])

This is an often useful alternative to make-declon-var.

We construct a new declon-var structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-declon-var

(defmacro change-declon-var (x &rest args)
          (std::change-aggregate 'declon-var
                                 x args
                                 '((:type . declon-var->type)
                                   (:declor . declon-var->declor)
                                   (:init . declon-var->init))
                                 'change-declon-var
                                 'nil))
