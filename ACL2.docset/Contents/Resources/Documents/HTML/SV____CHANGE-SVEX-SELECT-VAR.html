Modifying constructor for svex-select-var structures.

Syntax
(change-svex-select-var x 
                        [:name <name>] 
                        [:width <width>])

This is an often useful alternative to make-svex-select-var.

We construct a new svex-select-var structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-svex-select-var

(defmacro
     change-svex-select-var (x &rest args)
     (std::change-aggregate 'svex-select-var
                            x args
                            '((:name . svex-select-var->name)
                              (:width . svex-select-var->width))
                            'change-svex-select-var
                            'remake-svex-select-var))
