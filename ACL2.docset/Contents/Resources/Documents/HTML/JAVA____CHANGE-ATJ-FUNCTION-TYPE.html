Modifying constructor for atj-function-type structures.

Syntax
(change-atj-function-type x 
                          [:inputs <inputs>] 
                          [:outputs <outputs>] 
                          [:arrays <arrays>])

This is an often useful alternative to make-atj-function-type.

We construct a new atj-function-type structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-atj-function-type

(defmacro
    change-atj-function-type (x &rest args)
    (std::change-aggregate 'atj-function-type
                           x args
                           '((:inputs . atj-function-type->inputs)
                             (:outputs . atj-function-type->outputs)
                             (:arrays . atj-function-type->arrays))
                           'change-atj-function-type
                           'nil))
