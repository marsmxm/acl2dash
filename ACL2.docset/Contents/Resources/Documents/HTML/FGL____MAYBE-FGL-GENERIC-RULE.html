Option type; fgl-generic-rule or nil.

This is an option type introduced by fty::defoption. Note that defoption is just a wrapper for defflexsum, so there are :none and :some member types, a case macro, and so forth.

Member Types
:none maybe-fgl-generic-rule-none
Represents that no maybe-fgl-generic-rule is available, i.e., Nothing or None.
:some maybe-fgl-generic-rule-some
An available maybe-fgl-generic-rule, i.e., Just val or Some val.
