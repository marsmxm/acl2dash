Modifying constructor for vl-keyguts structures.

Syntax
(change-vl-keyguts x 
                   [:type <type>])

This is an often useful alternative to make-vl-keyguts.

We construct a new vl-keyguts structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.
