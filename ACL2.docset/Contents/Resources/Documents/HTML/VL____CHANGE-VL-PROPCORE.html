Modifying constructor for vl-propcore structures.

Syntax
(change-vl-propcore x 
                    [:guts <guts>])

This is an often useful alternative to make-vl-propcore.

We construct a new vl-propcore structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-propcore

(defmacro
     change-vl-propcore (x &rest args)
     (std::change-aggregate 'vl-propcore
                            x args '((:guts . vl-propcore->guts))
                            'change-vl-propcore
                            'nil))
