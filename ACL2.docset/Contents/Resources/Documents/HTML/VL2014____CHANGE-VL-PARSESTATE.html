Modifying constructor for vl-parsestate structures.

Syntax
(change-vl-parsestate x 
                      [:usertypes <usertypes>] 
                      [:warnings <warnings>])

This is an often useful alternative to make-vl-parsestate.

We construct a new vl-parsestate structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-parsestate

(defmacro
    change-vl-parsestate (x &rest args)
    (std::change-aggregate 'vl-parsestate
                           x args
                           '((:usertypes . vl-parsestate->usertypes)
                             (:warnings . vl-parsestate->warnings))
                           'change-vl-parsestate
                           'remake-vl-parsestate))
