Modifying constructor for scratchobj-cinst structures.

Syntax
(change-scratchobj-cinst x 
                         [:val <val>])

This is an often useful alternative to make-scratchobj-cinst.

We construct a new scratchobj-cinst structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.
