Lift atj-type-meet to lists.

Signature
(atj-type-list-meet x y) → glb
Arguments
x — Guard (atj-type-listp x).
y — Guard (atj-type-listp y).
Returns
glb — Type (atj-maybe-type-listp glb).

This is done element-wise. When the shorter list is exhausted, we return nil.

We show that this agrees with atj-type-bottom-list-meet over lists of ATJ types. Note that atj-type-bottom-list-meet has been defined just to show the semilattice properties, but we always want to use, as arguments, lists of ATJ types without nils of the same length (so we add a length equality requirement to the guard).

Definitions and Theorems

Function: atj-type-list-meet

(defun atj-type-list-meet (x y)
       (declare (xargs :guard (and (atj-type-listp x)
                                   (atj-type-listp y))))
       (declare (xargs :guard (= (len x) (len y))))
       (let ((__function__ 'atj-type-list-meet))
            (declare (ignorable __function__))
            (cond ((endp x) nil)
                  ((endp y) nil)
                  (t (cons (atj-type-meet (car x) (car y))
                           (atj-type-list-meet (cdr x)
                                               (cdr y)))))))

        (implies (and (atj-type-listp x)

                 (equal (atj-type-list-meet x y)
