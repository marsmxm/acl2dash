(hyp-tuplelist-p x) recognizes lists where every element satisfies hyp-tuple-p.

Signature
(hyp-tuplelist-p x)  std::bool

This is an ordinary std::deflist. It is "loose" in that it does not care whether x is nil-terminated.

Definitions and Theorems

Function: hyp-tuplelist-p

(defun hyp-tuplelist-p (x)
       (declare (xargs :guard t))
       (let ((__function__ 'hyp-tuplelist-p))
            (declare (ignorable __function__))
            (if (consp x)
                (and (hyp-tuple-p (car x))
                     (hyp-tuplelist-p (cdr x)))
                t)))
