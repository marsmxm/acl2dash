Modifying constructor for binary-op-or structures.

Syntax
(change-binary-op-or x 
                     )

This is an often useful alternative to make-binary-op-or.

We construct a new binary-op-or structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-binary-op-or

(defmacro change-binary-op-or (x &rest args)
          (std::change-aggregate 'binary-op-or
