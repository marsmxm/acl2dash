Modifying constructor for funcall-result-err structures.

Syntax
(change-funcall-result-err x 
                           [:get <get>])

This is an often useful alternative to make-funcall-result-err.

We construct a new funcall-result-err structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-funcall-result-err

(defmacro
     change-funcall-result-err (x &rest args)

                            'nil))
