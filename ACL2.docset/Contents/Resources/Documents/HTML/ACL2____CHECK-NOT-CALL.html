Check if a term is a call of not.

Signature
(check-not-call term) → (mv yes/no arg)
Arguments
term — Guard (pseudo-termp term).
Returns
yes/no — Type (booleanp yes/no).
arg — Type (pseudo-termp arg), given the guard.

If it is, the first result is t and the other result is the argument. Otherwise, all the results are nil.

Function: check-not-call

(defun check-not-call (term)
       (declare (xargs :guard (pseudo-termp term)))
       (let ((__function__ 'check-not-call))
            (declare (ignorable __function__))
            (case-match term (('not arg) (mv t arg))
                        (& (mv nil nil)))))
