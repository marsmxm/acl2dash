Modifying constructor for character-list-result-ok structures.

Syntax
(change-character-list-result-ok x 
                                 [:get <get>])

This is an often useful alternative to make-character-list-result-ok.

We construct a new character-list-result-ok structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
