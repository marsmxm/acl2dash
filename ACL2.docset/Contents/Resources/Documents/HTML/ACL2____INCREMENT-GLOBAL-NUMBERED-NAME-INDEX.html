Increment by 1 the global index for numbered names.

This macro generates an event to increment the index by 1.

Macro: increment-global-numbered-name-index

(defmacro increment-global-numbered-name-index nil
          '(table global-numbered-name-index 'index
                  (1+ (get-global-numbered-name-index world))))
