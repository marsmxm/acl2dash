Raw constructor for vcd-vector-p structures.

Syntax:

(vcd-vector bits idcode)

This is the lowest-level constructor for vcd-vector-p structures. It simply conses together a structure with the specified fields.

Note: It's generally better to use macros like make-vcd-vector or change-vcd-vector instead. These macros lead to more readable and robust code, because you don't have to remember the order of the fields.

The vcd-vector-p structures we create here are just constructed with ordinary cons. If you want to create honsed structures, see honsed-vcd-vector instead.

Definition

This is an ordinary constructor function introduced by defaggregate.

Function: vcd-vector

(defun vcd-vector (bits idcode)
       (declare (xargs :guard (and (stringp idcode))))
       (cons :vcd-vector (cons bits idcode)))
