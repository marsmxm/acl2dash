Modifying constructor for string-element-escape structures.

Syntax
(change-string-element-escape x 
                              [:get <get>])

This is an often useful alternative to make-string-element-escape.

We construct a new string-element-escape structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-string-element-escape

(defmacro
     change-string-element-escape
     (x &rest args)
     (std::change-aggregate 'string-element-escape
                            x args
                            '((:get . string-element-escape->get))
                            'change-string-element-escape
                            'nil))
