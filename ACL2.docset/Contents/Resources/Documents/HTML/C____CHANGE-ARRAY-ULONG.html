Modifying constructor for array-ulong structures.

Syntax
(change-array-ulong x 
                    [:get <get>])

This is an often useful alternative to make-array-ulong.

We construct a new array-ulong structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-array-ulong

(defmacro change-array-ulong (x &rest args)
          (std::change-aggregate 'array-ulong
                                 x args '((:get . array-ulong->get))
                                 'change-array-ulong
                                 'nil))
