Basic constructor macro for vl-typedef structures.

Syntax
(make-vl-typedef [:name <name>] 
                 [:type <type>] 
                 [:minloc <minloc>] 
                 [:maxloc <maxloc>] 
                 [:atts <atts>] 
                 [:warnings <warnings>] 
                 [:comments <comments>])

This is the usual way to construct vl-typedef structures. It simply conses together a structure with the specified fields.

This macro generates a new vl-typedef structure from scratch. See also change-vl-typedef, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by defprod.

Macro: make-vl-typedef

(defmacro make-vl-typedef (&rest args)
          (std::make-aggregate 'vl-typedef
                               args
                               '((:name)
                                 (:type)
                                 (:minloc)
                                 (:maxloc)
                                 (:atts)
                                 (:warnings)
                                 (:comments))
                               'make-vl-typedef
                               nil))

Function: vl-typedef

(defun
     vl-typedef
     (name type
           minloc maxloc atts warnings comments)
     (declare (xargs :guard (and (stringp name)
                                 (vl-datatype-p type)
                                 (vl-location-p minloc)
                                 (vl-location-p maxloc)
                                 (vl-atts-p atts)
                                 (vl-warninglist-p warnings)
                                 (vl-commentmap-p comments))))
     (declare (xargs :guard t))
     (let ((__function__ 'vl-typedef))
          (declare (ignorable __function__))
          (b* ((name (mbe :logic (str-fix name) :exec name))
               (type (mbe :logic (vl-datatype-fix type)
                          :exec type))
               (minloc (mbe :logic (vl-location-fix minloc)
                            :exec minloc))
               (maxloc (mbe :logic (vl-location-fix maxloc)
                            :exec maxloc))
               (atts (mbe :logic (vl-atts-fix atts)
                          :exec atts))
               (warnings (mbe :logic (vl-warninglist-fix warnings)
                              :exec warnings))
               (comments (mbe :logic (vl-commentmap-fix comments)
                              :exec comments)))
              (cons :vl-typedef (list (cons 'name name)
                                      (cons 'type type)
                                      (cons 'minloc minloc)
                                      (cons 'maxloc maxloc)
                                      (cons 'atts atts)
                                      (cons 'warnings warnings)
                                      (cons 'comments comments))))))
