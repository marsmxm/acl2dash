(ecutname-list-p x) recognizes lists where every element satisfies ecutnames-p.

Signature
(ecutname-list-p x)  std::bool

This is an ordinary std::deflist. It is "loose" in that it does not care whether x is nil-terminated.

(defun ecutname-list-p (x)
