Concatenate two reversed elabtraversals to get a reversed traversal that (when followed reversed) to the same place as following y reversed, then x reversed.

Returns
xy — Type (vl-elabtraversal-p xy).

       (let ((__function__ 'vl-elabpaths-append))
            (declare (ignorable __function__))
            (cond ((atom x) (vl-elabtraversal-fix y))
                  ((vl-elabinstruction-case (car x) :root)

                         (vl-elabtraversal-fix y)))
                  (t (cons (vl-elabinstruction-fix (car x))
                           (vl-elabpaths-append (cdr x) y))))))
