Modifying constructor for aexp-add structures.

Syntax
(change-aexp-add x 
                 [:left <left>] 
                 [:right <right>])

This is an often useful alternative to make-aexp-add.

We construct a new aexp-add structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-aexp-add

(defmacro change-aexp-add (x &rest args)
          (std::change-aggregate 'aexp-add
                                 x args
                                 '((:left . aexp-add->left)
                                   (:right . aexp-add->right))
                                 'change-aexp-add
                                 'nil))
