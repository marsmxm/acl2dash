Modifying constructor for type-sequence structures.

Syntax
(change-type-sequence x 
                      [:element <element>])

This is an often useful alternative to make-type-sequence.

We construct a new type-sequence structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-type-sequence

(defmacro
     change-type-sequence (x &rest args)
     (std::change-aggregate 'type-sequence
                            x args
                            '((:element . type-sequence->element))
                            'change-type-sequence
                            'nil))
