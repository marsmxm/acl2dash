Modifying constructor for g-concrete structures.

Syntax
(change-g-concrete x 
                   [:val <val>])

This is an often useful alternative to make-g-concrete.

We construct a new g-concrete structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-g-concrete

(defmacro change-g-concrete (x &rest args)
          (std::change-aggregate 'g-concrete
                                 x args '((:val . g-concrete->val))
                                 'change-g-concrete
                                 'nil))
