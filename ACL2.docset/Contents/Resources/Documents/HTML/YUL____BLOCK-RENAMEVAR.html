Check if two blocks are related by variable renaming.

Signature
(block-renamevar old new ren) → _
Arguments
old — Guard (blockp old).
new — Guard (blockp new).
ren — Guard (renamingp ren).
Returns
_ — Type (resulterr-optionp _).

We process the list of statements, discarding the final renaming list, because the scope of a block ends at the end of the block.
