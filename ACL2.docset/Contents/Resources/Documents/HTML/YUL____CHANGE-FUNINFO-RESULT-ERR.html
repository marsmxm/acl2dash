Modifying constructor for funinfo-result-err structures.

Syntax
(change-funinfo-result-err x 
                           [:get <get>])

This is an often useful alternative to make-funinfo-result-err.

We construct a new funinfo-result-err structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
