Modifying constructor for iconst-tysuffix-long structures.

Syntax
(change-iconst-tysuffix-long x 
                             )

This is an often useful alternative to make-iconst-tysuffix-long.

We construct a new iconst-tysuffix-long structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
