Fixtype of rules and nil.

This is an option type introduced by fty::defoption. Note that defoption is just a wrapper for fty::defflexsum, so there are :none and :some member types, a case macro, and so forth.

Member Types
:none → maybe-rule-none
Represents that no maybe-rule is available, i.e., Nothing or None.
:some → maybe-rule-some
An available maybe-rule, i.e., Just val or Some val.
