Fixtype of functions and nil.

This is an option type introduced by fty::defoption. Note that defoption is just a wrapper for fty::defflexsum, so there are :none and :some member types, a case macro, and so forth.

Member Types
:none → maybe-function-none
Represents that no maybe-function is available, i.e., Nothing or None.
:some → maybe-function-some
An available maybe-function, i.e., Just val or Some val.
