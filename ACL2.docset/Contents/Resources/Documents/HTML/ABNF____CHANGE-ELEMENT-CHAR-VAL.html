Modifying constructor for element-char-val structures.

Syntax
(change-element-char-val x 
                         [:get <get>])

This is an often useful alternative to make-element-char-val.

We construct a new element-char-val structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-element-char-val

(defmacro
     change-element-char-val (x &rest args)
     (std::change-aggregate 'element-char-val
                            x args '((:get . element-char-val->get))
                            'change-element-char-val
                            'nil))
