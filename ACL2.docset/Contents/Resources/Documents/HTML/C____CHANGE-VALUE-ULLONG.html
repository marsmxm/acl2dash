Modifying constructor for value-ullong structures.

Syntax
(change-value-ullong x 
                     [:get <get>])

This is an often useful alternative to make-value-ullong.

We construct a new value-ullong structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-value-ullong

(defmacro
     change-value-ullong (x &rest args)
     (std::change-aggregate 'value-ullong
                            x args '((:get . value-ullong->get))
                            'change-value-ullong
                            'nil))
