Modifying constructor for value-uchar structures.

Syntax
(change-value-uchar x 
                    [:get <get>])

This is an often useful alternative to make-value-uchar.

We construct a new value-uchar structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-value-uchar

(defmacro change-value-uchar (x &rest args)
          (std::change-aggregate 'value-uchar
                                 x args '((:get . value-uchar->get))
                                 'change-value-uchar
                                 'nil))
