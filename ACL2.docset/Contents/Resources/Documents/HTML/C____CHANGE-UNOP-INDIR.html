Modifying constructor for unop-indir structures.

Syntax
(change-unop-indir x 
                   )

This is an often useful alternative to make-unop-indir.

We construct a new unop-indir structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-unop-indir

(defmacro change-unop-indir (x &rest args)
          (std::change-aggregate 'unop-indir
                                 x args 'nil
                                 'change-unop-indir
                                 'nil))
