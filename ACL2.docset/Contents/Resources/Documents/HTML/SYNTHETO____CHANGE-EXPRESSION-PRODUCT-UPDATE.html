Modifying constructor for expression-product-update structures.

Syntax
(change-expression-product-update x 
                                  [:type <type>] 
                                  [:target <target>] 
                                  [:fields <fields>])

This is an often useful alternative to make-expression-product-update.

We construct a new expression-product-update structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-expression-product-update

(defmacro change-expression-product-update
          (x &rest args)
          (std::change-aggregate
               'expression-product-update
               x args
               '((:type . expression-product-update->type)
                 (:target . expression-product-update->target)
                 (:fields . expression-product-update->fields))
               'change-expression-product-update
               'nil))
