Basic constructor macro for vl-sequence structures.

Syntax
(make-vl-sequence [:name <name>] 
                  [:ports <ports>] 
                  [:decls <decls>] 
                  [:expr <expr>] 
                  [:loc <loc>])

This is the usual way to construct vl-sequence structures. It simply conses together a structure with the specified fields.

This macro generates a new vl-sequence structure from scratch. See also change-vl-sequence, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by defprod.

Function: vl-sequence

(defun vl-sequence (name ports decls expr loc)
       (declare (xargs :guard (and (stringp name)
                                   (vl-propportlist-p ports)
                                   (vl-vardecllist-p decls)
                                   (vl-propexpr-p expr)
                                   (vl-location-p loc))))
       (declare (xargs :guard t))
       (let ((__function__ 'vl-sequence))
            (declare (ignorable __function__))
            (b* ((name (mbe :logic (str-fix name) :exec name))
                 (ports (mbe :logic (vl-propportlist-fix ports)
                             :exec ports))
                 (decls (mbe :logic (vl-vardecllist-fix decls)
                             :exec decls))
                 (expr (mbe :logic (vl-propexpr-fix expr)
                            :exec expr))
                 (loc (mbe :logic (vl-location-fix loc)
                           :exec loc)))
                (cons :vl-sequence (list (cons 'name name)
                                         (cons 'ports ports)
                                         (cons 'decls decls)
                                         (cons 'expr expr)
                                         (cons 'loc loc))))))
