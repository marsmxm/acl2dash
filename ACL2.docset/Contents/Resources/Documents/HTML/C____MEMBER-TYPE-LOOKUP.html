Look up a member by name in a list of member types.

Signature
(member-type-lookup name members) → type
Arguments
name — Guard (identp name).
members — Guard (member-type-listp members).
Returns
type — Type (type-optionp type).

We search from the beginning and stop at the first hit; since the names are unique in well-formed lists, the exact search strategy makes no difference We return the type of the member if the search is successful.

Definitions and Theorems

Function: member-type-lookup

(defun member-type-lookup (name members)
       (declare (xargs :guard (and (identp name)
                                   (member-type-listp members))))
       (let ((__function__ 'member-type-lookup))
            (declare (ignorable __function__))
            (b* (((when (endp members)) nil)
                 ((when (equal (ident-fix name)
                               (member-type->name (car members))))
                  (member-type->type (car members))))
                (member-type-lookup name (cdr members)))))

Theorem: type-optionp-of-member-type-lookup

(defthm type-optionp-of-member-type-lookup
        (b* ((type (member-type-lookup name members)))
            (type-optionp type))
        :rule-classes :rewrite)
