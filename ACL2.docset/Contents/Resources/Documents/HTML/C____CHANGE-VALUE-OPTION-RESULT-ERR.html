Modifying constructor for value-option-result-err structures.

Syntax
(change-value-option-result-err x 
                                [:get <get>])

This is an often useful alternative to make-value-option-result-err.

We construct a new value-option-result-err structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
