(bitscan-rev src) returns the bit position of the most significant bit in src that is set, or 0 when src is zero (and hence has no such bit).

Signature
(bitscan-rev src)  position
Arguments
src Guard (natp src).
Returns
position Type (natp position).

Examples:

(bitscan-rev #b0001) = 0
(bitscan-rev #b0011) = 1
(bitscan-rev #b0101) = 2
(bitscan-rev #b1001) = 3

Function: bitscan-rev

(defun bitscan-rev (src)
       (declare (xargs :guard (natp src)))
       (let ((__function__ 'bitscan-rev))
            (declare (ignorable __function__))
            (if (zp src)
                0
                (let ((next (ash src -1)))
                     (if (= next 0)
                         0 (+ 1 (bitscan-rev next)))))))
