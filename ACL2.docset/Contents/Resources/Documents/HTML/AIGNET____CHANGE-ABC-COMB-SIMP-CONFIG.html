Modifying constructor for abc-comb-simp-config structures.

Syntax
(change-abc-comb-simp-config x 
                             [:script <script>] 
                             [:quiet <quiet>])

This is an often useful alternative to make-abc-comb-simp-config.

We construct a new abc-comb-simp-config structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
