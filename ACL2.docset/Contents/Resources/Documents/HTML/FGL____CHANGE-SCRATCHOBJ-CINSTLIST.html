Modifying constructor for scratchobj-cinstlist structures.

Syntax
(change-scratchobj-cinstlist x 
                             [:val <val>])

This is an often useful alternative to make-scratchobj-cinstlist.

We construct a new scratchobj-cinstlist structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.
