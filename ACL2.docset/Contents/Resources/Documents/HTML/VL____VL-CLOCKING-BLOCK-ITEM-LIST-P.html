(vl-clocking-block-item-list-p x) recognizes lists where every element satisfies vl-clocking-block-item-p.

Signature
(vl-clocking-block-item-list-p x)  *

This is an ordinary deflist. It is "loose" in that it does not care whether x is nil-terminated.
