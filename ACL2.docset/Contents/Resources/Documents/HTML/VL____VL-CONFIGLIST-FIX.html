(vl-configlist-fix x) is a usual fty list fixing function.

Signature
(vl-configlist-fix x) → fty::newx
Arguments
x — Guard (vl-configlist-p x).
Returns
fty::newx — Type (vl-configlist-p fty::newx).

In the logic, we apply vl-config-fix to each member of the x. In the execution, none of that is actually necessary and this is just an inlined identity function.
