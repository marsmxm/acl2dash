A copying macro that lets you create new glcp-obj-ctrex-p structures, based on existing structures.

Syntax:

(change-glcp-obj-ctrex x 
                       [:descrip <descrip>] 
                       [:genv <genv>] 
                       [:obj-alist <obj-alist>] 
                       [:dont-care-spec <dont-care-spec>])

This is a sometimes useful alternative to make-glcp-obj-ctrex. It constructs a new glcp-obj-ctrex-p structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by std::defaggregate.

Macro: change-glcp-obj-ctrex

(defmacro change-glcp-obj-ctrex (x &rest args)
          (std::change-aggregate
               'glcp-obj-ctrex
               x args
               '((:descrip . glcp-obj-ctrex->descrip)
                 (:genv . glcp-obj-ctrex->genv)
                 (:obj-alist . glcp-obj-ctrex->obj-alist)
                 (:dont-care-spec . glcp-obj-ctrex->dont-care-spec))
               'change-glcp-obj-ctrex
               'remake-glcp-obj-ctrex))
