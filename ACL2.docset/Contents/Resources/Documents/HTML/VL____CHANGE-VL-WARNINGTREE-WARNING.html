Modifying constructor for vl-warningtree-warning structures.

Syntax
(change-vl-warningtree-warning x 
                               [:warning <warning>])

This is an often useful alternative to make-vl-warningtree-warning.

We construct a new vl-warningtree-warning structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-warningtree-warning

(defmacro change-vl-warningtree-warning
          (x &rest args)
          (std::change-aggregate
               'vl-warningtree-warning
               x args
               '((:warning . vl-warningtree-warning->warning))
               'change-vl-warningtree-warning
               'nil))
