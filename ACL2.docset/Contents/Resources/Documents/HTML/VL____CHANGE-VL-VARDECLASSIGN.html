A copying macro that lets you create new vl-vardeclassign-p structures, based on existing structures.

Syntax:

(change-vl-vardeclassign x 
                         [:id <id>] 
                         [:dims <dims>] 
                         [:rhs <rhs>])

This is a sometimes useful alternative to make-vl-vardeclassign. It constructs a new vl-vardeclassign-p structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defaggregate.

Macro: change-vl-vardeclassign

(defmacro change-vl-vardeclassign (x &rest args)
          (std::change-aggregate 'vl-vardeclassign
                                 x args
                                 '((:id . vl-vardeclassign->id)
                                   (:dims . vl-vardeclassign->dims)
                                   (:rhs . vl-vardeclassign->rhs))
                                 'change-vl-vardeclassign
                                 'remake-vl-vardeclassign))
