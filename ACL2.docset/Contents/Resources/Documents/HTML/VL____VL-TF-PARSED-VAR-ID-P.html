Temporary structure used when parsing list_of_tf_variable_identifiers.

(vl-tf-parsed-var-id-p x) is a defaggregate of the following fields.

  • name Invariant (vl-idtoken-p name).
  • udims Invariant (vl-dimensionlist-p udims).
  • default Invariant (vl-maybe-expr-p default).
