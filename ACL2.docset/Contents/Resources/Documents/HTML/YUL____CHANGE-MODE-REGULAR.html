Modifying constructor for mode-regular structures.

Syntax
(change-mode-regular x 
                     )

This is an often useful alternative to make-mode-regular.

We construct a new mode-regular structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-mode-regular

(defmacro change-mode-regular (x &rest args)
          (std::change-aggregate 'mode-regular
                                 x args 'nil

                                 'nil))
