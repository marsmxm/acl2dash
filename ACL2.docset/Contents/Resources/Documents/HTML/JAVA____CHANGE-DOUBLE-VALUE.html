Modifying constructor for double-value structures.

Syntax
(change-double-value x 
                     [:double <double>])

This is an often useful alternative to make-double-value.

We construct a new double-value structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-double-value

(defmacro
     change-double-value (x &rest args)
     (std::change-aggregate 'double-value
                            x
                            args '((:double . double-value->double))
                            'change-double-value
                            'nil))
