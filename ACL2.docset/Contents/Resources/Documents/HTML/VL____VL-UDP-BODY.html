Raw constructor for vl-udp-body-p structures.

Syntax:

(vl-udp-body init table)

This is the lowest-level constructor for vl-udp-body-p structures. It simply conses together a structure with the specified fields.

Note: It's generally better to use macros like make-vl-udp-body or change-vl-udp-body instead. These macros lead to more readable and robust code, because you don't have to remember the order of the fields.

The vl-udp-body-p structures we create here are just constructed with ordinary cons. If you want to create honsed structures, see honsed-vl-udp-body instead.

Definition

This is an ordinary constructor function introduced by defaggregate.

Function: vl-udp-body

(defun vl-udp-body (init table)
       (declare (xargs :guard (and (vl-maybe-expr-p init)
                                   (vl-udptable-p table))))
       (cons init table))
