Modifying constructor for outcome-transformation-failure structures.

Syntax
(change-outcome-transformation-failure x 
                                       [:message <message>])

This is an often useful alternative to make-outcome-transformation-failure.

We construct a new outcome-transformation-failure structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

(defmacro change-outcome-transformation-failure
