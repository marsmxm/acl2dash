Modifying constructor for literal-character structures.

Syntax
(change-literal-character x 
                          [:value <value>])

This is an often useful alternative to make-literal-character.

We construct a new literal-character structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-literal-character

(defmacro
     change-literal-character (x &rest args)
     (std::change-aggregate 'literal-character
                            x args
                            '((:value . literal-character->value))
                            'change-literal-character
                            'nil))
