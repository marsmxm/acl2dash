Basic constructor macro for type-array structures.

Syntax
(make-type-array [:of <of>] 
                 [:size <size>])

This is the usual way to construct type-array structures. It simply conses together a structure with the specified fields.

This macro generates a new type-array structure from scratch. See also change-type-array, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by fty::defprod.
