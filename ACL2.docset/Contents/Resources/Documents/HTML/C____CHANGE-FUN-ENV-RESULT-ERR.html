Modifying constructor for fun-env-result-err structures.

Syntax
(change-fun-env-result-err x 
                           [:get <get>])

This is an often useful alternative to make-fun-env-result-err.

We construct a new fun-env-result-err structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
