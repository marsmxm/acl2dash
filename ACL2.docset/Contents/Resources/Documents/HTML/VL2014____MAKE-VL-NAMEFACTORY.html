Basic constructor macro for vl-namefactory structures.

Syntax
(make-vl-namefactory [:mod <mod>] 
                     [:namedb <namedb>])

This is the usual way to construct vl-namefactory structures. It simply conses together a structure with the specified fields.

This macro generates a new vl-namefactory structure from scratch. See also change-vl-namefactory, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by defprod.

Macro: make-vl-namefactory

(defmacro make-vl-namefactory (&rest args)
          (std::make-aggregate 'vl-namefactory
                               args '((:mod) (:namedb))
                               'make-vl-namefactory
                               nil))
