Turn a type name into a type.

Signature
(type-name-to-type tyname) → type
Arguments
tyname — Guard (tynamep tyname).
Returns
type — Type (typep type).

A type name denotes a type [C:6.7.7/2]. This ACL2 function returns the denoted type.

Definitions and Theorems

Function: type-name-to-type

(defun
 type-name-to-type (tyname)
 (declare (xargs :guard (tynamep tyname)))
 (let
  ((__function__ 'type-name-to-type))
  (declare (ignorable __function__))
  (b*
   ((tyspecseq (tyname->specs tyname))
    (type
     (tyspecseq-case
      tyspecseq
      :void (type-void)
      :char (type-char)
      :schar (type-schar)
      :sshort (type-sshort)
      :sint (type-sint)
      :slong (type-slong)
      :sllong (type-sllong)
      :uchar (type-uchar)
      :ushort (type-ushort)
      :uint (type-uint)
      :ulong (type-ulong)
      :ullong (type-ullong)
      :struct
      (prog2$
       (raise
        "Internal error: ~
                                              struct ~x0 not supported yet."
        tyspecseq.tag)
       (irr-type)))))
   (if (tyname->pointerp tyname)
       (type-pointer type)
       type))))
