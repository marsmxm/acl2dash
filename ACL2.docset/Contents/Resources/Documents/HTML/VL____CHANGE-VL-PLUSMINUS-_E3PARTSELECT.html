Modifying constructor for vl-partselect-plusminus structures.

Syntax
(change-vl-plusminus->partselect x 
                                 [:plusminus <plusminus>])

This is an often useful alternative to make-vl-plusminus->partselect.

We construct a new vl-partselect-plusminus structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-plusminus->partselect

(defmacro change-vl-plusminus->partselect
