A copying macro that lets you create new vl-udp-body-p structures, based on existing structures.

Syntax:

(change-vl-udp-body x 
                    [:init <init>] 
                    [:table <table>])

This is a sometimes useful alternative to make-vl-udp-body. It constructs a new vl-udp-body-p structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defaggregate.

Macro: change-vl-udp-body

(defmacro change-vl-udp-body (x &rest args)
          (std::change-aggregate 'vl-udp-body
                                 x args
                                 '((:init . vl-udp-body->init)
                                   (:table . vl-udp-body->table))
                                 'change-vl-udp-body
                                 'remake-vl-udp-body))
