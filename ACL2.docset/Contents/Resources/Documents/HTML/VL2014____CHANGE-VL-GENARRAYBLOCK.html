Modifying constructor for vl-genarrayblock structures.

Syntax
(change-vl-genarrayblock x 
                         [:index <index>] 
                         [:elems <elems>])

This is an often useful alternative to make-vl-genarrayblock.

We construct a new vl-genarrayblock structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-genarrayblock

(defmacro
     change-vl-genarrayblock (x &rest args)
     (std::change-aggregate 'vl-genarrayblock
                            x args
                            '((:index . vl-genarrayblock->index)
                              (:elems . vl-genarrayblock->elems))
                            'change-vl-genarrayblock
                            'nil))
