Signature
(atj-gen-values values jvar-value-base jvar-value-index) 
  → 
(mv block exprs new-jvar-value-index)
Arguments
values — Guard (true-listp values).
jvar-value-base — Guard (stringp jvar-value-base).
jvar-value-index — Guard (posp jvar-value-index).
Returns
block — Type (jblockp block).
exprs — Type (and (jexpr-listp exprs) (equal (len exprs) (len values))).
new-jvar-value-index — Type (posp new-jvar-value-index), given (posp jvar-value-index).
