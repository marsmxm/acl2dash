Fixtype of (ATC's model of) arrays of type signed long long.

This is a product type introduced by fty::defprod.

Additional Requirements

The following invariant is enforced on the fields:

(and (type-case elemtype :sllong) (consp elements))

Theorem: sllong-arrayp-alt-def

(defthm sllong-arrayp-alt-def
        (equal (sllong-arrayp x)
               (and (valuep x)
                    (value-case x :array)
                    (equal (value-array->elemtype x)
                           (type-sllong))
                    (sllong-listp (value-array->elements x)))))

Theorem: sllong-array->elements-alt-def

(defthm sllong-array->elements-alt-def
        (implies (sllong-arrayp array)
                 (equal (sllong-array->elements array)
                        (value-array->elements array))))
