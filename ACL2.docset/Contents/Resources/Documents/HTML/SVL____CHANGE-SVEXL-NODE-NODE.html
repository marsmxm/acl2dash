Modifying constructor for svexl-node-node structures.

Syntax
(change-svexl-node-node x 
                        [:node-id <node-id>])

This is an often useful alternative to make-svexl-node-node.

We construct a new svexl-node-node structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-svexl-node-node

(defmacro
     change-svexl-node-node (x &rest args)
