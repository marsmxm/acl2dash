Modifying constructor for value-sshort structures.

Syntax
(change-value-sshort x 
                     [:get <get>])

This is an often useful alternative to make-value-sshort.

We construct a new value-sshort structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-value-sshort

(defmacro
     change-value-sshort (x &rest args)
     (std::change-aggregate 'value-sshort
                            x args '((:get . value-sshort->get))
                            'change-value-sshort
                            'nil))
