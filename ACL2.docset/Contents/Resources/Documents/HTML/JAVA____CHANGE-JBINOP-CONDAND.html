Modifying constructor for jbinop-condand structures.

Syntax
(change-jbinop-condand x 
                       )

This is an often useful alternative to make-jbinop-condand.

We construct a new jbinop-condand structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
