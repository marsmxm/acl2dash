Basic constructor macro for sint structures.

Syntax
(make-sint [:get <get>])

This is the usual way to construct sint structures. It simply conses together a structure with the specified fields.

This macro generates a new sint structure from scratch. See also change-sint, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by fty::defprod.

Macro: make-sint

(defmacro make-sint (&rest args)
          (std::make-aggregate 'sint
                               args '((:get))
                               'make-sint
                               nil))

(defun sint (get)
       (declare (xargs :guard (sint-integerp get)))
       (declare (xargs :guard t))
       (let ((__function__ 'sint))
            (declare (ignorable __function__))
            (b* ((get (mbe :logic (sint-integer-fix get)
                           :exec get)))
                (cons :sint (list get)))))
