Coerces a oct-digit-char-p character into a number.

Signature
(oct-digit-char-value x) → val
Arguments
x — Guard (oct-digit-char-p x).
Returns
val — Type (natp val).

For instance, (oct-digit-char-value #\a) is 10. For any non-oct-digit-char-p, 0 is returned.
