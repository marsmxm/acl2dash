Basic constructor macro for integral-value-byte structures.

Syntax
(make-integral-value-byte [:get <get>])

This is the usual way to construct integral-value-byte structures. It simply conses together a structure with the specified fields.

This macro generates a new integral-value-byte structure from scratch. See also change-integral-value-byte, which can "change" an existing structure, instead.
