Modifying constructor for maybe-theorem-some structures.

Syntax
(change-maybe-theorem-some x 
                           [:val <val>])

This is an often useful alternative to make-maybe-theorem-some.

We construct a new maybe-theorem-some structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

This is an ordinary change- macro introduced by fty::defprod.
