Constructor macro for wcp-example-app-p structures.

Syntax:

(make-wcp-example-app [:instrule <instrule>] 
                      [:bindings <bindings>])

This is our preferred way to construct wcp-example-app-p structures. It simply conses together a structure with the specified fields.

This macro generates a new wcp-example-app-p structure from scratch. See also change-wcp-example-app, which can "change" an existing structure, instead.

The wcp-example-app-p structures we create here are just constructed with ordinary cons. If you want to create honsed structures, see make-honsed-wcp-example-app instead.

Definition

This is an ordinary make- macro introduced by std::defaggregate.

Macro: make-wcp-example-app

(defmacro make-wcp-example-app (&rest args)
          (std::make-aggregate 'wcp-example-app
                               args '((:instrule) (:bindings))
                               'make-wcp-example-app
                               nil))
