Representation for implicitly specified value parameter types.

This is a product type, introduced by deftagsum in support of vl-paramtype.

Fields
range — vl-maybe-range
The range for this parameter, if provided.
sign — vl-maybe-exprtype
The signedness for this parameter, if provided.
default — vl-maybe-expr
The default value for this parameter, if provided.
