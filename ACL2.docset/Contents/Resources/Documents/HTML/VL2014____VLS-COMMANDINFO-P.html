(vls-commandinfo-p x) is a defaggregate of the following fields.

  • fn — Name of the function that will execute this command. Should have the signature (fn [args] data) → string.
        Invariant (symbolp fn).
  • args — Names of the arguments.
        Invariant (symbol-listp args).
  • type — Content type produced by this command.
        Invariant (vls-commandtype-p type).
