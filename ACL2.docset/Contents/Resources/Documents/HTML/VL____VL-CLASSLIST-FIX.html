(vl-classlist-fix x) is a usual fty list fixing function.

Signature
(vl-classlist-fix x)  fty::newx
Arguments
x Guard (vl-classlist-p x).
Returns
fty::newx Type (vl-classlist-p fty::newx).

In the logic, we apply vl-class-fix to each member of the x. In the execution, none of that is actually necessary and this is just an inlined identity function.
