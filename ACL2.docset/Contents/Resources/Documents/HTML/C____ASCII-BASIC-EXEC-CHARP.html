Fixtype of the ACL2 ASCII characters that correspond to the members of the basic execution character set in C.

Signature
(ascii-basic-exec-charp x) → yes/no
Returns
yes/no — Type (booleanp yes/no).

These are the ones for the basic source character set, plus null character, alert, backspace, carriage return, and new line [C:5.2.1/2-3].

The ACL2 character with ASCII code 0 corresponds to the null character. The ACL2 character with ASCII code 7 corresponds to alert. The ACL2 character with ASCII code 8 corresponds to backspace. The ACL2 charater #\Return has ASCII code 13 and corresponds to carriage return. The ACL2 character #\Newline has ASCII code 10 and corresponds to new line.
