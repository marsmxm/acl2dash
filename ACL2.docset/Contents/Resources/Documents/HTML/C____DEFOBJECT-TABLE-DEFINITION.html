Definition of the table of shallowly embedded C external objects.

The keys are strings that are the symbol-names of symbols that represent the names of the objects. The name of each such symbol is a portable ASCII C identifier but this constraint is not enforced in the table's guard. The keys in the table are unique.

The values are the information about the objects See defobject-info.
