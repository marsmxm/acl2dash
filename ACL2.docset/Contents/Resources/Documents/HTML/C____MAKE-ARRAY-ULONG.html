Basic constructor macro for array-ulong structures.

Syntax
(make-array-ulong [:get <get>])

This is the usual way to construct array-ulong structures. It simply conses together a structure with the specified fields.

This macro generates a new array-ulong structure from scratch. See also change-array-ulong, which can "change" an existing structure, instead.

This is an ordinary make- macro introduced by fty::defprod.
