Semantics of characters in case-sensitive character value notations.

Signature
(nat-match-sensitive-char-p nat char) → yes/no
Arguments
nat — Guard (natp nat).
char — Guard (characterp char).
Returns
yes/no — Type (booleanp yes/no).

A natural number matches a character in a case-sensitive character value notation iff the natural number is the character's code.

Function: nat-match-sensitive-char-p

(defun nat-match-sensitive-char-p (nat char)
       (declare (xargs :guard (and (natp nat) (characterp char))))
       (b* ((nat (mbe :logic (nfix nat) :exec nat)))
           (equal nat (char-code char))))
