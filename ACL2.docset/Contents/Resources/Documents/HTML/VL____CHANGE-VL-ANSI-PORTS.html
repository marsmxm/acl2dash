Modifying constructor for vl-ansi-ports structures.

Syntax
(change-vl-ansi-ports x 
                      [:decls <decls>])

This is an often useful alternative to make-vl-ansi-ports.

We construct a new vl-ansi-ports structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-ansi-ports

(defmacro
     change-vl-ansi-ports (x &rest args)
     (std::change-aggregate 'vl-ansi-ports
                            x
                            args '((:decls . vl-ansi-ports->decls))
                            'change-vl-ansi-ports
                            'nil))
