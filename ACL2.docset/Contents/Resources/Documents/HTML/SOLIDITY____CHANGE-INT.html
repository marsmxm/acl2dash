Modifying constructor for int structures.

Syntax
(change-int x 
            [:size <size>] 
            [:value <value>])

This is an often useful alternative to make-int.

We construct a new int structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-int

(defmacro change-int (x &rest args)
          (std::change-aggregate 'int
                                 x args
                                 '((:size . int->size)
                                   (:value . int->value))
                                 'change-int
                                 'nil))
