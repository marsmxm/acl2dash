Generate a shallowly embedded ACL2 call of a Java primitive binary operation.

Signature
(atj-gen-shallow-jprim-binop-call 
     fn left-block right-block left-expr 
     right-expr src-types dst-types) 
 
  → 
(mv block expr)
Arguments
fn — Guard (atj-jprim-binop-fn-p fn).
left-block — Guard (jblockp left-block).
right-block — Guard (jblockp right-block).
left-expr — Guard (jexprp left-expr).
right-expr — Guard (jexprp right-expr).
src-types — Guard (atj-type-listp src-types).
dst-types — Guard (atj-type-listp dst-types).
Returns
block — Type (jblockp block).
expr — Type (jexprp expr).

This is called only if :guards is t. This is called after translating the operands to Java. The resulting blocks and expressions are passed as parameters here.

We generate a Java binary expression whose operator corresponds to the function.

Function: atj-gen-shallow-jprim-binop-call

(defun
 atj-gen-shallow-jprim-binop-call
 (fn left-block right-block left-expr
     right-expr src-types dst-types)
 (declare (xargs :guard (and (atj-jprim-binop-fn-p fn)
                             (jblockp left-block)
                             (jblockp right-block)
                             (jexprp left-expr)
                             (jexprp right-expr)
                             (atj-type-listp src-types)
                             (atj-type-listp dst-types))))
 (declare (xargs :guard (and (consp src-types)
                             (consp dst-types))))
 (let
  ((__function__ 'atj-gen-shallow-jprim-binop-call))
  (declare (ignorable __function__))
  (b*
   ((binop (atj-jprim-binop-fn-to-jbinop fn))
    (expr (jexpr-binary binop left-expr right-expr))
    (block (append (jblock-fix left-block)
                   (jblock-fix right-block))))
   (mv
      block
      (atj-adapt-expr-to-type expr (atj-type-list-to-type src-types)
                              (atj-type-list-to-type dst-types)
                              t)))))
