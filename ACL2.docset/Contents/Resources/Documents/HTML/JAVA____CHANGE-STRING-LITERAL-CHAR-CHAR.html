Modifying constructor for string-literal-char-char structures.

Syntax
(change-string-literal-char-char x 
                                 [:get <get>])

This is an often useful alternative to make-string-literal-char-char.

We construct a new string-literal-char-char structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-string-literal-char-char
