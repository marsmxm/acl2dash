Modifying constructor for jexpr-array structures.

Syntax
(change-jexpr-array x 
                    [:array <array>] 
                    [:index <index>])

This is an often useful alternative to make-jexpr-array.

We construct a new jexpr-array structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-jexpr-array

(defmacro change-jexpr-array (x &rest args)
          (std::change-aggregate 'jexpr-array
                                 x args
                                 '((:array . jexpr-array->array)
                                   (:index . jexpr-array->index))
                                 'change-jexpr-array
                                 'nil))
