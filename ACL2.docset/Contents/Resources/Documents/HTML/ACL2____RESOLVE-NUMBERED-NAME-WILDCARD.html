Resolve the wildcard in a numbered name (if any) to the largest index in use for the name's base.

Signature
(resolve-numbered-name-wildcard name wrld) → resolved-name
Arguments
name — Guard (symbolp name).
wrld — Guard (plist-worldp wrld).
Returns
resolved-name — Type (symbolp resolved-name).

If name is a numbered name with base base and the wildcard as index, return the numbered name with base base and index i, where i is the result of max-numbered-name-index-in-use. Otherwise, return name.
