Modifying constructor for expression-option-none structures.

Syntax
(change-expression-option-none x 
                               )

This is an often useful alternative to make-expression-option-none.

We construct a new expression-option-none structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-expression-option-none

(defmacro change-expression-option-none
          (x &rest args)
          (std::change-aggregate 'expression-option-none
                                 x args 'nil
                                 'change-expression-option-none
                                 'nil))
