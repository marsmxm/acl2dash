Modifying constructor for tag-info-union structures.

Syntax
(change-tag-info-union x 
                       )

This is an often useful alternative to make-tag-info-union.

We construct a new tag-info-union structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
