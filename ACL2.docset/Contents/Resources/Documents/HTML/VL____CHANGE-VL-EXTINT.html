Modifying constructor for vl-extint structures.

Syntax
(change-vl-extint x 
                  [:value <value>])

This is an often useful alternative to make-vl-extint.

We construct a new vl-extint structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-extint

(defmacro
     change-vl-extint (x &rest args)
     (std::change-aggregate 'vl-extint
                            x args '((:value . vl-extint->value))
                            'change-vl-extint
                            'nil))
