Modifying constructor for binop-ne structures.

Syntax
(change-binop-ne x 
                 )

This is an often useful alternative to make-binop-ne.

We construct a new binop-ne structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-binop-ne

(defmacro change-binop-ne (x &rest args)
          (std::change-aggregate 'binop-ne
