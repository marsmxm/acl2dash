Modifying constructor for pinst-dot structures.

Syntax
(change-pinst-dot x 
                  [:width <width>])

This is an often useful alternative to make-pinst-dot.

We construct a new pinst-dot structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-pinst-dot

(defmacro
     change-pinst-dot (x &rest args)
     (std::change-aggregate 'pinst-dot
                            x args '((:width . pinst-dot->width))
                            'change-pinst-dot
                            'remake-pinst-dot))
