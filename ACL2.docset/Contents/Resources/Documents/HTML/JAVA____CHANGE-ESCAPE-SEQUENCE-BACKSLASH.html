Modifying constructor for escape-sequence-backslash structures.

Syntax
(change-escape-sequence-backslash x 
                                  )

This is an often useful alternative to make-escape-sequence-backslash.

We construct a new escape-sequence-backslash structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-escape-sequence-backslash

(defmacro change-escape-sequence-backslash
          (x &rest args)
          (std::change-aggregate 'escape-sequence-backslash
                                 x args 'nil
                                 'change-escape-sequence-backslash
                                 'nil))
