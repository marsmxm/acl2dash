Modifying constructor for string-element-char structures.

Syntax
(change-string-element-char x 
                            [:get <get>])

This is an often useful alternative to make-string-element-char.

We construct a new string-element-char structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-string-element-char

(defmacro
     change-string-element-char
     (x &rest args)
     (std::change-aggregate 'string-element-char
                            x args
                            '((:get . string-element-char->get))
                            'change-string-element-char
                            'nil))
