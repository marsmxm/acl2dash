Modifying constructor for vl-unparam-instkey structures.

Syntax
(change-vl-unparam-instkey x 
                           [:modname <modname>] 
                           [:ifportalist <ifportalist>] 
                           [:param-actualkeys <param-actualkeys>])

This is an often useful alternative to make-vl-unparam-instkey.

We construct a new vl-unparam-instkey structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-unparam-instkey

(defmacro
  change-vl-unparam-instkey (x &rest args)
  (std::change-aggregate
       'vl-unparam-instkey
       x args
       '((:modname . vl-unparam-instkey->modname)
         (:ifportalist . vl-unparam-instkey->ifportalist)
         (:param-actualkeys . vl-unparam-instkey->param-actualkeys))
       'change-vl-unparam-instkey
       'nil))
