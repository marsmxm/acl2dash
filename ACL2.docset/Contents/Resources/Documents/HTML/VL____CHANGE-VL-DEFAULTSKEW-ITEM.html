Modifying constructor for vl-defaultskew-item structures.

Syntax
(change-vl-defaultskew-item x 
                            [:inputp <inputp>] 
                            [:skew <skew>])

This is an often useful alternative to make-vl-defaultskew-item.

We construct a new vl-defaultskew-item structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.
