Representation of a (non-global) clocking block declaration.

This is a product type introduced by defprod.

Fields
defaultp — booleanp
Was the default keyword provided?
name — maybe-string
Name of the clocking block, if provided. Per SystemVerilog-2012 14.3, only default clocking blocks can be unnamed.
event — vl-evatomlist
The clocking event expression for this block. Should always be non-empty and is typically a single edge expression like @(posedge clk).
iskew — vl-maybe-clkskew
Default input clocking skew, if applicable.
oskew — vl-maybe-clkskew
Default output clocking skew, if applicable.
clkassigns — vl-clkassignlist
Clocking assignments like input #3 foo.
properties — vl-propertylist
Any properties declared in this block.
sequences — vl-sequencelist
Any sequences declared in this block.
loc — vl-location
atts — vl-atts
