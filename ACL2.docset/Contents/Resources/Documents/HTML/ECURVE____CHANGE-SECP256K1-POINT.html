Modifying constructor for secp256k1-point structures.

Syntax
(change-secp256k1-point x 
                        [:x <x>] 
                        [:y <y>])

This is an often useful alternative to make-secp256k1-point.

We construct a new secp256k1-point structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-secp256k1-point

(defmacro change-secp256k1-point (x &rest args)
          (std::change-aggregate 'secp256k1-point
                                 x args
                                 '((:x . secp256k1-point->x)
                                   (:y . secp256k1-point->y))
                                 'change-secp256k1-point
                                 'nil))
