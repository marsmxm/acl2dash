Modifying constructor for escape-carriage-return structures.

Syntax
(change-escape-carriage-return x 
                               )

This is an often useful alternative to make-escape-carriage-return.

We construct a new escape-carriage-return structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-escape-carriage-return

(defmacro change-escape-carriage-return
          (x &rest args)
          (std::change-aggregate 'escape-carriage-return
                                 x args 'nil
                                 'change-escape-carriage-return
                                 'nil))
