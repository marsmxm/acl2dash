A copying macro that lets you create new evmac-appcondp structures, based on existing structures.

Syntax:

(change-evmac-appcond x 
                      [:name <name>] 
                      [:formula <formula>])

This is a sometimes useful alternative to make-evmac-appcond. It constructs a new evmac-appcondp structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by std::defaggregate.

Macro: change-evmac-appcond

(defmacro
     change-evmac-appcond (x &rest args)
     (std::change-aggregate 'evmac-appcond
                            x args
                            '((:name . evmac-appcond->name)
                              (:formula . evmac-appcond->formula))
                            'change-evmac-appcond
                            'nil))
