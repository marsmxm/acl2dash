Modifying constructor for eoutcome-result-ok structures.

Syntax
(change-eoutcome-result-ok x 
                           [:get <get>])

This is an often useful alternative to make-eoutcome-result-ok.

We construct a new eoutcome-result-ok structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
