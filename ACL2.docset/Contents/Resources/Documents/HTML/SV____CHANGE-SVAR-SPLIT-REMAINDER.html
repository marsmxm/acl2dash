Modifying constructor for svar-split-remainder structures.

Syntax
(change-svar-split-remainder x 
                             [:var <var>])

This is an often useful alternative to make-svar-split-remainder.

We construct a new svar-split-remainder structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-svar-split-remainder

(defmacro
     change-svar-split-remainder
     (x &rest args)
     (std::change-aggregate 'svar-split-remainder
                            x args
                            '((:var . svar-split-remainder->var))
                            'change-svar-split-remainder
                            'nil))
