A single member of an enum.

This is a product type introduced by defprod.

Fields
name — stringp
Name of this enumeration item. For instance, in enum { red, yellow, green }, the individual enumitems would be named "red", "yellow", and "green".
range — vl-maybe-range
For simple enumeration items this is nil, but for a fancy item, e.g., for enum { color[6:2] }, the range would be [6:2]. These might later be converted into their expanded out names, e.g., color6, color5, ..., color2.
value — vl-maybe-expr
For simple enumeration items without an explicit value expression this is just nil. For fancier items with explicit values like enum { foo=5, ... } this is the right-hand side expression, i.e., 5.
