Merge nested if statements and blocks.

Our goal here is to flatten out a statement by removing any nested if structures, for instance, we might rewrite:

a1 <= z1;             -->    a1 <= z1;
if (c1)                      if (c1) a2 <= z2;
begin                        if (c1 && c2) a3 <= z3;
  a2 <= z2;                  if (c1 && c3) a4 <= z4;
  if (c2) a3 <= z3;          if (c1 && c3) a5 <= z5;
  if (c3)
  begin
     a4 <= z4;
     a5 <= z5;
  end
end

We expect expressions to be sized, and that unelse has been run to remove else expressions.
