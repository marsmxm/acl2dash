Generate a shallowly embedded ACL2 call of a Java primitive array creation from components.

Signature
(atj-gen-shallow-jprimarr-new-init-call 
     fn comp-blocks 
     comp-exprs src-types dst-types) 
 
  → 
(mv block expr)
Arguments

comp-blocks — Guard (jblock-listp comp-blocks).
comp-exprs — Guard (jexpr-listp comp-exprs).
src-types — Guard (atj-type-listp src-types).

This is called only if :guards is t. This is called after translating the components to Java. The resulting blocks and expressions are passed as parameters here.

If the :guards input is t, the functions that model Java primitive array creation from components (i.e. byte-array-new-init etc.) are treated specially. The type annotation pre-translation step (see atj-type-annotate-term) requires these functions to be called on (translated) list calls, and, as part of the type annotation process, it ``removes'' the list, making its arguments directly arguments of the function (i.e. byte-array-new-init etc.), making it a function with a variable number of arguments (syntactically). After generating expressions for all these arguments, we generate a Java array creation expression with an initializer consisting of those generated expressions; we convert the resulting expression, as needed, to match the destination type.

Function: atj-gen-shallow-jprimarr-new-init-call

(defun
 atj-gen-shallow-jprimarr-new-init-call
 (fn comp-blocks
     comp-exprs src-types dst-types)
 (declare (xargs :guard (and (atj-jprimarr-new-init-fn-p fn)
                             (jblock-listp comp-blocks)
                             (jexpr-listp comp-exprs)
                             (atj-type-listp src-types)
                             (atj-type-listp dst-types))))
 (declare (xargs :guard (and (consp src-types)
                             (consp dst-types))))
 (let
  ((__function__ 'atj-gen-shallow-jprimarr-new-init-call))
  (declare (ignorable __function__))
  (b*
   ((block (flatten (jblock-list-fix comp-blocks)))
    (jtype (atj-jprimarr-new-init-fn-to-comp-jtype fn))
    (expr (jexpr-newarray-init jtype comp-exprs)))
   (mv
      block
      (atj-adapt-expr-to-type expr (atj-type-list-to-type src-types)
                              (atj-type-list-to-type dst-types)
                              t)))))
