Modifying constructor for value-array structures.

Syntax
(change-value-array x 
                    [:elemtype <elemtype>] 
                    [:elements <elements>])

This is an often useful alternative to make-value-array.

We construct a new value-array structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-value-array

(defmacro
     change-value-array (x &rest args)
     (std::change-aggregate 'value-array
                            x args
                            '((:elemtype . value-array->elemtype)
                              (:elements . value-array->elements))
                            'change-value-array
                            'nil))
