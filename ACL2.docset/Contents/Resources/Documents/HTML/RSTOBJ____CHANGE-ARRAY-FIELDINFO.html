A copying macro that lets you create new array-fieldinfo-p structures, based on existing structures.

Syntax:

(change-array-fieldinfo x 
                        [:tr-key <tr-key>] 
                        [:size-key <size-key>])

This is a sometimes useful alternative to make-array-fieldinfo. It constructs a new array-fieldinfo-p structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by std::defaggregate.

Macro: change-array-fieldinfo

(defmacro
   change-array-fieldinfo (x &rest args)
   (std::change-aggregate 'array-fieldinfo
                          x args
                          '((:tr-key . array-fieldinfo->tr-key)
                            (:size-key . array-fieldinfo->size-key))
                          'change-array-fieldinfo
                          'remake-array-fieldinfo))
