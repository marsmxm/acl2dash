Modifying constructor for svex-substconfig structures.

Syntax
(change-svex-substconfig x 
                         [:alist <alist>] 
                         [:simp <simp>])

This is an often useful alternative to make-svex-substconfig.

We construct a new svex-substconfig structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-svex-substconfig

(defmacro
     change-svex-substconfig (x &rest args)
     (std::change-aggregate 'svex-substconfig
