The slice size (or an indicator that there is no size) for a streaming expression.

This is a tagged union type, introduced by deftagsum.

Member Tags → Types
:expr → vl-slicesize-expr
A slice size that is an expression, e.g., {<< 16 {a,b}} has an expression slice size of 16.
:type → vl-slicesize-type
A slice size that is a datatype, e.g., {<< byte {a,b}} has a slice size of byte.
:none → vl-slicesize-none
An indication that this streaming expression does not have any slice size, e.g., {<< {a}}.
