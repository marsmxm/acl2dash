Modifying constructor for vl-propnexttime structures.

Syntax
(change-vl-propnexttime x 
                        [:strongp <strongp>] 
                        [:expr <expr>] 
                        [:prop <prop>])

This is an often useful alternative to make-vl-propnexttime.

We construct a new vl-propnexttime structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-propnexttime

(defmacro
     change-vl-propnexttime (x &rest args)
     (std::change-aggregate 'vl-propnexttime
                            x args
                            '((:strongp . vl-propnexttime->strongp)
                              (:expr . vl-propnexttime->expr)
                              (:prop . vl-propnexttime->prop))
                            'change-vl-propnexttime
                            'nil))
