Modifying constructor for cgraph-edge structures.

Syntax
(change-cgraph-edge x 
                    [:match-vars <match-vars>] 
                    [:rule <rule>] 
                    [:subst <subst>])

This is an often useful alternative to make-cgraph-edge.

We construct a new cgraph-edge structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-cgraph-edge

(defmacro
    change-cgraph-edge (x &rest args)
    (std::change-aggregate 'cgraph-edge
                           x args

                             (:subst . cgraph-edge->subst))
