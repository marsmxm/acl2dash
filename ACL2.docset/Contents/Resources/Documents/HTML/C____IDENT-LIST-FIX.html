(ident-list-fix x) is a usual ACL2::fty list fixing function.

Signature
(ident-list-fix x) → fty::newx
Arguments
x — Guard (ident-listp x).
Returns
fty::newx — Type (ident-listp fty::newx).

In the logic, we apply ident-fix to each member of the x. In the execution, none of that is actually necessary and this is just an inlined identity function.
