Modifying constructor for function-definer-quantified structures.

Syntax
(change-function-definer-quantified x 
                                    [:quantifier <quantifier>] 
                                    [:variables <variables>] 
                                    [:matrix <matrix>])

This is an often useful alternative to make-function-definer-quantified.

We construct a new function-definer-quantified structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-function-definer-quantified

(defmacro
     change-function-definer-quantified
     (x &rest args)
     (std::change-aggregate
          'function-definer-quantified
          x args
          '((:quantifier .
                         function-definer-quantified->quantifier)
            (:variables .
                        function-definer-quantified->variables)
            (:matrix . function-definer-quantified->matrix))
          'change-function-definer-quantified
          'nil))
