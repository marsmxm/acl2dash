An 12-bit unsigned bitstruct type.

This is a bitstruct type introduced by defbitstruct, represented as a unsigned 12-bit integer.

Fields
sce — bit
res1 — 7bits
lme — bit
res2 — bit
lma — bit
nxe — bit

Source: Intel Manual, Feb-14, Vol. 3A, Section 2.2.1
