Modifying constructor for vl-eventtriggerstmt structures.

Syntax
(change-vl-eventtriggerstmt x 
                            [:id <id>] 
                            [:atts <atts>])

This is an often useful alternative to make-vl-eventtriggerstmt.

We construct a new vl-eventtriggerstmt structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-eventtriggerstmt

(defmacro
     change-vl-eventtriggerstmt
     (x &rest args)
     (std::change-aggregate 'vl-eventtriggerstmt
                            x args
