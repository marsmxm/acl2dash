(vl-seltrace-fix x) is a usual fty list fixing function.

Signature
(vl-seltrace-fix x)  fty::newx
Arguments
x Guard (vl-seltrace-p x).
Returns
fty::newx Type (vl-seltrace-p fty::newx).

In the logic, we apply vl-selstep-fix to each member of the x. In the execution, none of that is actually necessary and this is just an inlined identity function.
