Modifying constructor for vl-sysfunname structures.

Syntax
(change-vl-sysfunname x 
                      [:name <name>])

This is an often useful alternative to make-vl-sysfunname.

We construct a new vl-sysfunname structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-sysfunname

(defmacro
     change-vl-sysfunname (x &rest args)
     (std::change-aggregate 'vl-sysfunname
                            x args '((:name . vl-sysfunname->name))
                            'change-vl-sysfunname
                            'remake-vl-sysfunname))
