Modifying constructor for maybe-rule-some structures.

Syntax
(change-maybe-rule-some x 
                        [:val <val>])

This is an often useful alternative to make-maybe-rule-some.

We construct a new maybe-rule-some structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
