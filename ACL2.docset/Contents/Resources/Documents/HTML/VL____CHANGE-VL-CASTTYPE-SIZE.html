Modifying constructor for vl-casttype-size structures.

Syntax
(change-vl-casttype-size x 
                         [:size <size>])

This is an often useful alternative to make-vl-casttype-size.

We construct a new vl-casttype-size structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-casttype-size

(defmacro
