Modifying constructor for value-cons structures.

Syntax
(change-value-cons x 
                   [:car <car>] 
                   [:cdr <cdr>])

This is an often useful alternative to make-value-cons.

We construct a new value-cons structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-value-cons

(defmacro change-value-cons (x &rest args)
          (std::change-aggregate 'value-cons
                                 x args
                                 '((:car . value-cons->car)
                                   (:cdr . value-cons->cdr))
                                 'change-value-cons
                                 'nil))
