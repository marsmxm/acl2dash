Modifying constructor for partsum-comp-str structures.

Syntax
(change-partsum-comp-str x 
                         [:val <val>])

This is an often useful alternative to make-partsum-comp-str.

We construct a new partsum-comp-str structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
