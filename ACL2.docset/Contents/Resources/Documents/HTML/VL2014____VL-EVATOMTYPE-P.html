Type of an item in an event control list.

Any particular atom in the event control list might have a posedge, negedge, or have no edge specifier at all, e.g., for plain atoms like a and b in always @(a or b).

This is an ordinary defenum.

Function: vl-evatomtype-p

(defun vl-evatomtype-p (x)
       (declare (xargs :guard t))
       (or (eq x ':vl-noedge)
           (eq x ':vl-posedge)
           (eq x ':vl-negedge)))
