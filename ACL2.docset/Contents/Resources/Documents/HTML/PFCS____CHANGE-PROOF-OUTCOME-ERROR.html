Modifying constructor for proof-outcome-error structures.

Syntax
(change-proof-outcome-error x 
                            )

This is an often useful alternative to make-proof-outcome-error.

We construct a new proof-outcome-error structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
