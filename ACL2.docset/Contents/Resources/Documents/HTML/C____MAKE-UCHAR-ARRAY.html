Basic constructor macro for uchar-array structures.

Syntax
(make-uchar-array [:elemtype <elemtype>] 
                  [:elements <elements>])

This is the usual way to construct uchar-array structures. It simply conses together a structure with the specified fields.

This macro generates a new uchar-array structure from scratch. See also change-uchar-array, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by fty::defprod.

(defun
 uchar-array (elemtype elements)
 (declare (xargs :guard (and (typep elemtype)
                             (uchar-listp elements))))
 (declare (xargs :guard (and (type-case elemtype :uchar)
                             (consp elements))))
 (let
     ((__function__ 'uchar-array))
     (declare (ignorable __function__))
     (b* ((elemtype (mbe :logic (type-fix elemtype)
                         :exec elemtype))
          (elements (mbe :logic (uchar-list-fix elements)
                         :exec elements)))
         (let ((elemtype (mbe :logic (if (type-case elemtype :uchar)
                                         elemtype (type-uchar))
                              :exec elemtype))
               (elements (mbe :logic (if (consp elements)
                                         elements (list (uchar 0)))
                              :exec elements)))
              (cons :array (list elemtype elements))))))
