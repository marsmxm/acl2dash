Modifying constructor for vl-paramvalue-expr structures.

Syntax
(change-vl-paramvalue-expr x 
                           [:expr <expr>])

This is an often useful alternative to make-vl-paramvalue-expr.

We construct a new vl-paramvalue-expr structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-paramvalue-expr

(defmacro
     change-vl-paramvalue-expr (x &rest args)
     (std::change-aggregate 'vl-paramvalue-expr
                            x args
                            '((:expr . vl-paramvalue-expr->expr))
                            'change-vl-paramvalue-expr
                            'nil))
