Modifying constructor for binary-op-lt structures.

Syntax
(change-binary-op-lt x 
                     )

This is an often useful alternative to make-binary-op-lt.

We construct a new binary-op-lt structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
