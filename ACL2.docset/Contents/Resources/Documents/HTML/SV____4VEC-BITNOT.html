Main correctness theorem: each result bit is just the ACL2::4v-not of the corresponding input bit.

Theorem: 4vec-bitnot-bits

(defthm 4vec-bitnot-bits
        (equal (4vec-idx->4v n (4vec-bitnot x))
               (acl2::4v-not (4vec-idx->4v n x))))

Theorem: 4vec-bitnot-of-3vec-fix-x

(defthm 4vec-bitnot-of-3vec-fix-x
        (equal (4vec-bitnot (3vec-fix x))
               (4vec-bitnot x)))

(defthm 4vec-bitnot-3vec-equiv-congruence-on-x
