Basic constructor macro for type-map structures.

Syntax
(make-type-map [:domain <domain>] 
               [:range <range>])

This is the usual way to construct type-map structures. It simply conses together a structure with the specified fields.

This macro generates a new type-map structure from scratch. See also change-type-map, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by fty::defprod.

Macro: make-type-map

(defmacro make-type-map (&rest args)

(defun type-map (domain range)
       (declare (xargs :guard (and (typep domain) (typep range))))
       (declare (xargs :guard t))
       (let ((__function__ 'type-map))
            (declare (ignorable __function__))
            (b* ((domain (mbe :logic (type-fix domain)
                              :exec domain))
                 (range (mbe :logic (type-fix range)
                             :exec range)))
                (cons :map (list domain range)))))
