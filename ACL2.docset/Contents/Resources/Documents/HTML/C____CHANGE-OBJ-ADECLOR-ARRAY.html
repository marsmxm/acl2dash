Modifying constructor for obj-adeclor-array structures.

Syntax
(change-obj-adeclor-array x 
                          [:decl <decl>] 
                          [:size <size>])

This is an often useful alternative to make-obj-adeclor-array.

We construct a new obj-adeclor-array structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-obj-adeclor-array

(defmacro
     change-obj-adeclor-array (x &rest args)
     (std::change-aggregate 'obj-adeclor-array
                            x args
                            '((:decl . obj-adeclor-array->decl)
                              (:size . obj-adeclor-array->size))
                            'change-obj-adeclor-array
                            'nil))
