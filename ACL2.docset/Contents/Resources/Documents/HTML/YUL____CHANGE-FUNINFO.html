Modifying constructor for funinfo structures.

Syntax
(change-funinfo x 
                [:inputs <inputs>] 
                [:outputs <outputs>] 
                [:body <body>])

This is an often useful alternative to make-funinfo.

We construct a new funinfo structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-funinfo

(defmacro change-funinfo (x &rest args)
          (std::change-aggregate 'funinfo
                                 x args
                                 '((:inputs . funinfo->inputs)
                                   (:outputs . funinfo->outputs)
                                   (:body . funinfo->body))
                                 'change-funinfo
                                 'nil))
