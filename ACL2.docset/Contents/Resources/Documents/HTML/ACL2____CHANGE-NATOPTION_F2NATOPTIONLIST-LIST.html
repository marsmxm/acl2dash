Modifying constructor for natoption/natoptionlist-list structures.

Syntax
(change-natoption/natoptionlist-list x 
                                     [:get <get>])

This is an often useful alternative to make-natoption/natoptionlist-list.

We construct a new natoption/natoptionlist-list structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
