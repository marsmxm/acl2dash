Representation of an immediate assertion statement.

This is a product type, introduced by deftagsum in support of vl-stmt.

Fields
assertion — vl-assertion
All of the data for the assertion. We split this out from the main vl-stmt structure so that module-level and statement-level assertions can easily share the same core representation.
atts — vl-atts
Any (* foo, bar = 1*) style attributes associated with this statement.
