Basic constructor macro for vl-interfaceport structures.

Syntax
(make-vl-interfaceport [:name <name>] 
                       [:ifname <ifname>] 
                       [:loc <loc>] 
                       [:modport <modport>] 
                       [:udims <udims>])

This is the usual way to construct vl-interfaceport structures. It simply conses together a structure with the specified fields.

This macro generates a new vl-interfaceport structure from scratch. See also change-vl-interfaceport, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by defprod.

Macro: make-vl-interfaceport

(defmacro make-vl-interfaceport (&rest args)

(defun
 vl-interfaceport
 (name ifname loc modport udims)
 (declare (xargs :guard (and (stringp name)
                             (stringp ifname)
                             (vl-location-p loc)
                             (maybe-stringp modport)
                             (vl-dimensionlist-p udims))))
 (declare (xargs :guard t))
 (let
  ((__function__ 'vl-interfaceport))
  (declare (ignorable __function__))
  (b*
   ((name (mbe :logic (str-fix name) :exec name))
    (ifname (mbe :logic (str-fix ifname)
                 :exec ifname))
    (loc (mbe :logic (vl-location-fix loc)
              :exec loc))
    (modport (mbe :logic (maybe-string-fix modport)
                  :exec modport))
    (udims (mbe :logic (vl-dimensionlist-fix udims)
                :exec udims)))
   (cons
       :vl-interfaceport
       (std::prod-cons
            (std::prod-cons name ifname)
            (std::prod-cons loc (std::prod-cons modport udims)))))))
