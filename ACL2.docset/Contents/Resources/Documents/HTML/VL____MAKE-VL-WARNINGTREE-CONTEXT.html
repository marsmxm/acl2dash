Basic constructor macro for vl-warningtree-context structures.

Syntax
(make-vl-warningtree-context [:context <context>] 
                             [:subtree <subtree>])

This is the usual way to construct vl-warningtree-context structures. It simply conses together a structure with the specified fields.

This macro generates a new vl-warningtree-context structure from scratch. See also change-vl-warningtree-context, which can "change" an existing structure, instead.

This is an ordinary make- macro introduced by defprod.
