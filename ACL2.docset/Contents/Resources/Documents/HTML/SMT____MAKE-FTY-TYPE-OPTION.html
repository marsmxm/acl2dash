Basic constructor macro for fty-type-option structures.

Syntax
(make-fty-type-option [:some-type <some-type>])

This is the usual way to construct fty-type-option structures. It simply conses together a structure with the specified fields.

This macro generates a new fty-type-option structure from scratch. See also change-fty-type-option, which can "change" an existing structure, instead.
