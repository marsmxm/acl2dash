Fixtype of Unicode input characters usable in character literals.

This corresponds to the grammar rule for single-character [JLS:3.10.4], whose definiens is input-character except for single quote and backslash, where input-character is defined in [JLS:3.4] as a unicode-input-character except for CR and LF.

Since unicode-input-character is captured by unicode-input-char-p in our formalization, we define a Unicode character for a character literal (as conveyed by the charlit part of the type name) as a Unicode input character that is not CR, LF, single quote, and backslash.

We prefer the nomenclature `Unicode character for a character literal', on which this type name is based, to the nomenclature `single character' suggested by the grammar.

       (let ((__function__ 'unicode-charlit-char-p))

(defun unicode-charlit-char-fix (x)
       (declare (xargs :guard (unicode-charlit-char-p x)))
