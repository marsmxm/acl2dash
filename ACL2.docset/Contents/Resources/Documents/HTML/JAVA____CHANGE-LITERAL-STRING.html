Modifying constructor for literal-string structures.

Syntax
(change-literal-string x 
                       [:get <get>])

This is an often useful alternative to make-literal-string.

We construct a new literal-string structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-literal-string

(defmacro
     change-literal-string (x &rest args)
     (std::change-aggregate 'literal-string
                            x args '((:get . literal-string->get))
                            'change-literal-string
                            'nil))
