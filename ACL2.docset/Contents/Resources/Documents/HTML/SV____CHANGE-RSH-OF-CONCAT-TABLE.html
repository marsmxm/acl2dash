Modifying constructor for rsh-of-concat-table structures.

Syntax
(change-rsh-of-concat-table x 
                            [:alist <alist>] 
                            [:alist-width <alist-width>] 
                            [:tail <tail>])

This is an often useful alternative to make-rsh-of-concat-table.

We construct a new rsh-of-concat-table structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-rsh-of-concat-table

(defmacro change-rsh-of-concat-table
          (x &rest args)
          (std::change-aggregate
               'rsh-of-concat-table
               x args
               '((:alist . rsh-of-concat-table->alist)
                 (:alist-width . rsh-of-concat-table->alist-width)
                 (:tail . rsh-of-concat-table->tail))
               'change-rsh-of-concat-table
               'nil))
