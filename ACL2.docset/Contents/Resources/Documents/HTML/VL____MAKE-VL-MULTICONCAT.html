Basic constructor macro for vl-multiconcat structures.

Syntax
(make-vl-multiconcat [:reps <reps>] 
                     [:parts <parts>] 
                     [:atts <atts>])

This is the usual way to construct vl-multiconcat structures. It simply conses together a structure with the specified fields.

This macro generates a new vl-multiconcat structure from scratch. See also change-vl-multiconcat, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by defprod.

Macro: make-vl-multiconcat

(defmacro make-vl-multiconcat (&rest args)
          (std::make-aggregate 'vl-multiconcat
                               args '((:reps) (:parts) (:atts))
                               'make-vl-multiconcat
                               nil))

(defun
 vl-multiconcat (reps parts atts)
 (declare (xargs :guard (and (vl-expr-p reps)
                             (vl-exprlist-p parts)
                             (vl-atts-p atts))))
 (declare (xargs :guard t))
 (let
     ((__function__ 'vl-multiconcat))
     (declare (ignorable __function__))
     (b* ((reps (mbe :logic (vl-expr-fix reps)
                     :exec reps))
          (parts (mbe :logic (vl-exprlist-fix parts)
                      :exec parts))
          (atts (mbe :logic (vl-atts-fix atts)
                     :exec atts)))
         (cons :vl-multiconcat
               (std::prod-cons reps (std::prod-cons parts atts))))))
