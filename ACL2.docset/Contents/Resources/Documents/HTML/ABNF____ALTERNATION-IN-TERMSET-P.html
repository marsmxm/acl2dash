Check if all the terminal value notations in an alternation denote values in a set.

Signature
(alternation-in-termset-p alternation termset) → yes/no
Arguments
alternation — Guard (alternationp alternation).
termset — Guard (nat-setp termset).
Returns
yes/no — Type (booleanp yes/no).

Function: alternation-in-termset-p

(defun alternation-in-termset-p
       (alternation termset)
       (declare (xargs :guard (and (alternationp alternation)
                                   (nat-setp termset))))
       (or (endp alternation)
           (and (concatenation-in-termset-p (car alternation)
                                            termset)
                (alternation-in-termset-p (cdr alternation)
                                          termset))))
