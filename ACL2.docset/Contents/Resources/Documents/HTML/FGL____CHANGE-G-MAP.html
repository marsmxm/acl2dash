Modifying constructor for g-map structures.

Syntax
(change-g-map x 
              [:tag <tag>] 
              [:alist <alist>])

This is an often useful alternative to make-g-map.

We construct a new g-map structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-g-map

(defmacro change-g-map (x &rest args)
          (std::change-aggregate 'g-map
                                 x args
                                 '((:tag . g-map->tag)
                                   (:alist . g-map->alist))
                                 'change-g-map
                                 'nil))
