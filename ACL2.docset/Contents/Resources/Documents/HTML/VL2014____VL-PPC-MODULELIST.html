Pretty print a list of modules with comments to ps.

Signature
(vl-ppc-modulelist x ss &key (ps 'ps)) → ps
Arguments
x — Guard (vl-modulelist-p x).
ss — Guard (vl-scopestack-p ss).

This just calls vl-ppc-module on each module.

Definitions and Theorems

Function: vl-ppc-modulelist-fn

(defun vl-ppc-modulelist-fn (x ss ps)
       (declare (xargs :stobjs (ps)))
       (declare (xargs :guard (and (vl-modulelist-p x)
                                   (vl-scopestack-p ss))))
       (let ((__function__ 'vl-ppc-modulelist))
            (declare (ignorable __function__))
            (if (atom x)
                ps
                (vl-ps-seq (vl-ppc-module (car x) ss)
                           (vl-ppc-modulelist (cdr x) ss)))))
