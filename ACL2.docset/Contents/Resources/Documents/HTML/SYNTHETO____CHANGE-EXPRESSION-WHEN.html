Modifying constructor for expression-when structures.

Syntax
(change-expression-when x 
                        [:test <test>] 
                        [:then <then>] 
                        [:else <else>])

This is an often useful alternative to make-expression-when.

We construct a new expression-when structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-expression-when

(defmacro change-expression-when (x &rest args)
          (std::change-aggregate 'expression-when
                                 x args
                                 '((:test . expression-when->test)
                                   (:then . expression-when->then)
                                   (:else . expression-when->else))
                                 'change-expression-when
                                 'nil))
