Modifying constructor for initializer structures.

Syntax
(change-initializer x 
                    [:field <field>] 
                    [:value <value>])

This is an often useful alternative to make-initializer.

We construct a new initializer structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-initializer

(defmacro change-initializer (x &rest args)
          (std::change-aggregate 'initializer
                                 x args
                                 '((:field . initializer->field)
                                   (:value . initializer->value))
                                 'change-initializer
                                 'nil))
