Modifying constructor for vl-partselect-range structures.

Syntax
(change-vl-range->partselect x 
                             [:range <range>])

This is an often useful alternative to make-vl-range->partselect.

We construct a new vl-partselect-range structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-range->partselect

(defmacro
    change-vl-range->partselect
    (x &rest args)
    (std::change-aggregate 'vl-range->partselect
                           x args
                           '((:range . vl-range->partselect->range))
                           'change-vl-range->partselect
                           'nil))
