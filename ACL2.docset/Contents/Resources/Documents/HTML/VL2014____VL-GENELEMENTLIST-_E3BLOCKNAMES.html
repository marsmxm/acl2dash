Collect all resolved block names in a vl-genelementlist-p.

Signature
(vl-genelementlist->blocknames x)  names
Arguments
x Guard (vl-genelementlist-p x).
Returns
names Type (string-listp names).

Note the elements may be unresolved or may not be blocks at all, in which case we do not add anything. In other words, the list of names returned may be shorter than the number of elements in the list.

Definitions and Theorems

Function: vl-genelementlist->blocknames

(defun
 vl-genelementlist->blocknames (x)
 (declare (xargs :guard (vl-genelementlist-p x)))
 (let
  ((__function__ 'vl-genelementlist->blocknames))
  (declare (ignorable __function__))
  (mbe
    :logic (if (consp x)
               (if (vl-genelement->blockname (car x))
                   (cons (vl-genelement->blockname (car x))
                         (vl-genelementlist->blocknames (cdr x)))
                   (vl-genelementlist->blocknames (cdr x)))
               nil)
    :exec
    (with-local-nrev (vl-genelementlist->blocknames-nrev x nrev)))))
