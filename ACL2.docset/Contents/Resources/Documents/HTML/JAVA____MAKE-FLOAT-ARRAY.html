Basic constructor macro for float-array structures.

Syntax
(make-float-array [:components <components>])

This is the usual way to construct float-array structures. It simply conses together a structure with the specified fields.

This macro generates a new float-array structure from scratch. See also change-float-array, which can "change" an existing structure, instead.

This is an ordinary make- macro introduced by fty::defprod.
