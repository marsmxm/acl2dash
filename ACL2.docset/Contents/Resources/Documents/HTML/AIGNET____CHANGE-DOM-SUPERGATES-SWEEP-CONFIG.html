Modifying constructor for dom-supergates-sweep-config structures.

Syntax
(change-dom-supergates-sweep-config x 
                                    [:gatesimp <gatesimp>])

This is an often useful alternative to make-dom-supergates-sweep-config.

We construct a new dom-supergates-sweep-config structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
