Look up a function in a function table.

Signature
(get-funtype name funtab)  funty?
Arguments
name Guard (identifierp name).
funtab Guard (funtablep funtab).
Returns
funty? Type (funtype-resultp funty?).

The lookup is by name. If a function is found, we return its type. Otherwise we return an error.

Definitions and Theorems

Function: get-funtype

(defun
 get-funtype (name funtab)
 (declare (xargs :guard (and (identifierp name)
                             (funtablep funtab))))
 (let
  ((__function__ 'get-funtype))
  (declare (ignorable __function__))
  (b* ((pair (omap::in (identifier-fix name)
                       (funtable-fix funtab))))
      (if (consp pair)
          (cdr pair)
          (err (list :function-not-found (identifier-fix name)))))))
