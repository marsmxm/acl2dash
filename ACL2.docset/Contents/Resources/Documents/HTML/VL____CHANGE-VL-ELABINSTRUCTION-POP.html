Modifying constructor for vl-elabinstruction-pop structures.

Syntax
(change-vl-elabinstruction-pop x 
                               [:levels <levels>])

This is an often useful alternative to make-vl-elabinstruction-pop.

We construct a new vl-elabinstruction-pop structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-elabinstruction-pop

(defmacro change-vl-elabinstruction-pop
