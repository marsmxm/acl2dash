(t-aig-ite* c a b) constructs a (more conservative) FAIG representing (if c a b), assuming these input FAIGs cannot evaluate to Z.

Signature
(t-aig-ite* c a b) â†’ *

This is a more-conservative version of t-aig-ite that emits X in the case that c is unknown, even when a = b. See 4v-ite for discussion related to this issue.

Definitions and Theorems

Function: t-aig-ite*

(defun t-aig-ite* (c a b)
       (declare (xargs :guard t))
       (let ((__function__ 't-aig-ite*))
            (declare (ignorable __function__))
            (b* (((faig a1 a0) a)
                 ((faig b1 b0) b)
                 ((faig c1 c0) c)
                 (x (aig-and c1 c0)))
                (cons (aig-or x
                              (aig-or (aig-and c1 a1)
                                      (aig-and c0 b1)))
                      (aig-or x
                              (aig-or (aig-and c1 a0)
                                      (aig-and c0 b0)))))))
