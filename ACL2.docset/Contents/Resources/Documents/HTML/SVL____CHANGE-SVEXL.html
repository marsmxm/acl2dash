Modifying constructor for svexl structures.

Syntax
(change-svexl x 
              [:top-node <top-node>] 
              [:node-array <node-array>])

This is an often useful alternative to make-svexl.

We construct a new svexl structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-svexl

(defmacro
     change-svexl (x &rest args)
     (std::change-aggregate 'svexl
                            x args
                            '((:top-node . svexl->top-node)
                              (:node-array . svexl->node-array))
                            'change-svexl
                            'nil))
