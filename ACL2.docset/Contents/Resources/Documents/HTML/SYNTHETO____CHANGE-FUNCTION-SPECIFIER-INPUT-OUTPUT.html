Modifying constructor for function-specifier-input-output structures.

Syntax
(change-function-specifier-input-output x 
                                        [:relation <relation>])

This is an often useful alternative to make-function-specifier-input-output.

We construct a new function-specifier-input-output structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-function-specifier-input-output

(defmacro
     change-function-specifier-input-output

     (std::change-aggregate
