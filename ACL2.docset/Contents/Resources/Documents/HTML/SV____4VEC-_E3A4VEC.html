Convert a plain 4vec into a symbolic a4vec.

Signature
(4vec->a4vec x)  res
Arguments
x Guard (4vec-p x).
Returns
res Type (a4vec-p res).

This is used, e.g., to create the initial a4vecs for any quoted constants in the svex we are bit-blasting.

Definitions and Theorems

Function: 4vec->a4vec

(defun 4vec->a4vec (x)
       (declare (xargs :guard (4vec-p x)))
       (let ((__function__ '4vec->a4vec))
            (declare (ignorable __function__))
            (b* (((4vec x)))
                (a4vec (aig-i2v x.upper)
                       (aig-i2v x.lower)))))
