Modifying constructor for pinst-keyline structures.

Syntax
(change-pinst-keyline x 
                      [:guts <guts>])

This is an often useful alternative to make-pinst-keyline.

We construct a new pinst-keyline structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-pinst-keyline

(defmacro
