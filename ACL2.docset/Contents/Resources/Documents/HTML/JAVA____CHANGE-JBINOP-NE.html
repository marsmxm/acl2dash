Modifying constructor for jbinop-ne structures.

Syntax
(change-jbinop-ne x 
                  )

This is an often useful alternative to make-jbinop-ne.

We construct a new jbinop-ne structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
