Basic constructor macro for data-value-plain structures.

Syntax
(make-data-value-plain [:get <get>])

This is the usual way to construct data-value-plain structures. It simply conses together a structure with the specified fields.

This macro generates a new data-value-plain structure from scratch. See also change-data-value-plain, which can "change" an existing structure, instead.
