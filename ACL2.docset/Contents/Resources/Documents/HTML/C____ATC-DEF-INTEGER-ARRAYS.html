Event to generate the core model of arrays of an integer type.

Signature
(atc-def-integer-arrays type) → event
Arguments
type — Guard (typep type).
Returns
event — Type (pseudo-event-formp event).

Here we generate the fixtype, and the internally used functions that use ACL2 integers as indices. Note that indices are 0-indexed. We also generate the function that returns the length of an array, as an ACL2 integer.
