Modifying constructor for array-schar structures.

Syntax
(change-array-schar x 
                    [:get <get>])

This is an often useful alternative to make-array-schar.

We construct a new array-schar structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-array-schar

(defmacro change-array-schar (x &rest args)
          (std::change-aggregate 'array-schar
                                 x args '((:get . array-schar->get))
                                 'change-array-schar
                                 'nil))
