Modifying constructor for jexpr-instanceof structures.

Syntax
(change-jexpr-instanceof x 
                         [:left <left>] 
                         [:right <right>])

This is an often useful alternative to make-jexpr-instanceof.

We construct a new jexpr-instanceof structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-jexpr-instanceof

(defmacro
     change-jexpr-instanceof (x &rest args)
     (std::change-aggregate 'jexpr-instanceof
                            x args
                            '((:left . jexpr-instanceof->left)
                              (:right . jexpr-instanceof->right))
                            'change-jexpr-instanceof
                            'nil))
