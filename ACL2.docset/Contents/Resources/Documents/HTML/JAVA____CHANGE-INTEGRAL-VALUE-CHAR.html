Modifying constructor for integral-value-char structures.

Syntax
(change-integral-value-char x 
                            [:get <get>])

This is an often useful alternative to make-integral-value-char.

We construct a new integral-value-char structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-integral-value-char

(defmacro
     change-integral-value-char
     (x &rest args)
     (std::change-aggregate 'integral-value-char
                            x args
                            '((:get . integral-value-char->get))
                            'change-integral-value-char
                            'nil))
