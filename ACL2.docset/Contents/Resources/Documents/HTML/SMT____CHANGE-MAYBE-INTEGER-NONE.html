Modifying constructor for maybe-integer-none structures.

Syntax
(change-maybe-integer-none x 
                           )

This is an often useful alternative to make-maybe-integer-none.

We construct a new maybe-integer-none structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-maybe-integer-none

(defmacro change-maybe-integer-none (x &rest args)
          (std::change-aggregate 'maybe-integer-none
                                 x args 'nil
                                 'change-maybe-integer-none
                                 'nil))
