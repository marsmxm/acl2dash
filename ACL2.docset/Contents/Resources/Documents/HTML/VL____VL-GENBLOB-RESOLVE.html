Substitutes the parameter values given by ledger throughout the given object.

Signature
(vl-genblob-resolve x elabindex 
                    ledger warnings &key (config 'config)) 
 
  → 
(mv warnings1 keylist new-x new-elabindex ledger)
Arguments
x — Guard (vl-genblob-p x).
elabindex — without the genblob's scope.
ledger — Guard (vl-unparam-ledger-p ledger).
warnings — Guard (vl-warninglist-p warnings).
config — Guard (vl-simpconfig-p config).
Returns
warnings1 — Type (vl-warninglist-p warnings1).
keylist — Type (vl-unparam-instkeylist-p keylist).
new-x — Type (vl-genblob-p new-x).
ledger — Type (vl-unparam-ledger-p ledger).
