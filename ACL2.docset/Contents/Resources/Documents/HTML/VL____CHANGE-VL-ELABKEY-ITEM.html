Modifying constructor for vl-elabkey-item structures.

Syntax
(change-vl-elabkey-item x 
                        [:name <name>])

This is an often useful alternative to make-vl-elabkey-item.

We construct a new vl-elabkey-item structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-elabkey-item

(defmacro
     change-vl-elabkey-item (x &rest args)
     (std::change-aggregate 'vl-elabkey-item
                            x
                            args '((:name . vl-elabkey-item->name))
                            'change-vl-elabkey-item
                            'nil))
