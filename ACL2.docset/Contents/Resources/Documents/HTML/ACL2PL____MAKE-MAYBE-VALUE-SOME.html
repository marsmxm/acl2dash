Basic constructor macro for maybe-value-some structures.

Syntax
(make-maybe-value-some [:val <val>])

This is the usual way to construct maybe-value-some structures. It simply conses together a structure with the specified fields.

This macro generates a new maybe-value-some structure from scratch. See also change-maybe-value-some, which can "change" an existing structure, instead.
