Packed representation of a line number and column number.

Signature
(vl-linecol line col) → linecol
Arguments
line — Guard (posp line).
col — Guard (natp col).
Returns
linecol — Type (vl-linecol-p linecol).

Semantically a vl-linecol is nothing more than a product type whose fields are a line number and a column number. The line number is a posp and the column number is a natp. We provide the usual defprod style interface for accessing this kind of structure.

For efficiency, we pack line/column numbers together into a single integer, which saves one cons per location. This can save a lot of memory when processing large files.
