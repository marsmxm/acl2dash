Modifying constructor for init-type-single structures.

Syntax
(change-init-type-single x 
                         [:get <get>])

This is an often useful alternative to make-init-type-single.

We construct a new init-type-single structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-init-type-single

(defmacro
     change-init-type-single (x &rest args)
     (std::change-aggregate 'init-type-single

                            'nil))
