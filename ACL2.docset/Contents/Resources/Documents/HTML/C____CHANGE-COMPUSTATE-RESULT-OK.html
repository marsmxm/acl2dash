Modifying constructor for compustate-result-ok structures.

Syntax
(change-compustate-result-ok x 
                             [:get <get>])

This is an often useful alternative to make-compustate-result-ok.

We construct a new compustate-result-ok structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
