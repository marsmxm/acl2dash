Modifying constructor for expr-grade-multiplicative structures.

Syntax
(change-expr-grade-multiplicative x 
                                  )

This is an often useful alternative to make-expr-grade-multiplicative.

We construct a new expr-grade-multiplicative structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
