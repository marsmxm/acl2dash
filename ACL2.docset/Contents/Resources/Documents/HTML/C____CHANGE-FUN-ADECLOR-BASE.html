Modifying constructor for fun-adeclor-base structures.

Syntax
(change-fun-adeclor-base x 
                         [:params <params>])

This is an often useful alternative to make-fun-adeclor-base.

We construct a new fun-adeclor-base structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
