Modifying constructor for renaming structures.

Syntax
(change-renaming x 
                 [:list <list>])

This is an often useful alternative to make-renaming.

We construct a new renaming structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-renaming

(defmacro change-renaming (x &rest args)
          (std::change-aggregate 'renaming
                                 x args '((:list . renaming->list))
                                 'change-renaming
                                 'nil))
