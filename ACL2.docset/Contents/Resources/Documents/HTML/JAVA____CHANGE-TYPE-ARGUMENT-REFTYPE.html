Modifying constructor for type-argument-reftype structures.

Syntax
(change-type-argument-reftype x 
                              [:get <get>])

This is an often useful alternative to make-type-argument-reftype.

We construct a new type-argument-reftype structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-type-argument-reftype

(defmacro
     change-type-argument-reftype
     (x &rest args)
     (std::change-aggregate 'type-argument-reftype

                            'change-type-argument-reftype
                            'nil))
