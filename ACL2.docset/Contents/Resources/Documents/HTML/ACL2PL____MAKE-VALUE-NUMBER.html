Basic constructor macro for value-number structures.

Syntax
(make-value-number [:get <get>])

This is the usual way to construct value-number structures. It simply conses together a structure with the specified fields.

This macro generates a new value-number structure from scratch. See also change-value-number, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by fty::defprod.

(defun value-number (get)
       (declare (xargs :guard (acl2-numberp get)))
       (declare (xargs :guard t))
       (let ((__function__ 'value-number))
            (declare (ignorable __function__))
            (b* ((get (mbe :logic (fix get) :exec get)))
                (cons :number (list get)))))
