Basic constructor macro for array-type-primitive structures.

Syntax
(make-array-type-primitive [:element <element>] 
                           [:dimensions <dimensions>])

This is the usual way to construct array-type-primitive structures. It simply conses together a structure with the specified fields.

This macro generates a new array-type-primitive structure from scratch. See also change-array-type-primitive, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by fty::defprod.
