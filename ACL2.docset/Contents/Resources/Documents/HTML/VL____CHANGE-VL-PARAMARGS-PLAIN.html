Modifying constructor for vl-paramargs-plain structures.

Syntax
(change-vl-paramargs-plain x 
                           [:args <args>])

This is an often useful alternative to make-vl-paramargs-plain.

We construct a new vl-paramargs-plain structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-paramargs-plain

(defmacro
     change-vl-paramargs-plain (x &rest args)
     (std::change-aggregate 'vl-paramargs-plain
                            x args
                            '((:args . vl-paramargs-plain->args))
                            'change-vl-paramargs-plain
                            'nil))
