Modifying constructor for type-sint structures.

Syntax
(change-type-sint x 
                  )

This is an often useful alternative to make-type-sint.

We construct a new type-sint structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-type-sint

(defmacro change-type-sint (x &rest args)
          (std::change-aggregate 'type-sint
