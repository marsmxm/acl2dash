Basic constructor macro for svl-aliasdb structures.

Syntax
(make-svl-aliasdb [:this <this>] 
                  [:sub <sub>])

This is the usual way to construct svl-aliasdb structures. It simply conses together a structure with the specified fields.

This macro generates a new svl-aliasdb structure from scratch. See also change-svl-aliasdb, which can "change" an existing structure, instead.
