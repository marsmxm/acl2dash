Recognize true lists of logical names.

Signature
(logical-name-listp names wrld) → yes/no
Arguments
wrld — Guard (plist-worldp wrld).
Returns
yes/no — Type (booleanp yes/no).

See logical-namep in the ACL2 source code.

While logical-namep is not boolean-valued, this function always returns a boolean.

We cannot use std::deflist to define this function because that macro attempts to prove that logical-namep is boolean-valued.

Function: logical-name-listp

(defun logical-name-listp (names wrld)
       (declare (xargs :guard (plist-worldp wrld)))
       (let ((__function__ 'logical-name-listp))
            (declare (ignorable __function__))
            (cond ((atom names) (null names))
                  (t (and (logical-namep (car names) wrld)
                          (logical-name-listp (cdr names)
                                              wrld))))))
