Fixtype of ATJ function type information and nil.

This is an option type introduced by fty::defoption. Note that defoption is just a wrapper for fty::defflexsum, so there are :none and :some member types, a case macro, and so forth.

Member Types
:none → atj-maybe-function-type-info-none
Represents that no atj-maybe-function-type-info is available, i.e., Nothing or None.
:some → atj-maybe-function-type-info-some
An available atj-maybe-function-type-info, i.e., Just val or Some val.
