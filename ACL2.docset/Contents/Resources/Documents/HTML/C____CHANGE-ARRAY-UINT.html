Modifying constructor for array-uint structures.

Syntax
(change-array-uint x 
                   [:get <get>])

This is an often useful alternative to make-array-uint.

We construct a new array-uint structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-array-uint

(defmacro change-array-uint (x &rest args)
          (std::change-aggregate 'array-uint
                                 x args '((:get . array-uint->get))
                                 'change-array-uint
                                 'nil))
