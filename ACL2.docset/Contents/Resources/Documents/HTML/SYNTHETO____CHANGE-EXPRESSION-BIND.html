Modifying constructor for expression-bind structures.

Syntax
(change-expression-bind x 
                        [:variables <variables>] 
                        [:value <value>] 
                        [:body <body>])

This is an often useful alternative to make-expression-bind.

We construct a new expression-bind structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-expression-bind

(defmacro
  change-expression-bind (x &rest args)
  (std::change-aggregate 'expression-bind
                         x args
                         '((:variables . expression-bind->variables)
                           (:value . expression-bind->value)
                           (:body . expression-bind->body))
                         'change-expression-bind
                         'nil))
