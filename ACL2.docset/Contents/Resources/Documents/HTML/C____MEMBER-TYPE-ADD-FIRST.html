Add a member type at the beginning of a list of member types.

Signature
(member-type-add-first name type members) → new-members
Arguments
name — Guard (identp name).
type — Guard (typep type).
members — Guard (member-type-listp members).
Returns
new-members — Type (member-type-list-optionp new-members).

We check that the a member with the same name is not already in the list, to maintain the well-formedness of the list.

Function: member-type-add-first

(defun member-type-add-first
       (name type members)
       (declare (xargs :guard (and (identp name)
                                   (typep type)
                                   (member-type-listp members))))
       (let ((__function__ 'member-type-add-first))
            (declare (ignorable __function__))
            (b* ((found (member-type-lookup name members))
                 ((when found)
                  (member-type-list-option-none)))
                (member-type-list-option-some
                     (cons (make-member-type :name name :type type)
                           members)))))
