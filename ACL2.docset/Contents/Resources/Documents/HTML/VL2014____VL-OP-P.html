Recognizer for valid operators.

(vl-op-p x) checks that x is one of the operators listed in the *vl-ops-table*. We prefer to use vl-op-p instead of looking up operators directly in the table, since this way we can disable vl-op-p and avoid large case splits.

(defun vl-op-p$inline (x)
       (declare (xargs :guard t))
       (let ((__function__ 'vl-op-p))

            (if (assoc x (vl-ops-table)) t nil)))

Theorem: type-when-vl-op-p

(defthm type-when-vl-op-p
        (implies (vl-op-p x)
                 (and (symbolp x)
                      (not (equal x t))
                      (not (equal x nil))))
        :rule-classes :compound-recognizer)
