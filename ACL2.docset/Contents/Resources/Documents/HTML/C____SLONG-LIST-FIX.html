(slong-list-fix x) is a usual ACL2::fty list fixing function.

Signature
(slong-list-fix x) → fty::newx
Arguments
x — Guard (slong-listp x).
Returns
fty::newx — Type (slong-listp fty::newx).

In the logic, we apply slong-fix to each member of the x. In the execution, none of that is actually necessary and this is just an inlined identity function.
