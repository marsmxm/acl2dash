Modifying constructor for tyspecseq-sshort structures.

Syntax
(change-tyspecseq-sshort x 
                         [:signed <signed>] 
                         [:int <int>])

This is an often useful alternative to make-tyspecseq-sshort.

We construct a new tyspecseq-sshort structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-tyspecseq-sshort

(defmacro
     change-tyspecseq-sshort (x &rest args)
