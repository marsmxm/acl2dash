Basic constructor macro for literal-hex-number structures.

Syntax
(make-literal-hex-number [:get <get>])

This is the usual way to construct literal-hex-number structures. It simply conses together a structure with the specified fields.

This macro generates a new literal-hex-number structure from scratch. See also change-literal-hex-number, which can "change" an existing structure, instead.

This is an ordinary make- macro introduced by fty::defprod.

(defun literal-hex-number (get)
       (declare (xargs :guard (hex-digit-listp get)))
       (declare (xargs :guard t))
       (let ((__function__ 'literal-hex-number))
            (declare (ignorable __function__))
            (b* ((get (mbe :logic (hex-digit-list-fix get)
                           :exec get)))
                (cons :hex-number (list get)))))
