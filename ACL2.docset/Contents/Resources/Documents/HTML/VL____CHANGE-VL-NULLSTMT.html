Modifying constructor for vl-nullstmt structures.

Syntax
(change-vl-nullstmt x 
                    [:atts <atts>])

This is an often useful alternative to make-vl-nullstmt.

We construct a new vl-nullstmt structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-nullstmt

(defmacro
     change-vl-nullstmt (x &rest args)
     (std::change-aggregate 'vl-nullstmt
                            x args '((:atts . vl-nullstmt->atts))
                            'change-vl-nullstmt
                            'remake-vl-nullstmt))
