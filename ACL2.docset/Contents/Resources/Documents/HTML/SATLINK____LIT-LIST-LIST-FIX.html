(lit-list-list-fix x) is a usual ACL2::fty list fixing function.

Signature
(lit-list-list-fix x) → fty::newx
Arguments
x — Guard (lit-list-listp x).
Returns
fty::newx — Type (lit-list-listp fty::newx).

In the logic, we apply lit-list-fix to each member of the x. In the execution, none of that is actually necessary and this is just an inlined identity function.
