Modifying constructor for vl-paramvalue-type structures.

Syntax
(change-vl-paramvalue-type x 
                           [:type <type>])

This is an often useful alternative to make-vl-paramvalue-type.

We construct a new vl-paramvalue-type structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-paramvalue-type

(defmacro
     change-vl-paramvalue-type (x &rest args)
     (std::change-aggregate 'vl-paramvalue-type
                            x args
                            '((:type . vl-paramvalue-type->type))
                            'change-vl-paramvalue-type
                            'nil))
