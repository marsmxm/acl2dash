Pretty-print a function definition.

Signature
(pprint-fundef fdef) → lines
Arguments
fdef — Guard (fundefp fdef).
Returns
lines — Type (msg-listp lines).

This is always at the top level, so we initialize the indentation level of the body to 1.

Definitions and Theorems

Function: pprint-fundef

(defun
 pprint-fundef (fdef)
 (declare (xargs :guard (fundefp fdef)))
 (let
  ((__function__ 'pprint-fundef))
  (declare (ignorable __function__))
  (b*
   (((fundef fdef) fdef))
   (append
    (list
     (pprint-line
      (msg
          "~@0 ~@1(~@2) {"
          (pprint-tyspecseq fdef.result)
          (pprint-ident fdef.name)
          (pprint-comma-sep (pprint-param-declon-list fdef.params)))
      0))
    (pprint-block-item-list fdef.body 1)
    (list (pprint-line "}" 0))))))
