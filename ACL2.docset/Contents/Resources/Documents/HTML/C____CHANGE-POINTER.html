Modifying constructor for pointer structures.

Syntax
(change-pointer x 
                [:address? <address?>] 
                [:reftype <reftype>])

This is an often useful alternative to make-pointer.

We construct a new pointer structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-pointer

(defmacro change-pointer (x &rest args)
          (std::change-aggregate 'pointer
                                 x args
                                 '((:address? . pointer->address?)
                                   (:reftype . pointer->reftype))
                                 'change-pointer
                                 'nil))
