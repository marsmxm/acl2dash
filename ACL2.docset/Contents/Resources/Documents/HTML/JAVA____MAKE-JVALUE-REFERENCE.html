Basic constructor macro for jvalue-reference structures.

Syntax
(make-jvalue-reference [:get <get>])

This is the usual way to construct jvalue-reference structures. It simply conses together a structure with the specified fields.

This macro generates a new jvalue-reference structure from scratch. See also change-jvalue-reference, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by fty::defprod.

Macro: make-jvalue-reference

(defmacro make-jvalue-reference (&rest args)
