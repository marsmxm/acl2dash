A copying macro that lets you create new propagate-limits-p structures, based on existing structures.

Syntax:

(change-propagate-limits x 
                         [:max-ops <max-ops>])

This is a sometimes useful alternative to make-propagate-limits. It constructs a new propagate-limits-p structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defaggregate.

Macro: change-propagate-limits

(defmacro
    change-propagate-limits (x &rest args)
    (std::change-aggregate 'propagate-limits
                           x args
                           '((:max-ops . propagate-limits->max-ops))
                           'change-propagate-limits
                           'nil))
