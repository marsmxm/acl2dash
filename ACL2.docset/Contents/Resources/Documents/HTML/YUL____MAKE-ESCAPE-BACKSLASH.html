Basic constructor macro for escape-backslash structures.

Syntax
(make-escape-backslash )

This is the usual way to construct escape-backslash structures. It simply conses together a structure with the specified fields.

This macro generates a new escape-backslash structure from scratch. See also change-escape-backslash, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by fty::defprod.
