Represents a single scoping operator (:: being applied to some interior scopeexpr.

This is a product type, introduced by defflexsum in support of vl-scopeexpr.

Fields
first — vl-scopename
The outer scope name, e.g., ape
paramargs — vl-maybe-paramargs
The parameter arguments for a class scope
rest — vl-scopeexpr
The inner scope expression, e.g., bat::cat.dog[3][2][1].elf.
