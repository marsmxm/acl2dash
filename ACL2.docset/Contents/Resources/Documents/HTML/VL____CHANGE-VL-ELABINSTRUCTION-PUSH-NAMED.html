Modifying constructor for vl-elabinstruction-push-named structures.

Syntax
(change-vl-elabinstruction-push-named x 
                                      [:key <key>])

This is an often useful alternative to make-vl-elabinstruction-push-named.

We construct a new vl-elabinstruction-push-named structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
