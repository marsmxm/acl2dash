Modifying constructor for vl-clkskew structures.

Syntax
(change-vl-clkskew x 
                   [:delay <delay>] 
                   [:edge <edge>])

This is an often useful alternative to make-vl-clkskew.

We construct a new vl-clkskew structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-clkskew

(defmacro change-vl-clkskew (x &rest args)
          (std::change-aggregate 'vl-clkskew
                                 x args
                                 '((:delay . vl-clkskew->delay)
                                   (:edge . vl-clkskew->edge))
                                 'change-vl-clkskew
                                 'remake-vl-clkskew))
