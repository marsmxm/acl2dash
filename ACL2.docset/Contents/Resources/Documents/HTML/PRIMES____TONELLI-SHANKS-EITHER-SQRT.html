Tonelli-Shanks modular square root. Finds a square root if a square root exists.

Signature
(tonelli-shanks-either-sqrt n p z)  sqrt
Arguments
n Guard (natp n).
p Guard (natp p).
z Guard (natp z).
Returns
sqrt Type (natp sqrt), given the guard.
Finds a square root of n modulo p if it exists, else returns 0. p must be an odd prime. z is a quadratic nonresidue in p.

Function: tonelli-shanks-either-sqrt

(defun tonelli-shanks-either-sqrt (n p z)
