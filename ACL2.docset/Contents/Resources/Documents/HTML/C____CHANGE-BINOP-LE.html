Modifying constructor for binop-le structures.

Syntax
(change-binop-le x 
                 )

This is an often useful alternative to make-binop-le.

We construct a new binop-le structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
