Modifying constructor for vl-timeliteral structures.

Syntax
(change-vl-timeliteral x 
                       [:quantity <quantity>] 
                       [:units <units>])

This is an often useful alternative to make-vl-timeliteral.

We construct a new vl-timeliteral structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-timeliteral

(defmacro
     change-vl-timeliteral (x &rest args)
     (std::change-aggregate 'vl-timeliteral
                            x args
                            '((:quantity . vl-timeliteral->quantity)
                              (:units . vl-timeliteral->units))
                            'change-vl-timeliteral
                            'remake-vl-timeliteral))
