Modifying constructor for stmt-type structures.

Syntax
(change-stmt-type x 
                  [:return-types <return-types>] 
                  [:variables <variables>])

This is an often useful alternative to make-stmt-type.

We construct a new stmt-type structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-stmt-type

(defmacro
  change-stmt-type (x &rest args)
  (std::change-aggregate 'stmt-type
                         x args
                         '((:return-types . stmt-type->return-types)
                           (:variables . stmt-type->variables))
                         'change-stmt-type
                         'nil))
