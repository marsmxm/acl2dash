Method names in a Java expression.

Signature
(jexpr-methods expr) → methods
Arguments
expr — Guard (jexprp expr).
Returns
methods — Type (string-listp methods).

We return all the method names in method calls.

Theorem: return-type-of-jexpr-methods.methods

(defthm return-type-of-jexpr-methods.methods
        (b* ((?methods (jexpr-methods expr)))
            (string-listp methods))
        :rule-classes :rewrite)
