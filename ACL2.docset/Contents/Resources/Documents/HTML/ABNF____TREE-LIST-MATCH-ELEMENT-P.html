A list of trees matches an element iff each tree matches the element.

Function: tree-list-match-element-p

(defun tree-list-match-element-p
       (trees element rules)
       (declare (xargs :guard (and (tree-listp trees)
                                   (elementp element)
                                   (rulelistp rules))))
       (or (atom trees)
           (and (tree-match-element-p (car trees)
                                      element rules)
                (tree-list-match-element-p (cdr trees)
                                           element rules))))
