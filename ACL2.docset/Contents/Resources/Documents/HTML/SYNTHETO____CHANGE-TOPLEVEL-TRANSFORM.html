Modifying constructor for toplevel-transform structures.

Syntax
(change-toplevel-transform x 
                           [:get <get>])

This is an often useful alternative to make-toplevel-transform.

We construct a new toplevel-transform structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-toplevel-transform

(defmacro
     change-toplevel-transform (x &rest args)
     (std::change-aggregate 'toplevel-transform
                            x
                            args '((:get . toplevel-transform->get))
                            'change-toplevel-transform
                            'nil))
