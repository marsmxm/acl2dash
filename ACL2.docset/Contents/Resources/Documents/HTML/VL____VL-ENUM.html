A SystemVerilog enum datatype, or an array of enums.

This is a product type, introduced by deftagsum in support of vl-datatype.

Fields
basetype — vl-datatype
The base type for the enum. Note that, in the SystemVerilog syntax, enums are only allowed to have certain base types that are very basic. But for simplicity, in our representation we just use an arbitrary vl-datatype here.
items — vl-enumitemlist
The items of the enumeration.
values — vl-exprlist
List of all valid values of this enum, generated by enumnames transform
pdims — vl-dimensionlist
Packed dimensions for this enum type.
udims — vl-dimensionlist
Unpacked dimensions for this enum type.
