Modifying constructor for array-result-ok structures.

Syntax
(change-array-result-ok x 
                        [:get <get>])

This is an often useful alternative to make-array-result-ok.

We construct a new array-result-ok structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-array-result-ok

(defmacro
