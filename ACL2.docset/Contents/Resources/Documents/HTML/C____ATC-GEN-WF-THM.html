Generate the theorem asserting the static well-formedness of the generated C code (referenced as the named constant).

Signature
(atc-gen-wf-thm proofs prog-const wf-thm print) 
  → 
(mv local-events exported-events)
Arguments
proofs — Guard (booleanp proofs).
prog-const — Guard (symbolp prog-const).
wf-thm — Guard (symbolp wf-thm).
print — Guard (evmac-input-print-p print).
Returns
local-events — A pseudo-event-form-listp.
exported-events — A pseudo-event-form-listp.

Since this is a ground theorem, we expect that it should be easily provable using just the executable counterpart of check-transunit, which is an executable function.

We generate singleton lists of events if :proofs is t, empty lists otherwise.

Definitions and Theorems

Function: atc-gen-wf-thm

(defun
 atc-gen-wf-thm
 (proofs prog-const wf-thm print)
 (declare (xargs :guard (and (booleanp proofs)
                             (symbolp prog-const)
                             (symbolp wf-thm)
                             (evmac-input-print-p print))))
 (let
  ((__function__ 'atc-gen-wf-thm))
  (declare (ignorable __function__))
  (b*
   (((unless proofs) (mv nil nil))
    ((mv local-event exported-event)
     (evmac-generate-defthm
          wf-thm
          :formula (cons 'equal
                         (cons (cons 'check-transunit
                                     (cons prog-const 'nil))
                               (cons ':wellformed 'nil)))
          :hints '(("Goal" :in-theory '((:e check-transunit))))
          :enable nil))
    (progress-start?
     (and
      (evmac-input-print->= print :info)
      (cons
          (cons 'cw-event
                (cons '"~%Generating the well-formedness theorem..."
                      'nil))
          'nil)))
    (progress-end?
         (and (evmac-input-print->= print :info)
              (cons (cons 'cw-event (cons '" done.~%" 'nil))
                    'nil)))
    (local-event (cons 'progn
                       (append progress-start?
                               (cons local-event progress-end?)))))
   (mv (list local-event)
       (list exported-event)))))
