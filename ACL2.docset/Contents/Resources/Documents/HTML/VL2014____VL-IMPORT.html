Representation of a single import item, i.e., import foo::bar;.

This is a product type introduced by defprod.

Fields
pkg — stringp
Package to import everything from, e.g., "foo".
part — vl-importpart
Either: a single name to import, e.g., "bar" above, or else the symbol :vl-import*, which means import everything, as in import foo::*;.
loc — vl-location
atts — vl-atts
