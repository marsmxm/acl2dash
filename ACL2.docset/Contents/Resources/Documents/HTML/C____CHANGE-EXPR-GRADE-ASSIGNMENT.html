Modifying constructor for expr-grade-assignment structures.

Syntax
(change-expr-grade-assignment x 
                              )

This is an often useful alternative to make-expr-grade-assignment.

We construct a new expr-grade-assignment structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
