Check if an octal numeral is well-formed.

Signature
(octdig/uscore-list-wfp dus) → yes/no
Arguments
dus — Guard (octdig/uscore-listp dus).
Returns
yes/no — Type (booleanp yes/no).

An octal-numeral in the grammar is a list of octal digits and underscores subject to the following constraints, derived from the grammar:

  • It must start with the digit 0, not an underscore or other digit.
  • It must end with a digit, not an underscore.
  • It must contain at least two digits (which implies that it cannot be confused with the decimal numeral 0).
