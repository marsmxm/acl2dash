Modifying constructor for maybe-symbol-value-none structures.

Syntax
(change-maybe-symbol-value-none x 
                                )

This is an often useful alternative to make-maybe-symbol-value-none.

We construct a new maybe-symbol-value-none structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
