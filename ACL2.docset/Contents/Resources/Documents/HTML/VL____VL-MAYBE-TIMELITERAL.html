Option type; vl-timeliteral or nil.

This is an option type introduced by defoption. Note that defoption is just a wrapper for defflexsum, so there are :none and :some member types, a case macro, and so forth.

Member Types
:none → vl-maybe-timeliteral-none
Represents that no vl-maybe-timeliteral is available, i.e., Nothing or None.
:some → vl-maybe-timeliteral-some
An available vl-maybe-timeliteral, i.e., Just val or Some val.
