Modifying constructor for vl-repeatstmt structures.

Syntax
(change-vl-repeatstmt x 
                      [:condition <condition>] 
                      [:body <body>] 
                      [:atts <atts>])

This is an often useful alternative to make-vl-repeatstmt.

We construct a new vl-repeatstmt structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-repeatstmt

(defmacro
    change-vl-repeatstmt (x &rest args)
    (std::change-aggregate 'vl-repeatstmt
                           x args
                           '((:condition . vl-repeatstmt->condition)
                             (:body . vl-repeatstmt->body)
                             (:atts . vl-repeatstmt->atts))
                           'change-vl-repeatstmt
                           'remake-vl-repeatstmt))
