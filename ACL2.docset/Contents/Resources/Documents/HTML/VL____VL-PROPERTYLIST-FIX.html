(vl-propertylist-fix x) is a usual fty list fixing function.

Signature
(vl-propertylist-fix x) → fty::newx
Arguments
x — Guard (vl-propertylist-p x).
Returns
fty::newx — Type (vl-propertylist-p fty::newx).

In the logic, we apply vl-property-fix to each member of the x. In the execution, none of that is actually necessary and this is just an inlined identity function.
