Construct a subsubsection of the user documentation of an event macro.

This provides more abstraction than using a specific heading.

Macro: evmac-subsubsection

(defmacro xdoc::evmac-subsubsection
          (title &rest content)
          (cons 'xdoc::&&
                (cons (cons 'xdoc::h5 (cons title 'nil))
                      content)))
