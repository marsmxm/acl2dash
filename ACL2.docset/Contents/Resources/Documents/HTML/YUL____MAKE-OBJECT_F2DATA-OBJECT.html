Basic constructor macro for object/data-object structures.

Syntax
(make-object/data-object [:get <get>])

This is the usual way to construct object/data-object structures. It simply conses together a structure with the specified fields.

This macro generates a new object/data-object structure from scratch. See also change-object/data-object, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by fty::defprod.
