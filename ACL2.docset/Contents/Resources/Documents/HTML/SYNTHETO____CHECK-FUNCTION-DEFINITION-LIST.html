Check if a list of function definitions is statically well-formed.

Signature
(check-function-definition-list fundefs ctxt) 
  → 
(mv err? obligs)
Arguments
fundefs — Guard (function-definition-listp fundefs).
ctxt — Guard (contextp ctxt).
Returns
obligs — Type (proof-obligation-listp obligs).

This is only used on function definitions inside a function recursion, i.e. not at the top level.

Definitions and Theorems

Function: check-function-definition-list

(defun
 check-function-definition-list
 (fundefs ctxt)
 (declare (xargs :guard (and (function-definition-listp fundefs)
                             (contextp ctxt))))
 (declare
  (xargs
   :guard
   (and
      (null (context->types ctxt))
      (subsetp-equal (function-definition-list->header-list fundefs)
                     (context->functions ctxt))
      (omap::empty (context->variables ctxt))
      (null (context->obligation-vars ctxt))
      (null (context->obligation-hyps ctxt)))))
 (let ((__function__ 'check-function-definition-list))
      (declare (ignorable __function__))
      (b* (((when (endp fundefs)) (mv nil nil))
           ((mv err? obligs)
            (check-function-definition-top/nontop (car fundefs)
                                                  ctxt))
           ((when err?) (mv err? nil))
           ((mv err? more-obligs)
            (check-function-definition-list (cdr fundefs)
                                            ctxt))
           ((when err?) (mv err? nil)))
          (mv nil (append obligs more-obligs)))))
