Variable renaming relation over function environments.

Signature
(funenv-renamevarp old new) → yes/no
Arguments
old — Guard (funenvp old).
new — Guard (funenvp new).
Returns
yes/no — Type (booleanp yes/no).

The two environments must have the same number of scopes, and the corresponding scopes must be related.

Function: funenv-renamevarp

(defun funenv-renamevarp (old new)
       (declare (xargs :guard (and (funenvp old) (funenvp new))))
       (let ((__function__ 'funenv-renamevarp))
            (declare (ignorable __function__))
            (or (and (endp old) (endp new))
                (and (consp old)
                     (consp new)
                     (funscope-renamevarp (car old)
                                          (car new))
                     (funenv-renamevarp (cdr old)
                                        (cdr new))))))
