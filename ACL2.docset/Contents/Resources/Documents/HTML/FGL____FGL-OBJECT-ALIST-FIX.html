(fgl-object-alist-fix x) is an ACL2::fty alist fixing function that follows the drop-keys strategy.

Signature
(fgl-object-alist-fix x) → fty::newx
Arguments
x — Guard (fgl-object-alist-p x).
Returns
fty::newx — Type (fgl-object-alist-p fty::newx).

Note that in the execution this is just an inline identity function.
