Turn a list of initializers into a variable substitution.

Signature
(initializers-to-variable-substitution inits) → subst
Arguments
inits — Guard (initializer-listp inits).
Returns
subst — Type (variable-substitutionp subst).

Each field name is regarded as a variable and the corresponding expression is associated to the variable, and will replace the variable when the substitution is applied.

Definitions and Theorems

Function: initializers-to-variable-substitution

(defun
 initializers-to-variable-substitution
 (inits)
 (declare (xargs :guard (initializer-listp inits)))
 (let
  ((__function__ 'initializers-to-variable-substitution))
  (declare (ignorable __function__))
  (cond
    ((endp inits) nil)
    (t (omap::update
            (initializer->field (car inits))
            (initializer->value (car inits))
            (initializers-to-variable-substitution (cdr inits)))))))
