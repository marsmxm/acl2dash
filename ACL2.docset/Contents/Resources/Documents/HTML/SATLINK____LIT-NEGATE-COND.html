Efficiently negate a literal.

Signature
(lit-negate-cond lit c) → new-lit
Arguments
lit — Guard (litp lit).
c — Guard (bitp c).
Returns
new-lit — Type (litp new-lit).

When c is 1, we negate lit. Otherwise, when c is 0, we return lit unchanged.
