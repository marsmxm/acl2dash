Modifying constructor for fgl-congruence-rune structures.

Syntax
(change-fgl-congruence-rune x 
                            [:name <name>])

This is an often useful alternative to make-fgl-congruence-rune.

We construct a new fgl-congruence-rune structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

(defmacro
