(expr-list-fix x) is a usual ACL2::fty list fixing function.

Signature
(expr-list-fix x) → fty::newx
Arguments
x — Guard (expr-listp x).
Returns
fty::newx — Type (expr-listp fty::newx).

In the logic, we apply expr-fix to each member of the x. In the execution, none of that is actually necessary and this is just an inlined identity function.
