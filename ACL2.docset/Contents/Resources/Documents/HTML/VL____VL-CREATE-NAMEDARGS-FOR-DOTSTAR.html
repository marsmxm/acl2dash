Create the arguments that .* expands to.

Signature
(vl-create-namedargs-for-dotstar missing ss warnings inst) 
  → 
(mv successp warnings new-args)
Arguments
missing — Names of submodule ports that aren't explicitly connected.
    Guard (string-listp missing).
ss — Scopestack for the superior module.
    Guard (vl-scopestack-p ss).
warnings — Warnings accumulator.
    Guard (vl-warninglist-p warnings).
inst — Context for warnings.
    Guard (vl-modinst-p inst).
Returns
successp — Type (booleanp successp).
warnings — Type (vl-warninglist-p warnings).
new-args — Type (vl-namedarglist-p new-args).

Function: vl-create-namedargs-for-dotstar

(defun
     vl-create-namedargs-for-dotstar
     (missing ss warnings inst)
     (declare (xargs :guard (and (string-listp missing)
                                 (vl-scopestack-p ss)
                                 (vl-warninglist-p warnings)
                                 (vl-modinst-p inst))))
     (let ((__function__ 'vl-create-namedargs-for-dotstar))
          (declare (ignorable __function__))
          (b* (((when (atom missing)) (mv t (ok) nil))
               ((mv okp1 warnings args1)
                (vl-create-namedarg-for-dotstar (car missing)
                                                ss warnings inst))
               ((mv okp2 warnings args2)
                (vl-create-namedargs-for-dotstar (cdr missing)
                                                 ss warnings inst)))
              (mv (and okp1 okp2)
                  warnings (append args1 args2)))))
