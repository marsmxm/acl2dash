Modifying constructor for float-value structures.

Syntax
(change-float-value x 
                    [:float <float>])

This is an often useful alternative to make-float-value.

We construct a new float-value structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-float-value

(defmacro
     change-float-value (x &rest args)
     (std::change-aggregate 'float-value
                            x args '((:float . float-value->float))
                            'change-float-value
                            'nil))
