Modifying constructor for vl-arguments-named structures.

Syntax
(change-vl-arguments-named x 
                           [:args <args>] 
                           [:starp <starp>])

This is an often useful alternative to make-vl-arguments-named.

We construct a new vl-arguments-named structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-arguments-named

(defmacro
     change-vl-arguments-named (x &rest args)
     (std::change-aggregate 'vl-arguments-named
                            x args
                            '((:args . vl-arguments-named->args)
                              (:starp . vl-arguments-named->starp))
                            'change-vl-arguments-named
                            'nil))
