Modifying constructor for escape-letter-t structures.

Syntax
(change-escape-letter-t x 
                        )

This is an often useful alternative to make-escape-letter-t.

We construct a new escape-letter-t structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
