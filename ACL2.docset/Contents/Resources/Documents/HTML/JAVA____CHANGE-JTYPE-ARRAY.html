Modifying constructor for jtype-array structures.

Syntax
(change-jtype-array x 
                    [:comp <comp>])

This is an often useful alternative to make-jtype-array.

We construct a new jtype-array structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-jtype-array

(defmacro
     change-jtype-array (x &rest args)
     (std::change-aggregate 'jtype-array
                            x args '((:comp . jtype-array->comp))
                            'change-jtype-array
                            'nil))
