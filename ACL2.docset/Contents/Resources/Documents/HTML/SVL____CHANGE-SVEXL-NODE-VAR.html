Modifying constructor for svexl-node-var structures.

Syntax
(change-svexl-node-var x 
                       [:name <name>])

This is an often useful alternative to make-svexl-node-var.

We construct a new svexl-node-var structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-svexl-node-var

(defmacro
     change-svexl-node-var (x &rest args)
