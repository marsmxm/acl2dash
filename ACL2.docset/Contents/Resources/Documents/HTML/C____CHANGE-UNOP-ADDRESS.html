Modifying constructor for unop-address structures.

Syntax
(change-unop-address x 
                     )

This is an often useful alternative to make-unop-address.

We construct a new unop-address structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-unop-address

(defmacro change-unop-address (x &rest args)
          (std::change-aggregate 'unop-address
                                 x args 'nil
                                 'change-unop-address
                                 'nil))
