Basic constructor macro for vl-echar-raw structures.

Syntax
(make-vl-echar-raw [:filename <filename>] 
                   [:pack <pack>])

This is the usual way to construct vl-echar-raw structures. It simply conses together a structure with the specified fields.

This macro generates a new vl-echar-raw structure from scratch. See also change-vl-echar-raw, which can "change" an existing structure, instead.
