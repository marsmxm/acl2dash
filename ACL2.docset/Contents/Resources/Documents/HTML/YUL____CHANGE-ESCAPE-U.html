Modifying constructor for escape-u structures.

Syntax
(change-escape-u x 
                 [:get <get>])

This is an often useful alternative to make-escape-u.

We construct a new escape-u structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-escape-u

(defmacro change-escape-u (x &rest args)
          (std::change-aggregate 'escape-u
                                 x args '((:get . escape-u->get))
                                 'change-escape-u
                                 'nil))
