Syntax
(change-def-parse-function-spec-rulename x 
                                         [:get <get>] 
                                         [:order <order>])

This is an often useful alternative to make-def-parse-function-spec-rulename.

We construct a new def-parse-function-spec-rulename structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
