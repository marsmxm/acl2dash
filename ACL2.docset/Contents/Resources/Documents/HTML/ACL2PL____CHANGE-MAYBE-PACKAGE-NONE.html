Modifying constructor for maybe-package-none structures.

Syntax
(change-maybe-package-none x 
                           )

This is an often useful alternative to make-maybe-package-none.

We construct a new maybe-package-none structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
