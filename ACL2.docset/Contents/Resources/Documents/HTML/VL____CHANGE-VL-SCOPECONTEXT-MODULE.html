Modifying constructor for vl-scopecontext-module structures.

Syntax
(change-vl-scopecontext-module x 
                               [:mod <mod>])

This is an often useful alternative to make-vl-scopecontext-module.

We construct a new vl-scopecontext-module structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.
