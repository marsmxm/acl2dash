(vl-portdecllist-fix x) is a usual ACL2::fty list fixing function.

Signature
(vl-portdecllist-fix x) → fty::newx
Arguments
x — Guard (vl-portdecllist-p x).
Returns
fty::newx — Type (vl-portdecllist-p fty::newx).

In the logic, we apply vl-portdecl-fix to each member of the x. In the execution, none of that is actually necessary and this is just an inlined identity function.
