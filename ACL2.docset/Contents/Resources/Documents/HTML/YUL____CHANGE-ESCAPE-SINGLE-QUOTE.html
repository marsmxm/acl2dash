Modifying constructor for escape-single-quote structures.

Syntax
(change-escape-single-quote x 
                            )

This is an often useful alternative to make-escape-single-quote.

We construct a new escape-single-quote structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-escape-single-quote

(defmacro change-escape-single-quote
          (x &rest args)
          (std::change-aggregate 'escape-single-quote
                                 x args 'nil
                                 'change-escape-single-quote
                                 'nil))
