(vl-string-self-escape-list-p x) recognizes lists where every element satisfies vl-string-self-escape-p.

Signature
(vl-string-self-escape-list-p x)  std::bool

This is an ordinary deflist. It is "loose" in that it does not care whether x is nil-terminated.

(defun vl-string-self-escape-list-p (x)

            (declare (ignorable __function__))

                     (vl-string-self-escape-list-p (cdr x)))
