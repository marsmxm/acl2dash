Modifying constructor for schar-array structures.

Syntax
(change-schar-array x 
                    [:elements <elements>])

This is an often useful alternative to make-schar-array.

We construct a new schar-array structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-schar-array

(defmacro
     change-schar-array (x &rest args)
     (std::change-aggregate 'schar-array
                            x args
                            '((:elements . schar-array->elements))
                            'change-schar-array
                            'nil))
