Modifying constructor for jstatem-throw structures.

Syntax
(change-jstatem-throw x 
                      [:expr <expr>])

This is an often useful alternative to make-jstatem-throw.

We construct a new jstatem-throw structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-jstatem-throw

(defmacro
     change-jstatem-throw (x &rest args)
     (std::change-aggregate 'jstatem-throw

                            'change-jstatem-throw
                            'nil))
