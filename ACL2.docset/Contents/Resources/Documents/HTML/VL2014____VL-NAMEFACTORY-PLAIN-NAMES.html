Generate a list of fresh names, using particular, preferred names if possible.

Signature
(vl-namefactory-plain-names names factory) 
  → 
(mv fresh-names new-factory)
Arguments
names — The names you would ideally like to use.
    Guard (string-listp names).
factory — Guard (vl-namefactory-p factory).
Returns
fresh-names — Fresh names that you may use. When possible, these are just names. If this is not possible due to name collisions, then some of the fresh_names may have additional indexes as in vl-namefactory-indexed-name and some notes may be printed.
    Type (string-listp fresh-names).
new-factory — Updated name factory where the fresh-names are marked as used.
    Type (vl-namefactory-p new-factory).

Function: vl-namefactory-plain-names

(defun vl-namefactory-plain-names
       (names factory)
       (declare (xargs :guard (and (string-listp names)
                                   (vl-namefactory-p factory))))
       (let ((__function__ 'vl-namefactory-plain-names))
            (declare (ignorable __function__))
            (b* (((when (atom names))
                  (mv nil (vl-namefactory-fix factory)))
                 ((mv name factory)
                  (vl-namefactory-plain-name (car names)
                                             factory))
                 ((mv rest factory)
                  (vl-namefactory-plain-names (cdr names)
                                              factory)))
                (mv (cons name rest) factory))))
