Determine if a character is a lower-case letter (a-z).

ACL2 has a built-in alternative to this function, common-lisp::lower-case-p, but it is irritating to use because it has standard-char-p guards. In contrast, down-alpha-p works on arbitrary characters.

Function: down-alpha-p$inline

(defun down-alpha-p$inline (x)
       (declare (type character x))
       (let ((acl2::__function__ 'down-alpha-p))
            (declare (ignorable acl2::__function__))
            (b* (((the (unsigned-byte 8) code)
                  (char-code x)))
                (and (<= (little-a) code)
                     (<= code (little-z))))))
