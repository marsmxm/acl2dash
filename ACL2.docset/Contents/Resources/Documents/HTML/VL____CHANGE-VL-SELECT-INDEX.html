Modifying constructor for vl-select-index structures.

Syntax
(change-vl-select-index x 
                        [:val <val>])

This is an often useful alternative to make-vl-select-index.

We construct a new vl-select-index structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-select-index

(defmacro
     change-vl-select-index (x &rest args)
     (std::change-aggregate 'vl-select-index
                            x args '((:val . vl-select-index->val))
                            'change-vl-select-index
                            'remake-vl-select-index))
