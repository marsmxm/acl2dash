Modifying constructor for expr-grade-xor structures.

Syntax
(change-expr-grade-xor x 
                       )

This is an often useful alternative to make-expr-grade-xor.

We construct a new expr-grade-xor structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
