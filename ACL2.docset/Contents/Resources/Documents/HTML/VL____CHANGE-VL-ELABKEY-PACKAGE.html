Modifying constructor for vl-elabkey-package structures.

Syntax
(change-vl-elabkey-package x 
                           [:name <name>])

This is an often useful alternative to make-vl-elabkey-package.

We construct a new vl-elabkey-package structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.
