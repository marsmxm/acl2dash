Modifying constructor for svstmt-assign structures.

Syntax
(change-svstmt-assign x 
                      [:writes <writes>] 
                      [:blockingp <blockingp>])

This is an often useful alternative to make-svstmt-assign.

We construct a new svstmt-assign structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-svstmt-assign

(defmacro
   change-svstmt-assign (x &rest args)
   (std::change-aggregate 'svstmt-assign
                          x args
                          '((:writes . svstmt-assign->writes)
                            (:blockingp . svstmt-assign->blockingp))
                          'change-svstmt-assign
                          'remake-svstmt-assign))
