Basic constructor macro for type-integer structures.

Syntax
(make-type-integer )

This is the usual way to construct type-integer structures. It simply conses together a structure with the specified fields.

This macro generates a new type-integer structure from scratch. See also change-type-integer, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by fty::defprod.
