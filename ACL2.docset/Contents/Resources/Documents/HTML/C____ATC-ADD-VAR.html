Add a variable with a type to the symbol table.

Signature
(atc-add-var var type inscope) → new-inscope
Arguments
var — Guard (symbolp var).
type — Guard (typep type).
inscope — Guard (atc-symbol-type-alist-listp inscope).
Returns
new-inscope — Type (atc-symbol-type-alist-listp new-inscope).

This is added to the innermost scope. The symbol table has always at least one scope.

This is always called after checking that the variable is not already in scope. So it unconditionally adds the variable without checking first.
