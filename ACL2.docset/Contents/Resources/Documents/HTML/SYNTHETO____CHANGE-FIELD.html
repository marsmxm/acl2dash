Modifying constructor for field structures.

Syntax
(change-field x 
              [:name <name>] 
              [:type <type>])

This is an often useful alternative to make-field.

We construct a new field structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-field

(defmacro change-field (x &rest args)
          (std::change-aggregate 'field
                                 x args
                                 '((:name . field->name)
                                   (:type . field->type))
                                 'change-field
                                 'nil))
