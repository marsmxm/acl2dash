Modifying constructor for vl-paramargs-named structures.

Syntax
(change-vl-paramargs-named x 
                           [:args <args>])

This is an often useful alternative to make-vl-paramargs-named.

We construct a new vl-paramargs-named structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-paramargs-named

(defmacro
     change-vl-paramargs-named (x &rest args)
     (std::change-aggregate 'vl-paramargs-named
                            x args
                            '((:args . vl-paramargs-named->args))
                            'change-vl-paramargs-named
                            'nil))
