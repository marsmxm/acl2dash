Check if all the terminal value notations in a repetition denote values in a set, or the repetition consists of zero instances.

Signature
(repetition-in-termset-p repetition termset) → yes/no
Arguments
repetition — Guard (repetitionp repetition).
termset — Guard (nat-setp termset).
Returns
yes/no — Type (booleanp yes/no).

Function: repetition-in-termset-p

(defun repetition-in-termset-p
       (repetition termset)
       (declare (xargs :guard (and (repetitionp repetition)
                                   (nat-setp termset))))
       (or (element-in-termset-p (repetition->element repetition)
                                 termset)
           (equal (repetition->range repetition)
                  (repeat-range 0 (nati-finite 0)))))
