Rewrite rules about certain functions distributing over if.

Definitions and Theorems

Theorem: car-of-if

(defthm car-of-if
        (equal (car (if a b c))
               (if a (car b) (car c))))

Theorem: mv-nth-of-if

(defthm mv-nth-of-if
        (equal (mv-nth n (if a b c))
               (if a (mv-nth n b) (mv-nth n c))))
