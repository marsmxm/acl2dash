Modifying constructor for maybe-type-definition-some structures.

Syntax
(change-maybe-type-definition-some x 
                                   [:val <val>])

This is an often useful alternative to make-maybe-type-definition-some.

We construct a new maybe-type-definition-some structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
