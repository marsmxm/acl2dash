Definitions and Theorems

Function: vl-datatype->pdims

(defun vl-datatype->pdims (x)
       (declare (xargs :guard (vl-datatype-p x)))
       (let ((__function__ 'vl-datatype->pdims))
            (declare (ignorable __function__))
            (vl-datatype-case x
                              :vl-coretype x.pdims
                              :vl-struct x.pdims
                              :vl-union x.pdims
                              :vl-enum x.pdims
                              :vl-usertype x.pdims)))

Theorem: vl-dimensionlist-p-of-vl-datatype->pdims

(defthm vl-dimensionlist-p-of-vl-datatype->pdims
        (b* ((pdims (vl-datatype->pdims x)))
            (vl-dimensionlist-p pdims))
        :rule-classes :rewrite)

Theorem: vl-datatype->pdims-of-vl-datatype-fix-x

(defthm vl-datatype->pdims-of-vl-datatype-fix-x
        (equal (vl-datatype->pdims (vl-datatype-fix x))
               (vl-datatype->pdims x)))

Theorem: vl-datatype->pdims-vl-datatype-equiv-congruence-on-x

(defthm vl-datatype->pdims-vl-datatype-equiv-congruence-on-x
        (implies (vl-datatype-equiv x x-equiv)
                 (equal (vl-datatype->pdims x)
                        (vl-datatype->pdims x-equiv)))
        :rule-classes :congruence)

These are goofy rules: normally we want to normalize things to (vl-datatype->pdims <term>), but if <term> is a call of one of the particular datatype constructors, we'll rewrite the other way so that we can simplify it to just whatever pdims are being given to the constructor.

Theorem: vl-datatype-pdims-when-vl-coretype

(defthm
  vl-datatype-pdims-when-vl-coretype
  (implies
       (vl-datatype-case x :vl-coretype)
       (and (implies (syntaxp (and (consp x)
                                   (eq (car x) 'vl-coretype)))
                     (equal (vl-datatype->pdims x)
                            (vl-coretype->pdims x)))
            (implies (syntaxp (not (and (consp x)
                                        (eq (car x) 'vl-coretype))))
                     (equal (vl-coretype->pdims x)
                            (vl-datatype->pdims x))))))
