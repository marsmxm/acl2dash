Modifying constructor for vl-elabkey-class structures.

Syntax
(change-vl-elabkey-class x 
                         [:name <name>])

This is an often useful alternative to make-vl-elabkey-class.

We construct a new vl-elabkey-class structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-elabkey-class

(defmacro
     change-vl-elabkey-class (x &rest args)
     (std::change-aggregate 'vl-elabkey-class
                            x
                            args '((:name . vl-elabkey-class->name))
                            'change-vl-elabkey-class
                            'nil))
