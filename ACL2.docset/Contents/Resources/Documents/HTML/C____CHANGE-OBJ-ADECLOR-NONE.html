Modifying constructor for obj-adeclor-none structures.

Syntax
(change-obj-adeclor-none x 
                         )

This is an often useful alternative to make-obj-adeclor-none.

We construct a new obj-adeclor-none structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
