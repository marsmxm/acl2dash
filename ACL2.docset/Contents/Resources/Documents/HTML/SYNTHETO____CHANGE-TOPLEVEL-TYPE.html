Modifying constructor for toplevel-type structures.

Syntax
(change-toplevel-type x 
                      [:get <get>])

This is an often useful alternative to make-toplevel-type.

We construct a new toplevel-type structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-toplevel-type

(defmacro
     change-toplevel-type (x &rest args)
     (std::change-aggregate 'toplevel-type
                            x args '((:get . toplevel-type->get))
                            'change-toplevel-type
                            'nil))
