Generate a list of distinct wires with a particular range.

Signature
(vl-make-list-of-netdecls n basename range)  wires
Arguments
n how many wires to generate.
Guard (natp n).
basename generate wires like basename_3.
Guard (stringp basename).
range range for each generated wire.
Guard (vl-maybe-range-p range).
Returns
wires Type (vl-vardecllist-p wires).

Definitions and Theorems

Function: vl-make-list-of-netdecls

(defun
 vl-make-list-of-netdecls
 (n basename range)
 (declare (xargs :guard (and (natp n)
                             (stringp basename)
                             (vl-maybe-range-p range))))
 (let
  ((__function__ 'vl-make-list-of-netdecls))
  (declare (ignorable __function__))
  (b*
   (((when (zp n)) nil)
    (range (vl-maybe-range-fix range))
    (type
     (hons-copy (make-vl-coretype :name :vl-logic
                                  :pdims (and range (list range)))))
    (decl1 (make-vl-vardecl :name (cat basename "_" (natstr n))
                            :type type
                            :nettype :vl-wire
                            :loc *vl-fakeloc*)))
   (cons decl1
         (vl-make-list-of-netdecls (- n 1)
                                   basename range)))))
