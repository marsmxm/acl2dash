Modifying constructor for statement-if structures.

Syntax
(change-statement-if x 
                     [:test <test>] 
                     [:body <body>])

This is an often useful alternative to make-statement-if.

We construct a new statement-if structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-statement-if

(defmacro change-statement-if (x &rest args)
          (std::change-aggregate 'statement-if
                                 x args
                                 '((:test . statement-if->test)
                                   (:body . statement-if->body))
                                 'change-statement-if
                                 'nil))
