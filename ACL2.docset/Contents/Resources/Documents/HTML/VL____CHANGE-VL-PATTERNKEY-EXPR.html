Modifying constructor for vl-patternkey-expr structures.

Syntax
(change-vl-patternkey-expr x 
                           [:key <key>])

This is an often useful alternative to make-vl-patternkey-expr.

We construct a new vl-patternkey-expr structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-patternkey-expr

(defmacro
