Constraint on the number of address keys in the wallet state.

Signature
(stat-addresses-bounded-p stat) → yes/no
Arguments
stat — Guard (statp stat).
Returns
yes/no — Type (booleanp yes/no).

Since all the address keys have non-hardened indices, the largest index is 2^{31}-1, so the maximum value of the addresses state component is 2^{31}.
