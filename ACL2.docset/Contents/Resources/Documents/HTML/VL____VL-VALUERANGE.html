A value or a range used in an inside expression. For instance, the 8 or [16:20] from a inside { 8, [16:20] }.

This is a tagged union type, introduced by deftagsum.

A range of values from an inside expression's set. For instance, the [16:20] part of a inside { 8, [16:20] }.
:valuerange-single → vl-valuerange-single
A single value from an inside expression's set. For instance, the 8 part of a inside { 8, [16:20] }.
