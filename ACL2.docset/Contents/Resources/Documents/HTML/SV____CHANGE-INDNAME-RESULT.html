Modifying constructor for indname-result structures.

Syntax
(change-indname-result x 
                       [:varname <varname>] 
                       [:decl <decl>])

This is an often useful alternative to make-indname-result.

We construct a new indname-result structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-indname-result

(defmacro
     change-indname-result (x &rest args)
     (std::change-aggregate 'indname-result
                            x args
                            '((:varname . indname-result->varname)
                              (:decl . indname-result->decl))
                            'change-indname-result
                            'remake-indname-result))
