Modifying constructor for expression-var structures.

Syntax
(change-expression-var x 
                       [:name <name>])

This is an often useful alternative to make-expression-var.

We construct a new expression-var structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-expression-var

(defmacro
     change-expression-var (x &rest args)
     (std::change-aggregate 'expression-var
                            x args '((:name . expression-var->name))
                            'change-expression-var
                            'nil))
