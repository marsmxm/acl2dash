Modifying constructor for wellformed-result-ok structures.

Syntax
(change-wellformed-result-ok x 
                             [:get <get>])

This is an often useful alternative to make-wellformed-result-ok.

We construct a new wellformed-result-ok structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
