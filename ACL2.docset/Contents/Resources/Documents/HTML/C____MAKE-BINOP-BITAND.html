Basic constructor macro for binop-bitand structures.

Syntax
(make-binop-bitand )

This is the usual way to construct binop-bitand structures. It simply conses together a structure with the specified fields.

This macro generates a new binop-bitand structure from scratch. See also change-binop-bitand, which can "change" an existing structure, instead.
