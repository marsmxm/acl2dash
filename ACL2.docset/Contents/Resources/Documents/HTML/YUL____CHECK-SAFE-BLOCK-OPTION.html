Check if an optional block is safe.

Signature
(check-safe-block-option block? vartab funtab) → modes
Arguments
block? — Guard (block-optionp block?).
vartab — Guard (vartablep vartab).
funtab — Guard (funtablep funtab).
Returns
modes — Type (mode-set-resultp modes).

If there is no block, the check succeeds and we return just regular mode. If there is a block, we check it.
