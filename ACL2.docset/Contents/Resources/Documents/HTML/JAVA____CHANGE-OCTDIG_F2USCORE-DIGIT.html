Modifying constructor for octdig/uscore-digit structures.

Syntax
(change-octdig/uscore-digit x 
                            [:get <get>])

This is an often useful alternative to make-octdig/uscore-digit.

We construct a new octdig/uscore-digit structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.
