(vl-defaultdisablelist-fix x) is a usual fty list fixing function.

Signature
(vl-defaultdisablelist-fix x)  fty::newx
Arguments
x Guard (vl-defaultdisablelist-p x).
Returns
fty::newx Type (vl-defaultdisablelist-p fty::newx).

In the logic, we apply vl-defaultdisable-fix to each member of the x. In the execution, none of that is actually necessary and this is just an inlined identity function.
