Split a filename into its basename and extension.

Signature
(vl-split-filename filename) → (mv basename extension)
Arguments
filename — E.g., foo.txt.
    Guard (stringp filename).
Returns
basename — E.g., foo.
    Type (stringp basename).
extension — E.g. txt, or nil when there is no extension.
    Type (maybe-stringp extension).

Definitions and Theorems

Function: vl-split-filename

(defun vl-split-filename (filename)
       (declare (xargs :guard (stringp filename)))
       (let ((__function__ 'vl-split-filename))
            (declare (ignorable __function__))
            (b* ((filename (string-fix filename))
                 (pos (str::strrpos "." filename))
                 ((unless pos) (mv filename nil))
                 (basename (subseq filename 0 pos))
                 (extension (subseq filename (+ 1 pos) nil)))
                (mv basename extension))))
