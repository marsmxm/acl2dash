Modifying constructor for maybe-rule-none structures.

Syntax
(change-maybe-rule-none x 
                        )

This is an often useful alternative to make-maybe-rule-none.

We construct a new maybe-rule-none structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

This is an ordinary change- macro introduced by fty::defprod.
