Fixtype of definitions of relations.

This is a product type introduced by fty::defprod.

Fields
name — symbolp
para — symbol-list
body — constraint-list

A relation definition consists of the name of the relation (any symbol), a list of formal parameters (any symbols), and a body consisting of a list of constraints. The constraints are taken conjunctively; but see the discussion in constraint about possible extensions to allow explicit propositional connectives (in which case the body of a definition would presumably be just a single constraint, which may be a conjunction).
