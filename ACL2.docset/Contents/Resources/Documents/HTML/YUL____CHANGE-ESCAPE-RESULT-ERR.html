Modifying constructor for escape-result-err structures.

Syntax
(change-escape-result-err x 
                          [:get <get>])

This is an often useful alternative to make-escape-result-err.

We construct a new escape-result-err structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-escape-result-err

(defmacro
     change-escape-result-err (x &rest args)
     (std::change-aggregate 'escape-result-err
                            x
                            args '((:get . escape-result-err->get))
                            'change-escape-result-err
                            'nil))
