Modifying constructor for vl-enumitem structures.

Syntax
(change-vl-enumitem x 
                    [:name <name>] 
                    [:range <range>] 
                    [:value <value>])

This is an often useful alternative to make-vl-enumitem.

We construct a new vl-enumitem structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-vl-enumitem

(defmacro change-vl-enumitem (x &rest args)
          (std::change-aggregate 'vl-enumitem
                                 x args
                                 '((:name . vl-enumitem->name)
                                   (:range . vl-enumitem->range)
                                   (:value . vl-enumitem->value))
                                 'change-vl-enumitem
                                 'remake-vl-enumitem))
