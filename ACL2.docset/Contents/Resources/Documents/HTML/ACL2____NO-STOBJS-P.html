Check if a named function has no input or output stobjs.

Signature
(no-stobjs-p fn wrld) → yes/no
Arguments
fn — Guard (symbolp fn).
wrld — Guard (plist-worldp wrld).
Returns
yes/no — Type (booleanp yes/no).

The function must not be in *stobjs-out-invalid*, because in that case its (output) stobjs depend on how it is called.

See no-stobjs-p+ for an enhanced variant of this utility.

Definitions and Theorems

Function: no-stobjs-p

(defun
  no-stobjs-p (fn wrld)
  (declare (xargs :guard (and (symbolp fn) (plist-worldp wrld))))
  (declare (xargs :guard (not (member-eq fn *stobjs-out-invalid*))))
  (let ((__function__ 'no-stobjs-p))
       (declare (ignorable __function__))
       (and (all-nils (stobjs-in fn wrld))
            (all-nils (stobjs-out fn wrld)))))
