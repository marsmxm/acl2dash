Modifying constructor for stmt-null structures.

Syntax
(change-stmt-null x 
                  )

This is an often useful alternative to make-stmt-null.

We construct a new stmt-null structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-stmt-null

(defmacro change-stmt-null (x &rest args)
          (std::change-aggregate 'stmt-null
                                 x args 'nil
                                 'change-stmt-null
                                 'nil))
