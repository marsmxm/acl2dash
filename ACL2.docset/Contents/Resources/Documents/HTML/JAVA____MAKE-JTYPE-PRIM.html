Basic constructor macro for jtype-prim structures.

Syntax
(make-jtype-prim [:type <type>])

This is the usual way to construct jtype-prim structures. It simply conses together a structure with the specified fields.

This macro generates a new jtype-prim structure from scratch. See also change-jtype-prim, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by fty::defprod.

Macro: make-jtype-prim

(defmacro make-jtype-prim (&rest args)
          (std::make-aggregate 'jtype-prim
                               args '((:type))
                               'make-jtype-prim

(defun jtype-prim (type)
       (declare (xargs :guard (primitive-typep type)))
       (declare (xargs :guard t))
       (let ((__function__ 'jtype-prim))
            (declare (ignorable __function__))
            (b* ((type (mbe :logic (primitive-type-fix type)
                            :exec type)))
                (cons :prim (list type)))))
