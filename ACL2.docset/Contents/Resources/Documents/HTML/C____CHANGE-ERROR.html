Modifying constructor for error structures.

Syntax
(change-error x 
              [:info <info>])

This is an often useful alternative to make-error.

We construct a new error structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-error

(defmacro change-error (x &rest args)
          (std::change-aggregate 'error
                                 x args '((:info . error->info))
                                 'change-error
                                 'nil))
