Eliminate param decls for useless params from each module, and build a map that explains what has been eliminated (i.e., what needs to be cleaned up from each module instance.)

Signature
(vl-modulelist-clean-params-aux x) → (mv x-prime map)
Arguments
x — Guard (vl-modulelist-p x).
Returns
x-prime — Type (vl-modulelist-p x-prime), given the guard.
map — Type (vl-useless-params-map-p map), given the guard.

Definitions and Theorems

Function: vl-modulelist-clean-params-aux

(defun vl-modulelist-clean-params-aux (x)
       (declare (xargs :guard (vl-modulelist-p x)))
       (let ((__function__ 'vl-modulelist-clean-params-aux))
            (declare (ignorable __function__))
            (b* (((when (atom x)) (mv nil nil))
                 ((mv car-prime car-entry)
                  (vl-module-clean-params (car x)))
                 ((mv cdr-prime map)
                  (vl-modulelist-clean-params-aux (cdr x)))
                 (map (if car-entry
                          (hons-acons (vl-module->name (car x))
                                      car-entry map)
                          map)))
                (mv (cons car-prime cdr-prime) map))))
