Retrieve the matrix of a function introduced via defun-sk, in untranslated form.

Signature
(defun-sk-imatrix fn wrld) → imatrix
Arguments
fn — Guard (defun-sk-namep fn wrld).
wrld — Guard (plist-worldp wrld).
Returns
imatrix — An untranslated term.

The i that starts imatrix in the name of this function conveys that the result is based on the defun-sk form that introduced fn.

Use defun-sk-matrix to retrieve the matrix in translated form.
