Remove the underscores from a list of octal digits and underscores.

Signature
(octdig/uscores-to-digits dus) → ds
Arguments
dus — Guard (octdig/uscore-listp dus).
Returns
ds — Type (oct-digit-listp ds).

Any underscores in an octal-numeral in the grammar are just used for separation. This function returns the underlying sequence of octal digits.
