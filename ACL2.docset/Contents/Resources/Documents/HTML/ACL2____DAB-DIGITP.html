Recognize valid digits for representing natural numbers as digits in the specified base.

Signature
(dab-digitp base x) → yes/no
Arguments
base — Guard (dab-basep base).
Returns
yes/no — Type (booleanp yes/no).

The fixing function for this predicate is dab-digit-fix.

Definitions and Theorems

Function: dab-digitp

(defun dab-digitp (base x)
       (declare (xargs :guard (dab-basep base)))
       (let ((__function__ 'dab-digitp))
            (declare (ignorable __function__))
            (and (natp x)
                 (< x (dab-base-fix base)))))
