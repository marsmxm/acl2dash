Fixtype of optional Syntheto type sums.

This is an option type introduced by fty::defoption. Note that defoption is just a wrapper for fty::defflexsum, so there are :none and :some member types, a case macro, and so forth.

Member Types
:none → maybe-type-sum-none
Represents that no maybe-type-sum is available, i.e., Nothing or None.
:some → maybe-type-sum-some
An available maybe-type-sum, i.e., Just val or Some val.
