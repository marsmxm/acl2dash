Modifying constructor for svex/index structures.

Syntax
(change-svex/index x 
                   [:expr <expr>] 
                   [:idx <idx>])

This is an often useful alternative to make-svex/index.

We construct a new svex/index structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-svex/index

(defmacro change-svex/index (x &rest args)
          (std::change-aggregate 'svex/index
                                 x args
                                 '((:expr . svex/index->expr)
                                   (:idx . svex/index->idx))
                                 'change-svex/index
                                 'nil))
