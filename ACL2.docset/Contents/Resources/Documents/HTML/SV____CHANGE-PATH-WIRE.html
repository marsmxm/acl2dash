Modifying constructor for path-wire structures.

Syntax
(change-path-wire x 
                  [:name <name>])

This is an often useful alternative to make-path-wire.

We construct a new path-wire structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-path-wire

(defmacro change-path-wire (x &rest args)
          (std::change-aggregate 'path-wire
                                 x args '((:name . path-wire->name))
                                 'change-path-wire
                                 'nil))
