Modifying constructor for numeric-value-char structures.

Syntax
(change-numeric-value-char x 
                           [:get <get>])

This is an often useful alternative to make-numeric-value-char.

We construct a new numeric-value-char structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-numeric-value-char

(defmacro
     change-numeric-value-char (x &rest args)
     (std::change-aggregate 'numeric-value-char
                            x
                            args '((:get . numeric-value-char->get))
                            'change-numeric-value-char
                            'nil))
