Modifying constructor for fgl-rune-formula structures.

Syntax
(change-fgl-rune-formula x 
                         [:name <name>])

This is an often useful alternative to make-fgl-rune-formula.

We construct a new fgl-rune-formula structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-fgl-rune-formula

(defmacro
     change-fgl-rune-formula (x &rest args)
     (std::change-aggregate 'fgl-rune-formula
                            x
                            args '((:name . fgl-rune-formula->name))
                            'change-fgl-rune-formula
                            'nil))
