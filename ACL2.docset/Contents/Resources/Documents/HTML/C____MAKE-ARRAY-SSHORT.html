Basic constructor macro for array-sshort structures.

Syntax
(make-array-sshort [:get <get>])

This is the usual way to construct array-sshort structures. It simply conses together a structure with the specified fields.

This macro generates a new array-sshort structure from scratch. See also change-array-sshort, which can "change" an existing structure, instead.

This is an ordinary make- macro introduced by fty::defprod.
