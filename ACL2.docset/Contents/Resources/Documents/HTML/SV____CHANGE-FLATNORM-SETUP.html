Modifying constructor for flatnorm-setup structures.

Syntax
(change-flatnorm-setup x 
                       [:monotonify <monotonify>])

This is an often useful alternative to make-flatnorm-setup.

We construct a new flatnorm-setup structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.

Macro: change-flatnorm-setup

(defmacro change-flatnorm-setup (x &rest args)
          (std::change-aggregate
               'flatnorm-setup
               x args
               '((:monotonify . flatnorm-setup->monotonify))
               'change-flatnorm-setup
               'nil))
