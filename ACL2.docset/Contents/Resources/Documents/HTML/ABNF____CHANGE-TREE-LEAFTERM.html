Modifying constructor for tree-leafterm structures.

Syntax
(change-tree-leafterm x 
                      [:get <get>])

This is an often useful alternative to make-tree-leafterm.

We construct a new tree-leafterm structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-tree-leafterm

(defmacro
     change-tree-leafterm (x &rest args)
     (std::change-aggregate 'tree-leafterm
                            x args '((:get . tree-leafterm->get))
                            'change-tree-leafterm
                            'nil))
