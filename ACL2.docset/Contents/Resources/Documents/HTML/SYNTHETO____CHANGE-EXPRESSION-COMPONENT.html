Modifying constructor for expression-component structures.

Syntax
(change-expression-component x 
                             [:multi <multi>] 
                             [:index <index>])

This is an often useful alternative to make-expression-component.

We construct a new expression-component structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-expression-component

(defmacro
    change-expression-component
    (x &rest args)
    (std::change-aggregate 'expression-component
                           x args
                           '((:multi . expression-component->multi)
                             (:index . expression-component->index))
                           'change-expression-component
                           'nil))
