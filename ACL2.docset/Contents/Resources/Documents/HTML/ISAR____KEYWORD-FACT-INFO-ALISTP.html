Recognize alists from keywords to fact information.

An alist of this type stores information about all the facts proved so far when running the proof.

Definitions and Theorems

Function: keyword-fact-info-alistp

(defun keyword-fact-info-alistp (x)
       (declare (xargs :guard t))
       (if (consp x)
           (and (consp (car x))
                (keywordp (caar x))
                (fact-infop (cdar x))
                (keyword-fact-info-alistp (cdr x)))
           (null x)))
