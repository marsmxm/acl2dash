Fixtype of error results.

This is a product type introduced by defprod.

Fields
info — ACL2::any-p
stack — ACL2::symbol-list

This is the fixtype of error results for defresult; see the introduction of defresult for background and motivation.

An error result consists of some unconstrained information and of a list of symbols that should be normally function names. This list is treated like a call stack trace by patbind-ok. The stack's top is the car of the list.
