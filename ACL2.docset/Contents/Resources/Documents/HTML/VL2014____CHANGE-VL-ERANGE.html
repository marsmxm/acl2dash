A copying macro that lets you create new vl-erange-p structures, based on existing structures.

Syntax:

(change-vl-erange x 
                  [:type <type>] 
                  [:left <left>] 
                  [:right <right>])

This is a sometimes useful alternative to make-vl-erange. It constructs a new vl-erange-p structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defaggregate.

Macro: change-vl-erange

(defmacro change-vl-erange (x &rest args)
          (std::change-aggregate 'vl-erange
                                 x args
                                 '((:type . vl-erange->type)
                                   (:left . vl-erange->left)
                                   (:right . vl-erange->right))
                                 'change-vl-erange
                                 'remake-vl-erange))
