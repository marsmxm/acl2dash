Fixtype of (ATC's model of) arrays of type signed short.

This is a product type introduced by fty::defprod.

Fields
elemtype — type
elements — sshort-list
Additional Requirements

The following invariant is enforced on the fields:

(and (type-case elemtype :sshort) (consp elements))

Theorem: sshort-arrayp-alt-def

(defthm sshort-arrayp-alt-def
        (equal (sshort-arrayp x)
               (and (valuep x)
                    (value-case x :array)
                    (equal (value-array->elemtype x)
                           (type-sshort))
                    (sshort-listp (value-array->elements x)))))

Theorem: sshort-array->elements-alt-def

(defthm sshort-array->elements-alt-def
        (implies (sshort-arrayp array)
                 (equal (sshort-array->elements array)
                        (value-array->elements array))))
