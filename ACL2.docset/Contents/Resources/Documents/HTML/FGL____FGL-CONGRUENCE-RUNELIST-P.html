(fgl-congruence-runelist-p x) recognizes lists where every element satisfies fgl-congruence-rune-p.

Signature
(fgl-congruence-runelist-p x)  *

This is an ordinary deflist. It is "strict" in that it requires x to be a "properly" nil-terminated list.
