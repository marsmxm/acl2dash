Modifying constructor for reference-type-variable structures.

Syntax
(change-reference-type-variable x 
                                [:get <get>])

This is an often useful alternative to make-reference-type-variable.

We construct a new reference-type-variable structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-reference-type-variable

(defmacro
     change-reference-type-variable
     (x &rest args)
     (std::change-aggregate 'reference-type-variable
                            x args
                            '((:get . reference-type-variable->get))
                            'change-reference-type-variable
                            'nil))
