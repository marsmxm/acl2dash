Raw constructor for vl-opinfo-p structures.

Syntax:

(vl-opinfo arity text)

This is the lowest-level constructor for vl-opinfo-p structures. It simply conses together a structure with the specified fields.

Note: It's generally better to use macros like make-vl-opinfo or change-vl-opinfo instead. These macros lead to more readable and robust code, because you don't have to remember the order of the fields.

The vl-opinfo-p structures we create here are just constructed with ordinary cons. If you want to create honsed structures, see honsed-vl-opinfo instead.

Definition

This is an ordinary constructor function introduced by defaggregate.

Function: vl-opinfo

(defun vl-opinfo (arity text)
       (declare (xargs :guard (and (maybe-natp arity)
                                   (maybe-stringp text))))
       (cons :vl-opinfo (cons (cons 'arity arity)
                              (cons (cons 'text text) nil))))
