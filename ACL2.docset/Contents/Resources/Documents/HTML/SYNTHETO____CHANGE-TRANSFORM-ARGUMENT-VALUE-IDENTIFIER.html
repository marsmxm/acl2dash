Modifying constructor for transform-argument-value-identifier structures.

Syntax
(change-transform-argument-value-identifier x 
                                            [:name <name>])

This is an often useful alternative to make-transform-argument-value-identifier.

We construct a new transform-argument-value-identifier structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-transform-argument-value-identifier

(defmacro change-transform-argument-value-identifier
          (x &rest args)
          (std::change-aggregate
               'transform-argument-value-identifier
               x args
               '((:name .
                        transform-argument-value-identifier->name))
               'change-transform-argument-value-identifier
               'nil))
