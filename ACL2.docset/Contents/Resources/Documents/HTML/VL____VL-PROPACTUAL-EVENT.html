This is a product type, introduced by deftagsum in support of vl-propactual.

Fields
name — maybe-string
Explicit name for this argument, if provided. For instance, in .foo(bar), this is the foo part. Some actuals may not have a name..
evatoms — vl-evatomlist
This isn't quite general enough, but event expressions are also used in vl-eventcontrols, so I'd like this to be compatible with that code, reuse its parser, etc. If we find that this isn't good enough, we should extend the eventcontrol representation too.
