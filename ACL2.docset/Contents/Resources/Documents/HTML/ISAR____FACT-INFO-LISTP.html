Recognize lists of information about facts.

Signature
(fact-info-listp x)  std::bool

This is an ordinary std::deflist. It is "strict" in that it requires x to be a "properly" nil-terminated list.

Function: fact-info-listp

(defun fact-info-listp (x)
       (declare (xargs :guard t))
       (let ((__function__ 'fact-info-listp))
            (declare (ignorable __function__))
            (if (consp x)
                (and (fact-infop (car x))
                     (fact-info-listp (cdr x)))
                (null x))))
