Basic constructor macro for short-array structures.

Syntax
(make-short-array [:components <components>])

This is the usual way to construct short-array structures. It simply conses together a structure with the specified fields.

This macro generates a new short-array structure from scratch. See also change-short-array, which can "change" an existing structure, instead.

This is an ordinary make- macro introduced by fty::defprod.
