Fixtype of (ATC's model of) arrays of type unsigned long.

This is a product type introduced by fty::defprod.

Fields
elemtype — type
elements — ulong-list
Additional Requirements

The following invariant is enforced on the fields:

(and (type-case elemtype :ulong) (consp elements))

Theorem: ulong-arrayp-alt-def

(defthm ulong-arrayp-alt-def
        (equal (ulong-arrayp x)
               (and (valuep x)
                    (value-case x :array)
                    (equal (value-array->elemtype x)
                           (type-ulong))
                    (ulong-listp (value-array->elements x)))))

Theorem: ulong-array->elements-alt-def

(defthm ulong-array->elements-alt-def
        (implies (ulong-arrayp array)
                 (equal (ulong-array->elements array)
                        (value-array->elements array))))
