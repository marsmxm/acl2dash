Modifying constructor for bexp-equal structures.

Syntax
(change-bexp-equal x 
                   [:left <left>] 
                   [:right <right>])

This is an often useful alternative to make-bexp-equal.

We construct a new bexp-equal structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-bexp-equal

(defmacro change-bexp-equal (x &rest args)
          (std::change-aggregate 'bexp-equal
                                 x args
                                 '((:left . bexp-equal->left)
                                   (:right . bexp-equal->right))
                                 'change-bexp-equal
                                 'nil))
