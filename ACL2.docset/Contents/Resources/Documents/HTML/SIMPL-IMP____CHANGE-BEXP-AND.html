Modifying constructor for bexp-and structures.

Syntax
(change-bexp-and x 
                 [:left <left>] 
                 [:right <right>])

This is an often useful alternative to make-bexp-and.

We construct a new bexp-and structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by fty::defprod.

Macro: change-bexp-and

(defmacro change-bexp-and (x &rest args)
          (std::change-aggregate 'bexp-and
                                 x args
                                 '((:left . bexp-and->left)
                                   (:right . bexp-and->right))
                                 'change-bexp-and
                                 'nil))
