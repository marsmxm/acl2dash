Basic constructor macro for sshort-array structures.

Syntax
(make-sshort-array [:elements <elements>])

This is the usual way to construct sshort-array structures. It simply conses together a structure with the specified fields.

This macro generates a new sshort-array structure from scratch. See also change-sshort-array, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by fty::defprod.

 sshort-array (elements)
 (declare (xargs :guard (sshort-listp elements)))
 (declare (xargs :guard (consp elements)))
 (let
  ((__function__ 'sshort-array))
  (declare (ignorable __function__))
  (b* ((elements (mbe :logic (sshort-list-fix elements)
                      :exec elements)))
