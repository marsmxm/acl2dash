Basic constructor macro for jliteral-integer structures.

Syntax
(make-jliteral-integer [:get <get>])

This is the usual way to construct jliteral-integer structures. It simply conses together a structure with the specified fields.

This macro generates a new jliteral-integer structure from scratch. See also change-jliteral-integer, which can "change" an existing structure, instead.

Definition

This is an ordinary make- macro introduced by fty::defprod.

Macro: make-jliteral-integer

(defmacro make-jliteral-integer (&rest args)

(defun jliteral-integer (get)
       (declare (xargs :guard (integer-literalp get)))
       (declare (xargs :guard t))
       (let ((__function__ 'jliteral-integer))
            (declare (ignorable __function__))
            (b* ((get (mbe :logic (integer-literal-fix get)
