(svex-overridelist-fix x) is a usual fty list fixing function.

Signature
(svex-overridelist-fix x) → fty::newx
Arguments
x — Guard (svex-overridelist-p x).
Returns
fty::newx — Type (svex-overridelist-p fty::newx).

In the logic, we apply svex-override-fix to each member of the x. In the execution, none of that is actually necessary and this is just an inlined identity function.
