Semantics of repetitions.

Signature
(tree-list-match-repetition-p trees repetition rules) → yes/no
Arguments
trees — Guard (tree-listp trees).
repetition — Guard (repetitionp repetition).
rules — Guard (rulelistp rules).
Returns
yes/no — Type (booleanp yes/no).

A list of trees matches a repetition iff the length of the list matches the range of the repetition and each tree in the list matches the element of the repetition.

Function: tree-list-match-repetition-p

(defun
 tree-list-match-repetition-p
 (trees repetition rules)
 (declare (xargs :guard (and (tree-listp trees)
                             (repetitionp repetition)
                             (rulelistp rules))))
 (and
   (numrep-match-repeat-range-p (len trees)
                                (repetition->range repetition))
   (tree-list-match-element-p trees (repetition->element repetition)
                              rules)))
