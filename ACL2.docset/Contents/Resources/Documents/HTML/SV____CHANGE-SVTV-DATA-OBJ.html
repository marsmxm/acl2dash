Syntax

                      [:namemap <namemap>] 
                      [:namemap-validp <namemap-validp>]

                      [:cycle-phases <cycle-phases>]

                      [:pipeline <pipeline>] 
                      [:pipeline-validp <pipeline-validp>])

This is an often useful alternative to make-svtv-data-obj.

We construct a new svtv-data-obj structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.

Definition

This is an ordinary change- macro introduced by defprod.
