Generate a shallowly embedded ACL2 call of a Java primitive unary operation.

Signature
(atj-gen-shallow-jprim-unop-call 
     fn 
     arg-block arg-expr src-types dst-types) 
 
  → 
(mv block expr)
Arguments
fn — Guard (atj-jprim-unop-fn-p fn).
arg-block — Guard (jblockp arg-block).
arg-expr — Guard (jexprp arg-expr).
src-types — Guard (atj-type-listp src-types).
dst-types — Guard (atj-type-listp dst-types).
Returns
block — Type (jblockp block).
expr — Type (jexprp expr).

This is called only if :guards is t. This is called after translating the operand to Java. The resulting block and expression are passed as parameters here.

We generate a Java unary expression whose operator corresponds to the function.

Definitions and Theorems

Function: atj-gen-shallow-jprim-unop-call

(defun
 atj-gen-shallow-jprim-unop-call
 (fn arg-block arg-expr src-types dst-types)
 (declare (xargs :guard (and (atj-jprim-unop-fn-p fn)
                             (jblockp arg-block)
                             (jexprp arg-expr)
                             (atj-type-listp src-types)
                             (atj-type-listp dst-types))))
 (declare (xargs :guard (and (consp src-types)
                             (consp dst-types))))
 (let
  ((__function__ 'atj-gen-shallow-jprim-unop-call))
  (declare (ignorable __function__))
  (b*
   ((unop (atj-jprim-unop-fn-to-junop fn))
    (expr (jexpr-unary unop arg-expr))
    (block (jblock-fix arg-block)))
   (mv
      block
      (atj-adapt-expr-to-type expr (atj-type-list-to-type src-types)
                              (atj-type-list-to-type dst-types)
                              t)))))
