Modifying constructor for vl-maybe-paramvalue-some structures.

Syntax
(change-vl-maybe-paramvalue-some x 
                                 [:val <val>])

This is an often useful alternative to make-vl-maybe-paramvalue-some.

We construct a new vl-maybe-paramvalue-some structure that is a copy of x, except that you can explicitly change some particular fields. Any fields you don't mention just keep their values from x.
